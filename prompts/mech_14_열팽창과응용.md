너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 01. 단열과 열팽창
- 토픽: 열팽창과 응용
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 물질은 온도가 올라가면 팽창하고 내려가면 수축한다. 고체의 선팽창: ΔL = αL₀ΔT (α: 선팽창 계수). 부피 팽창: ΔV = βV₀ΔT (β ≈ 3α). 열팽창 계수는 물질마다 다르며, 바이메탈은 열팽창 계수가 다른 두 금속을 접합하여 온도에 따라 휘어지는 원리를 이용한다.

## 씬 구성
- 3가지 시나리오: 선팽창 / 부피 팽창 / 바이메탈
- 온도 슬라이더로 팽창/수축을 실시간 관찰
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 선팽창

### 3D
- 금속 막대: BoxGeometry(L₀, 0.3, 0.3), 0xcccccc (금속 은색)
- 눈금자: BoxGeometry 아래, 0~L₀ 눈금 표시 (mm 단위)
- 온도 표시: 디지털 온도계 + 빨간 수은 온도계 3D

### 팽창 시각화
- 온도 슬라이더: -50°C ~ 300°C (기본값 20°C)
- 온도 증가 → 막대 길이 증가 (scale.x 변화)
  - ΔL = αL₀ΔT
  - 새 길이 L = L₀(1 + αΔT)
- 막대 색상: 온도에 따라 파랑(저온) → 빨강(고온) 그라데이션
- ΔL 표시: 늘어난 부분을 빨간 영역으로 표시 + 수치 라벨

### 물질별 비교
- 3개 막대를 나란히 배치:
  - 알루미늄 (α = 23×10⁻⁶/K): 0xaabbcc, 가장 많이 팽창
  - 구리 (α = 17×10⁻⁶/K): 0xcc8855
  - 인바 합금 (α = 1.2×10⁻⁶/K): 0x888888, 거의 안 팽창
- 같은 온도 변화에서 팽창량 차이 비교

### 실생활 예시
- 철도 레일 이음새: 여름에 팽창 → 이음새 간격 필요
- 다리 신축 이음장치: 열팽창 흡수
- 유리병 뚜껑: 뜨거운 물로 금속 뚜껑만 팽창 → 쉽게 열림

## 시나리오 2: 부피 팽창

### 3D
- 플라스크(둥근 바닥 플라스크): SphereGeometry(r=1) + CylinderGeometry(가느다란 관)
- 내부 액체: 빨간 색소 물, CylinderGeometry
- 온도 증가 → 액체 부피 팽창 → 관 안 액체 높이 상승
- 온도 감소 → 액체 수축 → 관 안 액체 높이 하강
- ΔV = βV₀ΔT

### 물(4°C 이상)의 이상 팽창
- 물: 0~4°C에서는 온도 올라가면 수축! (밀도 최대 4°C)
- 4°C 이상에서는 정상 팽창
- 그래프: 물의 밀도 vs 온도 (4°C에서 꺾임)
- "물의 이상 팽창: 겨울에 호수가 위에서부터 어는 까닭" 라벨

## 시나리오 3: 바이메탈

### 3D
- 바이메탈 strip: 두 금속판을 접합
  - 상단 금속 A(높은 α): BoxGeometry(4, 0.1, 0.5), 0xcc8855 (구리색)
  - 하단 금속 B(낮은 α): BoxGeometry(4, 0.1, 0.5), 0xaaaaaa (철색)
  - 두 판이 접합된 상태
- 온도 증가 시:
  - A가 B보다 많이 팽창 → 아래쪽으로 휜다
  - 곡률이 온도에 비례
  - 바이메탈 strip이 곡선 형태로 변형 (BufferGeometry 정점 변형)
- 온도 감소 시: 반대쪽으로 휜다

### 바이메탈 응용
- 온도 조절기(서모스탯): 바이메탈이 휘어지면 전기 회로 차단/연결
- 화재 경보기: 온도 상승 → 바이메탈 휨 → 경보 작동
- 자동차 냉각 시스템: 온도에 따라 밸브 개폐

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 선팽창 / 부피 팽창 / 바이메탈
- "온도" 슬라이더: -50~300°C (기본값 20, 1 단계)
- "물질" 드롭다운 (선팽창): 알루미늄 / 구리 / 철 / 인바 / 비교 모드
- "초기 길이 L₀" 슬라이더: 1~5m (기본값 2)
- "물질별 비교" 토글: ON/OFF
- "팽창량 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📏 열팽창과 응용
━━━━━━━━━━━━━━
시나리오: [선팽창/부피팽창/바이메탈]
온도: [값]°C (ΔT = [값]°C)

선팽창:
  ΔL = αL₀ΔT
  물질: [이름] (α = [값]×10⁻⁶/K)
  L₀ = [값]m
  ΔL = [값]mm
  L = L₀ + ΔL = [값]m

부피 팽창:
  ΔV = βV₀ΔT (β ≈ 3α)

바이메탈:
  두 금속의 α 차이 → 온도 변화 시 휨
  → 서모스탯, 화재 경보기
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물질은 온도가 올라가면 팽창합니다. 선팽창 ΔL=αL₀ΔT이며, 열팽창 계수 α는 물질마다 다릅니다. 바이메탈은 α가 다른 두 금속을 접합하여 온도에 따라 휘어지는 원리를 이용합니다."
- 탐구 질문:
  1. "길이 2m인 알루미늄 막대의 온도를 100°C 올리면 길이 변화는 몇 mm인가요?"
  2. "철도 레일에 이음새 간격을 두는 까닭을 열팽창으로 설명해 보세요."
  3. "물이 4°C에서 밀도가 최대인 성질이 겨울 호수 생태계에 어떤 영향을 미치나요?"
- 실생활 예시: 철도 레일 이음새, 다리 신축 이음, 바이메탈 서모스탯, 수은 온도계
- 교과서 연계: "역학과 에너지 II편 p64~69 — 단열과 열팽창"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: mech_14_thermal_expansion.html
