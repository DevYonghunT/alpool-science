너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 산과 염기
- 토픽: 산과 염기 — 이온화와 지시약
- 대상: 고등학교 1학년
- 핵심 개념: 산은 물에 녹아서 이온화하므로 수용액 상태에서 전류가 흐른다. 산의 수용액에서 산성을 나타내는 이온은 수소 이온(H⁺)이다. 염기도 물에 녹아서 이온화하여 수용액에서 전류가 흐른다. 염기의 수용액에서 염기성을 나타내는 이온은 수산화 이온(OH⁻)이다.

## 씬 구성
- 화면을 좌/우로 분할:
  - 좌측 50%: 3D 비커 + 수용액 + 이온 입자 모형
  - 우측 50%: 지시약 색 변화 패널 (HTML overlay + 3D)
- 산/염기/중성 전환
- 배경: 0xf0f0f5 (밝은 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 비커 + 이온 모형

### 산 (염산 HCl 수용액)
- 비커: 투명 CylinderGeometry
- 수용액 색상: 무색 (투명)
- 이온 입자:
  - H⁺ (수소 이온): SphereGeometry(r=0.08), 0xff4444 (빨강), "H⁺" 라벨, 빠르게 이동
  - Cl⁻ (염화 이온): SphereGeometry(r=0.15), 0x44cc44 (초록), "Cl⁻" 라벨
  - 이온이 수용액 내에서 자유롭게 이동하는 애니메이션
- 이온화 과정 애니메이션:
  1. HCl 분자(H-Cl 결합)가 물에 들어감
  2. 물 분자와 만나 H⁺와 Cl⁻로 분리
  3. 이온들이 수용액 내에서 자유롭게 이동
- 화학 반응식: `HCl → H⁺ + Cl⁻`
- 교과서: "염산에서 산성을 나타내는 이온은 수소 이온(H⁺)이다" (p48)
- 추가 예시: H₂SO₄ → 2H⁺ + SO₄²⁻, CH₃COOH → H⁺ + CH₃COO⁻, H₂CO₃ → 2H⁺ + CO₃²⁻

### 염기 (수산화 나트륨 NaOH 수용액)
- 이온 입자:
  - Na⁺ (나트륨 이온): SphereGeometry(r=0.12), 0xffaa44 (주황), "Na⁺" 라벨
  - OH⁻ (수산화 이온): SphereGeometry(r=0.1), 0x4488ff (파랑), "OH⁻" 라벨
- 이온화: `NaOH → Na⁺ + OH⁻`
- 교과서: "수산화 나트륨 수용액에서 염기성을 나타내는 이온은 수산화 이온(OH⁻)이다" (p49)
- 추가 예시: KOH → K⁺ + OH⁻, Ca(OH)₂ → Ca²⁺ + 2OH⁻, NH₄OH → NH₄⁺ + OH⁻

## 지시약 색 변화 패널 (교과서 그림 I-25, p47)

### 리트머스 종이
- 3D 직사각형(PlaneGeometry) 2장:
  - 푸른색 리트머스 종이: 산성에서 **붉은색**으로 변화, 염기성에서 변화 없음
  - 붉은색 리트머스 종이: 염기성에서 **푸른색**으로 변화, 산성에서 변화 없음
- 색상 변화 애니메이션 (수용액에 담그면 서서히 색 변화)

### 페놀프탈레인 용액
- 3D 비커에 용액 색상:
  - 산성: 무색
  - 염기성: 붉은색 0xff3377
  - 중성: 무색

### BTB 용액
- 3D 비커에 용액 색상:
  - 산성: 노란색 0xddcc00
  - 중성: 초록색 0x44aa44
  - 염기성: 파란색 0x2266bb

### 전기 전도성 확인
- 간이 전기 전도성 측정기: 전극 2개 + LED
- 산/염기 수용액 → LED 켜짐 (이온이 있어 전류가 흐름)
- 증류수 → LED 꺼짐

## 산과 염기의 성질 표 (교과서 p47)
- 산성: 전류가 흐름, 신맛, 탄산 칼슘과 반응하면 CO₂ 기체, 금속과 반응하면 H₂ 기체
- 염기성: 전류가 흐름, 쓴맛, 미끈거림, 단백질 녹임

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "용액 종류" 드롭다운: 염산(HCl) / 황산(H₂SO₄) / 아세트산 / 수산화 나트륨(NaOH) / 수산화 칼륨(KOH) / 수산화 칼슘 / 증류수
- "지시약" 드롭다운: 리트머스 / 페놀프탈레인 / BTB
- "이온화 재생" 버튼: 이온화 애니메이션 재생
- "이온 표시" 토글: ON/OFF
- "전기 전도성 확인" 토글: ON/OFF
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 용액 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, InstancedMesh로 이온 파티클 관리
