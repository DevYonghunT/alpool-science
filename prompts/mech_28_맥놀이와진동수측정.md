너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 04. 음파의 간섭과 소음 제어
- 토픽: 맥놀이와 진동수 측정
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 진동수가 약간 다른 두 음파가 중첩되면 소리의 세기가 주기적으로 변하는 맥놀이(beat) 현상이 나타난다. 맥놀이 진동수 f_beat = |f₁ - f₂|이다. 맥놀이를 이용하면 미지의 진동수를 정밀하게 측정할 수 있다. 피아노 조율사는 맥놀이를 이용하여 악기를 조율한다.

## 씬 구성
- 화면 상단: 3D 두 소리굽쇠(tuning fork) + 음파 시각화
- 화면 하단: 3개 파형 그래프 (f₁, f₂, 합성파) Canvas 2D
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 소리굽쇠

### 소리굽쇠 1
- U자형: CylinderGeometry(0.05, 0.05, 2) × 2 (양쪽 가지) + CylinderGeometry(0.06, 0.06, 1) (손잡이)
- 색상: 0xcccccc (금속 은색)
- 진동 애니메이션: 양쪽 가지가 좌우로 미세하게 진동 (amplitude ∝ 소리 세기)
- "f₁ = [값]Hz" 라벨
- 음파: RingGeometry 동심원 확장, 0x4488ff

### 소리굽쇠 2
- 동일 형태, 0xddaa44 (금색)
- "f₂ = [값]Hz" 라벨
- 음파: RingGeometry 동심원, 0xff8844

### 맥놀이 시각화
- 두 음파가 겹치는 영역에서:
  - 보강 간섭 순간: 밝은 초록 글로우 (소리 큼)
  - 상쇄 간섭 순간: 어두움 (소리 작음/없음)
  - 이 패턴이 주기적으로 반복 → 맥놀이
- 맥놀이 주기 T_beat = 1/f_beat = 1/|f₁-f₂|

### 스피커/귀 표시
- 관측자 위치에서 소리 세기가 주기적으로 변하는 것을 시각화
- 볼륨 아이콘: 🔊↔🔇 반복

## 파형 그래프 (하단 Canvas 2D)

### 파형 1
- y₁ = Asin(2πf₁t), 파란 선
- 진동수 f₁ 표시

### 파형 2
- y₂ = Asin(2πf₂t), 빨간 선
- 진동수 f₂ 표시

### 합성파 (맥놀이)
- y = y₁ + y₂ = 2Acos(2π·(f₁-f₂)/2·t)·sin(2π·(f₁+f₂)/2·t)
- 초록 굵은 선
- **포락선**(envelope): 2Acos(2π·f_beat/2·t) — 점선으로 표시
  - 포락선 주기 = 맥놀이 주기 = 1/|f₁-f₂|
  - 소리가 커졌다 작아졌다 반복
- f₁과 f₂ 차이가 작을수록: 맥놀이 느리게 (분간 쉬움)
- f₁과 f₂ 차이가 클수록: 맥놀이 빠르게

## 진동수 차이에 따른 맥놀이 변화
- f₁ = 440Hz 고정
- f₂ 슬라이더: 435~445Hz
  - f₂ = 440Hz: 맥놀이 없음 (완전히 같은 음)
  - f₂ = 441Hz: 1초에 1번 맥놀이 (f_beat = 1Hz)
  - f₂ = 445Hz: 1초에 5번 맥놀이 (f_beat = 5Hz)
- "진동수가 같아질수록 맥놀이가 느려진다 → 조율 완료" 라벨

## 피아노 조율 응용
- 피아노 건반 3D 모형 (BoxGeometry × 몇 개)
- 기준음(소리굽쇠 440Hz)과 피아노 음을 동시에 발생
- 맥놀이가 사라질 때까지 피아노 현의 장력 조절 (슬라이더)
- 맥놀이 = 0 → "✅ 조율 완료!" 메시지

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "f₁" 슬라이더: 400~500Hz (기본값 440, 1 단계)
- "f₂" 슬라이더: 400~500Hz (기본값 443, 1 단계)
- "진폭 A" 슬라이더: 0.3~1.5 (기본값 1)
- "소리 재생" 토글: ON/OFF (Web Audio)
- "포락선 표시" 토글: ON/OFF (기본값 ON)
- "피아노 조율" 모드 토글: ON/OFF
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🎵 맥놀이와 진동수 측정
━━━━━━━━━━━━━━━
f₁ = [값]Hz, f₂ = [값]Hz

맥놀이 진동수:
  f_beat = |f₁ - f₂| = [값]Hz
  (1초에 [값]번 소리가 커졌다 작아짐)

맥놀이 주기:
  T_beat = 1/f_beat = [값]s

합성파:
  y = 2Acos(πΔf·t)sin(π(f₁+f₂)t)
  포락선 진동수 = Δf/2
  반송파 진동수 = (f₁+f₂)/2

f₁ = f₂일 때: 맥놀이 없음 → 조율 완료!
```
