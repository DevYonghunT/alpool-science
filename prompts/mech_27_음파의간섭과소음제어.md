너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 04. 음파의 간섭과 소음 제어
- 토픽: 음파의 간섭과 소음 제어 (ANC)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 두 음파가 같은 공간에서 만나면 중첩되어 간섭이 일어난다. 같은 위상으로 만나면 보강 간섭(소리 커짐), 반대 위상으로 만나면 상쇄 간섭(소리 작아짐/소멸)이 일어난다. 능동 소음 제어(ANC)는 소음과 반대 위상의 음파를 발생시켜 상쇄 간섭으로 소음을 줄이는 기술이다.

## 씬 구성
- 화면 좌측 60%: 3D 두 스피커 + 음파 간섭 패턴 시각화
- 화면 우측 40%: 합성파 그래프 + 간섭 패턴 맵 (Canvas 2D)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (위에서 내려다보기)
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 음파 간섭

### 두 스피커
- 스피커 1: BoxGeometry(0.5, 0.8, 0.3), 0x444444 + CircleGeometry(0.25) 전면, 좌측 배치
- 스피커 2: 동일, 우측 배치
- 스피커 간격 d: 슬라이더로 조절 (1~6m)

### 원형 음파
- 각 스피커에서 동심원(RingGeometry) 확장: 0x44ff88 (스피커 1) / 0xff8844 (스피커 2) 반투명
- 일정 간격으로 새 원 생성 → 확장 → 소멸

### 간섭 패턴 (위에서 내려다보기)
- 2D 평면(PlaneGeometry)에 간섭 패턴을 텍스처로 렌더링:
  - 보강 간섭 지점: 밝은 색 (0x44ff44)
  - 상쇄 간섭 지점: 어두운 색 (0x000000)
  - 경로차 = nλ → 보강 (밝음)
  - 경로차 = (n+½)λ → 상쇄 (어두움)
- 쌍곡선 형태의 간섭 무늬 패턴
- 파장 λ 또는 스피커 간격 d 변화 → 패턴 변화

### 특정 점에서의 간섭
- 마우스 클릭으로 공간의 한 점 선택 → 그 점에서의:
  - 스피커 1까지 거리 r₁
  - 스피커 2까지 거리 r₂
  - 경로차 |r₁-r₂|
  - 간섭 유형: 보강(경로차=nλ) / 상쇄(경로차=(n+½)λ)
  - 합성파 진폭 표시

## 합성파 그래프 (우측 상단 Canvas 2D)
- 3개 파형:
  - 파동 1: y₁ = Asin(kx₁-ωt), 파란 선
  - 파동 2: y₂ = Asin(kx₂-ωt), 빨간 선
  - 합성파: y = y₁+y₂, 초록 굵은 선
- 위상 차이 슬라이더: 0°~360°
  - 0° (같은 위상): 보강 → 합성파 진폭 2A
  - 180° (반대 위상): 상쇄 → 합성파 진폭 0
  - 90°: 중간

## 능동 소음 제어 (ANC) 시각화

### 3D
- 소음원: 스피커/기계, 0xff4444, 소음파(빨간 사인파) 발생
- ANC 장치: 마이크(소음 감지) + 스피커(반대 위상 음파 발생)
  - ANC 스피커: 0x4444ff, 반소음파(파란 사인파, 위상 반전) 발생
- 합성 결과: 소음+반소음 = 거의 무음!
  - 합성파(초록 사인파)의 진폭이 0에 가까움
- "ANC ON" 상태: 소음 사라짐 → "🔇 소음 제거!" 라벨
- "ANC OFF" 상태: 소음 그대로

### ANC 원리 다이어그램
- 소음 파형(빨강) + 반소음 파형(파랑, 위상 반전) = 합성(초록, 0)
- 상쇄 간섭 활용
- "소음과 정확히 반대 위상의 음파를 만들어 소음을 상쇄" 라벨

### ANC 실생활
- 노이즈 캔슬링 이어폰/헤드폰: 마이크로 외부 소음 감지 → 반대 위상 음파 발생
- 자동차 실내 소음 제어
- 산업용 소음 감소 시스템

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 두 스피커 간섭 / ANC 소음 제어
- "진동수 f" 슬라이더: 200~2000Hz (기본값 440)
- "스피커 간격" 슬라이더: 1~6m (기본값 3)
- "위상 차이" 슬라이더: 0~360° (기본값 0)
- "ANC" 토글 (ANC 모드): ON/OFF
- "간섭 패턴 맵" 토글: ON/OFF (기본값 ON)
- "음파 표시" 토글: ON/OFF (기본값 ON)
- "소리 재생" 토글: ON/OFF (Web Audio)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔊 음파의 간섭과 소음 제어
━━━━━━━━━━━━━━━━━
모드: [간섭/ANC]
f = [값]Hz, λ = v/f = [값]m
스피커 간격 d = [값]m

간섭 조건:
  보강: 경로차 = nλ (n=0,1,2...)
    → 소리 커짐 (진폭 2A)
  상쇄: 경로차 = (n+½)λ
    → 소리 소멸 (진폭 0)

ANC (능동 소음 제어):
  소음 + 반소음(위상 반전) = 0
  → 상쇄 간섭으로 소음 제거
```

- 실생활 예시: 노이즈 캔슬링 헤드폰, 콘서트홀 음향 설계, 소음 방벽, 비행기 소음 제어
