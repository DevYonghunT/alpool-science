너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 산화와 환원
- 토픽: 광합성·연소·제련 — 산소의 역할
- 대상: 고등학교 1학년
- 핵심 개념: 자연과 인류의 역사에 큰 변화를 가져온 반응인 광합성, 연소, 철의 제련에는 모두 산소가 관여한다. 광합성은 식물이 빛에너지를 흡수해 이산화 탄소와 물로부터 포도당과 산소를 만드는 과정이다. 연소는 물질이 대기 중의 산소와 만나 열에너지를 방출하는 반응이다. 철의 제련은 산화 철(III)에서 산소를 제거하고 순수한 철을 얻는 과정이다.

## 씬 구성
- 3가지 반응을 탭으로 전환하며 각각의 입자 모형을 3D로 시각화한다.
- 화면 중앙: 3D 입자 모형 애니메이션
- 화면 하단: 화학 반응식 (HTML overlay)
- 배경: 0x1a1a2e (어두운 배경, 입자 모형이 잘 보이게)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(중심, 1.0)

## 3가지 반응 상세

### 광합성 (교과서 그림 I-20, p38)
- 3D 입자 모형:
  - 반응물: CO₂ 6개(탄소 0x444444 + 산소 0xff3333 × 2) + H₂O 6개(수소 0xffffff × 2 + 산소 0xff3333)
  - 생성물: C₆H₁₂O₆ 1개(포도당, 탄소 6개+수소 12개+산소 6개 결합체) + O₂ 6개(산소 0xff3333 × 2)
  - 빛에너지: 노란 광선 파티클이 위에서 내려옴
- 애니메이션: 빛에너지 흡수 → CO₂와 H₂O가 분해 → 재결합하여 포도당+O₂ 생성 (3초)
- 화학 반응식: `6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂` (빛에너지 흡수)
- 산화/환원 표시: CO₂의 탄소가 **환원**, H₂O의 산소가 **산화**
- 배경에 나뭇잎 실루엣
- 교과서: "식물은 빛에너지를 흡수하고 이를 이용해 이산화 탄소와 물로부터 포도당과 산소를 만드는 광합성을 한다" (p38)

### 연소 (교과서 그림 I-21, p39)
- 3D 입자 모형:
  - 반응물: 화석 연료(탄소 체인 C-C-C-H...) + O₂ 여러 개
  - 생성물: CO₂ + H₂O
  - 열에너지: 빨간-주황 파티클 방출
- 애니메이션: 연료+산소 → 충돌 → CO₂ + H₂O + 열에너지 방출 (2초)
- 화학 반응식 (뷰테인 예시): `2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O` (열에너지 방출)
- 산화/환원 표시: 탄소가 **산화**, 산소가 **환원**
- 배경에 불꽃 효과(파티클)
- 교과서: "물질이 대기 중의 산소와 만나 연소하면 열에너지를 방출한다" (p39)

### 철의 제련 (교과서 그림 I-22, p39)
- 3D 입자 모형:
  - 반응물: Fe₂O₃(산화 철(III), 철 0x886644 + 산소 0xff3333) + CO(일산화 탄소, 탄소+산소)
  - 생성물: Fe(순수 철, 0xaaaaaa 은색) + CO₂
- 애니메이션: 산화 철에서 산소가 분리 → CO와 결합하여 CO₂ → 순수 철 남음 (3초)
- 화학 반응식: `Fe₂O₃ + 3CO → 2Fe + 3CO₂`
- 산화/환원 표시: 철이 **환원**(산소를 잃음), 일산화 탄소가 **산화**(산소를 얻음)
- 배경에 용광로 실루엣
- 교과서: "인류가 화석 연료의 연소를 이용해 산화 철(III)에서 산소를 제거하고 순수한 철을 얻었다" (p39)

## 공통 시각화 규칙
- 원자 모형: SphereGeometry, 원소별 색상 고정
  - 탄소(C): 0x444444 (회색-검정), r=0.15
  - 산소(O): 0xff3333 (빨강), r=0.12
  - 수소(H): 0xffffff (흰색), r=0.08
  - 철(Fe): 0xaa8866 (갈색→환원 후 0xcccccc 은색), r=0.18
- 결합: CylinderGeometry 가느다란 막대로 원자 간 연결
- 산화 반응: 빨간 화살표 "산화" 라벨
- 환원 반응: 파란 화살표 "환원" 라벨
- 산소의 이동을 점선 화살표로 강조

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "반응 종류" 드롭다운: 광합성 / 연소 / 철의 제련
- "반응 재생" 버튼: 해당 반응 애니메이션 재생
- "산화/환원 표시" 토글: ON/OFF (기본값 ON)
- "입자 크기" 슬라이더: 0.5x~2x (기본값 1x)
- "자동 전환" 버튼: 3가지 반응 순서대로 전환 (각 6초)
- "초기화" 버튼
