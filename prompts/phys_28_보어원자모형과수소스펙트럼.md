너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 2. 원자와 에너지 준위 > 원자와 스펙트럼
- 토픽: 보어 원자 모형과 수소 스펙트럼
- 대상: 고등학교 (선택과목)
- 핵심 개념: 보어는 수소 원자에서 전자가 특정한 궤도(에너지 준위)에서만 존재할 수 있다고 제안했다. 전자가 높은 에너지 준위에서 낮은 에너지 준위로 전이할 때 에너지 차이에 해당하는 빛(광자)을 방출하며, 이것이 수소의 선 스펙트럼이 된다. 에너지 준위는 양자화되어 있으며 En = -13.6/n² eV (n = 1, 2, 3, ...)이다.

## 씬 구성
- 화면 좌측 50%: 3D 보어 원자 모형 (원자핵 + 전자 궤도 + 전이 애니메이션)
- 화면 우측 50%: 에너지 준위 다이어그램 + 수소 스펙트럼 (Canvas 2D)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) + PointLight(원자핵, 0.5)

## 3D 보어 원자 모형

### 원자핵
- SphereGeometry(r=0.3, 16, 16), 0xff4444 (빨강), emissive 0.3
- "p⁺" 라벨 (양성자)
- 중심에 고정

### 전자 궤도 (n = 1, 2, 3, 4, 5)
- 각 궤도: TorusGeometry(r_n, 0.015, 8, 64)
  - n=1: r=1.5, 0x4488ff
  - n=2: r=3.0, 0x44aaff
  - n=3: r=4.5, 0x44ccff
  - n=4: r=6.0, 0x44eeff
  - n=5: r=7.5, 0x88ffff
- 궤도 옆 라벨: "n=1", "n=2", ... + 에너지 값 "E₁=-13.6eV"
- 궤도 반투명 (opacity 0.3), 선택된 궤도만 밝게

### 전자
- SphereGeometry(r=0.12), 0xffff44 (노란색), emissive 0.8
- 현재 궤도 위를 원운동 (requestAnimationFrame)
- 전자의 궤도 반지름에 따라 공전 속도 변화 (안쪽 빠르게)

### 전이 애니메이션 (핵심!)
- **방출(높→낮 전이)**: 전자가 높은 궤도 → 낮은 궤도로 이동
  1. 전자가 현재 궤도에서 점프 시작 (0.5초)
  2. 전이 궤적: 나선형 또는 직선으로 낮은 궤도로 이동
  3. 전이 순간 광자 방출: 색상 구(SphereGeometry r=0.08)가 바깥으로 발사
     - 광자 색상 = 전이 에너지에 해당하는 파장의 색 (wavelengthToColor 함수)
     - Hα(n=3→2): 0xff3333 빨강 (656nm)
     - Hβ(n=4→2): 0x00cccc 청록 (486nm)
     - Hγ(n=5→2): 0x4444ff 파랑 (434nm)
     - 라이먼 계열(n→1): 자외선 (보라-자외)
  4. 방출된 광자가 스크린/스펙트럼에 기록

- **흡수(낮→높 전이)**: 광자가 원자에 흡수되어 전자가 높은 궤도로 올라감
  1. 외부에서 광자(색상 구)가 원자핵 방향으로 날아옴
  2. 원자에 흡수 (광자 사라짐)
  3. 전자가 높은 궤도로 점프

## 에너지 준위 다이어그램 (우측 상단, Canvas 2D)

### 에너지 준위
- Y축: 에너지 (eV), -13.6 ~ 0
- 수평선: n=1(-13.6eV), n=2(-3.4eV), n=3(-1.51eV), n=4(-0.85eV), n=5(-0.54eV), n=∞(0eV)
- 각 준위 옆에 "n=[값], E=[값]eV" 라벨
- 이온화 에너지: n=∞(0eV) 위에 "이온화" 라벨

### 전이 화살표
- 방출 전이: 위→아래 빨간 화살표 + 광자 에너지 ΔE = hf
- 흡수 전이: 아래→위 파란 화살표
- 현재 선택된 전이를 밝게 강조

### 스펙트럼 계열
- 라이먼 계열 (n→1): 자외선
- 발머 계열 (n→2): 가시광선 (Hα, Hβ, Hγ, Hδ)
- 파셴 계열 (n→3): 적외선
- 선택한 계열의 전이 화살표를 그룹으로 표시

## 수소 스펙트럼 (우측 하단, Canvas 2D)
- 가시광선 영역 400~700nm 바
- 수소 방출 스펙트럼: 검은 배경 + 밝은 선
  - Hα: 656.3nm (빨강)
  - Hβ: 486.1nm (청록)
  - Hγ: 434.0nm (파랑)
  - Hδ: 410.2nm (보라)
- 전이 발생 시 해당 파장의 선이 밝아지는 애니메이션

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "전자 현재 궤도 n" 슬라이더: 1~5 (기본값 1, 정수)
- "전이 대상 궤도" 슬라이더: 1~5
- "전이 실행" 버튼: 현재→대상 전이 애니메이션 (방출 또는 흡수)
- "스펙트럼 계열" 드롭다운: 라이먼(n→1) / 발머(n→2) / 파셴(n→3) / 자유 선택
- "자동 전이" 버튼: 발머 계열 Hα→Hβ→Hγ→Hδ 순서대로
- "에너지 준위 표시" 토글: ON/OFF (기본값 ON)
- "스펙트럼 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼
