너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 1. 생태계와 환경 > 지구 환경의 변화
- 토픽: 엘니뇨와 라니냐
- 대상: 고등학교 1학년
- 핵심 개념: 엘니뇨는 열대 태평양 동쪽 해역의 해수면 온도가 평년보다 높아지는 현상이고, 라니냐는 반대로 낮아지는 현상이다. 엘니뇨와 라니냐는 전 세계적으로 이상 기후를 유발하며, 생태계에도 영향을 미친다. 지구 온난화로 인해 엘니뇨와 라니냐의 발생 빈도와 강도가 변할 수 있다.

## 씬 구성
- 화면 중앙: 3D 태평양 단면도 (해수면 온도, 대기 순환, 해류)
- 3가지 모드: 평상시 / 엘니뇨 / 라니냐
- 배경: 0x112244 (깊은 바다색)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.6)

## 3D 태평양 단면도

### 바다
- BoxGeometry(20, 3, 8) — 태평양 단면 (좌: 남미 동태평양, 우: 아시아·호주 서태평양)
- 해수면: PlaneGeometry(20, 8), 상단
- 해수 온도를 색상으로 표시:
  - 차가운 물: 0x0044aa (짙은 파랑)
  - 따뜻한 물: 0xff4400 (빨강-주황)
  - 중간: 0x44aa88 (청록)
- 해수 온도 분포가 모드에 따라 변화

### 대륙
- 좌측: 남아메리카 대륙 실루엣 (BoxGeometry 갈색 0x886644)
- 우측: 아시아/호주 대륙 실루엣 (BoxGeometry 갈색)
- 각 대륙 위에 라벨

### 무역풍 (대기 순환)
- 화살표 애니메이션: 동→서 방향 바람 (파란색 화살표 파티클)
- 평상시: 강한 무역풍 (화살표 크고 빠름)
- 엘니뇨: 약한 무역풍 (화살표 작고 느림)
- 라니냐: 매우 강한 무역풍 (화살표 매우 크고 빠름)

### 해류 (수온약층 변화)
- 수온약층: 곡선(TubeGeometry), 따뜻한 물/차가운 물 경계
- 평상시: 서쪽 깊고 동쪽 얕은 기울기
- 엘니뇨: 수온약층이 평탄해짐 (동쪽 깊어짐)
- 라니냐: 수온약층 기울기 더 심해짐

### 용승 (동태평양)
- 파란 화살표: 해저→표면 방향 (차가운 물 올라옴)
- 평상시: 적당한 용승
- 엘니뇨: 용승 약화 (화살표 작아짐)
- 라니냐: 용승 강화 (화살표 커짐)

## 3가지 모드 상세

### 평상시
- 무역풍: 동→서 강함
- 서태평양: 따뜻한 물 축적 (빨간색), 적운 구름, 비 많음
- 동태평양: 차가운 물 용승 (파란색), 맑은 날씨
- 수온약층: 서쪽 깊고 동쪽 얕음

### 엘니뇨
- 무역풍: 약해짐
- 서태평양: 따뜻한 물 감소, 건조
- 동태평양: 따뜻한 물 유입 (빨간색으로 변화), 구름+비 증가
- 수온약층: 평탄해짐
- 영향: "동태평양 해수면 온도 상승, 남미 홍수, 호주 가뭄" 라벨

### 라니냐
- 무역풍: 매우 강함
- 서태평양: 더 많은 따뜻한 물, 폭우
- 동태평양: 더 차가운 물 용승, 극심한 건조
- 수온약층: 기울기 극심
- 영향: "동태평양 해수면 온도 하강, 남미 가뭄, 호주 홍수" 라벨

## 해수면 온도 지도 (상단 HTML overlay)
- 태평양 적도 부근 해수면 온도를 색상 바로 표시
- 파랑(차가움) → 초록(평균) → 빨강(따뜻함)
- 모드에 따라 색상 분포 변화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 평상시 / 엘니뇨 / 라니냐
- "무역풍 세기" 슬라이더: 0~2 (0=없음, 1=평상시, 2=강함)
- "해수면 온도 표시" 토글: ON/OFF (기본값 ON)
- "대기 순환 표시" 토글: ON/OFF (기본값 ON)
- "영향 표시" 토글: ON/OFF
- "자동 전환" 버튼: 평상시→엘니뇨→라니냐 (각 5초)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌊 엘니뇨와 라니냐
━━━━━━━━━━━━━━
현재 모드:      [평상시/엘니뇨/라니냐]
무역풍 세기:    [약/보통/강]
동태평양 수온:  [평년 대비 +/-N°C]
용승 강도:      [약/보통/강]

엘니뇨:
  동태평양 해수면 온도 ↑
  무역풍 약화, 용승 약화
  → 남미 홍수, 호주 가뭄

라니냐:
  동태평양 해수면 온도 ↓
  무역풍 강화, 용승 강화
  → 남미 가뭄, 호주 홍수
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "엘니뇨는 열대 태평양 동쪽 해역의 해수면 온도가 평년보다 높아지는 현상이고, 라니냐는 반대로 낮아지는 현상입니다. 이들은 전 세계적으로 이상 기후를 유발하며 생태계에도 영향을 미칩니다."
- 탐구 질문:
  1. "엘니뇨가 발생하면 동태평양의 해수면 온도와 무역풍은 어떻게 변하나요?"
  2. "엘니뇨와 라니냐가 전 세계 기후에 미치는 영향의 차이점을 설명해 보세요."
  3. "지구 온난화가 엘니뇨/라니냐에 미치는 영향은 무엇일까요?"
- 실생활 예시: 2023~2024 슈퍼 엘니뇨, 한국의 이상 기후(폭염/한파), 페루 어업 피해
- 교과서 연계: "미래엔 통합과학2 II편 p88~89 — 엘니뇨와 라니냐, 이상 기후"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 색상/파티클 속성 변경 (dispose 불필요)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
- 파일명: sci2_19_el_nino_la_nina.html
