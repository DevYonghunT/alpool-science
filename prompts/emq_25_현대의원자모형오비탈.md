너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 04. 현대의 원자 모형
- 토픽: 현대의 원자 모형 — 오비탈
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 현대 원자 모형에서 전자는 정해진 궤도를 도는 것이 아니라 특정 공간 영역(오비탈)에 존재할 확률로 기술된다. 오비탈은 4가지 양자수(주양자수 n, 부양자수 l, 자기양자수 mₗ, 스핀양자수 mₛ)로 결정된다. s 오비탈(l=0)은 구형, p 오비탈(l=1)은 아령형(3개: px, py, pz), d 오비탈(l=2)은 복잡한 형태(5개)를 가진다. |ψ(r,θ,φ)|²이 전자의 존재 확률 밀도이다.

## 씬 구성
- 화면 중앙: 3D 오비탈 형태 시각화 (확률 밀도 분포)
- 좌측 하단: 양자수 정보 패널 (HTML overlay)
- 우측: 에너지 준위 다이어그램 (Canvas 2D / HTML)
- 배경: 0x050515
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + PointLight(0.5, 원점)

## 3D 오비탈 시각화

### 원자핵
- SphereGeometry(r=0.15), MeshStandardMaterial, 0xff4444 emissive 0xff2222
- 원점에 배치, 항상 표시

### s 오비탈 (l=0)
- 1s: 구형 확률 분포, SphereGeometry(r=2), 0x4488ff opacity 0.2
  - 내부에 Points(2000개)로 확률 밀도 표현
  - 중심에서 밀도 높고, 바깥으로 갈수록 감소 (지수 감소)
  - 입자 색상: 0x66bbff, 크기 0.03
- 2s: 더 큰 구 + 내부에 절점(node) 구면 1개
  - 절점 표시: 투명 구면(SphereGeometry, wireframe, 0xffaa44)
  - "절점: ψ=0인 곳, 전자 존재 확률 0" 라벨
- 3s: 더 큰 구 + 절점 구면 2개

### p 오비탈 (l=1)
- px: x축 방향 아령 모양 (±x 방향에 확률 밀도 집중)
  - 양의 로브: 0x4488ff, 음의 로브: 0xff6644 (파동 함수 부호 표현)
  - Points로 각 로브 내부 확률 분포
- py: y축 방향 아령
- pz: z축 방향 아령
- mₗ = -1, 0, +1 → 3개 오비탈 방향
- 절평면(nodal plane) 표시: PlaneGeometry, 0xffaa44 opacity 0.1

### d 오비탈 (l=2)
- dxy: xy평면에 네잎클로버 형태
- dxz: xz평면에 네잎클로버
- dyz: yz평면에 네잎클로버
- dx²-y²: x, y축 방향 네잎클로버
- dz²: z축 방향 도넛+양쪽 로브
- mₗ = -2, -1, 0, +1, +2 → 5개 오비탈

### 확률 밀도 시각화 방식
- "점 분포" 모드: Points로 |ψ|²에 비례하는 밀도로 점 배치
- "등확률면" 모드: MeshStandardMaterial, 특정 |ψ|² 값의 등위면(isosurface) 반투명 표시
- "단면" 모드: 특정 평면(xy, xz, yz)에서의 |ψ|² 컬러맵 (높은 확률=밝음)

## 에너지 준위 다이어그램 (우측)
- 수직축: 에너지 (위로 갈수록 높음)
- 수평 선: 각 오비탈의 에너지 준위
  - 1s (가장 낮음)
  - 2s, 2p
  - 3s, 3p, 3d
  - 4s, 4p, 4d, 4f
- 축퇴(degeneracy): 같은 n, l → 같은 에너지 (수소 원자)
  - 2p: 3개 오비탈(px, py, pz) → 같은 에너지 선 3개
  - 3d: 5개 오비탈 → 같은 에너지 선 5개
- 현재 선택된 오비탈 하이라이트
- 클릭하면 해당 오비탈 3D 표시

## 양자수 정보 패널

### 4가지 양자수
- n (주양자수): 1, 2, 3, ... → 에너지 준위, 크기
- l (부양자수): 0 ~ n-1 → 오비탈 형태 (0=s, 1=p, 2=d, 3=f)
- mₗ (자기양자수): -l ~ +l → 오비탈 방향
- mₛ (스핀양자수): +1/2 또는 -1/2 → 전자 스핀
- 현재 선택: n=[값], l=[값](=[s/p/d/f]), mₗ=[값]

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "주양자수 n" 슬라이더: 1~4 (기본값 1)
- "부양자수 l" 슬라이더: 0~(n-1) (기본값 0, n에 따라 범위 변경)
- "자기양자수 mₗ" 슬라이더: -l~+l (기본값 0)
- "오비탈" 드롭다운: 1s / 2s / 2p / 3s / 3p / 3d / ... (양자수와 연동)
- "시각화 방식" 드롭다운: 점 분포 / 등확률면 / 단면
- "단면 평면" 드롭다운 (단면 모드): xy / xz / yz
- "절점 표시" 토글: ON/OFF (기본값 ON)
- "에너지 다이어그램" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚛️ 현대의 원자 모형 — 오비탈
━━━━━━━━━━━━━━━━━━━
오비탈: [n][s/p/d/f] (mₗ=[값])

양자수:
  n = [값] (주양자수: 에너지·크기)
  l = [값] (부양자수: 형태)
  mₗ = [값] (자기양자수: 방향)
  mₛ = ±1/2 (스핀양자수)

오비탈 형태:
  s (l=0): 구형, 방향 없음
  p (l=1): 아령형, 3방향 (px,py,pz)
  d (l=2): 클로버형, 5방향

확률 밀도:
  |ψ(r,θ,φ)|² = 전자 존재 확률
  절점: ψ=0, 확률=0인 면/구
  절점 수: n-l-1개 (방사 절점)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "현대 원자 모형에서 전자는 궤도(orbit)가 아닌 오비탈(orbital)에 존재합니다. 오비탈은 전자가 발견될 확률이 높은 공간 영역이며, 양자수(n, l, mₗ)에 의해 에너지·형태·방향이 결정됩니다."
- 탐구 질문:
  1. "보어 모형의 '궤도'와 현대 원자 모형의 '오비탈'은 어떻게 다른가요?"
  2. "2p 오비탈이 3개인 까닭을 자기양자수 mₗ로 설명해 보세요."
  3. "s 오비탈에서 주양자수 n이 커지면 절점(node)의 수는 어떻게 변하나요?"
- 실생활 예시: 화학 결합(오비탈 겹침), 분광학(전자 전이), MRI(핵스핀), LED(에너지 준위 전이)
- 교과서 연계: "전자기와 양자 III편 p112~117 — 현대의 원자 모형, 오비탈과 양자수"
