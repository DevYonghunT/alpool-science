너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 01. 거울과 렌즈
- 토픽: 거울의 반사와 상 (평면/볼록/오목)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 평면 거울에서 상은 거울 뒤에 같은 거리에 정립 허상이 맺힌다. 오목 거울은 빛을 모아 초점에 집중시키며, 물체 위치에 따라 실상 또는 허상이 맺힌다. 볼록 거울은 빛을 퍼뜨려 항상 축소된 정립 허상이 맺힌다. 거울 방정식: 1/f = 1/a + 1/b (f: 초점 거리, a: 물체 거리, b: 상 거리). 배율 m = -b/a.

## 씬 구성
- 화면 중앙: 3D 거울 + 물체 + 광선 추적(ray tracing) 시각화
- 3가지 거울: 평면 / 오목(concave) / 볼록(convex)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 거울과 광선 추적

### 거울
- 평면 거울: PlaneGeometry(3, 3), MeshStandardMaterial(metalness 0.9, roughness 0.05), 0xddddee
- 오목 거울: SphereGeometry(r=R, θ=π/3) 안쪽 반사면, 0xddddee 반사
  - 곡률 반지름 R, 초점 거리 f = R/2
  - 초점 F: SphereGeometry(r=0.05), 0xff4444, "F" 라벨
  - 곡률 중심 C: SphereGeometry(r=0.05), 0x4444ff, "C" 라벨
- 볼록 거울: SphereGeometry(r=R) 바깥쪽 반사면
  - 초점 F: 거울 뒤쪽 (가상 초점)
- 주축: CylinderGeometry 수평 점선, 0xaaaaaa

### 물체 (화살표)
- 수직 화살표: CylinderGeometry(0.03, 0.03, h) + ConeGeometry 촉, 0x44ff44 (초록)
- 물체 거리 a: 슬라이더로 조절 (주축을 따라 위치 이동)
- "물체" 라벨

### 광선 추적 (핵심! 3개 주요 광선)
- **광선 1 (평행 광선)**: 주축에 평행하게 입사 → 반사 후 초점 F를 통과 (오목) / F에서 발산하는 것처럼 (볼록)
  - 색상: 0xff4444 (빨강)
- **광선 2 (초점 광선)**: 초점 F를 통과하여 입사 → 반사 후 주축에 평행
  - 색상: 0x4444ff (파랑)
- **광선 3 (중심 광선)**: 곡률 중심 C를 향하여 입사 → 같은 경로로 되돌아감
  - 색상: 0x44ff44 (초록)
- 3개 광선이 만나는 점 = 상의 위치!
- 광선: CylinderGeometry 또는 Line, 반사점에서 꺾임

### 상 (이미지)
- 3개 광선이 만나는 위치에 상 화살표 표시
- 실상: 반사 광선이 실제로 만남 → 실선 화살표, 0xff8844 (주황)
- 허상: 반사 광선의 연장선이 만남 → 점선 화살표, 0xff8844 점선
- 상의 크기/방향: 배율 m = -b/a에 따라
  - |m| > 1: 확대 / |m| < 1: 축소
  - m > 0: 정립 / m < 0: 도립

## 오목 거울: 물체 위치별 상의 변화 (핵심!)

### 물체가 C 바깥 (a > 2f)
- 실상, 도립, 축소, C와 F 사이에 맺힘

### 물체가 C에 (a = 2f)
- 실상, 도립, 같은 크기, C에 맺힘

### 물체가 F와 C 사이 (f < a < 2f)
- 실상, 도립, 확대, C 바깥에 맺힘

### 물체가 F에 (a = f)
- 상이 무한대에 맺힘 (평행 반사광)

### 물체가 F 안쪽 (a < f)
- 허상, 정립, 확대, 거울 뒤에 맺힘 (화장 거울!)

## 볼록 거울
- 항상: 허상, 정립, 축소
- 넓은 범위를 볼 수 있음 (자동차 사이드 미러, 편의점 보안 거울)

## 거울 방정식 계산 (우측 패널)
- 1/f = 1/a + 1/b → b = af/(a-f)
- m = -b/a = -f/(a-f)
- 입력: a, f → 출력: b, m, 상의 종류(실/허, 도립/정립, 확대/축소)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "거울" 드롭다운: 평면 / 오목 / 볼록
- "초점 거리 f" 슬라이더 (오목/볼록): 1~8 (기본값 3)
- "물체 거리 a" 슬라이더: 1~15 (기본값 6)
- "물체 높이" 슬라이더: 0.5~3 (기본값 1.5)
- "광선 표시" 토글: ON/OFF (기본값 ON)
- "상 표시" 토글: ON/OFF (기본값 ON)
- "초점/곡률중심 표시" 토글: ON/OFF (기본값 ON)
- "거울 방정식" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🪞 거울의 반사와 상
━━━━━━━━━━━━━━
거울: [평면/오목/볼록]
f = [값], a = [값]

거울 방정식: 1/f = 1/a + 1/b
  b = af/(a-f) = [값]
  배율 m = -b/a = [값]

상의 종류:
  [실상/허상], [정립/도립], [확대/축소]
  크기: |m|×물체높이 = [값]

오목 거울:
  a > 2f → 실상, 도립, 축소
  a = 2f → 실상, 도립, 같은 크기
  f < a < 2f → 실상, 도립, 확대
  a < f → 허상, 정립, 확대
```

- 실생활 예시: 화장 거울(오목), 자동차 미러(볼록), 태양열 집열기(오목 반사경), 치과 거울
