너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 05. 레이저
- 토픽: 광통신과 광섬유
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 광섬유는 코어(높은 굴절률)와 클래딩(낮은 굴절률)으로 구성되며, 전반사를 이용하여 빛 신호를 먼 거리까지 전달한다. 광통신은 전기 신호를 빛 신호로 변환하여 광섬유를 통해 전송하고, 수신측에서 다시 전기 신호로 변환한다. 파장 분할 다중화(WDM)로 하나의 광섬유에 여러 파장의 빛을 동시에 전송할 수 있다.

## 씬 구성
- 화면 좌측 50%: 3D 광섬유 단면/측면 + 전반사 시각화
- 화면 우측 50%: 광통신 시스템 블록 다이어그램 + WDM (HTML/Canvas)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 광섬유

### 광섬유 구조 (단면)
- 코어: CylinderGeometry(0.15, 0.15, 10), 0x88ccff 반투명 (n_코어 = 1.62)
- 클래딩: CylinderGeometry(0.25, 0.25, 10), 0xddddee 반투명 (n_클래딩 = 1.52)
- 피복: CylinderGeometry(0.3, 0.3, 10), 0xffaa44 (주황, 보호 피복)
- 단면도 모드: 원형 단면에서 코어/클래딩/피복 동심원 표시

### 전반사 시각화 (핵심!)
- 곡선 광섬유: TubeGeometry로 S자 또는 구부러진 경로
- 빛 파티클(SphereGeometry r=0.03, 0xffdd44):
  - 코어 내부에서 지그재그로 이동
  - 코어-클래딩 경계에서 전반사 반복
  - 반사될 때마다 작은 반사 표시 (반짝 효과)
- 임계각 표시: sinθ_c = n_클래딩/n_코어
  - θ_c보다 큰 각도로 입사 → 전반사 (빛 유지)
  - θ_c보다 작은 각도 → 일부 빛 손실 (굴절로 빠져나감)
- 입사각 슬라이더: 전반사 조건 확인

### 광섬유가 구부러져도 빛이 전달되는 원리
- 곡률 슬라이더: 직선~크게 구부러짐
- 적당히 구부러짐: 전반사 유지 → 빛 전달 ✅
- 너무 급하게 구부러짐: 전반사 조건 깨짐 → 빛 손실 ❌

## 광통신 시스템 블록 다이어그램 (우측)

### 송신부
1. 전기 신호(디지털 데이터): 0과 1의 디지털 파형
2. 레이저 다이오드(LED): 전기 → 빛 변환
   - "0": 빛 OFF, "1": 빛 ON (또는 세기 변조)
3. 광섬유에 빛 신호 입력

### 전송부
4. 광섬유: 전반사로 빛 신호 전달 (수십~수백 km)
5. 증폭기: 장거리 전송 시 신호 감쇠 보상

### 수신부
6. 광 검출기(포토다이오드): 빛 → 전기 변환
7. 전기 신호 복원: 디지털 데이터

### 3D 시각화
- 각 블록을 BoxGeometry로 배치, 화살표로 연결
- 데이터 파티클: 전기 → 빛 → 전기 변환 과정을 색상 변화로

## 파장 분할 다중화 (WDM)

### 원리
- 하나의 광섬유에 여러 파장의 빛을 동시에 전송
  - λ₁ = 1310nm (빨강 표시): 채널 1
  - λ₂ = 1490nm (초록 표시): 채널 2
  - λ₃ = 1550nm (파랑 표시): 채널 3
- 프리즘/회절격자로 합성(MUX) → 광섬유 전송 → 분리(DEMUX)
- "대역폭 증가: 하나의 광섬유로 여러 채널 동시 전송" 라벨

### 3D
- 광섬유 내부에 3가지 색상 파티클이 동시에 이동
- 합성기(MUX): 3개 입력 → 1개 출력
- 분리기(DEMUX): 1개 입력 → 3개 출력

## 광섬유 vs 구리선 비교
| 특성 | 광섬유 | 구리선 |
|------|--------|--------|
| 전송 속도 | 매우 빠름 | 느림 |
| 대역폭 | 매우 넓음 | 좁음 |
| 전자기 간섭 | 없음 | 있음 |
| 무게 | 가벼움 | 무거움 |
| 거리 | 장거리 | 단거리 |

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 광섬유 전반사 / 광통신 시스템 / WDM
- "입사각" 슬라이더 (전반사): 0~90° (기본값 60)
- "광섬유 곡률" 슬라이더: 0~5 (기본값 1)
- "코어 굴절률" 슬라이더: 1.4~1.8 (기본값 1.62)
- "클래딩 굴절률" 슬라이더: 1.3~1.6 (기본값 1.52)
- "데이터 전송" 버튼 (광통신): 01101 디지털 신호 전송 시뮬레이션
- "WDM 채널 수" 슬라이더: 1~4
- "비교 표" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌐 광통신과 광섬유
━━━━━━━━━━━━━━
보기: [전반사/광통신/WDM]

광섬유:
  코어 n₁ = [값], 클래딩 n₂ = [값]
  임계각 θ_c = arcsin(n₂/n₁) = [값]°
  θ > θ_c → 전반사 (빛 유지)

광통신 시스템:
  전기 → 레이저(빛) → 광섬유
  → 검출기(전기) → 데이터 복원

WDM (파장 분할 다중화):
  여러 파장 → 1개 광섬유에 동시 전송
  대역폭 = N × 단일 채널 대역폭

장점: 고속, 대용량, 전자기 간섭 없음
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "광섬유는 전반사를 이용하여 빛 신호를 먼 거리까지 전달합니다. 광통신은 전기→빛→전기 변환으로 고속 대용량 데이터를 전송합니다. WDM으로 하나의 광섬유에 여러 채널을 동시에 전송할 수 있습니다."
- 탐구 질문:
  1. "광섬유에서 코어의 굴절률이 클래딩보다 큰 까닭을 전반사와 관련지어 설명해 보세요."
  2. "광섬유를 너무 급하게 구부리면 빛이 손실되는 까닭은?"
  3. "광통신이 구리선 통신보다 전자기 간섭에 강한 까닭은?"
- 실생활 예시: 인터넷 해저 케이블, FTTH(가정용 광섬유), 내시경(의료), 장식 광섬유 조명
- 교과서 연계: "전자기와 양자 II편 p84~91 — 레이저, 광통신, 광섬유"
