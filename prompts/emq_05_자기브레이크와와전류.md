너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 04. 전자기 유도의 활용
- 토픽: 자기 브레이크와 와전류
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 도체가 자기장 속에서 운동하면 도체 내부에 유도 전류(와전류)가 발생한다. 렌츠 법칙에 의해 와전류는 도체의 운동을 방해하는 방향으로 흐르므로 제동력이 작용한다. 이를 이용한 것이 자기 브레이크(eddy current brake)이다. 와전류는 인덕션 레인지, 금속 탐지기 등에도 활용된다.

## 씬 구성
- 3가지 시나리오: 자기 브레이크 / 자석 낙하(구리관) / 인덕션 레인지
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 시나리오 1: 자기 브레이크

### 3D
- 회전 금속 원판: CylinderGeometry(3, 3, 0.1, 32), 0xcccccc (알루미늄)
  - 원판이 회전축을 중심으로 회전
- 전자석(자기 브레이크): BoxGeometry(1, 1, 0.5) × 2, 원판 양쪽에 배치
  - N극(빨강)/S극(파랑) 표시
  - 자기장: 원판을 관통하는 화살표

### 와전류 시각화
- 자석이 원판에 가까이 있을 때:
  - 원판 표면에 와전류(소용돌이 형태) 곡선 화살표 표시 (노란색)
  - 와전류에 의한 자기장 → 자석의 자기장과 반대 → 제동력
  - 원판 회전 속도 서서히 감소 (브레이크 효과)
- 자석 ON/OFF 토글:
  - ON: 와전류 발생 → 원판 감속
  - OFF: 와전류 없음 → 자유 회전 유지

### 렌츠 법칙 설명
- "와전류는 자기 선속 변화를 방해하는 방향으로 흐른다"
- "→ 원판의 운동을 방해 → 제동력 (자기 브레이크)"

## 시나리오 2: 자석의 구리관 낙하

### 3D
- 구리관: CylinderGeometry(0.8, 0.8, 6, 32, 1, true), 0xcc8855 (구리색), 수직
- 자석: CylinderGeometry(0.3, 0.3, 0.6), N극(빨강)/S극(파랑)
- 일반 물체(비교): SphereGeometry(r=0.3), 0x888888

### 낙하 비교
- 일반 물체: 자유 낙하 (빠르게 떨어짐)
- 자석: 구리관 안에서 매우 천천히 떨어짐!
  - 자석 주위 자기장이 구리관 벽에 와전류 유도
  - 와전류의 자기장이 자석의 낙하를 방해 (렌츠 법칙)
  - 종단 속도에 도달하여 등속으로 천천히 내려감
- 두 물체 동시 낙하 비교 애니메이션
- "자석이 천천히 떨어지는 까닭 = 와전류에 의한 제동력" 라벨

## 시나리오 3: 인덕션 레인지

### 3D
- 인덕션: BoxGeometry(3, 0.2, 3), 0x222222 (검정 상판)
  - 내부 코일: TorusGeometry, 0xcc6633 (구리)
  - 교류 전류 → 변하는 자기장 발생
- 금속 냄비: CylinderGeometry(1, 1, 0.8), 0x888888
  - 냄비 바닥에 와전류 유도 → 열 발생 (줄 열)
  - 와전류 시각화: 냄비 바닥에 소용돌이 곡선
  - 열 파티클: 빨간-주황 파티클 상승
- 유리 냄비: 와전류 발생 안 됨 → 가열 안 됨 → "❌" 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 자기 브레이크 / 구리관 낙하 / 인덕션 레인지
- "자기장 세기" 슬라이더: 0.1~2T (기본값 0.5)
- "회전 속도" 슬라이더 (브레이크): 1~10 RPM
- "브레이크 ON/OFF" 토글
- "와전류 표시" 토글: ON/OFF (기본값 ON)
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔧 자기 브레이크와 와전류
━━━━━━━━━━━━━━━━
시나리오: [브레이크/구리관/인덕션]

와전류 (eddy current):
  도체가 자기장 속에서 운동하면
  도체 내부에 소용돌이 형태 유도 전류 발생

렌츠 법칙:
  와전류는 자기 선속 변화를 방해
  → 운동 방해 방향 → 제동력

자기 브레이크:
  와전류의 제동력으로 감속
  마찰 없이 비접촉 제동!

인덕션 레인지:
  교류 → 변하는 자기장 → 와전류
  → 줄 열 발생 → 금속 냄비만 가열
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "도체가 자기장 속에서 운동하면 와전류가 발생하고, 렌츠 법칙에 의해 운동을 방해하는 제동력이 작용합니다. 자기 브레이크, 인덕션 레인지, 금속 탐지기 등에 활용됩니다."
- 탐구 질문:
  1. "자석이 구리관 안에서 천천히 떨어지는 까닭을 와전류와 렌츠 법칙으로 설명해 보세요."
  2. "인덕션 레인지 위에 유리 냄비를 올리면 가열되지 않는 까닭은 무엇인가요?"
  3. "자기 브레이크가 일반 마찰 브레이크보다 유리한 점은 무엇인가요?"
- 실생활 예시: 롤러코스터 자기 브레이크, 인덕션 레인지, 금속 탐지기, 전기 계량기
- 교과서 연계: "전자기와 양자 I편 p26~31 — 전자기 유도의 활용, 와전류"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: emq_05_eddy_current_brake.html
