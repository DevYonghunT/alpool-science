너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 1. 전기장과 전기 에너지 > 소비 전력과 전기 안전
- 토픽: 옴의 법칙과 소비 전력
- 대상: 고등학교 (선택과목)
- 핵심 개념: 옴의 법칙에 의하면 도체에 흐르는 전류의 세기는 도체 양 끝의 전압에 비례하고, 도체의 저항에 반비례한다. V = IR. 전기 기구가 단위 시간에 소비하는 전기 에너지를 소비 전력이라 하며 P = VI = I²R = V²/R이다. 가정에서 사용하는 전기 기구의 전력량 = 소비 전력 × 사용 시간이다.

## 씬 구성
- 화면 좌측 50%: 3D 전기 회로 (전지 + 저항 + 전선 + 전구)
- 화면 우측 50%: V-I 그래프 + 전력 계산기 (Canvas 2D / HTML overlay)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 전기 회로

### 전지 (전원)
- BoxGeometry(0.8, 1.2, 0.4), 0x444444 (검정)
- 양극(+): 빨간 표시, 음극(−): 파란 표시
- 전압 V: 라벨 표시 "[값]V", 슬라이더로 조절
- 전압 슬라이더: 1~12V (기본값 6, 1 단계)

### 저항 (부하)
- CylinderGeometry(0.15, 0.15, 0.8), 색띠 패턴(4색 띠 저항)
  - 색띠 색상: 저항값에 따라 자동 변경
- 저항값 R: 라벨 표시 "[값]Ω", 슬라이더로 조절
- 저항 슬라이더: 1~100Ω (기본값 10)

### 전선
- TubeGeometry 곡선, 0xcc6633 (구리색)
- 전지 양극 → 저항 → 전지 음극 연결
- 전선 내부 전자 흐름: 작은 노란 구(SphereGeometry r=0.03) 파티클이 전류 방향으로 이동
- 전류 세기에 따라 전자 이동 속도 변화

### 전구 (시각적 표현)
- SphereGeometry(r=0.25, 투명 유리) + 필라멘트
- 전류에 따라 밝기 변화: emissiveIntensity = I/I_max
- I가 크면 밝게, I가 0이면 꺼짐

### 전류계/전압계
- 전류계: CircleGeometry + 바늘, 전류값 표시 "I = [값]A"
- 전압계: CircleGeometry + 바늘, 전압값 표시 "V = [값]V"

## V-I 그래프 (우측 Canvas 2D)
- X축: 전압 V (0~12V)
- Y축: 전류 I (0~5A)
- 직선: I = V/R (기울기 = 1/R)
- R 값에 따라 기울기 변화
- 현재 (V, I) 위치에 빨간 점
- 여러 R 값의 직선을 겹쳐 표시 가능

## 소비 전력 계산기 (우측 하단)
- P = VI = [값]W
- P = I²R = [값]W
- P = V²/R = [값]W
- 세 공식 결과가 동일함 확인
- 전력량 W = Pt = [값]Wh (사용 시간 슬라이더)
- 전기요금 예상: [값]원 (1kWh = 약 100원 기준)

## 전기 안전 (교과서 내용)
- "전기 안전" 토글 시:
  - 과전류: 전류가 너무 크면 전선 과열 → 빨간 글로우 + 경고 ⚠️
  - 누전: 전류가 인체로 흐를 때의 위험 표시
  - 합선: 저항 없이 전원 직접 연결 → 큰 전류 → 화재 위험
  - 접지: 접지선 표시, 안전 확보
  - 차단기: 과전류 시 자동 차단 애니메이션

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "전압 V" 슬라이더: 1~12V (기본값 6, 1 단계)
- "저항 R" 슬라이더: 1~100Ω (기본값 10, 1 단계)
- "전자 흐름 표시" 토글: ON/OFF (기본값 ON)
- "전류계/전압계 표시" 토글: ON/OFF (기본값 ON)
- "V-I 그래프에 점 기록" 버튼: 현재 (V,I)를 그래프에 기록
- "사용 시간" 슬라이더: 1~24시간 (전력량 계산용)
- "전기 안전" 토글: ON/OFF
- "초기화" 버튼

## 코드 규칙

- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리

- 성능: requestAnimationFrame 루프, 전자 파티클 최대 50개
