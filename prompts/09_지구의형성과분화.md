너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 별의 진화와 원소의 생성
- 토픽: 지구의 형성과 분화
- 대상: 고등학교 1학년
- 핵심 개념: 미행성체들이 충돌하면서 원시 지구가 형성되었다. 원시 지구는 대부분이 녹아 마그마 바다를 형성했고, 밀도 차이에 의해 철과 같은 무거운 물질이 중심부로 가라앉아 핵을, 산소와 규소 등의 가벼운 물질이 떠올라 맨틀과 지각을 형성했다. 이후 지표면이 식어 원시 지각과 원시 바다가 형성되었고 바다에서 생명체가 탄생했다.

## 씬 구성
- 화면 중앙에 지구의 3D 단면도를 렌더링한다.
- 단면도: 지구를 반으로 잘라(ClippingPlane 사용) 내부 층을 보여줌
- 배경: 우주 0x000010, 별 파티클 Points(800개)
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls (지구를 회전하며 관찰)
- 조명: AmbientLight(0.2) + DirectionalLight(0.6, 약간 위에서 비추기) + PointLight(핵 위치, 붉은 발광)

## 형성 과정 5단계 상세 (교과서 그림 II-13, p58)

### 단계 1: 미행성체 충돌 → 원시 지구
- 3D: 작은 암석 덩어리(IcosahedronGeometry, 20개)들이 화면 바깥에서 중심을 향해 날아와 충돌
- 충돌 애니메이션:
  1. 미행성체 하나가 중심의 작은 구(r=0.3)에 충돌
  2. 충돌 시: 밝은 주황 플래시 + 파편 파티클(Points 15개)
  3. 충돌 후: 중심 구의 반지름이 약간 증가 (0.3 → 0.35 → ... → 2.0)
  4. 5~6회 충돌 반복하여 원시 지구(r=2.0) 완성
- 원시 지구 색상: 0xcc4400 (뜨거운 붉은색, 표면에 충돌 흔적 밝은 점들)
- 교과서: "미행성체들이 충돌하면서 만들어진 원시 지구" (p58 그림 II-13)

### 단계 2: 마그마 바다 형성
- 3D: 원시 지구 전체가 녹은 상태
  - 표면: SphereGeometry(r=2.0), ShaderMaterial로 용암 효과
    - 기본 색상: 0xff4400 (짙은 주황) + 0xffcc00 (밝은 노랑) 노이즈 패턴
    - 시간에 따라 노이즈 패턴이 느리게 흐름 (마그마 대류 효과)
  - 내부: 균일하게 빨강-주황 혼합, 아직 층 분리 안 됨
  - 표면에서 가스/수증기 파티클(Points 100개) 위로 올라가는 효과 (원시 대기 형성)
- 발광: 전체적으로 emissive 0xff4400, 강한 빨간 PointLight
- 교과서: "원시 지구는 대부분이 녹아 마그마 바다를 형성했다" (p58)

### 단계 3: 밀도 차이에 의한 층 분화
- 3D: 지구 단면도에서 물질이 분리되는 애니메이션
  - 무거운 물질(철, 니켈) 표현:
    - 작은 회색-은색 구(SphereGeometry r=0.05, 0x888888) 30개
    - 표면/중간에서 서서히 중심으로 가라앉는 애니메이션 (2초)
    - 중심에 모여 핵(SphereGeometry r=0.7, 0xaa7744 금속 갈색) 형성
  - 가벼운 물질(산소, 규소) 표현:
    - 작은 연갈색 구(r=0.04, 0xccaa88) 30개
    - 중심에서 서서히 표면으로 떠오르는 애니메이션 (2초)
    - 외곽에 모여 맨틀+지각 층 형성
- 분화 완료 후 단면도 층 구조:
  | 층 | 원소 | 색상 | 반지름 범위 | 상태 |
  |---|------|------|-----------|------|
  | 내핵 | 철, 니켈 | 0xddaa44 밝은 금속 | r=0~0.35 | 고체 |
  | 외핵 | 철, 니켈 | 0xcc7722 어두운 금속 | r=0.35~0.7 | 액체 |
  | 맨틀 | 산소, 규소, 마그네슘 | 0xcc5533 붉은 갈색 | r=0.7~1.8 | 고체(유동성) |
  | 지각 | 산소, 규소, 알루미늄 | 0x886644 갈색 | r=1.8~2.0 | 고체 |
- 각 층 경계에 흰색 점선 원
- 각 층에 이름 + 주요 원소 라벨
- 교과서: "철과 같은 무거운 물질이 중심부로 가라앉아 핵을 이루고, 산소와 규소 등의 가벼운 물질이 떠올라 맨틀과 지각을 형성했다" (p58)

### 단계 4: 원시 지각 + 원시 바다 형성
- 3D: 지구 외부 뷰 (단면도 해제, 전체 구로 표시)
  - 지표면 냉각: 표면 색상이 서서히 변화 (용암 주황 0xff4400 → 암석 갈색 0x664422 → 대륙+바다)
    - 냉각 애니메이션: 3초에 걸쳐 색상 전환
  - 냉각 완료 후:
    - 대륙: 갈색-녹색 패치 0x886644 + 0x447744
    - 바다: 파란색 영역 0x2255aa
    - 구름: 반투명 흰색 SphereGeometry(r=2.05, opacity 0.15) 느리게 회전
  - 비 애니메이션: 작은 파란 파티클(Points 200개)이 하늘에서 지표로 내려옴 → 낮은 곳에 고임 → 원시 바다 형성
- 교과서: "이후 지표면이 식어 원시 지각과 원시 바다가 형성되었다" (p58)

### 단계 5: 최초의 생명체 탄생
- 3D: 바다 속 장면
  - 배경을 우주에서 바다 속으로 전환 (0x000010 → 0x003355)
  - 해저: PlaneGeometry, 0x332211 진한 갈색
  - 열수 분출구: CylinderGeometry(r=0.3, h=1), 0x444444, 꼭대기에서 뜨거운 파티클(Points 50개, 0xff6600) 위로 솟아오름
  - 최초의 생명체: 작은 구(r=0.05, 0x44ff44 연초록) 여러 개가 열수 분출구 주변에서 천천히 움직임
  - 기포 파티클: SphereGeometry(r=0.02~0.05, 반투명 0xaaddff) 아래→위로 떠오름
- 교과서: "바다에서 생명체가 탄생했다" (p58)
- 핵심 메시지: "결국 지구와 생명체의 역사는 빅뱅에서 시작된 우주 역사의 일부분이라고 할 수 있다" (p58)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "형성 단계" 슬라이더: 0 ~ 4 (5단계, 정수)
  - 0: 미행성체 충돌 / 1: 마그마 바다 / 2: 층 분화 / 3: 지각+바다 / 4: 생명체 탄생
- "단면도 보기" 토글: ON/OFF (기본값 ON, 단계 2~3에서 특히 유용)
  - ON: ClippingPlane으로 지구 반구 잘라 내부 표시
  - OFF: 외부만 표시
- "분화 애니메이션 재생" 버튼 (단계 2에서만 활성):
  - 무거운/가벼운 물질의 이동을 슬로모션으로 반복 재생
- "자동 재생" 버튼: 단계 0→4 자동 진행 (각 단계 5초)
- "지구 회전" 토글: ON/OFF (기본값 ON, 지구 자전 애니메이션)
- "초기화" 버튼: 단계 0, 기본 카메라

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- ClippingPlane: renderer.localClippingEnabled = true, 지구 재질에 clippingPlanes 설정
- 마그마 효과: ShaderMaterial 또는 노이즈 기반 텍스처 애니메이션
- 밀도 분화 파티클: BufferGeometry + Points, position 속성 애니메이션
- 단계 전환 시 이전 객체 dispose()
- 성능: requestAnimationFrame 루프
- 파일명: earth_formation_differentiation_sim.html
