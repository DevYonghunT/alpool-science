너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 지질 시대의 환경과 생물 변화
- 토픽: 화석의 형성 과정
- 대상: 고등학교 1학년
- 핵심 개념: 지질 시대에 살았던 생물의 유해나 흔적이 지층 속에 남아 있는 것을 화석이라고 한다. 화석 연구를 통해 지층의 생성 시기와 환경, 생물의 진화 과정 등 지구의 역사를 이해할 수 있다.

## 씬 구성
- 화면 중앙에 지층 단면을 3D로 렌더링한다.
- 4단계 타임라인으로 화석 형성 과정을 애니메이션으로 보여준다.
- 배경: 0xddccaa (따뜻한 베이지, 지층 느낌)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 화석 형성 4단계 (교과서 그림 I-1, p14)

### 단계 1: 생물의 죽음
- 3D: 수면(PlaneGeometry, 반투명 파란색 0x4488aa opacity 0.3) 아래 해저 바닥(PlaneGeometry 0xaa9977)
- 물고기 실루엣(BoxGeometry 조합: 몸통+꼬리+지느러미): 해저 바닥 위에 눕혀진 상태
- 물 속에 작은 파티클(Points 100개) 부유
- 라벨: "① 과거에 살았던 생물의 유해가 땅속에 묻힌다"

### 단계 2: 퇴적물 쌓임
- 3D: 물고기 위에 퇴적물 층이 서서히 쌓이는 애니메이션
- 퇴적물: BoxGeometry 넓고 얇은 층, 3~4겹 (각각 색상 약간 다름: 0xbb9966, 0xaa8855, 0x998844, 0x887733)
- 층이 위에서 아래로 내려오며 쌓임 (각 2초 간격)
- 라벨: "② 그 위로 퇴적물이 쌓여 오랜 세월이 지나면 화석이 형성된다"

### 단계 3: 화석 형성
- 3D: 여러 층의 퇴적층(지층)이 단단하게 굳어진 모습
- 지층 사이에 물고기 화석이 보이게 (반투명 지층 or 단면 표시)
- 물고기가 뼈만 남은 형태로 변화 (색상: 0xddddcc 연한 회색)
- 지층 색상 그라데이션: 아래=어두운 갈색, 위=밝은 갈색
- 라벨: "③ 화석은 지층 속에 보존된다"

### 단계 4: 화석 노출
- 3D: 지층이 풍화·침식으로 깎여 화석이 드러나는 장면
- 상단 지층이 서서히 사라지는 애니메이션 (scale.y → 0 또는 opacity → 0)
- 화석이 지표면에 노출
- 발굴 도구(망치, 붓 3D 모형) 배치
- 라벨: "④ 이후 지층이 풍화·침식을 받아 지층이 깎이면서 화석이 드러난다"

## 화석 종류 전환
- GUI에서 화석 종류를 선택할 수 있다:
  - 물고기 화석 (기본)
  - 삼엽충 화석: BoxGeometry 타원형 세그먼트
  - 공룡 발자국 화석: 바닥에 움푹 파인 형태(CylinderGeometry 오목)
  - 고사리 화석: 잎 형태(PlaneGeometry + 잎맥 텍스처)
  - 조개 화석: SphereGeometry 반구형
- 화석 종류에 따라 지층 환경 정보 변경

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "형성 단계" 슬라이더: 0~3 (4단계, 정수)
- "화석 종류" 드롭다운: 물고기 / 삼엽충 / 공룡 발자국 / 고사리 / 조개
- "자동 재생" 버튼: 단계 0→3 자동 진행 (각 단계 4초)
- "지층 투명도" 슬라이더: 0.3~1.0 (화석을 잘 보기 위해)
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 단계 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_02_fossil_formation.html
