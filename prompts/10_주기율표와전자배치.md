너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 원소의 주기성과 결합
- 토픽: 주기율표와 전자 배치
- 대상: 고등학교 1학년
- 핵심 개념: 주기율표에서 같은 주기에 속하는 원소는 전자가 채워진 전자 껍질의 수가 같고, 같은 족에 속하는 원소는 원자가 전자의 수가 같다. 원자가 전자는 원소의 화학적 성질을 결정하므로 같은 족 원소는 유사한 화학적 성질을 가진다. 원소들의 성질이 주기성을 나타내는 것은 원자가 전자 수가 주기적으로 변하기 때문이다.

## 씬 구성
- 화면을 좌우 분할:
  - 좌측(폭 55%): 인터랙티브 주기율표 (HTML overlay, CSS Grid)
  - 우측(폭 45%): 3D 보어 원자 모형 (Three.js)
- 우측 3D 영역:
  - 배경: 0x0a0a1e (짙은 남색)
  - 원자 모형을 화면 중앙에 배치
  - 카메라: PerspectiveCamera(FOV 50) + OrbitControls (원자 모형을 마우스로 회전)
  - 조명: AmbientLight(0.3) + PointLight(원자핵 위치, 0xffffff, 세기 1.5)

## 주기율표 (좌측 HTML overlay)
- CSS Grid로 1~3주기(원자 번호 1~18번) 주기율표 렌더링
- 각 칸 구성:
  - 상단 좌측: 원자 번호 (작은 폰트, 10px)
  - 중앙: 원소 기호 (큰 폰트, 22px, bold)
  - 하단: 원소 이름 한글 (작은 폰트, 10px)
- 칸 색상:
  - 1족(알칼리 금속: Li, Na, K): 0xff6644 연한 빨강
  - 2족(알칼리 토금속: Be, Mg): 0xffaa44 주황
  - 13~16족: 0x44aaff 연한 파랑 (비금속 일부), 0xcccccc 회색 (금속)
  - 17족(할로젠: F, Cl): 0x44ddaa 청록
  - 18족(비활성 기체: He, Ne, Ar): 0xaa88ff 보라
- 칸 인터랙션:
  - 호버: 칸 테두리 밝게 강조 + 살짝 확대(transform: scale(1.1))
  - 클릭: 해당 원소의 3D 보어 원자 모형이 우측에 렌더링
  - 현재 선택된 칸: 밝은 노란 테두리(3px solid #ffdd44)

### 주기율표 강조 표시
- "같은 주기 강조" 버튼 클릭 시: 선택된 원소와 같은 주기(가로줄) 원소들의 배경색을 밝게 강조 + "전자 껍질 수가 같음" 라벨
- "같은 족 강조" 버튼 클릭 시: 선택된 원소와 같은 족(세로줄) 원소들의 배경색을 밝게 강조 + "원자가 전자 수가 같음 → 유사한 화학적 성질" 라벨

## 3D 보어 원자 모형 (우측, 교과서 그림 II-16~17, p64~65)
선택된 원소에 따라 보어 원자 모형을 3D로 렌더링한다.

### 원자핵
- SphereGeometry(r=0.4, 32, 32)
- 색상: 0xff4444 (빨강)
- 원자핵 위에 전하량 라벨: "+[원자 번호]" (예: "+11" for Na)
- 내부에 양성자(빨강 작은 구 r=0.06) + 중성자(파랑 작은 구 r=0.06) 밀집 배치
  - 양성자 수 = 원자 번호
  - 중성자 수 ≈ 원자 번호 (간략화)

### 전자 껍질
- TorusGeometry(반지름, 튜브 두께=0.01, radialSegments=64)로 원형 궤도 표시
- 궤도 색상: 0x4488cc (연한 파랑), opacity 0.4
- 전자 껍질 3개:
  - 1번 껍질(K): 반지름 1.0 (최대 2개 전자)
  - 2번 껍질(L): 반지름 1.8 (최대 8개 전자)
  - 3번 껍질(M): 반지름 2.6 (최대 8개 전자, 원자 번호 3~18 기준)

### 전자
- SphereGeometry(r=0.08, 16, 16)
- 색상: 0x44aaff (밝은 파랑), emissive 0x2266aa
- 각 전자가 해당 껍질 궤도 위를 원운동 (속도는 안쪽 껍질이 빠르게)
  - 1번 껍질 전자: 1.5 rad/s
  - 2번 껍질 전자: 1.0 rad/s
  - 3번 껍질 전자: 0.7 rad/s
- 전자 배치 규칙 (교과서 p64):
  - 에너지 준위가 낮은 전자 껍질부터 채움
  - 1번 껍질: 최대 2개
  - 2번 껍질: 최대 8개
  - 3번 껍질: 최대 8개 (원자 번호 3~18 범위)

### 전자 채우기 애니메이션
- 원소를 클릭하면 전자가 하나씩 껍질에 채워지는 애니메이션 (0.3초/전자)
- 1번 껍질 → 2번 껍질 → 3번 껍질 순서
- 새 전자가 추가될 때: 바깥에서 날아와 궤도에 합류하는 효과
- 모든 전자가 채워진 후: 원자가 전자(최외각 전자)를 노란색(0xffdd44)으로 강조 변경 + 깜빡임 효과

### 원자가 전자 강조
- 가장 바깥 전자 껍질의 전자를 노란색으로 표시
- 원자가 전자 수를 큰 라벨로 표시: "원자가 전자: [N]개"
- 같은 족 원소는 원자가 전자 수가 같다는 것을 좌측 주기율표에서도 동시 강조

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "채우기 애니메이션" 토글: ON/OFF (기본값 ON)
  - ON: 원소 클릭 시 전자가 하나씩 채워지는 애니메이션
  - OFF: 즉시 전체 전자 배치 표시
- "원자가 전자 강조" 토글: ON/OFF (기본값 ON)
  - ON: 최외각 전자 노란색 강조
  - OFF: 모든 전자 동일 색상
- "같은 주기 강조" 버튼: 주기율표에서 같은 주기 강조
- "같은 족 강조" 버튼: 주기율표에서 같은 족 강조
- "전자 공전 속도" 슬라이더: 0x ~ 5x (기본값 1x), 0.5 단계
- "원자핵 상세" 토글: ON/OFF (기본값 OFF)
  - ON: 원자핵 내부 양성자+중성자 개별 표시
  - OFF: 원자핵을 단일 빨간 구로 표시
- "초기화" 버튼: 수소(H) 선택, 기본 카메라

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "원소들의 성질이 주기성을 나타내는 것은 규칙에 따라 원자의 전자 배치가 달라지면서 원소의 화학적 성질을 결정하는 원자가 전자 수가 주기적으로 변하기 때문입니다."
- 탐구 질문:
  1. "1족 원소(Li, Na, K)의 원자가 전자 수는 모두 몇 개인가요? 이 원소들의 공통적인 성질은 무엇인가요?"
  2. "18족 원소(He, Ne, Ar)는 왜 다른 원소와 잘 반응하지 않나요? (힌트: 최외각 전자 껍질)"
  3. "새롭게 발견한 원소가 알칼리 금속과 비슷한 성질을 가진다면 원자가 전자 수는 얼마일까요?"
- 원소 분류 (교과서 p61):
  - 금속 원소: 광택, 열/전기 전도, 외력에 변형 (주기율표 왼쪽+가운데)
  - 비금속 원소: 광택 없음, 열/전기 비전도 (주기율표 오른쪽)
  - 비활성 기체(18족): 다른 원소와 잘 반응하지 않음
- 교과서 연계: "미래엔 통합과학1 II편 p60~65 — 주기율표, 전자 배치, 원자가 전자, 같은 족 원소의 유사성"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 주기율표: CSS Grid (HTML overlay), position: absolute, z-index: 10
- 전자 공전: requestAnimationFrame에서 각도 업데이트
- 원소 전환 시 이전 원자 모형 dispose() + 새 모형 생성
- 전자 채우기 애니메이션: setTimeout 체이닝 또는 자체 timeline 구현
- 성능: 전자 최대 18개, SphereGeometry 재사용(geometry 공유)
- 파일명: periodic_table_electron_config_sim.html
