너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 2. 에너지와 열 > 에너지 효율
- 토픽: 에너지 효율과 영구 기관
- 대상: 고등학교 (선택과목)
- 핵심 개념: 에너지 효율은 공급한 에너지에 대해 유용하게 사용된 에너지의 비율이다. 에너지 효율 = (유용한 에너지 / 공급한 에너지) × 100%. 에너지 전환 과정에서 에너지의 총량은 보존되지만, 열에너지 등으로 일부가 소모되어 효율은 항상 100% 미만이다. 외부에서 에너지를 공급받지 않고 영원히 일을 할 수 있는 영구 기관은 에너지 보존 법칙에 의해 불가능하다.

## 씬 구성
- 화면을 좌/우로 분할:
  - 좌측 50%: 3D 에너지 전환 장치 (전구/모터/열기관 등)
  - 우측 50%: 에너지 흐름 Sankey 다이어그램 (HTML overlay)
- 여러 장치의 효율 비교
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 에너지 효율 비교 장치 (5가지)

### 1. 백열전구 (효율 ~5%)
- 3D: SphereGeometry(r=0.3, 투명 유리) + 필라멘트(CylinderGeometry 코일), 0xffdd44 emissive
- 에너지 입력: 전기 에너지 100J
- 유용 출력: 빛에너지 5J
- 폐열: 열에너지 95J (빨간 파티클, 주변으로 방출)

### 2. LED 전구 (효율 ~40%)
- 3D: BoxGeometry(0.2, 0.1, 0.2) 칩 + 투명 커버, 0x44ddff 발광
- 에너지 입력: 전기 에너지 100J
- 유용 출력: 빛에너지 40J
- 폐열: 열에너지 60J

### 3. 전기 모터 (효율 ~85%)
- 3D: CylinderGeometry 모터 본체 + 회전축 + 프로펠러
- 에너지 입력: 전기 에너지 100J
- 유용 출력: 운동 에너지 85J
- 폐열: 열에너지 15J

### 4. 내연 기관 (효율 ~25%)
- 3D: phys_12에서 사용한 내연 기관 간략 모형
- 에너지 입력: 화학 에너지(연료) 100J
- 유용 출력: 운동 에너지 25J
- 폐열: 열에너지 75J

### 5. 태양 전지 (효율 ~20%)
- 3D: PlaneGeometry 태양 전지판, 0x1a1a44
- 에너지 입력: 빛에너지(태양) 100J
- 유용 출력: 전기 에너지 20J
- 폐열: 열에너지 80J

## Sankey 다이어그램 (우측)
- 각 장치에 대해:
  - 왼쪽 입력: 넓은 막대 (공급 에너지, 100%)
  - 위쪽 출력: 유용한 에너지 막대 (초록, [효율]%)
  - 아래쪽 출력: 폐열 막대 (빨강, [100-효율]%)
  - 막대 폭 = 에너지 비율에 비례
- 5가지 장치를 나란히 비교하는 모드도 지원

## 영구 기관 시뮬레이션 (교과서 내용)

### 영구 기관 시도 1: 자체 순환 물레방아
- 3D: 물레방아(원형 바퀴 + 물통들) 회전
- 물이 한쪽으로 쏟아져 회전 시작 → 한 바퀴 돌아 원래 위치
- 마찰로 인해 에너지 감소 → 점점 느려짐 → 정지
- "❌ 영구 기관 실패 — 마찰과 공기 저항으로 에너지 손실"

### 영구 기관 시도 2: 자석 기관
- 3D: 원형 레일 위 자석(SphereGeometry, 빨강/파랑) + 구슬
- 구슬이 자석의 인력/척력으로 이동 시도
- 자기장은 보존력 → 한 바퀴 돌면 에너지 변화 0 → 추가 가속 없음
- "❌ 영구 기관 실패 — 에너지 보존 법칙"

### 교훈 메시지
- "에너지 보존 법칙에 의해 외부에서 에너지를 공급받지 않고 영원히 일을 하는 장치는 불가능하다"
- "에너지는 전환 과정에서 항상 일부가 열에너지로 손실되므로 효율은 100% 미만이다"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "장치" 드롭다운: 백열전구 / LED / 전기모터 / 내연기관 / 태양전지 / 전체비교
- "영구 기관" 드롭다운: 물레방아 / 자석 기관 / 표시 안 함
- "에너지 입력" 슬라이더: 50~500J (기본값 100)
- "효율 조절" 슬라이더: 1~100% (선택 장치의 효율 임의 조정)
- "Sankey 다이어그램" 토글: ON/OFF (기본값 ON)
- "영구 기관 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📊 에너지 효율과 영구 기관
━━━━━━━━━━━━━━━━━━
현재 장치: [장치명]

에너지 효율:
  η = (유용한 에너지 / 공급 에너지) × 100%
  η = [값]J / [값]J × 100% = [값]%

장치별 효율:
  백열전구: ~5%
  LED:     ~40%
  모터:    ~85%
  내연기관: ~25%
  태양전지: ~20%

영구 기관:
  에너지 보존 법칙에 의해 불가능
  η < 100% (항상)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "에너지 효율은 공급한 에너지에 대해 유용하게 사용된 에너지의 비율입니다. 에너지 전환 과정에서 일부가 열에너지로 소모되므로 효율은 항상 100% 미만이며, 영구 기관은 에너지 보존 법칙에 의해 불가능합니다."
- 탐구 질문:
  1. "백열전구보다 LED가 에너지 효율이 높은 까닭은 무엇인가요?"
  2. "자동차 내연 기관의 효율이 약 25%라면, 나머지 75%의 에너지는 어디로 갔나요?"
  3. "영구 기관이 불가능한 까닭을 에너지 보존 법칙으로 설명해 보세요."
- 실생활 예시: 에너지 효율 등급(가전제품), 하이브리드 자동차(회생 제동), 열병합 발전
- 교과서 연계: "물리학 I편 p68~71 — 에너지 효율, 영구 기관, 에너지 보존"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 장치 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
- 파일명: phys_13_energy_efficiency.html
