너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 평형과 구조물의 안정성
- 토픽: 돌림힘과 돌림힘의 평형
- 대상: 고등학교 (선택과목)
- 핵심 개념: 회전축에서 일정한 거리만큼 떨어진 지점에 힘을 가해 물체의 회전 운동을 변화시키는 물리량을 돌림힘이라고 한다. 돌림힘의 크기 τ = rF (τ: 돌림힘, r: 팔 길이, F: 힘의 크기)이다. 한 물체에 여러 돌림힘이 작용할 때 물체가 회전하지 않는 상태를 돌림힘의 평형이라고 하며, 이때 시계 방향 돌림힘의 합 = 반시계 방향 돌림힘의 합이다.

## 씬 구성
- 화면 중앙: 3D 시소(막대 저울) + 회전축 + 무게추
- 사용자가 무게추를 배치하여 돌림힘의 평형을 찾는 인터랙티브 시뮬레이션
- 배경: 0xeef0f5 (밝은 실험실)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 시소 (저울대)

### 받침대(지렛목)
- CylinderGeometry(0.15, 0.3, 0.5, 3) — 삼각형 단면, 0x888888
- 바닥 중앙에 배치

### 저울대 (막대)
- BoxGeometry(8, 0.15, 0.4), 0xcc9944 (나무색)
- 받침대 위에 올려져 회전 가능
- 회전축: 중심점 (받침대 꼭대기)
- 눈금 표시: -4, -3, -2, -1, 0, +1, +2, +3, +4 (회전축으로부터의 거리)

### 무게추
- SphereGeometry(r=0.2), 0xff4444 (빨강) / 0x4444ff (파랑) / 0x44aa44 (초록)
- 무게추 위에 질량 라벨: "1kg", "2kg", "3kg" 등
- 최대 6개의 무게추를 저울대 위 임의 위치에 배치 가능
- 무게추를 드래그하여 위치 변경 (Raycaster 활용)

### 물리 시뮬레이션
- 돌림힘 계산: 각 무게추의 τ = r × mg
- 좌측 돌림힘 합 vs 우측 돌림힘 합 비교
- 시계 방향 > 반시계 방향: 저울대 시계 방향으로 기울어짐 (rotation.z 변화)
- 알짜 돌림힘 = 0: 저울대 수평 유지 → "⚖️ 돌림힘의 평형!" 표시
- 기울어짐 각도: 알짜 돌림힘에 비례 (최대 ±30°)
- 부드러운 회전 애니메이션 (lerp 보간)

## 돌림힘 시각화
- 각 무게추에서 회전축까지 팔 길이 r: 빨간 점선으로 표시
- 각 무게추의 중력(mg): 아래 방향 화살표
- 각 무게추의 돌림힘 τ = rF: 회전 방향 곡선 화살표 (시계/반시계)
- 돌림힘 크기를 화살표 크기로 표현

## 예제 시나리오 (프리셋)
### 시나리오 1: 단순 평형
- 좌측 2m 위치에 3kg, 우측 3m 위치에 2kg
- τ좌 = 2×3g = 6g, τ우 = 3×2g = 6g → 평형

### 시나리오 2: 불평형
- 좌측 4m에 1kg, 우측 1m에 2kg
- τ좌 = 4g, τ우 = 2g → 좌측으로 기울어짐

### 시나리오 3: 여닫이문
- 문 회전축 + 미는 위치에 따른 돌림힘 변화
- 팔 길이가 길수록 작은 힘으로 문을 열 수 있음 시연

### 시나리오 4: 스패너(렌치)
- 손잡이 끝(팔 길이 큼)에서 힘을 주면 적은 힘으로 볼트를 돌림
- 팔 길이 짧으면 큰 힘 필요

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "프리셋" 드롭다운: 자유 배치 / 단순 평형 / 불평형 / 여닫이문 / 스패너
- "무게추 1 질량" 슬라이더: 0~5kg (0.5 단계)
- "무게추 1 위치" 슬라이더: -4~+4m (0.5 단계)
- "무게추 2 질량" 슬라이더: 0~5kg
- "무게추 2 위치" 슬라이더: -4~+4m
- "무게추 3 질량" 슬라이더: 0~5kg (기본값 0 = 사용 안 함)
- "무게추 3 위치" 슬라이더: -4~+4m
- "돌림힘 화살표 표시" 토글: ON/OFF (기본값 ON)
- "팔 길이 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔄 돌림힘과 돌림힘의 평형
━━━━━━━━━━━━━━━━━
돌림힘 τ = r × F [N·m]
  r: 팔 길이 (회전축~힘 작용점)
  F: 힘의 크기

좌측 돌림힘:
  추1: τ₁ = [r]m × [m]kg × 9.8 = [값]N·m
  추2: τ₂ = [r]m × [m]kg × 9.8 = [값]N·m
  합계: [값]N·m (반시계 방향)

우측 돌림힘:
  추3: τ₃ = [r]m × [m]kg × 9.8 = [값]N·m
  합계: [값]N·m (시계 방향)

알짜 돌림힘: [값]N·m
상태: [평형 / 좌측 기울어짐 / 우측 기울어짐]
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "돌림힘의 크기는 τ = rF입니다. 팔 길이(r)가 길수록, 힘(F)이 클수록 돌림힘이 커집니다. 시계 방향 돌림힘의 합과 반시계 방향 돌림힘의 합이 같으면 물체는 회전하지 않는 돌림힘의 평형 상태가 됩니다."
- 탐구 질문:
  1. "여닫이문을 열 때 회전축에서 먼 곳을 미는 것이 유리한 까닭을 돌림힘으로 설명해 보세요."
  2. "시소에서 몸무게가 다른 두 사람이 평형을 이루려면 어떻게 앉아야 하나요?"
  3. "크레인이 무거운 물체를 들어올릴 때 크레인 반대편에 무거운 추를 다는 까닭은 무엇인가요?"
- 실생활 예시: 시소, 문 손잡이 위치, 스패너(렌치), 모빌, 크레인
- 교과서 연계: "물리학 I편 p14~17 — 돌림힘, 돌림힘의 평형 (그림 I-2~5)"
