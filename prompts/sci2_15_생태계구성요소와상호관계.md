너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 1. 생태계와 환경 > 생물과 환경
- 토픽: 생태계 구성요소와 상호 관계
- 대상: 고등학교 1학년
- 핵심 개념: 군집 안에서 생물은 다른 생물과 어울려 살아가는 한편 주변 환경과도 서로 영향을 주고받으며 살아가는 체계를 형성하는데, 이를 생태계라고 한다. 생태계는 생물요소(생산자, 소비자, 분해자)와 비생물요소(빛, 공기, 온도, 물, 토양)로 이루어져 있다.

## 씬 구성
- 화면 중앙에 연못 생태계를 3D로 렌더링한다. (교과서 해 보기: 생태계구성요소 구분하기, p72~73)
- 각 구성요소를 클릭하면 정보 패널이 표시된다.
- 배경: 0x88ccdd (밝은 하늘)
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls (위에서 비스듬히 내려다보기)
- 조명: AmbientLight(0.4) + DirectionalLight(0.7) + PointLight(태양 위치, 0.3)

## 연못 생태계 3D (교과서 그림 스타일)

### 비생물요소
- 빛(태양): SphereGeometry(r=0.5), emissive 0xffdd00, 하늘 위에 배치, 광선 파티클
- 물(연못): CircleGeometry(r=5), 0x4488aa opacity 0.5, 연못 형태
- 토양: PlaneGeometry 연못 주변, 0x886644
- 공기: 투명 (라벨만 표시)
- 온도: 온도계 아이콘 (수식 패널에서 표시)
- 각 비생물요소에 마우스 호버 시 노란 강조 + 이름 표시

### 생물요소 — 생산자
- 수련: PlaneGeometry 원형 잎 2~3개, 0x44aa33, 물 위에 배치
- 부들: CylinderGeometry 줄기 + 갈색 원통형 꽃, 물가에 3개
- 검정말(수초): 물속에 CylinderGeometry 여러 가닥, 0x338833
- 부레옥잠: SphereGeometry 부유 형태, 0x55bb55
- 각 식물에 "생산자" 라벨 (초록)

### 생물요소 — 소비자
- 오리: BoxGeometry 조합 실루엣(몸통+머리+부리), 0xddaa44, 물 위에서 이동
- 개구리: SphereGeometry 몸 + BoxGeometry 다리, 0x55aa33, 연못가에 배치
- 붕어: BoxGeometry 물고기 형태, 0xff8844, 물속에서 이동
- 짚신벌레: SphereGeometry 작은 타원, 0xddddaa, 물속에서 빠르게 이동
- 다슬기: SphereGeometry 나선형, 0x887766, 바닥에 배치
- 각 동물에 "1차 소비자"/"2차 소비자" 라벨

### 생물요소 — 분해자
- 버섯: CylinderGeometry 줄기 + SphereGeometry 갓, 0xaa7744, 토양 위에 2~3개
- "분해자" 라벨 (갈색)

## 상호 관계 시각화
### 비생물요소 → 생물요소
- 빛 → 식물: 노란 화살표 (광합성)
- 물 → 모든 생물: 파란 화살표
- 온도 → 개구리: "변온 동물" 라벨

### 생물요소 → 비생물요소
- 식물 → 공기: 초록 화살표 (O₂ 방출)
- 분해자 → 토양: 갈색 화살표 (유기물 분해)

### 생물요소 ↔ 생물요소
- 생산자 → 1차 소비자 → 2차 소비자: 먹이 관계 화살표 (빨간)
- 분해자 ← 모든 생물 사체: 점선 화살표

- GUI에서 "상호 관계 표시" 토글로 화살표 ON/OFF

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "강조 모드" 드롭다운: 전체 / 생산자만 / 소비자만 / 분해자만 / 비생물요소만
- "상호 관계 표시" 토글: ON/OFF (기본값 OFF)
- "먹이 관계 표시" 토글: ON/OFF
- "이름 표시" 토글: ON/OFF (기본값 ON)
- "계절" 드롭다운: 봄 / 여름 / 가을 / 겨울 (계절에 따라 생태계 색상/생물 변화)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌿 생태계 구성요소와 상호 관계
━━━━━━━━━━━━━━━━━━━
생태계: 연못 생태계

비생물요소: 빛, 공기, 온도, 물, 토양

생물요소:
  생산자: 수련, 부들, 검정말, 부레옥잠
    → 태양의 빛에너지로 양분을 만듦
  소비자: 오리, 개구리, 붕어, 짚신벌레
    → 다른 생물을 먹어 양분을 얻음
  분해자: 버섯, 곰팡이
    → 사체나 배설물을 분해

양분과 에너지의 흐름:
  생산자 → 1차 소비자 → 2차 소비자
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "생태계는 생물요소와 비생물요소로 이루어져 있습니다. 생물요소는 양분과 에너지가 이동하는 단계에 따라 생산자, 소비자, 분해자로 구분됩니다. 생태계를 구성하는 요소들은 서로 영향을 주고받는 밀접한 관계를 맺고 있습니다."
- 탐구 질문:
  1. "연못 생태계에서 비생물요소를 모두 찾아 써 보세요."
  2. "연못 생태계의 생물요소를 생산자, 소비자, 분해자로 분류해 보고, 그렇게 분류한 까닭을 설명해 보세요."
  3. "지렁이가 사는 땅에서 농작물이 잘 자라는 까닭은 무엇인가요?"
- 실생활 예시: 도시 공원 생태계, 갯벌 생태계, 학교 연못
- 교과서 연계: "미래엔 통합과학2 II편 p72~75 — 생태계구성요소, 생물과 환경의 상호 관계"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 강조 효과 리셋 (dispose 불필요, 속성만 변경)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_15_ecosystem_components.html
