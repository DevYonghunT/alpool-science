너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 2. 자연의 구성 물질 > 물질의 전기적 성질
- 토픽: 도체, 반도체, 부도체의 전기적 성질
- 대상: 고등학교 1학년
- 핵심 개념: 물질은 전기적 성질에 따라 도체, 반도체, 부도체로 구분한다. 도체(금속)는 자유 전자가 많아 전류가 잘 흐른다. 부도체(고무, 유리)는 자유 전자가 거의 없어 전류가 흐르지 않는다. 반도체(규소, 게르마늄)는 조건(온도, 불순물)에 따라 전기 전도성이 변한다. 에너지 띠 모델에서 도체는 전도띠와 원자가띠가 겹치고, 부도체는 띠틈이 크며, 반도체는 띠틈이 작다.

## 씬 구성
- 화면을 좌우 분할:
  - 좌측(55%): 전기 회로 실험 3D 장면
  - 우측(45%): 에너지 띠 모델 시각화 (HTML overlay + Canvas 2D)
- 좌측 3D 장면:
  - 배경: 0xf5f5f5 (밝은 실험실)
  - 바닥: PlaneGeometry(12, 8), 0xe0e0e0
  - 조명: AmbientLight(0.5) + DirectionalLight(0.6) + 그림자 활성화
  - 카메라: PerspectiveCamera(FOV 45) + OrbitControls

## 전기 회로 실험 장치 (교과서 탐구 p88)

### 회로 구성
- 전지: BoxGeometry(1.0, 0.5, 0.3), 0xdd4444(+극) + 0x4444dd(−극), "1.5V" 라벨
- 스위치: BoxGeometry(0.3, 0.15, 0.15) + 레버(BoxGeometry 가느다란 막대), ON/OFF 애니메이션
- 디지털 전류계: BoxGeometry(1.2, 0.8, 0.3), 0x222222 (검정 케이스)
  - 화면: PlaneGeometry(0.8, 0.4), 녹색 LED 숫자 표시 (HTML overlay)
  - 전류값 실시간 표시: "0.00 A" ~ "2.50 A"
- 전선: TubeGeometry(CatmullRomCurve3), 0x444444, 전지→스위치→물체→전류계→전지 순환
- 물체 삽입 위치: 전선 중간에 집게 2개(CylinderGeometry + SphereGeometry), 금속색 0xccaa44

### 5가지 물체 (교과서 탐구 p88)
드롭다운으로 선택하여 집게 사이에 삽입:

| 물체 | 3D 형상 | 색상 | 유형 | 전류 |
|------|---------|------|------|------|
| 클립 | BoxGeometry(곡선형 와이어) | 0xcccccc 은색 | 도체 | 2.1 A |
| 연필심 | CylinderGeometry(r=0.05, h=2) | 0x333333 흑연색 | 도체(약) | 0.3 A |
| 지우개 | BoxGeometry(0.6, 0.3, 0.2) | 0xff8888 분홍 | 부도체 | 0.00 A |
| 나무 막대 | CylinderGeometry(r=0.15, h=2) | 0xaa8855 나무색 | 부도체 | 0.00 A |
| 발광 다이오드(LED) | SphereGeometry(r=0.15) + 다리 2개 | 0xff0000 빨강 | 반도체 | 0.02 A (방향 의존) |

### 전류 흐름 시각화
- 스위치 ON + 도체 삽입 시:
  - 전선 위에 작은 노란 구(전자 파티클, r=0.03) 여러 개가 (−)극→(+)극 방향으로 이동
  - 전류가 클수록 파티클 속도 빠르고 수 많음
  - 전구 없이 전류계 값으로 표시
- 스위치 ON + 부도체 삽입 시:
  - 전자 파티클이 물체 앞에서 정지 (막힌 효과)
  - 전류계: "0.00 A"
- LED(반도체) 삽입 시:
  - 순방향: 약한 전류 흐름 + LED 발광 (emissive ON)
  - 역방향: 전류 안 흐름 (정류 특성)
  - "극성 반전" 버튼으로 순/역방향 전환

## 에너지 띠 모델 (우측 HTML overlay + Canvas 2D)

### 3가지 물질 에너지 띠 비교 (가로 나란히)

#### 도체 (구리)
- 원자가띠(VB): 파란 직사각형, 하단부
- 전도띠(CB): 빨간 직사각형, 상단부
- **겹침**: 원자가띠와 전도띠가 겹쳐서 전자가 자유롭게 이동
- 전자 파티클: 겹친 영역에서 자유롭게 이동하는 파란 점
- 라벨: "도체 — 띠틈 없음, 전자 자유 이동"

#### 반도체 (규소)
- 원자가띠(VB): 파란 직사각형
- 전도띠(CB): 빨간 직사각형
- **작은 띠틈**: VB와 CB 사이에 작은 간격 (높이 ~20px)
  - 띠틈 라벨: "띠틈 (작음) ~1.1 eV"
- 온도 슬라이더와 연동:
  - 낮은 온도: 전자가 VB에만 있음 → 부도체처럼 행동
  - 높은 온도: 일부 전자가 VB→CB로 점프(화살표 애니메이션) → 전류 흐름
  - 점프하는 전자 수가 온도에 비례
- 라벨: "반도체 — 작은 띠틈, 온도↑ → 전도성↑"

#### 부도체 (고무)
- 원자가띠(VB): 파란 직사각형
- 전도띠(CB): 빨간 직사각형
- **큰 띠틈**: VB와 CB 사이에 큰 간격 (높이 ~80px)
  - 띠틈 라벨: "띠틈 (큼) ~5 eV 이상"
- 전자: VB에만 고정, CB로 올라갈 수 없음
- 라벨: "부도체 — 큰 띠틈, 전자 이동 불가"

### 온도 효과 (반도체)
- 온도 슬라이더 (0°C ~ 500°C): 반도체의 전도띠로 올라가는 전자 수 변화
- 0°C: 전자 0개 → 전류 0
- 100°C: 전자 약간 → 약한 전류
- 300°C: 전자 많음 → 강한 전류
- 에너지 띠 다이어그램에서 전자 점프 애니메이션 + 좌측 전류계 값 연동

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "물체 선택" 드롭다운: 클립(도체) / 연필심(도체) / 지우개(부도체) / 나무 막대(부도체) / LED(반도체)
- "스위치" 토글: ON/OFF (기본값 OFF)
- "온도" 슬라이더: 0 ~ 500 (기본값 25), 단위 °C — 반도체 전도성 변화
- "LED 극성" 버튼 (LED 선택 시): 순방향 ↔ 역방향 전환
- "에너지 띠 표시" 토글: ON/OFF (기본값 ON)
- "전자 파티클 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 물질의 전기적 성질
━━━━━━━━━━━━━━━━━
선택 물체:     [물체명]
물질 분류:     [도체 / 반도체 / 부도체]
전류:          [값] A
스위치:        [ON / OFF]
온도:          [값] °C

에너지 띠:
  도체:   띠틈 없음 → 전자 자유 이동 → 전류 ✓
  반도체: 띠틈 작음 → 조건에 따라 전류 가변
  부도체: 띠틈 큼   → 전자 이동 불가 → 전류 ✗

반도체 특성:
  온도↑ → 전도성↑
  불순물 첨가 → 전도성↑ (도핑)
```

- 실생활 예시: 전선(구리 도체), 고무 장갑(부도체, 감전 방지), CPU(반도체 규소), LED(반도체 발광), 태양전지(반도체)
- 교과서 연계: "미래엔 통합과학1 II편 p88~93 — 물질의 전기적 성질, 도체/반도체/부도체, 탐구(여러 가지 물체의 전기적 성질 비교)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원
- 전자 파티클: BufferGeometry + Points, 전선 경로 따라 이동
- 에너지 띠 다이어그램: Canvas 2D API로 렌더링 (정밀 제어)
- 온도-전도성 관계: 지수 함수 근사 (반도체 전도도 ∝ exp(-Eg/2kT))
- 물체 전환 시 3D 물체 교체 + 전류계 값 업데이트
- 성능: requestAnimationFrame 루프, 전자 파티클 최대 30개
- 파일명: conductor_semiconductor_insulator_sim.html
