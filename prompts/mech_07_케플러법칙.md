너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 04. 케플러 법칙과 중력
- 토픽: 케플러 법칙 (타원 궤도, 면적 속도, 조화 법칙)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 케플러 제1법칙(타원 궤도): 행성은 태양을 한 초점으로 하는 타원 궤도를 따라 운동한다. 케플러 제2법칙(면적 속도 일정): 행성과 태양을 잇는 선분이 같은 시간 동안 쓸어 지나가는 면적은 일정하다(근일점에서 빠르고, 원일점에서 느림). 케플러 제3법칙(조화 법칙): T²∝a³ (T: 공전 주기, a: 반장축).

## 씬 구성
- 화면 중앙: 3D 태양계 (태양 + 행성 타원 궤도)
- 3가지 탭: 제1법칙(타원 궤도) / 제2법칙(면적 속도) / 제3법칙(조화 법칙)
- 배경: 0x000010 (우주)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (위에서 내려다보기)
- 조명: AmbientLight(0.1) + PointLight(태양 위치, 2.0)

## 3D 태양계

### 태양
- SphereGeometry(r=0.8), emissive 0xffcc00, emissiveIntensity 1.0
- PointLight(0xffdd44, 2.0) 태양 위치에
- 태양 코로나: SphereGeometry(r=1.0), 0xffee44 opacity 0.1

### 행성 (기본: 지구)
- SphereGeometry(r=0.15), 0x2266aa (파란 지구)
- 타원 궤도를 따라 공전: EllipseCurve로 궤도 경로 생성
- 궤도: Line(BufferGeometry), 0x444488 반투명

### 타원 궤도 파라미터
- 반장축 a: 궤도의 긴 반지름 (슬라이더로 조절)
- 이심률 e: 타원의 찌그러짐 (0=원, 0~0.9)
  - e=0: 원형 궤도
  - e=0.5: 뚜렷한 타원
  - e=0.9: 매우 찌그러진 타원 (혜성 궤도)
- 반단축 b = a√(1-e²)
- 근일점: 태양에 가장 가까운 점 (a(1-e))
- 원일점: 태양에서 가장 먼 점 (a(1+e))

## 케플러 제1법칙: 타원 궤도

### 시각화
- 타원 궤도: EllipseCurve + Line
- 태양: 타원의 한 초점에 위치 (중심이 아님!)
- 다른 초점: 작은 점으로 표시
- 근일점/원일점 라벨
- 이심률 슬라이더로 궤도 형태 변화
  - e=0: 태양이 중심 → 원궤도
  - e 증가: 태양이 한쪽으로 치우침 → 타원
- "태양은 타원의 중심이 아니라 한 초점에 위치" 강조

## 케플러 제2법칙: 면적 속도 일정

### 시각화 (핵심!)
- 같은 시간 간격 Δt 동안 행성-태양 선분이 쓸어 지나가는 면적 표시
- 면적: 부채꼴 모양 반투명 삼각형(BufferGeometry)
  - 근일점 근처: 좁고 긴 부채꼴 (빠르게 이동)
  - 원일점 근처: 넓고 짧은 부채꼴 (느리게 이동)
  - **면적은 동일!** (색상으로 강조: 모두 같은 면적)
- 면적 수치 표시: "ΔA₁ = ΔA₂ = ΔA₃ = [값]" 라벨
- 행성 속도 벡터: 크기가 위치에 따라 변화 (근일점에서 길고, 원일점에서 짧음)
- "근일점에서 빠르고 원일점에서 느리다" 텍스트

## 케플러 제3법칙: 조화 법칙

### 시각화
- 여러 행성(3~5개)의 궤도를 동시에 표시
  - 수성: a=0.39AU, T=0.24년, 0xaaaaaa
  - 금성: a=0.72AU, T=0.62년, 0xddaa66
  - 지구: a=1.00AU, T=1.00년, 0x2266aa
  - 화성: a=1.52AU, T=1.88년, 0xcc4422
  - 목성: a=5.20AU, T=11.86년, 0xdd9944
- 모든 행성이 동시에 공전 (안쪽 빠르게, 바깥 느리게)
- T²/a³ 계산 표: 모든 행성에서 같은 값(≈1 for AU, 년) 확인
- T²-a³ 그래프 (Canvas 2D): 직선 관계 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "법칙" 드롭다운: 제1법칙(타원) / 제2법칙(면적) / 제3법칙(조화) / 전체
- "반장축 a" 슬라이더: 2~10 (기본값 5)
- "이심률 e" 슬라이더: 0~0.9 (기본값 0.3, 0.05 단계)
- "행성 수" 슬라이더 (제3법칙): 1~5 (기본값 3)
- "면적 표시 간격" 슬라이더 (제2법칙): 1~5초
- "시뮬레이션 속도" 슬라이더: 0.5x~5x
- "궤도 표시" 토글: ON/OFF (기본값 ON)
- "속도 벡터" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🪐 케플러 법칙
━━━━━━━━━━━━━━
현재: [제1/2/3법칙]
반장축 a = [값]AU, 이심률 e = [값]

제1법칙 (타원 궤도):
  행성은 태양을 한 초점으로 하는
  타원 궤도를 따라 운동
  근일점: a(1-e) = [값]AU
  원일점: a(1+e) = [값]AU

제2법칙 (면적 속도 일정):
  ΔA/Δt = 일정
  근일점: 빠르게 / 원일점: 느리게

제3법칙 (조화 법칙):
  T²/a³ = k (일정)
  T = [값]년, a = [값]AU
  T²/a³ = [값]
```
