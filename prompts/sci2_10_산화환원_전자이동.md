너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 산화와 환원
- 토픽: 산화와 환원 — 전자 이동에 의한 반응
- 대상: 고등학교 1학년
- 핵심 개념: 산소가 관여하지 않을 때도 산화·환원 반응이 일어난다. 반응이 일어날 때 물질이 전자를 잃는 반응을 산화라 하고, 물질이 전자를 얻는 반응을 환원이라고 한다. 아연판을 황산 구리(II) 수용액에 넣으면 아연(Zn)이 전자를 잃어 산화되어 아연 이온(Zn²⁺)으로 변하고, 구리 이온(Cu²⁺)이 전자를 얻어 환원되어 구리(Cu)로 변한다.

## 씬 구성
- 화면 중앙: 비커 + 수용액 + 아연판 3D 모형
- 화면 하단: 입자 수준 확대 뷰 (전자 이동 애니메이션)
- 교과서 그림 I-23 (p42): 아연판과 황산 구리(II) 수용액의 반응
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(1.0)

## 3D 실험 장치
### 비커
- CylinderGeometry(1.2, 1.2, 2, 32, 1, true) — 투명 유리
- MeshPhysicalMaterial(transmission: 0.8, roughness: 0.05, ior: 1.5)
- 비커 내부에 수용액: CylinderGeometry(1.15, 1.15, 1.6), 색상 변화

### 수용액 (황산 구리(II))
- 초기: 0x2266bb (진한 파란색, Cu²⁺ 이온 때문)
- 반응 진행: 서서히 0x88bbcc (연한 파란색)으로 변화 (Cu²⁺ 감소)
- 교과서: "아연 이온이 수용액에 녹아들어 가면서 수용액의 푸른색이 점점 옅어진다"

### 아연판
- BoxGeometry(0.1, 1.8, 1.2), 초기 색상 0xbbbbbb (은색)
- 비커 안에 세로로 넣은 상태
- 반응 진행: 표면에 구리(0xcc6633 붉은 갈색)가 석출되어 쌓임
- 석출 애니메이션: 아연판 표면에 작은 구리 구(SphereGeometry r=0.03)가 하나씩 달라붙음

## 입자 수준 확대 뷰 (하단 또는 분할 화면)
### 반응 전
- 아연 원자(Zn): SphereGeometry(r=0.2), 0xbbbbbb (은색), 격자 배열 (아연판)
- 구리 이온(Cu²⁺): SphereGeometry(r=0.18), 0x3388cc (파란색), "2+" 라벨, 수용액에 부유
- 황산 이온(SO₄²⁻): 작은 표시 (선택적)

### 반응 애니메이션 (5초)
1. 아연 원자(Zn)에서 **전자 2개**(작은 노란 구 SphereGeometry r=0.05, 0xffff00)가 분리
2. 전자 2개가 아연 → 구리 이온 방향으로 이동 (궤적 표시: 노란 점선)
3. 아연이 Zn²⁺ 이온(0xbbccdd, "2+" 라벨)이 되어 수용액으로 이동
4. Cu²⁺가 전자 2개를 받아 Cu(0xcc6633 구리색)가 되어 아연판 표면에 부착
- 화학 반응식: `Zn + Cu²⁺ → Zn²⁺ + Cu`
- 산화(Zn → Zn²⁺, 전자 잃음) + 환원(Cu²⁺ → Cu, 전자 얻음) 동시 표시

### 추가 예시: 마그네슘 연소 (교과서 p43)
- Mg + O₂ → MgO
- 마그네슘(Mg)이 전자를 잃어 Mg²⁺, 산소(O₂)가 전자를 얻어 O²⁻
- 산소가 관여하는 반응도 전자의 이동으로 설명할 수 있음

### 추가 예시: 구리와 질산 은 수용액 (교과서 p43)
- Cu + 2Ag⁺ → Cu²⁺ + 2Ag
- 구리가 전자를 잃어 산화, 은 이온이 전자를 얻어 환원

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "반응" 드롭다운: 아연+황산구리 / 마그네슘 연소 / 구리+질산은
- "반응 재생" 버튼: 전자 이동 애니메이션 재생
- "전자 이동 표시" 토글: ON/OFF (기본값 ON)
- "입자 뷰" 토글: 실험 장치 뷰 / 입자 확대 뷰 / 분할 뷰
- "반응 속도" 슬라이더: 0.5x~3x
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 산화와 환원 — 전자의 이동
━━━━━━━━━━━━━━━━━━━
현재 반응:    [반응명]
화학 반응식:  [반응식]
산화 물질:   [물질] (전자를 잃음)
환원 물질:   [물질] (전자를 얻음)

규칙:
  산화: 물질이 전자를 잃는 반응
  환원: 물질이 전자를 얻는 반응
  → 산화와 환원은 항상 동시에 일어난다

산소의 이동과 전자의 이동:
  산화 = 산소를 얻음 = 전자를 잃음
  환원 = 산소를 잃음 = 전자를 얻음
```

- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js

- 성능: requestAnimationFrame 루프
