너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 03. 편광
- 토픽: 편광과 편광 필터
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 자연광은 모든 방향으로 진동하는 횡파이다. 편광 필터를 통과하면 특정 방향으로만 진동하는 편광이 된다. 말뤼스 법칙: 편광이 편광 필터를 통과할 때 투과 강도 I = I₀cos²θ (θ: 편광 방향과 필터 축 사이각). θ = 0°이면 I = I₀(전부 통과), θ = 90°이면 I = 0(완전 차단). 편광은 LCD, 편광 선글라스, 3D 영화 등에 활용된다.

## 씬 구성
- 화면 중앙: 3D 빛의 전파 + 편광 필터(1~2개) + 전기장 진동 시각화
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2)

## 3D 편광 시각화

### 자연광 (비편광)
- 빛의 전파 방향: x축 (오른쪽 →)
- 전기장 진동: y, z 축 모든 방향 (360°)
  - 여러 방향의 화살표(ArrowHelper)가 빛 전파 방향에 수직으로 진동
  - 다양한 색상의 화살표가 무작위 방향으로
- "자연광: 모든 방향으로 진동하는 횡파" 라벨

### 편광 필터 1 (편광자)
- PlaneGeometry(3, 3), 0x888888 반투명 + 수직 줄무늬(편광축 방향)
- 자연광이 편광 필터를 통과하면:
  - 편광축 방향으로만 진동하는 빛(편광)이 됨
  - 편광: 한 방향 화살표만 남음
  - 투과 강도: 자연광의 1/2 (I₀/2)
- 필터 회전 슬라이더: 0~180° → 편광 방향 변화

### 편광 필터 2 (분석자)
- 편광된 빛이 두 번째 필터를 통과
- 말뤼스 법칙: I = I₀cos²θ
  - θ = 0° (평행): I = I₀ (전부 통과, 밝음)
  - θ = 45°: I = I₀/2 (절반 통과)
  - θ = 90° (수직): I = 0 (완전 차단, 어두움!)
- 분석자 각도 슬라이더: 0~180°
- 통과 후 전기장 진동: 분석자 편광축 방향으로 축소

### 투과광 밝기
- 스크린 또는 LED: SphereGeometry emissive
- 밝기 = I/I₀ = cos²θ에 비례
- θ에 따라 밝기 실시간 변화
- θ = 90°: 완전히 꺼짐 → "편광자와 분석자가 수직 → 빛 완전 차단" 라벨

## 말뤼스 법칙 그래프 (우측 Canvas 2D)
- X축: 분석자 각도 θ (0~180°)
- Y축: 투과 강도 I/I₀
- 곡선: cos²θ 패턴
- 현재 θ에 빨간 마커
- θ = 0°: I/I₀ = 1 (최대)
- θ = 90°: I/I₀ = 0 (최소)
- θ = 45°: I/I₀ = 0.5

## 3개 편광 필터 (흥미로운 현상!)
- 편광자(0°) + 분석자(90°): 빛 완전 차단
- 그 사이에 45° 필터를 추가하면: 빛이 일부 통과!
  - I = I₀/2 × cos²45° × cos²45° = I₀/8
- "수직 편광 필터 사이에 45° 필터를 넣으면 빛이 다시 통과?" → 양자역학적 사고 연결
- 3D에서 필터 3개 배치 + 투과 과정 시각화

## 편광의 활용
- LCD 원리: 편광 필터 2개 + 액정(전압에 따라 편광 방향 회전)
  - 전압 ON: 편광 회전 → 빛 통과 (밝음)
  - 전압 OFF: 편광 회전 안 됨 → 빛 차단 (어두움)
- 편광 선글라스: 수평 편광(반사광) 차단 → 눈부심 감소
- 3D 영화: 좌/우 눈에 다른 편광 → 입체감

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "설정" 드롭다운: 필터 1개 / 필터 2개 / 필터 3개 / LCD 원리
- "편광자 각도" 슬라이더: 0~180° (기본값 0)
- "분석자 각도" 슬라이더: 0~180° (기본값 0)
- "중간 필터 각도" 슬라이더 (3개 모드): 0~90° (기본값 45)
- "전기장 진동 표시" 토글: ON/OFF (기본값 ON)
- "말뤼스 그래프" 토글: ON/OFF (기본값 ON)
- "LCD 전압" 토글 (LCD 모드): ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🕶️ 편광과 편광 필터
━━━━━━━━━━━━━━
설정: [1개/2개/3개/LCD]
편광자: [값]°, 분석자: [값]°

말뤼스 법칙:
  I = I₀cos²θ
  θ = [값]° (편광-분석자 사이각)
  I/I₀ = cos²[값]° = [값]

θ=0° → I=I₀ (전부 통과)
θ=45° → I=I₀/2
θ=90° → I=0 (완전 차단)

편광: 특정 방향으로만 진동하는 빛
자연광 → 편광자 → 편광 (I₀/2)
```
