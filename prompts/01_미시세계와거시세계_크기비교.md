너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: I. 과학의 기초 > 1. 과학의 기본량 > 시간과 공간
- 토픽: 미시 세계와 거시 세계 크기 비교
- 대상: 고등학교 1학년
- 핵심 개념: 자연 세계는 원자 수준의 아주 작은 규모를 나타내는 미시 세계와 그보다 훨씬 큰 규모를 나타내는 거시 세계로 구분할 수 있다. 자연에서 일어나는 다양한 현상이나 물체의 크기는 시간과 공간의 규모가 다르므로 각 현상마다 관찰이나 측정하는 방법이 다르다.

## 씬 구성
- 화면 중앙에 현재 스케일에 해당하는 대표 물체를 3D로 렌더링한다.
- 물체는 MeshStandardMaterial로 표면 질감을 표현한다.
- 줌인/줌아웃 시 물체가 부드럽게 확대/축소되며 다음 물체로 전환된다 (requestAnimationFrame 기반 lerp 보간 애니메이션, 전환 시간 0.8초).
- 각 스케일 단계에서 이전 물체는 페이드아웃, 새 물체는 페이드인.
- 배경색은 스케일에 따라 자연스럽게 그라데이션 전환:
  - 미시 세계(10⁻¹⁵ ~ 10⁻⁶m): 짙은 남색 → 보라색 (0x0a0a2e → 0x2d1b69), 미세한 파티클 효과(원자/분자 느낌)
  - 중간 세계(10⁻⁵ ~ 10³m): 하늘색 → 연초록 (0x87CEEB → 0xE0F0FF), 자연환경 느낌
  - 거시 세계(10⁴ ~ 10²⁶m): 검은 우주 (0x000011 → 0x000000), 별 파티클(Points, 500개)
- 바닥: 중간 세계 스케일에서만 GridHelper 표시.
- 조명: AmbientLight(0.5) + DirectionalLight(0.7) + 그림자 비활성화(성능)
- 카메라: PerspectiveCamera, FOV 60, 자동 줌 (물체 크기에 맞춰 카메라 거리 자동 조절)
- OrbitControls: 마우스 드래그로 물체를 360도 회전하여 관찰 가능. **enableZoom = false** (마우스 휠은 스케일 전환 전용으로 사용하므로 OrbitControls의 줌 기능은 비활성화)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "스케일 (10^n m)" 슬라이더: -15 ~ 26 (기본값 0, 정수 단계)
  - 슬라이더 이동 시 물체 전환 애니메이션 실행
  - 슬라이더 값 변경 시 하단 정보 패널 실시간 업데이트
- "자동 투어" 버튼: 클릭하면 10⁻¹⁵m에서 10²⁶m까지 3초 간격으로 자동 전환
  - 투어 중 "일시정지" 버튼으로 변경
  - 투어 중에도 OrbitControls로 물체 회전 가능
- "투어 속도" 슬라이더: 1초 ~ 5초 (기본값 3초), 0.5초 단계
- "초기화" 버튼: 스케일 10⁰m(농구대)로 리셋, 카메라 위치 초기화

### 마우스/터치 인터랙션
- 마우스 휠 위/아래: 스케일 한 단계씩 줌인/줌아웃 (슬라이더 연동)
- 마우스 드래그: OrbitControls로 물체 360도 회전
- 물체 클릭(Raycaster): 해당 물체 상세 정보 팝업 표시

## 시각화 규칙

### 물체 라벨 (항상 표시)
- 물체 위 중앙에 HTML overlay로 물체 이름(한글) 표시
  - 폰트: 'Pretendard', sans-serif, 24px, bold, 흰색
  - 텍스트 그림자: 2px 2px 4px rgba(0,0,0,0.5)
- 물체 아래에 실제 크기 표시
  - 폰트: 16px, 흰색, 반투명
  - 예: "약 7 μm", "반지름 6,400 km"

### 스케일 바 (화면 좌측)
- 세로 방향 로그 스케일 바 (HTML overlay)
- 현재 위치를 밝은 점으로 강조
- 스케일 바 위에 "미시 세계" / "거시 세계" 구분선(10⁻⁴m 부근) 표시
- 현재 스케일이 미시 세계이면 "🔬 미시 세계" 라벨 강조 (파란색)
- 현재 스케일이 거시 세계이면 "🔭 거시 세계" 라벨 강조 (노란색)

### 전환 애니메이션
- 스케일 변경 시: 현재 물체 → 축소(0.8초) → 배경색 전환(0.5초) → 새 물체 확대(0.8초)
- 자동 투어 시: 동일 애니메이션 + 물체별 1초 정지

### 크기 비교 보조선
- 각 물체 옆에 이전 스케일 물체의 크기를 점으로 표시 (상대적 크기 비교)
- 예: 지구(10⁷m) 옆에 달(10⁶m)을 실제 비율 크기의 작은 구로 함께 표시

## 정보 패널
화면 좌측 하단에 반투명 HTML overlay 패널(배경: rgba(0,0,0,0.7), 둥근 모서리 12px, padding 20px)로 표시:

```
🔍 자연 세계의 규모
━━━━━━━━━━━━━━━
현재 스케일:  10^[n] m
물체:        [물체명]
실제 크기:   [값] [단위]
세계 구분:   [미시 세계 / 거시 세계]

📏 크기 비교:
이 물체는 [이전 물체]보다 약 [배수]배 크다.
```
모든 값은 스케일 변경 시 실시간 업데이트된다.

## 교육적 요소
화면 상단 우측에 토글 가능한 "📖 학습 가이드" 버튼 (클릭 시 패널 슬라이드):
- 핵심 원리: "자연 세계는 원자 수준의 아주 작은 미시 세계와 그보다 훨씬 큰 거시 세계로 구분할 수 있습니다. 물체의 크기에 따라 관찰과 측정 방법이 달라집니다."
- 탐구 질문:
  1. "이 시뮬레이션에서 크기가 가장 작은 물체와 가장 큰 물체는 무엇인가요?"
  2. "적혈구(약 7μm)에서 지구(반지름 6,400km)까지 크기는 약 몇 배 차이가 나나요?"
  3. "미시 세계의 물체를 관찰하려면 어떤 도구가 필요할까요?"
- 실생활 예시: 전자 현미경(미시 세계 관찰), 천체 망원경(거시 세계 관측), 레이저 거리 측정기(정밀 측정)
- 교과서 연계: "미래엔 통합과학1 I편 p14~17 — 시간과 공간, 자연 세계의 규모"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 성능: requestAnimationFrame 루프, 불필요한 재생성 방지, 물체 전환 시 이전 물체 dispose() 호출
- 파일명: micro_macro_scale_sim.html
