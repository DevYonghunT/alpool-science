너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 2. 에너지 > 발전과 에너지 전환
- 토픽: 전자기 유도와 발전기 원리
- 대상: 고등학교 1학년
- 핵심 개념: 자석이나 코일을 움직여 코일을 통과하는 자기장이 변하면 코일에 전류가 흐르는 전자기 유도 현상이 나타난다. 발전기는 전자기 유도 현상을 이용해 자석이나 코일의 운동 에너지를 전기 에너지로 전환하는 장치이다. 발전소에서는 터빈을 회전시켜 발전기를 작동한다.

## 씬 구성
- 화면 중앙: 3D 전자기 유도 실험 장치 / 발전기 모형
- 3가지 보기 모드: 자석 이동 / 코일 이동 / 발전기 구조
- 배경: 0xf0f0f5 (실험실 밝은 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 모드 1: 자석을 움직일 때 (교과서 그림 II-19)

### 3D 실험 장치
- 코일: TorusGeometry(r=1, tube=0.08, 32, 100) × 여러 개 나란히, 0xcc6633 (구리색)
  - 코일 양 끝에 전선 연결 → 검류계(또는 LED)
- 막대 자석: BoxGeometry(0.4, 0.4, 2), N극 0xff3333(빨강) / S극 0x3333ff(파랑)
  - "N" / "S" 라벨
- 검류계: CircleGeometry + 바늘(회전) — 전류 감지 시 바늘 움직임
- LED: SphereGeometry(r=0.1), 전류 흐를 때 0xffff44 발광

### 자기장 시각화
- 자석 주변 자기력선: TubeGeometry 곡선, N극→S극 방향, 파란 반투명
- 코일을 통과하는 자기장: 화살표 파티클 (자석 이동 시 변화)

### 애니메이션
- 자석이 코일 안으로 들어가는 슬라이더 (또는 자동)
- 자석이 코일에 접근: 자기장 변화 → 전류 발생 → LED 켜짐 + 검류계 바늘 이동
- 자석이 멈추면: 자기장 변화 없음 → 전류 0 → LED 꺼짐
- 자석이 멀어지면: 자기장 변화 (반대 방향) → 반대 전류 → 검류계 바늘 반대로
- 교과서: "자석이나 코일을 움직여 코일을 통과하는 자기장이 변하면 코일에 전류가 흐른다" (p102~103)

## 모드 2: 코일을 움직일 때

### 동일 장치, 이번엔 자석 고정 + 코일 이동
- 결과 동일: 코일과 자석의 상대 운동이 중요
- 교과서: "코일을 움직여도 전자기 유도가 일어난다"

## 모드 3: 발전기 구조 (교과서 그림 II-21, p104)

### 발전기 3D 모형
- 고정자(코일): TorusGeometry 여러 겹, 0xcc6633, 외부에 고정
- 회전자(자석): BoxGeometry N/S 극 자석, 내부에서 회전
- 터빈: CylinderGeometry 축 + PlaneGeometry 날개 6~8개
- 터빈과 자석이 같은 축으로 연결

### 발전기 작동 애니메이션
- 터빈 회전 → 자석 회전 → 코일을 통과하는 자기장 변화 → 전류 발생
- 전구(SphereGeometry, emissive)가 터빈 회전 속도에 따라 밝기 변화
- 터빈 회전 속도 슬라이더

### 에너지 전환 경로
- "운동 에너지(터빈) → 운동 에너지(자석 회전) → 전기 에너지(전자기 유도) → 빛/열에너지(전구)"
- 화살표로 에너지 전환 경로 표시

### 발전소 종류별 터빈 구동 방식
- 풍력: 바람 → 터빈 직접 회전
- 수력: 물 → 터빈 직접 회전
- 화력: 화석 연료 연소 → 증기 → 터빈 회전
- 핵발전: 핵분열 → 증기 → 터빈 회전
- 각 방식을 아이콘으로 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 자석 이동 / 코일 이동 / 발전기 구조
- "자석 위치" 슬라이더 (모드 1): -3~3 (코일 안/밖)
- "터빈 회전 속도" 슬라이더 (모드 3): 0~10 RPM
- "자기력선 표시" 토글: ON/OFF (기본값 ON)
- "에너지 전환 경로" 토글: ON/OFF
- "발전소 종류" 드롭다운 (모드 3): 풍력 / 수력 / 화력 / 핵발전
- "초기화" 버튼

```
━━━━━━━━━━━━━━━━━━━
현재 모드:      [자석 이동/코일 이동/발전기]
전류 발생:      [있음/없음]
전류 방향:      [정방향/역방향/—]

전자기 유도:
  코일을 통과하는 자기장이 변하면
  코일에 전류가 흐른다

발전기 원리:
  운동 에너지(터빈)
  → 자석 회전 (자기장 변화)
  → 전기 에너지 (전자기 유도)

발전소: [종류]가 터빈을 회전시킴
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_22_electromagnetic_induction.html
