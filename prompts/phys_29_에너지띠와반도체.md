너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 2. 원자와 에너지 준위 > 에너지띠와 반도체
- 토픽: 에너지띠와 반도체 (n형/p형)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 고체 물질에서 원자들이 모이면 개별 에너지 준위가 넓어져 에너지띠를 형성한다. 가전자띠(원자가 전자가 채워진 띠)와 전도띠(전자가 자유롭게 이동할 수 있는 띠) 사이의 간격을 띠틈(band gap)이라 한다. 도체는 띠틈이 없거나 매우 작고, 부도체는 매우 크며, 반도체는 중간 크기이다. 반도체에 불순물을 도핑하면 n형(전자 과잉) 또는 p형(양공 과잉) 반도체가 된다.

## 씬 구성
- 화면 좌측 60%: 3D 에너지띠 다이어그램 (도체/반도체/부도체 비교)
- 화면 우측 40%: 불순물 반도체 (n형/p형) 시각화
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 에너지띠 다이어그램 (좌측)

### 3D 에너지띠 블록
각 물질의 에너지띠를 BoxGeometry 블록으로 표시:

#### 도체 (구리)
- 가전자띠: BoxGeometry(2, 1.5, 0.5), 0x4488ff (파랑)
- 전도띠: BoxGeometry(2, 1.5, 0.5), 0xff8844 (주황)
- 두 띠가 겹침 (띠틈 = 0) 또는 가전자띠가 반쯤 차 있음
- 전자(노란 구)가 자유롭게 전도띠를 이동
- 라벨: "도체 — 띠틈 없음, 전류 잘 흐름"

#### 반도체 (규소)
- 가전자띠: BoxGeometry(2, 1.5, 0.5), 0x4488ff
- 전도띠: BoxGeometry(2, 1.5, 0.5), 0xff8844
- 띠틈: 약간의 간격 (BoxGeometry 투명, 높이 0.5), "Eg ≈ 1.1eV" 라벨
- 상온에서 일부 전자가 열에너지로 전도띠로 올라감 (소수)
- 온도 슬라이더: 온도 올리면 전도띠의 전자 수 증가 (전도성 증가)
- 라벨: "반도체 — 중간 띠틈, 온도↑ 전도성↑"

#### 부도체 (다이아몬드)
- 가전자띠: BoxGeometry(2, 1.5, 0.5), 0x4488ff
- 전도띠: BoxGeometry(2, 1.5, 0.5), 0xff8844
- 띠틈: 큰 간격 (높이 1.5), "Eg ≈ 5.5eV" 라벨
- 전자가 전도띠에 거의 없음 (전류 안 흐름)
- 라벨: "부도체 — 큰 띠틈, 전류 거의 안 흐름"

### 전자 시각화
- 가전자띠: 전자(노란 구)가 빽빽하게 채움
- 전도띠: 도체는 많이, 반도체는 적게, 부도체는 거의 없음
- 전기장 적용(화살표): 전도띠의 전자가 이동 → 전류 흐름 시각화

## 불순물 반도체 (우측)

### 순수 반도체 (진성 반도체, Si)
- 3D 격자: SphereGeometry(0.15) × 16개 (4×4), Si 원자, 0x888888
- 공유 결합: CylinderGeometry 연결선, 0xaaaaaa
- 각 Si 원자: 가전자 4개 표시 (작은 파란 구)
- 에너지 다이어그램: 가전자띠 꽉 참, 전도띠 비어있음

### n형 반도체 (인(P) 도핑)
- Si 격자에 P 원자 1개 치환: 0xff6644 (주황), "P" 라벨
- P는 가전자 5개 → 4개는 결합에 사용, 1개 남음 (여분 전자)
- 여분 전자: 큰 노란 구(0xffff44), 격자에서 자유롭게 이동
- 에너지 다이어그램: 띠틈 안에 "도너 준위" (전도띠 바로 아래)
- "n형: 전자가 전류의 주 운반자" 라벨
- 전기장 적용 → 여분 전자 이동 → 전류

### p형 반도체 (붕소(B) 도핑)
- Si 격자에 B 원자 1개 치환: 0x44aaff (파랑), "B" 라벨
- B는 가전자 3개 → 결합에 1개 부족 → 양공(hole) 생성
- 양공: 빈 원(RingGeometry r=0.12, 0xff4444), 전자가 빠진 자리
- 주변 전자가 양공으로 이동하면 양공이 반대 방향으로 이동하는 것처럼 보임
- 에너지 다이어그램: 띠틈 안에 "억셉터 준위" (가전자띠 바로 위)
- "p형: 양공이 전류의 주 운반자" 라벨
- 전기장 적용 → 양공이 전기장 방향으로 이동 → 전류

## 온도 효과
- 온도 슬라이더: 0K ~ 500K
- 온도↑ → 반도체 전도띠 전자 수↑ → 전도성↑
- 반대로 도체는 온도↑ → 전도성↓ (격자 진동 증가로 전자 산란)
- 이 차이를 시각적으로 비교

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "물질" 드롭다운: 도체(Cu) / 반도체(Si) / 부도체(다이아몬드) / 전체 비교
- "반도체 종류" 드롭다운: 순수(진성) / n형(P도핑) / p형(B도핑)
- "온도" 슬라이더: 0~500K (기본값 300)
- "전기장 적용" 토글: ON/OFF (전자/양공 이동 시각화)
- "에너지띠 다이어그램" 토글: ON/OFF (기본값 ON)
- "격자 구조" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
💎 에너지띠와 반도체
━━━━━━━━━━━━━━━
물질: [도체/반도체/부도체]
반도체 종류: [순수/n형/p형]
온도: [값]K

에너지띠:
  도체: 띠틈 ≈ 0 → 전류 잘 흐름
  반도체: Eg ≈ 1.1eV → 온도↑ 전도성↑
  부도체: Eg ≈ 5.5eV → 전류 거의 안 흐름

불순물 반도체:
  n형: Si에 P(5족) 도핑 → 여분 전자
  p형: Si에 B(3족) 도핑 → 양공
  도핑 → 전도성 크게 향상
```
