너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 03. 자기장과 로런츠 힘
- 토픽: 로런츠 힘과 대전 입자의 원운동
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 자기장 속에서 운동하는 전하는 로런츠 힘 F = qvB(v⊥B일 때)를 받으며, 이 힘은 속도와 자기장 모두에 수직이다. 균일한 자기장에서 대전 입자는 등속 원운동을 하며, 원운동 반지름 r = mv/(qB)이다. 이를 이용한 질량 분석기는 이온의 질량을 측정한다. 지구 자기장에서 태양풍의 대전 입자가 극지방으로 모여 오로라가 발생한다.

## 씬 구성
- 화면 중앙: 3D 균일 자기장 영역 + 대전 입자의 원운동 궤적
- 3가지 시나리오: 기본 원운동 / 질량 분석기 / 오로라(지구 자기장)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 시나리오 1: 자기장에서 대전 입자의 원운동

### 3D 자기장 영역
- 균일 자기장: 화면 안쪽(z축) 방향 (⊗ 표시) 또는 밖으로(⊙ 표시)
- 자기장 표시: 격자 점에 ⊗(파란 X) 또는 ⊙(파란 점) 아이콘
- 자기장 영역: BoxGeometry(10, 10, 1), 0x223344 반투명

### 대전 입자
- SphereGeometry(r=0.15), 0xff4444 (+전하) 또는 0x4444ff (-전하)
- 초기 속도 v: 자기장에 수직 방향(x축) → 로런츠 힘 → 원운동
- 원운동 궤적: 원형 경로를 따라 등속 원운동
  - 잔상(trail): 반투명 구 또는 Line으로 원형 궤적 표시

### 로런츠 힘 벡터
- 속도 v: 초록 화살표, 접선 방향
- 자기장 B: 파란 화살표, z축 방향
- 로런츠 힘 F = qv×B: 빨간 화살표, 원의 중심 방향 (구심력)
- 3개 벡터가 항상 수직 관계 (서로 90°)
- "F는 v와 B 모두에 수직 → 속력 변화 없이 방향만 변화 → 원운동" 강조

### 원운동 파라미터
- 반지름: r = mv/(qB)
- 주기: T = 2πm/(qB) (속력 무관! = 사이클로트론 주파수)
- m, v, q, B 슬라이더 → 반지름/주기 실시간 변화
  - v↑ → r↑ (더 큰 원)
  - B↑ → r↓ (더 작은 원)
  - m↑ → r↑, T↑

### 나선 운동 (v에 B 방향 성분이 있을 때)
- 속도에 자기장 평행 성분 추가 → 원운동 + 직진 = 나선 궤적
- 나선 궤적 3D 시각화 (TubeGeometry 헬릭스)

## 시나리오 2: 질량 분석기

### 3D
- 이온 발생기: BoxGeometry, 0x888888 (이온 빔 발사)
- 속도 선별기: 전기장 E + 자기장 B → qE = qvB → v = E/B (속도 선별)
- 자기장 영역: 반원형 궤적
- 다른 질량의 이온:
  - 가벼운 이온: 작은 반지름 원 (빨간 궤적)
  - 무거운 이온: 큰 반지름 원 (파란 궤적)
  - r = mv/(qB) → m ∝ r (같은 v, q, B에서)
- 검출기: 반원 끝에서 다른 위치에 도달 → 질량 구분
- "질량 분석기: r = mv/(qB) → 반지름으로 질량 측정" 라벨

## 시나리오 3: 오로라

### 3D
- 지구: SphereGeometry(r=3), 0x2266aa
- 지구 자기장: 자기력선(TubeGeometry 곡선), N극→S극
- 태양풍 대전 입자: 작은 파티클(Points 50개), 0xffcc44
  - 자기력선을 따라 나선 운동하며 극지방으로 이동
- 극지방 도착 시: 대기와 충돌 → 초록/보라 발광 (오로라!)
  - 발광 파티클: 0x44ff66 (초록), 극지방 상공에

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 기본 원운동 / 질량 분석기 / 오로라
- "전하 종류" 드롭다운: (+) / (−)
- "속력 v" 슬라이더: 1~10 (기본값 5)
- "자기장 B" 슬라이더: 0.1~2T (기본값 0.5)
- "질량 m" 슬라이더: 1~10 (상대값, 기본값 1)
- "나선 운동" 토글: ON/OFF (v의 B방향 성분 추가)
- "벡터 표시" 토글: v/B/F 화살표 ON/OFF (기본값 ON)
- "궤적 표시" 토글: ON/OFF (기본값 ON)
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌀 로런츠 힘과 대전 입자의 원운동
━━━━━━━━━━━━━━━━━━━
시나리오: [원운동/질량분석기/오로라]
q = [값]C, v = [값]m/s, B = [값]T, m = [값]kg

로런츠 힘:
  F = qvB (v⊥B일 때)
  방향: v와 B 모두에 수직

원운동:
  구심력 = 로런츠 힘 → qvB = mv²/r
  반지름: r = mv/(qB) = [값]m
  주기: T = 2πm/(qB) = [값]s
  (주기는 속력에 무관!)

질량 분석기: r ∝ m
  → 반지름으로 질량 측정
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "자기장 속에서 운동하는 전하는 로런츠 힘 F=qvB를 받습니다. 이 힘은 속도와 자기장에 수직이므로 속력은 변하지 않고 방향만 변합니다. 균일한 자기장에서 대전 입자는 등속 원운동합니다."
- 탐구 질문:
  1. "로런츠 힘이 대전 입자의 속력을 변화시키지 않는 까닭을 설명해 보세요. (힌트: 힘과 이동 방향)"
  2. "질량 분석기에서 무거운 이온의 원운동 반지름이 더 큰 까닭은 무엇인가요?"
  3. "오로라가 주로 극지방에서 관측되는 까닭을 지구 자기장과 관련지어 설명해 보세요."
- 실생활 예시: 질량 분석기(동위원소 분석), 사이클로트론(입자 가속), 오로라, CRT 모니터
- 교과서 연계: "전자기와 양자 I편 p21~25 — 자기장과 로런츠 힘 (그림 I-15~20)"
