너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 1. 빛과 물질의 이중성 > 빛의 굴절과 광학 기술
- 토픽: 빛의 굴절과 전반사
- 대상: 고등학교 (선택과목)
- 핵심 개념: 빛이 한 매질에서 다른 매질로 진행할 때 경계면에서 진행 방향이 꺾이는 현상을 굴절이라 한다. 스넬 법칙: n₁sinθ₁ = n₂sinθ₂. 빛이 굴절률이 큰 매질에서 작은 매질로 진행할 때 입사각이 임계각보다 크면 전반사가 일어난다. 광섬유(광케이블)는 전반사를 이용한다.

## 씬 구성
- 화면 중앙: 3D 두 매질 경계면 + 빛의 굴절/반사/전반사 시각화
- 2가지 모드: 굴절 법칙 / 전반사와 광섬유
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) — 어둡게 (빛이 잘 보이게)

## 모드 1: 스넬의 굴절 법칙

### 3D 두 매질
- 상단 매질(매질 1): BoxGeometry(8, 4, 4), 반투명 0x88ccff opacity 0.15 (공기/물)
- 하단 매질(매질 2): BoxGeometry(8, 4, 4), 반투명 0x4488aa opacity 0.25 (물/유리)
- 경계면: PlaneGeometry(8, 4), 0xffffff opacity 0.1 수평면
- 법선(수직선): CylinderGeometry 점선, 경계면에 수직, 0xaaaaaa

### 빛의 경로
- 입사광: CylinderGeometry(0.03, 0.03, 가변), 0xffdd44 (노란빛) emissive
  - 입사각 θ₁ 슬라이더로 조절 (0~89°)
  - 경계면까지 직선
- 굴절광: CylinderGeometry, 0xffdd44 emissive (약간 연하게)
  - 굴절각 θ₂ = arcsin((n₁/n₂)sinθ₁) 자동 계산
  - 경계면에서 꺾여서 진행
- 반사광: CylinderGeometry, 0xffdd44 opacity 0.3 (반투명)
  - 반사각 = 입사각 (항상)
- 각도 호(Arc): 입사각, 굴절각을 호로 표시 + 각도 라벨

### 매질 조합 선택
- 공기(n=1.00) → 물(n=1.33)
- 공기(n=1.00) → 유리(n=1.50)
- 물(n=1.33) → 유리(n=1.50)
- 유리(n=1.50) → 공기(n=1.00) ← 전반사 가능
- 다이아몬드(n=2.42) → 공기(n=1.00) ← 전반사 쉽게 발생

### 파면(wavefront) 시각화 (옵션)
- 입사광의 파면(수직 선분)이 경계면에 도달하면서 꺾이는 과정
- 매질 1에서 빠른 속도 → 매질 2에서 느린 속도 → 파면 기울어짐 → 굴절
- "빛의 속도가 매질에 따라 다르기 때문에 굴절" 설명

## 모드 2: 전반사와 광섬유

### 전반사 시각화
- 매질: 유리(n=1.5) → 공기(n=1.0)
- 임계각: θc = arcsin(n₂/n₁) = arcsin(1/1.5) ≈ 41.8°
- 입사각 슬라이더:
  - θ < θc: 굴절광 존재 (약한 반사 + 굴절)
  - θ = θc: 굴절각 = 90° (경계면을 따라 진행)
  - θ > θc: **전반사** — 굴절광 없음, 빛이 100% 반사
  - 전반사 시 밝은 반사광 + "전반사!" 라벨 + 이펙트

### 광섬유 3D
- 광섬유: CylinderGeometry(0.3, 0.3, 10) 코어, 0x88ccff 반투명
  - 코어(n=1.6): 내부, 높은 굴절률
  - 클래딩(n=1.4): 외부, 낮은 굴절률
- 곡선 형태: TubeGeometry 곡선 경로 (S자 또는 U자)
- 빛이 코어 내부에서 전반사를 반복하며 진행:
  - 노란 파티클(빛 입자)이 지그재그로 이동
  - 경계면에서 반사될 때마다 작은 반사 표시
- 빛이 곡선을 따라 먼 곳까지 전달
- "광섬유: 전반사를 이용해 빛 신호를 전달" 라벨

### 광섬유 활용
- 인터넷 통신: 광케이블로 데이터 전송
- 의료: 내시경
- 조명: 광섬유 조명

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 굴절 법칙 / 전반사와 광섬유
- "매질 조합" 드롭다운: 공기→물 / 공기→유리 / 물→유리 / 유리→공기 / 다이아몬드→공기
- "입사각 θ₁" 슬라이더: 0~89° (기본값 30, 1 단계)
- "파면 표시" 토글: ON/OFF
- "법선 표시" 토글: ON/OFF (기본값 ON)
- "각도 표시" 토글: ON/OFF (기본값 ON)
- "광섬유 곡률" 슬라이더 (모드 2): 0~3 (직선~곡선)
- "초기화" 버튼
