너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 1. 지구시스템 > 지구시스템의 구성과 상호작용
- 토픽: 탄소의 순환
- 대상: 고등학교 1학년
- 핵심 개념: 탄소는 생명체의 몸을 이루는 기본 원소로 지구시스템의 각 권역을 순환한다. 기권에서는 CO₂/메테인, 지권에서는 탄산칼슘/화석 연료, 수권에서는 탄산 이온, 생물권에서는 유기물 형태로 존재한다. 광합성(기권→생물권), 호흡(생물권→기권), 화석연료 연소(지권→기권), 용해(기권→수권), 석회암 형성(수권→지권) 등의 과정으로 순환한다. 탄소의 순환에서도 에너지의 흐름이 일어난다.

## 씬 구성
- 전체 화면에 4개 권역(기권, 수권, 지권, 생물권)을 2D+3D 하이브리드로 배치한다.
- 교과서 그림 III-8 (p109) 스타일의 순환 다이어그램을 3D로 재해석한다.
- 레이아웃:
  - 상단(높이 30%): 기권 — 하늘색 영역, CO₂ 분자 파티클
  - 중앙 좌측(높이 40%, 폭 30%): 생물권 — 나무, 동물 실루엣
  - 중앙 우측(높이 40%, 폭 30%): 수권 — 바다, 해양 생물
  - 하단(높이 30%): 지권 — 암석, 화석 연료, 석회암
- 권역 사이에 탄소 이동 경로를 화살표로 표시하고, 탄소(C) 파티클이 경로를 따라 이동한다.
- 배경: 0x1a2a3a (짙은 청회색)
- 조명: AmbientLight(0.4) + DirectionalLight(0.5)
- 카메라: OrthographicCamera (2D 다이어그램 뷰) + PerspectiveCamera 전환 가능

## 4개 권역 3D 표현

### 기권
- 영역: PlaneGeometry(전체 폭, 높이 30%), 0x88ccee, opacity 0.15
- CO₂ 분자: 검은 구(C, r=0.06) + 빨간 구 2개(O, r=0.05) 결합, 10~15개 랜덤 이동
- CH₄(메테인): 검은 구(C) + 흰 구 4개(H), 3~5개
- 라벨: "기권 — CO₂, CH₄"
- CO₂ 농도 표시: "CO₂: [값] ppm" (슬라이더와 연동)

### 생물권
- 나무: CylinderGeometry(줄기) + SphereGeometry(잎, 0x228b22), 3그루
- 동물: 단순 실루엣(BoxGeometry 조합), 0x8B4513 갈색, 2마리
- 나무 잎에 "광합성" 라벨, 동물에 "호흡" 라벨
- 유기물 표현: 나무/동물 내부에 C 파티클(검은 작은 구) 포함

### 수권
- 바다: BoxGeometry(폭 30%, 높이 40%, 깊이 2), 0x2255aa, opacity 0.5
- 해양 생물: 물고기 실루엣(삼각형 BufferGeometry) 3~5마리
- 탄산 이온: "CO₃²⁻" 라벨이 붙은 파란 구 5~8개, 바다 내부에서 느리게 이동
- 해저: 석회암 암석(IcosahedronGeometry, 0xddddcc), 2~3개

### 지권
- 영역: PlaneGeometry(전체 폭, 높이 30%), 0x664422, opacity 0.15
- 화석 연료: 검은 큐브(BoxGeometry, 0x222222) "석탄/석유" 라벨, 2~3개
- 석회암: 밝은 갈색 큐브(0xddccaa) "CaCO₃" 라벨
- 화산: ConeGeometry(r=0.8, h=1.5, 0x664444), 꼭대기에 빨간 파티클(용암)

## 탄소 이동 경로 6가지 (교과서 그림 III-8)

각 경로에 화살표(ArrowHelper 또는 CurveGeometry + 삼각형 머리)와 C 파티클(검은 구 r=0.04)이 경로를 따라 이동하는 애니메이션:

| # | 경로 | 방향 | 화살표 색 | 과정 |
|---|------|------|----------|------|
| 1 | 기권 → 생물권 | ↓좌 | 0x44aa44 초록 | **광합성**: CO₂ + H₂O → 포도당 + O₂ |
| 2 | 생물권 → 기권 | ↑좌 | 0xff6644 주황 | **호흡**: 유기물 + O₂ → CO₂ + H₂O |
| 3 | 지권 → 기권 | ↑중앙 | 0xff4422 빨강 | **화석연료 연소 / 화산 활동**: 탄소 → CO₂ |
| 4 | 기권 → 수권 | ↓우 | 0x4488ff 파랑 | **용해**: CO₂가 해수에 녹아 탄산 이온 형성 |
| 5 | 수권 → 지권 | ↓ | 0x886644 갈색 | **석회암 형성**: 탄산 이온 → CaCO₃ 침전 |
| 6 | 생물권 → 지권 | ↓좌 | 0x444444 회색 | **화석연료 형성**: 생물 유해 매몰 → 석탄/석유 |

### 각 경로 애니메이션 상세
- C 파티클(검은 구 r=0.04): 곡선 경로(CubicBezierCurve3)를 따라 이동, 속도 2초/경로
- 경로 시작점에서 생성, 도착점에서 사라짐
- 각 경로에 2~3개 C 파티클이 동시에 다른 위치에서 이동 (연속 흐름 느낌)
- 경로 위에 과정 라벨 표시 (예: "광합성", "호흡", "연소")

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "화석 연료 연소량" 슬라이더: 0 ~ 10 (기본값 3)
  - 연소량↑ → 경로 3(지권→기권) C 파티클 수↑ → 기권 CO₂ 농도↑
  - CO₂ 농도 표시 실시간 변화
  - 기권 영역이 점점 불투명해짐(온실효과 시각화)
- "광합성 활동" 슬라이더: 0 ~ 10 (기본값 5)
  - 광합성↑ → 경로 1(기권→생물권) C 파티클 수↑ → CO₂ 농도↓
  - 나무가 더 크고 녹색으로 변화
- "순환 속도" 슬라이더: 0.5x ~ 5x (기본값 1x)
- "경로 선택" 드롭다운: 전체 / 광합성만 / 호흡만 / 연소만 / 용해만 / 석회암만 / 화석연료만
  - 선택된 경로만 밝게, 나머지 페이드
- "에너지 흐름 표시" 토글: ON/OFF (기본값 OFF)
  - ON: 광합성(태양 에너지 흡수 ↓), 호흡(열 에너지 방출 ↑), 연소(열+빛 에너지 방출 ↑) 에너지 화살표 추가 표시
- "CO₂ 농도 그래프" 토글: ON/OFF (기본값 OFF)
  - ON: 하단에 시간에 따른 CO₂ 농도 변화 그래프 표시 (연소량과 광합성 슬라이더 값에 따라 실시간 변화)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌿 탄소의 순환
━━━━━━━━━━━━
CO₂ 농도:      [값] ppm
화석연료 연소:  [값]/10
광합성 활동:    [값]/10

탄소의 분포:
  기권: CO₂, CH₄ (기체)
  수권: CO₃²⁻ (탄산 이온)
  지권: CaCO₃ (석회암), 화석 연료
  생물권: 유기물 (C-H-O-N)

주요 순환 경로:
  광합성: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂
  호흡:   C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O
  연소:   CₓHᵧ + O₂ → CO₂ + H₂O

탄소 분포량 최다: 지권의 석회암
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "탄소는 기권, 수권, 지권, 생물권 사이를 끊임없이 순환합니다. 화석 연료 연소로 기권의 CO₂ 농도가 증가하면 지구 온난화의 원인이 됩니다."
- 탐구 질문:
  1. "광합성과 호흡은 탄소 순환에서 어떤 역할을 하나요?"
  2. "화석 연료를 많이 태우면 기권의 CO₂ 농도는 어떻게 변하나요?"
  3. "탄소의 분포량이 가장 많은 곳은 어디인가요?" (답: 지권의 석회암)
- 실생활 예시: 화석 연료 사용(CO₂ 증가), 식물 조림(CO₂ 흡수), 해양 산성화(CO₂ 용해), 탄소 포집 기술(CCS)
- 교과서 연계: "미래엔 통합과학1 III편 p109 — 탄소의 순환(그림 III-8)"

## 코드 규칙
- 단일 HTML 파일, Three.js CDN, OrbitControls CDN, lil-gui CDN
- ES6 모듈, 한국어 주석, 반응형, 모바일 터치 지원
- C 파티클: BufferGeometry + Points, 곡선 경로 CubicBezierCurve3 위 이동
- 권역 배경: PlaneGeometry + MeshBasicMaterial(반투명)
- CO₂ 분자: Group(SphereGeometry 3개)
- 화살표: ArrowHelper 또는 자체 곡선 + 삼각형 머리
- CO₂ 농도 그래프: Canvas 2D API
- 성능: requestAnimationFrame, C 파티클 최대 50개
- 파일명: carbon_cycle_sim.html
