너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 05. 정상파와 악기
- 토픽: 정상파와 공명
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 같은 진동수의 두 파동이 반대 방향으로 진행하면서 중첩되면 정상파(standing wave)가 형성된다. 정상파에서 진동이 최대인 점을 배(antinode), 진동이 0인 점을 마디(node)라 한다. 고유 진동수로 진동시키면 공명이 일어나 진폭이 최대가 된다. 줄의 양 끝이 고정된 경우 L = nλ/2 (n = 1, 2, 3, ...).

## 씬 구성
- 화면 좌측 60%: 3D 줄/관 위의 정상파 시각화
- 화면 우측 40%: 고조파 비교 + 진동수 관계 (Canvas 2D)
- 2가지 모드: 현(줄) 정상파 / 관(공기 기둥) 정상파
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.5)

## 모드 1: 현(줄)의 정상파

### 3D 줄
- 줄: TubeGeometry 곡선 (사인파 형태), 0xcc6633 (구리색), 양 끝 고정
- 줄 길이 L: 슬라이더로 조절
- 양 끝 고정점: SphereGeometry(r=0.08), 0x444444
- 줄이 정상파 형태로 진동: y(x,t) = 2Asin(nπx/L)cos(ωt)
  - BufferGeometry의 position을 매 프레임 갱신

### n차 고조파 (n = 1, 2, 3, 4, 5)
- **기본 진동 (n=1)**: 배 1개, 마디 2개(양 끝)
  - L = λ/2 → λ₁ = 2L → f₁ = v/(2L)
- **2차 고조파 (n=2)**: 배 2개, 마디 3개
  - L = λ → λ₂ = L → f₂ = 2f₁
- **3차 고조파 (n=3)**: 배 3개, 마디 4개
  - L = 3λ/2 → λ₃ = 2L/3 → f₃ = 3f₁
- **n차**: 배 n개, 마디 (n+1)개, f_n = nf₁
- 고조파 번호 n 슬라이더: 1~5

### 마디/배 표시
- 마디(node): 빨간 점, 정상파에서 진동 = 0인 위치
- 배(antinode): 초록 점, 정상파에서 진동 최대인 위치
- 라벨: "마디", "배"

### 두 파동의 중첩 시각화 (정상파 형성 원리)
- "형성 원리 보기" 토글:
  - 오른쪽으로 진행하는 파동 (파란 선): y₁ = Asin(kx-ωt)
  - 왼쪽으로 진행하는 파동 (빨간 선): y₂ = Asin(kx+ωt)
  - 합성 (정상파, 초록 굵은 선): y = 2Asin(kx)cos(ωt)
  - 두 파동이 반대 방향으로 진행하면서 중첩 → 정상파 형성

## 모드 2: 관(공기 기둥)의 정상파

### 열린 관 (양쪽 열림)
- CylinderGeometry(0.3, 0.3, L), 투명 0xaaaacc
- 양쪽 끝 = 배 (공기 자유 진동)
- 기본 진동: L = λ/2, f₁ = v/(2L)
- n차: f_n = nf₁ (모든 정수 배음)

### 닫힌 관 (한쪽 닫힘)
- 한쪽 끝 = 마디 (닫힌 끝, 공기 진동 불가)
- 다른 끝 = 배 (열린 끝)
- 기본 진동: L = λ/4, f₁ = v/(4L)
- 홀수 배음만: f_n = nf₁ (n = 1, 3, 5, ...)

### 공기 분자 진동
- 관 내부 공기 분자를 파티클로 표시
- 배 위치: 공기 분자 큰 폭으로 진동
- 마디 위치: 공기 분자 진동 없음 (밀도만 변화)
- 밀/소 패턴: 종파 특성 시각화 (색상/밀도)

## 공명 시각화
- 진동수 슬라이더를 연속적으로 조절
- 고유 진동수(f₁, f₂, f₃...)에서만 정상파 형성 → 진폭 최대!
- 고유 진동수가 아닌 경우: 불규칙 진동, 작은 진폭
- "공명: 고유 진동수로 진동시킬 때 진폭이 최대" 라벨
- 진폭-진동수 그래프: f₁, f₂, f₃에서 피크

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 현(줄) / 열린 관 / 닫힌 관
- "고조파 n" 슬라이더: 1~5 (기본값 1, 정수)
- "줄/관 길이 L" 슬라이더: 0.5~3m (기본값 1)
- "파동 속력 v" 슬라이더: 100~500m/s (기본값 340, 공기 중 음속)
- "형성 원리" 토글: 두 진행파 + 합성 표시 ON/OFF
- "마디/배 표시" 토글: ON/OFF (기본값 ON)
- "공명 스캔" 버튼: 진동수를 연속으로 변화시켜 공명 확인
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🎸 정상파와 공명
━━━━━━━━━━━━━━
모드: [현/열린관/닫힌관]
L = [값]m, v = [값]m/s, n = [값]

현 (양쪽 고정):
  L = nλ/2 → λ_n = 2L/n
  f_n = nv/(2L) = n × f₁
  f₁ = v/(2L) = [값]Hz

열린 관 (양쪽 열림):
  f_n = nv/(2L) (모든 n)

닫힌 관 (한쪽 닫힘):
  f_n = nv/(4L) (홀수 n만)
  f₁ = v/(4L) = [값]Hz

정상파: 배 [n]개, 마디 [n+1]개
공명: 고유 진동수에서 진폭 최대
```

- 탐구 질문:

  3. "와인잔에 노래를 불러 깨뜨리는 것이 가능한 까닭을 공명으로 설명해 보세요."
- 실생활 예시: 기타줄(현의 정상파), 관악기(관의 정상파), 다리 공명(타코마 다리 붕괴), 전자레인지(정상파)
