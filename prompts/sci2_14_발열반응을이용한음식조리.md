너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 에너지의 흡수와 방출
- 토픽: 발열 반응을 이용한 음식 조리
- 대상: 고등학교 1학년
- 핵심 개념: 물과 산화 칼슘의 반응은 에너지를 방출하는 발열 반응이다. 이 반응에서 발생하는 열을 이용하면 가열 장치 없이 음식을 조리할 수 있다. 산화 칼슘 주머니의 개수를 달리하여 물의 온도 변화를 측정하면, 산화 칼슘의 양이 많을수록 더 많은 에너지가 방출되어 온도가 높아진다.

## 씬 구성
- 화면 중앙: 3D 실험 장치 (반응 용기 + 조리 용기 + 온도 센서)
- 화면 우측: 실시간 온도 변화 그래프 (Canvas 2D)
- 교과서 탐구 (p58~59): 물과 산화 칼슘을 이용한 음식 조리 방법 설계하고 실험하기
- 배경: 0xf5f0e8
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 3D 실험 장치 (교과서 탐구 과정, p58)

### 반응 용기
- BoxGeometry(1.5, 1.0, 1.5) — 사각형 용기, 0xddddcc (스테인리스)
- 내부에 물 (PlaneGeometry 수면, 0x88bbdd opacity 0.6)
- 산화 칼슘 주머니: BoxGeometry(0.3, 0.2, 0.3) × N개, 0xeeeeee (흰색 부직포)
  - 주머니 개수: 1~5개 슬라이더로 조절
  - 주머니를 물에 넣으면 발열 반응 시작

### 조리 용기
- CylinderGeometry(0.6, 0.6, 0.8) — 스테인리스 조리 용기, 0xcccccc
- 물 300mL 표시
- 반응 용기 위에 겹쳐 놓은 상태
- 온도 센서 꽂힌 모습: CylinderGeometry(0.02, 0.02, 1) 가느다란 막대

### 온도 센서
- 스마트 기기와 연결된 온도 센서 표현
- 디지털 온도 표시: 현재 온도 큰 숫자로 표시

## 반응 애니메이션
1. 산화 칼슘 주머니를 반응 용기에 넣는다 (드롭 애니메이션, 1초)
2. 물 120mL를 붓는다 (물 파티클 쏟아짐 효과, 1초)
3. 발열 반응 시작: 증기 파티클(Points, 0xdddddd 흰색, 위로 올라감) 발생
4. 온도 상승 애니메이션: 20°C → 최고 온도 (주머니 수에 따라 다름)
   - 1개: ~35°C
   - 2개: ~50°C
   - 3개: ~65°C
   - 4개: ~78°C
   - 5개: ~90°C
5. 조리 용기 위의 물도 온도 상승 (열 전달)
6. 반응 용기 주변 열기 효과 (빨간-주황 파티클)

## 실시간 온도 변화 그래프 (우측, Canvas 2D)
- X축: 시간 (0~300초)
- Y축: 온도 (0~100°C)
- 주머니 개수별로 다른 색상의 곡선:
  - 1개: 파란색
  - 3개: 초록색
  - 5개: 빨간색
- 시간에 따라 실시간으로 그래프가 그려짐
- 최고 온도 지점에 마커 표시

## 음식 조리 결과
- 주머니 3개 이상: "달걀찜 조리 가능!" 메시지 + 달걀찜 3D 모형
- 주머니 5개: "라면도 가능!" + 뜨거운 물 표현
- 주머니 1개: "온도가 충분하지 않습니다" 경고

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "산화 칼슘 주머니 수" 슬라이더: 1~5 (기본값 3, 정수)
- "실험 시작" 버튼: 발열 반응 애니메이션 시작
- "시간 배속" 슬라이더: 1x~10x (기본값 5x)
- "온도 그래프 겹치기" 토글: 이전 실험 그래프와 겹쳐 표시
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)

- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 실험 초기화 시 이전 객체 dispose() 또는 속성 리셋
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_14_exothermic_cooking.html
