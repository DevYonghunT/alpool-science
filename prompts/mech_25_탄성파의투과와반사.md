너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 02. 탄성파
- 토픽: 탄성파의 투과와 반사
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 파동이 서로 다른 매질의 경계면에 도달하면 일부는 반사되고 일부는 투과된다. 밀한 매질(빽빽한)에서 소한 매질(느슨한)로 진행하면 반사파는 입사파와 같은 위상이고, 소한→밀한으로 진행하면 반사파는 위상이 반전된다. 초음파는 매질 경계에서 반사되는 성질을 이용하여 태아 검진, 수중 탐사(소나), 비파괴 검사 등에 활용된다.

## 씬 구성
- 화면 중앙: 3D 두 매질 경계면 + 파동의 투과/반사 시각화
- 2가지 모드: 밀한→소한 / 소한→밀한
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 파동 투과/반사

### 두 매질
- 매질 1 (왼쪽): BoxGeometry(8, 2, 2), 0x4488aa 반투명 opacity 0.15
  - 내부에 입자(SphereGeometry) 격자 → 밀도 시각화
- 매질 2 (오른쪽): BoxGeometry(8, 2, 2), 0xaa8844 반투명 opacity 0.15
  - 입자 밀도가 매질 1과 다름
- 경계면: PlaneGeometry, 0xffffff opacity 0.2, 중앙에 수직

### 파동 시각화 (횡파로 표현)
- 입사파: 왼쪽에서 오른쪽으로 진행, 0x4488ff (파랑)
  - 입자들이 y방향으로 진동하며 파동 전파
  - 사인파 곡선
- 반사파: 경계면에서 되돌아옴, 0xff4444 (빨강)
  - 진폭이 입사파보다 작음 (에너지 일부가 투과되므로)
- 투과파: 경계면을 통과하여 매질 2로 진행, 0x44ff44 (초록)
  - 진폭이 입사파보다 작음
  - 매질 2의 파동 속력이 다르므로 파장 변화 (진동수는 일정)

### 위상 변화 (핵심!)
- **밀한→소한 (고정단 반사 비유)**: 반사파 위상 반전 없음 (같은 위상)
  - 반사파가 입사파와 같은 방향으로 시작
- **소한→밀한 (자유단 반사 비유)**: 반사파 위상 반전! (반대 위상)
  - 반사파가 입사파와 반대 방향으로 시작
  - 위상 반전을 색상/화살표로 강조
- 토글로 두 경우 비교

### 경계면에서의 에너지
- 입사 에너지 = 반사 에너지 + 투과 에너지 (에너지 보존)
- 에너지 막대그래프: 입사(100%) = 반사([N]%) + 투과([N]%)
- 매질 밀도 차이가 클수록 반사 비율↑ (전반사에 가까워짐)

## 용수철 모형 (줄 파동)
- 대안 시각화: 긴 줄(Line 또는 TubeGeometry)에 파동 전파
  - 한쪽 끝을 흔들면 파동이 진행
  - 고정단(벽에 고정): 위상 반전 반사
  - 자유단(링에 연결, 자유 이동): 위상 보존 반사
- 고정단/자유단 토글

## 초음파 활용 (실생활)
- "초음파 응용" 토글:
  - 태아 초음파 검진: 초음파가 조직 경계에서 반사 → 이미지 생성
  - 소나(SONAR): 수중에서 음파 발사 → 반사파 수신 → 거리 측정
  - 비파괴 검사: 금속 내부 균열 탐지
- 각 응용의 간단한 3D 다이어그램

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "경우" 드롭다운: 밀한→소한 / 소한→밀한 / 고정단 반사 / 자유단 반사
- "입사 진폭" 슬라이더: 0.3~1.5 (기본값 0.8)
- "입사 진동수" 슬라이더: 0.5~3Hz (기본값 1)
- "매질 밀도 차이" 슬라이더: 1~5 (기본값 2)
- "파동 발사" 버튼
- "위상 표시" 토글: ON/OFF
- "에너지 막대" 토글: ON/OFF (기본값 ON)
- "초음파 응용" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔊 탄성파의 투과와 반사
━━━━━━━━━━━━━━━━
경우: [밀한→소한 / 소한→밀한]

입사파: A_i = [값], f = [값]Hz
반사파: A_r = [값] (반사율 [값]%)
투과파: A_t = [값] (투과율 [값]%)

위상 변화:
  밀한→소한: 반사파 위상 보존
  소한→밀한: 반사파 위상 반전!

에너지 보존:
  입사 = 반사 + 투과
  
경계면에서:
  진동수 f는 변하지 않음
  파장 λ = v/f (매질에 따라 v 다름)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "파동이 서로 다른 매질의 경계면에 도달하면 일부는 반사, 일부는 투과됩니다. 소한→밀한 매질로 진행할 때 반사파의 위상이 반전됩니다. 투과 시 진동수는 변하지 않지만 파장은 변합니다."
- 탐구 질문:
  1. "고정단에서 반사된 파동의 위상이 반전되는 까닭을 작용 반작용으로 설명해 보세요."
  2. "초음파 검진에서 조직 경계면이 뚜렷할수록 이미지가 선명한 까닭은 무엇인가요?"
  3. "파동이 매질 경계를 통과할 때 변하는 물리량과 변하지 않는 물리량은 각각 무엇인가요?"
- 실생활 예시: 초음파 검진, 소나(잠수함), 지진파 분석(지구 내부 구조), 박쥐 반향 정위
- 교과서 연계: "역학과 에너지 III편 p100~107 — 탄성파의 전파, 투과, 반사"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 경우 전환 시 파동 리셋 (입자 위치 초기화)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 입자 InstancedMesh 60개 (매질 1 + 매질 2)
- 파일명: mech_25_wave_reflection_transmission.html
