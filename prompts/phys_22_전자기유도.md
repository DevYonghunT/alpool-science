너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 2. 전기와 자기의 상호작용 > 전자기 유도와 에너지 전달
- 토픽: 전자기 유도 (패러데이 법칙)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 자석이나 코일을 움직여 코일을 통과하는 자기장(자기 선속)이 변하면 코일에 전류가 유도되는 전자기 유도 현상이 나타난다. 유도 기전력의 크기는 자기 선속의 시간 변화율에 비례한다(패러데이 법칙). 유도 전류의 방향은 자기 선속의 변화를 방해하는 방향이다(렌츠 법칙).

## 씬 구성
- 화면 중앙: 3D 자석 + 코일 + 검류계/LED
- 3가지 모드: 자석 이동 / 코일 이동 / 자기장 변화(전자석)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 3D 실험 장치

### 코일
- TorusGeometry(1.5, 0.06, 8, 64) × 10~20겹 (겹 수 조절 가능)
- 색상: 0xcc6633 (구리색)
- 고정 위치 (또는 이동 모드에서 이동)
- 코일 양 끝에 전선 → 검류계/LED 연결

### 막대 자석
- BoxGeometry(0.6, 0.6, 3), N극 0xff4444 / S극 0x4444ff
- "N" / "S" 라벨
- 코일에 가까이/멀리 이동 (슬라이더 또는 드래그)

### 자기력선
- TubeGeometry 곡선, 자석 N극→S극, 0x44ff88 반투명
- 자석 위치에 따라 코일을 통과하는 자기력선 수 변화 시각화

### 검류계 / LED
- 검류계: CircleGeometry + 바늘(회전), 0번 중앙
  - 유도 전류 방향에 따라 바늘이 좌/우로 움직임
  - 전류 크기에 따라 바늘 각도 변화
- LED: SphereGeometry(r=0.1), 유도 전류 흐를 때 0xffff44 발광

## 3가지 모드

### 모드 1: 자석을 코일에 넣기/빼기
- 자석 위치 슬라이더: -5(먼 곳) ~ 0(코일 안) ~ +5(반대쪽 먼 곳)
- **자석이 접근할 때**: 코일 통과 자기 선속 증가 → 유도 전류 발생 → 검류계 바늘 이동
- **자석이 코일 안에 정지**: 자기 선속 변화 없음 → 유도 전류 = 0 → 바늘 중앙
- **자석이 멀어질 때**: 자기 선속 감소 → 반대 방향 유도 전류 → 바늘 반대로
- **N극/S극 방향 전환**: 유도 전류 방향 반대

### 모드 2: 코일을 자석에 가까이/멀리
- 자석 고정, 코일 이동
- 결과 동일: 상대 운동이 중요

### 모드 3: 전자석의 전류 변화
- 코일 옆에 전자석(솔레노이드 + 철심)
- 전자석 전류를 ON/OFF 또는 세기 변경
- 전류 변화 → 자기장 변화 → 코일에 유도 전류
- "스위치 넣는 순간"과 "끄는 순간"에만 유도 전류 발생

## 자기 선속 그래프 (우측 Canvas 2D)
- X축: 시간 t
- Y축 상: 자기 선속 Φ (코일 통과)
- Y축 하: 유도 기전력 ε = -ΔΦ/Δt
- 자석 이동에 따라 실시간으로 그래프가 그려짐
- Φ가 변하는 구간에서만 ε ≠ 0

## 렌츠 법칙 시각화
- 유도 전류의 방향: 자기 선속 변화를 방해하는 방향
- 예: 자석 N극이 접근 → 자기 선속 증가 → 코일이 N극을 만들어 반발(자기 선속 증가 방해)
- 코일 끝에 N/S 표시로 렌츠 법칙 시각화
- "유도 전류는 자기 선속 변화를 방해하는 방향" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 자석 이동 / 코일 이동 / 전자석 전류 변화
- "자석 위치" 슬라이더 (모드 1): -5~+5 (기본값 -3)
- "자석 극성" 토글: N극 먼저 / S극 먼저
- "코일 감은 수" 슬라이더: 5~30 (기본값 15)
- "자석 세기" 슬라이더: 0.1~1T (기본값 0.5)
- "자동 왕복" 버튼: 자석이 자동으로 코일을 통과하며 왕복
- "자기력선 표시" 토글: ON/OFF (기본값 ON)
- "렌츠 법칙 표시" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 전자기 유도 (패러데이 법칙)
━━━━━━━━━━━━━━━━━━
모드: [자석 이동 / 코일 이동 / 전자석]
자석 위치: [값]
코일 감은 수 N = [값]

자기 선속: Φ = BA = [값] Wb
유도 기전력: ε = -NΔΦ/Δt = [값]V
유도 전류: [값]A ([방향])

패러데이 법칙:
  ε = -NΔΦ/Δt
  자기 선속이 변해야 유도 전류 발생

렌츠 법칙:
  유도 전류는 자기 선속 변화를
  방해하는 방향으로 흐른다
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "코일을 통과하는 자기 선속이 변하면 코일에 유도 전류가 발생합니다(전자기 유도). 유도 기전력은 자기 선속의 시간 변화율에 비례하며(패러데이 법칙), 유도 전류 방향은 변화를 방해합니다(렌츠 법칙)."
- 탐구 질문:
  1. "자석이 코일 안에 정지해 있을 때 유도 전류가 흐르지 않는 까닭은 무엇인가요?"
  2. "코일의 감은 수를 늘리면 유도 기전력은 어떻게 변하나요?"
  3. "자석을 더 빠르게 움직이면 유도 전류는 어떻게 변하나요?"
- 실생활 예시: 발전기, 전기 기타 픽업, 금속 탐지기, 교통 카드(NFC)
- 교과서 연계: "물리학 II편 p112~117 — 전자기 유도, 패러데이 법칙, 렌츠 법칙"
