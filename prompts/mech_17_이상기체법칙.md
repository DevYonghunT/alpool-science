너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 02~03. 열에 의한 물질의 변화 / 열역학 제1법칙
- 토픽: 이상 기체 법칙과 과정
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 이상 기체 상태 방정식 PV = nRT (P: 압력, V: 부피, n: 몰수, R: 기체 상수 8.314 J/(mol·K), T: 절대 온도). 등온 과정(T 일정): PV = 일정 (보일 법칙). 등적 과정(V 일정): P/T = 일정 (샤를 법칙). 등압 과정(P 일정): V/T = 일정. 단열 과정: Q = 0, PV^γ = 일정.

## 씬 구성
- 화면 좌측 50%: 3D 실린더+피스톤 + 기체 분자 입자 모형
- 화면 우측 50%: P-V 다이어그램 (Canvas 2D)
- 4가지 과정: 등온 / 등적 / 등압 / 단열
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 실린더 + 피스톤

### 실린더
- CylinderGeometry(1.2, 1.2, 4, 32, 1, true), 0xaaaaaa (금속 회색), 투명 (내부 보이게)
- 바닥: CircleGeometry, 0x888888 (밀폐)
- 벽에 눈금: 부피 표시 (리터)

### 피스톤
- CylinderGeometry(1.18, 1.18, 0.3), 0x666688
- 위아래로 이동 가능 (부피 V 변화)
- 피스톤 위에 추(무게) 배치 가능 → 외부 압력 변화

### 기체 분자 입자
- SphereGeometry(r=0.06) × 30~50개, 0x44cc88
- 실린더 내부에서 랜덤 방향으로 이동 (벽/피스톤과 충돌하면 반사)
- 온도↑: 이동 속도↑ (빠르게), 색상 빨강쪽
- 온도↓: 이동 속도↓ (느리게), 색상 파랑쪽
- 벽과 충돌 시 작은 플래시 → 압력 시각화

### 가열/냉각 장치
- 실린더 아래 히터: BoxGeometry, 0xff4444 (가열 시 빨간 글로우)
- 냉각 표시: 파란 글로우

## 4가지 열역학 과정

### 등온 과정 (T 일정)
- 피스톤이 천천히 올라감/내려감 (부피 변화)
- 온도 일정 → 분자 속도 변화 없음
- P ∝ 1/V: 부피↑ → 압력↓, 부피↓ → 압력↑
- P-V 그래프: 쌍곡선 (PV = 일정)
- 열 공급 필요: Q = W (내부 에너지 변화 없음)

### 등적 과정 (V 일정)
- 피스톤 고정 (잠금), 부피 변하지 않음
- 가열 → 온도↑, 압력↑ (분자 속도 증가 → 벽 충돌 강해짐)
- P ∝ T: P-V 그래프에서 수직선
- W = 0 (부피 변화 없으므로), Q = ΔU (열 = 내부 에너지 변화)

### 등압 과정 (P 일정)
- 피스톤 위에 일정한 추 (외부 압력 일정)
- 가열 → 온도↑, 부피↑ (피스톤 올라감)
- V ∝ T: P-V 그래프에서 수평선
- Q = ΔU + W: 열 = 내부 에너지 변화 + 기체가 한 일

### 단열 과정 (Q = 0)
- 실린더 단열재로 감싸기 (단열벽 표시)
- 피스톤을 빠르게 압축: 온도↑ (분자 속도 증가), 압력↑
- 피스톤을 빠르게 팽창: 온도↓, 압력↓
- PV^γ = 일정 (γ = Cp/Cv, 이상 기체 γ ≈ 1.4)
- P-V 그래프: 등온 곡선보다 급한 기울기
- 디젤 엔진: 단열 압축으로 고온 → 연료 자연 발화

## P-V 다이어그램 (우측 Canvas 2D)
- X축: 부피 V (리터)
- Y축: 압력 P (atm 또는 kPa)
- 4가지 과정별 곡선:
  - 등온: 쌍곡선 (빨강)
  - 등적: 수직선 (파랑)
  - 등압: 수평선 (초록)
  - 단열: 등온보다 급한 곡선 (보라)
- 현재 상태 (P, V) 점을 빨간 마커로 표시
- 과정 진행에 따라 경로를 실시간으로 그림
- **그래프 아래 면적 = 기체가 한 일 W** (음영으로 표시)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "과정" 드롭다운: 등온 / 등적 / 등압 / 단열
- "초기 온도 T" 슬라이더: 200~600K (기본값 300)
- "초기 부피 V" 슬라이더: 1~10L (기본값 5)
- "초기 압력 P" 슬라이더: 0.5~5atm (기본값 1)
- "부피 변화" 슬라이더 (등온/단열): 현재 부피의 50~200%
- "온도 변화" 슬라이더 (등적/등압): ±200K
- "과정 실행" 버튼: 해당 과정 애니메이션 재생
- "분자 표시" 토글: ON/OFF (기본값 ON)
- "P-V 그래프 면적" 토글: ON/OFF (일 W 표시)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔬 이상 기체 법칙
━━━━━━━━━━━━━━
과정: [등온/등적/등압/단열]
PV = nRT
P = [값]atm, V = [값]L, T = [값]K

등온 (T 일정): PV = 일정
등적 (V 일정): P/T = 일정, W=0
등압 (P 일정): V/T = 일정
단열 (Q = 0): PV^γ = 일정

열역학 제1법칙:
  Q = ΔU + W
  Q: 흡수열, ΔU: 내부E변화, W: 일
  현재: Q=[값]J, ΔU=[값]J, W=[값]J
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "이상 기체 상태 방정식 PV=nRT. 등온(PV=일정), 등적(P/T=일정), 등압(V/T=일정), 단열(PV^γ=일정) 과정에서 기체의 상태 변화를 이해하고 P-V 다이어그램으로 표현합니다."
- 탐구 질문:
  1. "등온 과정에서 기체를 압축하면 압력은 어떻게 변하나요? (PV=일정)"
  2. "등적 과정에서 열을 가하면 내부 에너지와 압력은 어떻게 변하나요?"
  3. "단열 압축으로 디젤 엔진에서 연료가 자연 발화하는 원리를 설명해 보세요."
- 실생활 예시: 자전거 펌프(단열 압축→뜨거움), 냉장고(등온 팽창), 디젤 엔진, 풍선
