너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 1. 전기장과 전기 에너지 > 전기장과 전위차
- 토픽: 전기장과 전기력선
- 대상: 고등학교 (선택과목)
- 핵심 개념: 전하를 띤 물체는 주위 공간에 전기장을 형성한다. 전기장은 양(+)전하가 받는 전기력의 방향으로 정의하며, 전기력선으로 시각화한다. 전기력선은 양전하에서 나와 음전하로 들어가며, 전기력선의 밀도가 높을수록 전기장이 강하다. 같은 부호의 전하 사이에는 척력, 다른 부호의 전하 사이에는 인력이 작용한다.

## 씬 구성
- 화면 중앙: 3D 공간에 점전하를 배치하고 전기장/전기력선을 시각화
- 4가지 배치: 양전하 1개 / 음전하 1개 / 같은 부호 2개 / 다른 부호 2개
- 배경: 0x0a0a20 (어두운 배경, 전기력선이 잘 보이게)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(0.5)

## 3D 전하 모형

### 점전하
- 양전하(+q): SphereGeometry(r=0.3, 16, 16), 0xff4444 (빨강), "+" 라벨
- 음전하(-q): SphereGeometry(r=0.3, 16, 16), 0x4444ff (파랑), "−" 라벨
- 전하량 크기에 따라 구 크기 변화 (q 슬라이더)
- 드래그로 전하 위치 이동 가능 (Raycaster)

### 전기력선 (핵심 시각화)
- 양전하에서 나가는 방향: TubeGeometry 곡선 + ConeGeometry 화살촉
- 음전하로 들어가는 방향
- 전기력선 색상: 0x44ff88 (초록) 또는 전기장 세기에 따라 색상 변화(약→강: 파랑→빨강)
- 전기력선 개수: 전하량에 비례 (q=1이면 8개, q=2이면 16개 등)
- 3D 공간에 방사형으로 퍼져나감 (2D 단면 모드도 지원)

### 전기장 벡터
- 공간의 격자 점에 작은 화살표(ArrowHelper)로 전기장 방향/크기 표시
- 화살표 길이 = 전기장 세기에 비례
- 화살표 색상: 세기에 따라 그라데이션 (약: 파랑 → 강: 빨강)
- "벡터장 표시" 토글로 ON/OFF

## 4가지 배치 모드

### 모드 1: 양전하 1개
- 전기력선이 +q에서 방사형으로 바깥으로 퍼짐
- 양전하에 가까울수록 전기력선 밀도↑ = 전기장 강함
- 멀어질수록 전기력선 밀도↓ = 전기장 약함

### 모드 2: 음전하 1개
- 전기력선이 바깥에서 −q 방향으로 모여듦

### 모드 3: 같은 부호 (+q, +q)
- 두 양전하 사이: 전기력선이 반발 (척력)
- 중간 지점에서 전기력선이 서로 밀어냄 → 전기장 약한 영역
- 척력 화살표 표시: 두 전하 사이에 빨간 양방향 화살표

### 모드 4: 다른 부호 (+q, −q) — 전기 쌍극자
- +q에서 나온 전기력선이 곡선을 그리며 −q로 들어감
- 아름다운 쌍극자 전기력선 패턴
- 인력 화살표 표시: 두 전하 사이에 초록 양방향 화살표

## 시험 전하(test charge) 시뮬레이션
- 작은 양전하(시험 전하, SphereGeometry r=0.1, 0xffff44)를 공간에 놓으면
- 전기장 방향으로 힘을 받아 이동하는 애니메이션
- 시험 전하의 궤적을 점선으로 표시
- "시험 전하 놓기" 버튼 + 마우스 클릭으로 위치 지정

## 전기장 세기 공식 (교과서)
- 점전하 q에 의한 전기장: E = kq/r² (k = 9×10⁹ N·m²/C²)
- 전기장 세기를 클릭한 위치에서 실시간 계산 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "배치" 드롭다운: +q 1개 / −q 1개 / +q +q / +q −q
- "전하량 q₁" 슬라이더: 1~5 (기본값 2, 정수)
- "전하량 q₂" 슬라이더 (2전하 모드): 1~5
- "전하 간 거리" 슬라이더 (2전하 모드): 1~8 (기본값 4)
- "전기력선 표시" 토글: ON/OFF (기본값 ON)
- "벡터장 표시" 토글: ON/OFF (기본값 OFF)
- "시험 전하 놓기" 버튼
- "2D 단면 모드" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 전기장과 전기력선
━━━━━━━━━━━━━━
배치: [모드명]
q₁ = [값]C, q₂ = [값]C
거리 r = [값]m

전기장 세기:
  E = kq/r²
  k = 9×10⁹ N·m²/C²

전기력선 규칙:
  • 양전하에서 나와 음전하로 들어감
  • 전기력선의 밀도 ∝ 전기장 세기
  • 전기력선끼리 교차하지 않음
  • 같은 부호 → 척력, 다른 부호 → 인력
```

  2. "같은 부호의 두 전하 사이에서 전기장이 약한 영역이 생기는 까닭은 무엇인가요?"

- 실생활 예시: 정전기(풍선-머리카락), 번개(전하 방전), 복사기(정전기 활용)

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 배치 전환 시 이전 전기력선 dispose() + 새로 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 전기력선 최대 32개 × 50 세그먼트
- 파일명: phys_14_electric_field_lines.html
