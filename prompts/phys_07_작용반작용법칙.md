너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 작용 반작용과 운동량 보존
- 토픽: 작용 반작용 법칙 (뉴턴 제3법칙)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 물체 A가 물체 B에 힘을 작용하면 물체 B도 물체 A에 크기가 같고 방향이 반대인 힘을 작용한다. 이를 작용 반작용 법칙 또는 뉴턴 제3법칙이라고 한다. 작용과 반작용은 항상 크기가 같고 방향이 반대이며, 동시에 작용하고, 서로 다른 물체에 작용한다.

## 씬 구성
- 화면 중앙: 3D 두 물체 사이의 상호작용 시각화
- 4가지 사례를 탭으로 전환
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 사례 1: 손으로 벽 밀기 (기본)

### 3D
- 벽: BoxGeometry(0.3, 3, 3), 0x888888 (회색 콘크리트)
- 사람 손: BoxGeometry 조합 (팔+손바닥), 0xddccbb (피부색)
- 힘 화살표:
  - **작용**: 손 → 벽 방향 (빨강 F₁, "작용"), 크기 조절 가능
  - **반작용**: 벽 → 손 방향 (파랑 F₂, "반작용"), 항상 F₁과 크기 같고 방향 반대
- 슬라이더로 미는 힘 조절: F₁이 변하면 F₂도 동시에 같은 크기로 변화
- 힘 수치 표시: "F₁ = [N]N → / F₂ = [N]N ←"
- **핵심 강조**: 두 힘이 서로 다른 물체에 작용하므로 상쇄되지 않음

## 사례 2: 두 사람이 밀기 (스케이트)

### 3D
- 빙판: PlaneGeometry(15, 8), 0xccddee (연한 파랑, 빙판)
- 사람 A: BoxGeometry 조합 실루엣, 0xff6644, 질량 60kg
- 사람 B: BoxGeometry 조합 실루엣, 0x4466ff, 질량 80kg
- A가 B를 밀면:
  - A → B 힘 (빨강 화살표): 작용
  - B → A 힘 (파랑 화살표): 반작용 (크기 동일)
  - A는 뒤로 밀려남 (가벼우므로 더 많이), B는 앞으로 밀려남 (무거우므로 적게)
  - a_A = F/m_A, a_B = F/m_B → a_A > a_B (질량 차이)
- 애니메이션: 밀기 → 두 사람 반대 방향으로 이동 (속도 다름)

## 사례 3: 로켓 추진

### 3D
- 로켓: CylinderGeometry(0.5, 0.3, 3) + ConeGeometry(0.5, 1) 상단, 0xdddddd (은색)
- 연소 가스: 파티클(Points 200개, 0xff6600 주황-빨강) 하방 분출
- 힘 화살표:
  - 로켓 → 가스 (아래 방향, 빨강): 작용 (가스를 밀어냄)
  - 가스 → 로켓 (위 방향, 파랑): 반작용 (로켓을 밀어올림) = 추진력
- 연소 버튼: 가스 분출 + 로켓 상승 애니메이션
- 가스 분출량(힘) 슬라이더 → 로켓 가속도 변화

## 사례 4: 힘 센서 실험 (교과서 탐구)

### 3D
- 두 개의 힘 센서: BoxGeometry(0.4, 0.2, 0.2) × 2, 0x44aa44
  - 센서 A: 디지털 표시 "F₁ = [값]N"
  - 센서 B: 디지털 표시 "F₂ = [값]N"
- 두 센서가 서로 맞대고 있음
- 한쪽을 밀면 양쪽 센서에 같은 크기의 힘 표시
- **항상 F₁ = F₂** 확인 → "작용 = 반작용" 실증
- 슬라이더로 미는 힘 조절

## 작용·반작용 vs 힘의 평형 비교 (하단 패널)
| 구분 | 작용·반작용 | 힘의 평형 |
|------|-----------|---------|
| 작용 대상 | 서로 다른 두 물체 | 같은 하나의 물체 |
| 크기 | 항상 같음 | 같음 |
| 방향 | 반대 | 반대 |
| 상쇄 여부 | 상쇄 안 됨 (다른 물체) | 상쇄됨 (같은 물체) |

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "사례" 드롭다운: 벽 밀기 / 스케이트 밀기 / 로켓 추진 / 힘 센서 실험
- "힘의 크기" 슬라이더: 1~20N (기본값 10)
- "사람 A 질량" 슬라이더 (사례 2): 40~100kg
- "사람 B 질량" 슬라이더 (사례 2): 40~100kg
- "가스 분출량" 슬라이더 (사례 3): 1~10
- "시뮬레이션 시작" 버튼
- "힘 화살표 표시" 토글: ON/OFF (기본값 ON)
- "비교 표 표시" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
↔️ 작용 반작용 법칙 (뉴턴 제3법칙)
━━━━━━━━━━━━━━━━━━━
현재 사례: [사례명]

작용: F₁ = [값]N ([방향])
반작용: F₂ = [값]N ([방향])
|F₁| = |F₂| ✅

뉴턴 제3법칙:
  물체 A가 B에 힘을 작용하면
  B도 A에 크기가 같고 방향이 반대인
  힘을 작용한다.

특징:
  • 크기 같고 방향 반대
  • 동시에 작용
  • 서로 다른 물체에 작용 → 상쇄 안 됨
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물체 A가 물체 B에 힘을 작용하면, B도 A에 크기가 같고 방향이 반대인 힘을 작용합니다. 이 두 힘은 서로 다른 물체에 작용하므로 상쇄되지 않습니다."
- 탐구 질문:
  1. "로켓이 우주 공간(진공)에서도 추진할 수 있는 까닭을 작용 반작용 법칙으로 설명해 보세요."
  2. "작용 반작용 관계에 있는 두 힘과 힘의 평형 관계에 있는 두 힘의 차이점은 무엇인가요?"
  3. "스케이트를 탄 두 사람이 서로 밀 때 가벼운 사람이 더 많이 밀려나는 까닭을 설명해 보세요."
- 실생활 예시: 걷기(발→지면/지면→발), 수영(손→물/물→손), 총 반동
- 교과서 연계: "물리학 I편 p38~41 — 작용 반작용 법칙, 로켓 추진"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
- 파일명: phys_07_action_reaction.html
