너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 2. 전기와 자기의 상호작용 > 전자기 유도와 에너지 전달
- 토픽: 상호 유도와 무선 충전
- 대상: 고등학교 (선택과목)
- 핵심 개념: 두 코일 사이에서 일어나는 전자기 유도를 상호 유도라 한다. 한 코일(1차 코일)에 흐르는 전류가 변하면 주위 자기장이 변하고, 이 변화가 인접한 다른 코일(2차 코일)에 유도 기전력을 발생시킨다. 무선 충전기는 상호 유도를 이용해 전기 에너지를 한 코일에서 다른 코일로 전달한다. NFC, 인덕션 레인지, 변압기도 같은 원리이다.

## 씬 구성
- 화면 중앙: 3D 두 코일(1차/2차) + 자기장 시각화
- 3가지 활용 사례: 무선 충전기 / 인덕션 레인지 / 변압기(NFC)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 기본 원리 시각화

### 두 코일
- 1차 코일(송신): TorusGeometry(1.5, 0.06, 8, 64) × 10겹, 0xff6644 (주황-빨강)
  - 전원(교류) 연결: 사인파 전류 흐름
  - 전류 파티클(노란 구)이 코일을 따라 이동 (방향이 주기적으로 바뀜 = 교류)
- 2차 코일(수신): TorusGeometry(1.5, 0.06, 8, 64) × 10겹, 0x4466ff (파랑)
  - 부하(LED/전구) 연결
  - 유도 전류에 의해 LED 점등

### 자기장 시각화
- 1차 코일에서 생성되는 자기장: 초록 화살표, 코일 축 방향
- 자기장이 2차 코일을 통과: 자기력선(TubeGeometry 곡선)이 1차→2차 연결
- 1차 전류 변화 → 자기장 변화 → 2차에 유도 전류
- 자기장 강도: 1차 전류 세기에 비례, 코일 간 거리에 반비례

### 에너지 전달 애니메이션
- 1차 코일 → 자기장(초록 파티클 이동) → 2차 코일 → LED 점등
- 에너지 전달 효율: 코일 간 거리/정렬에 따라 변화
  - 거리 가까울수록: 효율 높음 → LED 밝음
  - 거리 멀수록: 효율 낮음 → LED 어두움
  - 코일이 어긋나면: 효율 감소

## 활용 사례 1: 무선 충전기

### 3D
- 충전기 패드: CylinderGeometry(2, 2, 0.3), 0x333333 (검정)
  - 내부에 1차 코일(보이게, 반투명)
- 휴대전화: BoxGeometry(0.8, 1.5, 0.1), 0x222244 (어두운 남색)
  - 내부에 2차 코일
  - 화면에 충전 아이콘(⚡) 표시
- 충전 과정:
  1. 충전기에 교류 전류 흐름 (1차 코일)
  2. 자기장 생성 → 2차 코일 통과
  3. 유도 전류 → 배터리 충전
- 휴대전화 올려놓기/들어올리기 → 충전 시작/중단

## 활용 사례 2: 인덕션 레인지

### 3D
- 인덕션: BoxGeometry(3, 0.2, 3), 0x222222 (검정 상판)
  - 내부 코일(보이게, 반투명)
- 냄비: CylinderGeometry(1, 1, 0.8), 0x888888 (금속)
  - 냄비 바닥이 2차 코일 역할 (금속에 유도 전류 → 열 발생)
- 열 효과: 냄비에서 증기 파티클(흰색) 올라옴
- "불 없이 조리" 라벨

## 활용 사례 3: NFC (근거리 무선 통신)

### 3D
- 휴대전화 2대: BoxGeometry × 2
  - 각각 내부에 코일 형태 안테나
- 가까이 가져가면:
  1. 1차(단말기) → 자기장 → 2차(카드/폰)
  2. 전자기 유도로 정보 전달
  3. "결제 완료 ✅" 표시

## 코일 간 거리/정렬 효과

### 거리 실험
- 코일 간 거리 슬라이더: 0.5~5
- 거리에 따른 자기 결합 계수(0~1) 변화
- LED 밝기 = 결합 계수에 비례
- 거리-효율 그래프 (Canvas 2D)

### 정렬 실험
- 2차 코일의 각도를 틸트 슬라이더로 조절 (0°=정렬 ~ 90°=수직)
- 정렬이 벗어나면 효율 감소

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "활용 사례" 드롭다운: 기본 원리 / 무선 충전기 / 인덕션 레인지 / NFC
- "1차 전류 세기" 슬라이더: 0.5~5A (기본값 2)
- "교류 주파수" 슬라이더: 1~10Hz (시각화용, 기본값 3)
- "코일 간 거리" 슬라이더: 0.5~5 (기본값 1)
- "2차 코일 각도" 슬라이더: 0~90° (기본값 0)
- "자기장 표시" 토글: ON/OFF (기본값 ON)
- "에너지 흐름 표시" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📡 상호 유도와 무선 충전
━━━━━━━━━━━━━━━━━
활용: [기본/무선충전/인덕션/NFC]
1차 전류: [값]A (교류 [값]Hz)
코일 간 거리: [값]
2차 유도 전류: [값]A
전달 효율: [값]%

상호 유도:
  1차 코일 전류 변화
  → 자기장 변화
  → 2차 코일에 유도 기전력
  → 유도 전류 → 에너지 전달

활용:
  무선 충전기, 인덕션 레인지,
  NFC, 변압기, 전기 기타 픽업
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "두 코일 사이에서 일어나는 전자기 유도를 상호 유도라 합니다. 한 코일의 전류가 변하면 자기장이 변하고, 이웃한 코일에 유도 기전력이 발생합니다. 무선 충전기, 인덕션 레인지, NFC 등이 이 원리를 활용합니다."
- 탐구 질문:
  1. "무선 충전기에서 에너지가 전달되는 과정을 상호 유도로 설명해 보세요."
  2. "인덕션 레인지 위에 유리 그릇을 올리면 가열되지 않는 까닭은 무엇인가요? (힌트: 유도 전류)"
  3. "코일 간 거리가 멀어지면 무선 충전 효율이 감소하는 까닭은 무엇인가요?"
- 실생활 예시: 스마트폰 무선 충전, 인덕션 레인지, 교통 카드(NFC), 전기 기타 픽업, 금속 탐지기
- 교과서 연계: "물리학 II편 p118~123 — 상호 유도, 무선 충전, 전자기 유도의 활용"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 활용 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 100개
- 파일명: phys_23_mutual_induction_wireless.html
