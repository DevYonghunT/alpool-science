너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 04. 전자기 유도의 활용
- 토픽: 전자기파의 발생과 수신
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 변하는 전기장은 자기장을 만들고, 변하는 자기장은 전기장을 만든다. 이 상호 작용으로 전기장과 자기장이 수직으로 진동하면서 빛의 속력으로 전파되는 것이 전자기파이다. 전자기파의 스펙트럼은 라디오파, 마이크로파, 적외선, 가시광선, 자외선, X선, 감마선으로 분류된다. LC 회로의 공진 주파수를 조절하여 특정 전자기파를 수신한다.

## 씬 구성
- 화면 좌측 50%: 3D 전자기파 전파 시각화 (E장/B장 수직 진동)
- 화면 우측 50%: 전자기파 스펙트럼 + LC 회로 (Canvas 2D / HTML)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 전자기파 전파

### 전기장 (E)
- 수직 방향(y축) 진동: 빨간 사인파 곡선 (TubeGeometry)
- 화살표 격자: 진행 방향(x축)을 따라 일정 간격으로 빨간 화살표(ArrowHelper)
- 각 화살표의 길이 = Esin(kx-ωt) → 실시간 진동

### 자기장 (B)
- 수평 방향(z축) 진동: 파란 사인파 곡선
- 화살표 격자: 파란 화살표, E와 수직
- 각 화살표의 길이 = Bsin(kx-ωt) → E와 동위상

### 전파 방향
- x축 방향: 초록 화살표 "전파 방향 c"
- **E ⊥ B ⊥ c** (세 방향이 모두 수직)
- 전자기파가 x축을 따라 이동하는 애니메이션
- c = 3×10⁸ m/s (빛의 속력)

### 발생원 (안테나)
- 쌍극 안테나: CylinderGeometry(0.05, 0.05, 2) 수직, 0xcc6633
- 교류 전류: 안테나에서 전자가 위아래로 진동 (노란 파티클)
- 진동하는 전하 → 변하는 전기장 → 변하는 자기장 → 전자기파 발생
- 안테나에서 전자기파가 퍼져나가는 애니메이션

## 전자기파 스펙트럼 (우측)
- 수평 막대: 파장/진동수에 따라 7가지 영역 (Canvas 2D)
  - 라디오파 (λ > 1m, f < 3×10⁸Hz): 0xff4444
  - 마이크로파 (λ: 1mm~1m): 0xff8844
  - 적외선 (λ: 700nm~1mm): 0xffcc44
  - 가시광선 (λ: 400~700nm): 무지개 그라데이션
  - 자외선 (λ: 10~400nm): 0x8844ff
  - X선 (λ: 0.01~10nm): 0x4488ff
  - 감마선 (λ < 0.01nm): 0x44ffff
- 각 영역 클릭 시 활용 예시 표시
- 진동수 슬라이더 → 스펙트럼 위 마커 이동 + 해당 영역 강조
- "파장 짧을수록 진동수 높고 에너지 큼" 관계

## LC 회로와 동조 (라디오 수신)
- LC 회로: 코일(L, TorusGeometry) + 축전기(C, 2개 평행판)
- 공진 주파수: f₀ = 1/(2π√LC)
- L 또는 C를 조절하면 f₀ 변화 → 특정 방송 주파수에 동조
- "라디오 다이얼을 돌리면 축전기 C가 변하여 수신 주파수 변경" 라벨
- 여러 방송 주파수(빨/초/파 전자기파)가 날아오는 시각화 → C 조절로 하나만 선택

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 전자기파 전파 / 스펙트럼 / LC 수신
- "파장 λ" 슬라이더: 0.01nm~10m (로그 스케일)
- "진동수 f" 슬라이더: 연동 (f=c/λ)
- "E/B 벡터 표시" 토글: ON/OFF (기본값 ON)
- "안테나 표시" 토글: ON/OFF
- "LC 용량 C" 슬라이더 (LC 모드): 1~100pF
- "LC 인덕턴스 L" 슬라이더: 1~100μH
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📡 전자기파의 발생과 수신
━━━━━━━━━━━━━━━━━
보기: [전파/스펙트럼/LC]

전자기파:
  E(전기장) ⊥ B(자기장) ⊥ 전파 방향
  c = fλ = 3×10⁸ m/s
  현재: f=[값]Hz, λ=[값]m
  영역: [라디오파/가시광선/X선/...]

E = hf (광자 에너지)
  파장↓ → 진동수↑ → 에너지↑

LC 회로 공진:
  f₀ = 1/(2π√LC) = [값]Hz
  L=[값]H, C=[값]F
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 보기 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, E/B 화살표 최대 30개씩
- 파일명: emq_07_electromagnetic_wave.html
