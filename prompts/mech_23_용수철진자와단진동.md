너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 01. 용수철 진자와 단진동
- 토픽: 용수철 진자와 단진동
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 평형 위치를 중심으로 주기적으로 왕복하는 운동을 단진동(SHM)이라 한다. 용수철에 매달린 물체의 운동이 대표적이다. 변위 x = Asin(ωt+φ), 속도 v = Aωcos(ωt+φ), 가속도 a = -ω²x = -Aω²sin(ωt+φ). 가속도는 항상 변위와 반대 방향(복원력 방향)이며 크기는 변위에 비례한다. 주기 T = 2π√(m/k), 각진동수 ω = √(k/m).

## 씬 구성
- 화면 좌측 50%: 3D 용수철 진자 (수직/수평)
- 화면 우측 50%: x-t, v-t, a-t 그래프 (Canvas 2D, 동시 표시)

## 3D 용수철 진자

### 수직 용수철 진자 (기본)
- 고정점: BoxGeometry(2, 0.2, 0.5), 0x888888, 상단 고정
- 용수철: 나선형 곡선 (TubeGeometry로 헬릭스 경로 생성)
  - 반지름 0.3, 감은 수 10~15, 색상 0xcc6633 (구리)
  - 물체 위치에 따라 용수철 길이(늘어남/줄어듦) 실시간 변화
- 물체: SphereGeometry(r=0.3), 0xff4444 (빨강), 용수철 끝에 매달림
  - "m = [값]kg" 라벨
- 평형 위치: 수평 점선 + "x = 0" 라벨
- 최대 변위: "x = +A" / "x = -A" 라벨

### 운동 시각화
- 물체가 상하로 단진동: x(t) = Asin(ωt)
- 속도 벡터: 초록 화살표, 물체에 접선 방향 (수직)
  - 평형 위치에서 최대, 최대 변위에서 0
- 가속도 벡터: 빨간 화살표, 항상 평형 위치 방향 (복원력 방향)
  - 최대 변위에서 최대, 평형 위치에서 0
- 복원력: F = -kx (훅의 법칙), 파란 화살표
- **핵심**: 가속도 a와 변위 x가 항상 반대 방향!

### 에너지 시각화
- 운동 에너지 Ek = ½mv²: 빨간 막대
- 탄성 위치 에너지 Ep = ½kx²: 파란 막대
- 총 에너지 E = Ek + Ep = ½kA² = 일정: 초록 수평선
- 평형 위치: Ek 최대, Ep = 0
- 최대 변위: Ek = 0, Ep 최대

## 수평 용수철 진자 (선택 모드)
- 마찰 없는 수평면 위에서 용수철에 연결된 물체
- 좌우로 단진동
- 수직과 동일한 물리, 중력 효과 없음

## 실시간 3개 그래프 (우측 Canvas 2D)

### x-t 그래프 (변위)
- 사인 곡선: x = Asin(ωt)
- 진폭 A, 주기 T = 2π/ω 표시
- 색상: 파란색

### v-t 그래프 (속도)
- 코사인 곡선: v = Aωcos(ωt)
- x보다 π/2(90°) 위상 앞섬
- 색상: 초록색

### a-t 그래프 (가속도)
- -사인 곡선: a = -Aω²sin(ωt) = -ω²x
- x와 반대 위상 (180° 차이)
- 색상: 빨간색

### 3개 그래프 동기화
- 세로 점선으로 현재 시각 표시 (3개 그래프에 동일)
- x가 최대일 때 v=0, a가 음의 최대 (복원력 최대)
- x=0일 때 v 최대, a=0

## a-x 그래프 (가속도 vs 변위)
- a = -ω²x → 원점 통과 직선, 기울기 = -ω²
- **기울기 = -ω²** → ω, T 결정 가능
- "단진동 판별: a = -ω²x (가속도 ∝ -변위)" 강조

## 주기와 변수 관계
- T = 2π√(m/k)
- k(용수철 상수) 슬라이더: k↑ → T↓ (빠른 진동)
- m(질량) 슬라이더: m↑ → T↑ (느린 진동)
- A(진폭): **주기에 영향 없음** (등시성)
- 두 진자를 다른 m 또는 k로 동시에 비교하는 모드

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "방향" 드롭다운: 수직 / 수평
- "질량 m" 슬라이더: 0.1~5kg (기본값 1, 0.1 단계)
- "용수철 상수 k" 슬라이더: 1~100N/m (기본값 20, 1 단계)
- "진폭 A" 슬라이더: 0.1~2m (기본값 0.5, 0.1 단계)
- "시뮬레이션 시작" 버튼
- "비교 모드" 토글: 두 진자 동시 (다른 m 또는 k)
- "벡터 표시" 토글: 속도/가속도/복원력 ON/OFF
- "에너지 막대" 토글: ON/OFF (기본값 ON)
- "a-x 그래프" 토글: ON/OFF
- "초기화" 버튼
