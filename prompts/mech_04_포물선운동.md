너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 02. 뉴턴 운동 법칙과 포물선 운동
- 토픽: 포물선 운동 (수평 투사와 비스듬히 던지기)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 포물선 운동은 수평 방향의 등속 운동과 수직 방향의 등가속도 운동(자유 낙하)이 동시에 일어나는 2차원 운동이다. 수평 투사: 초기 수평 속도 v₀로 던지면 x = v₀t, y = ½gt². 비스듬히 던지기: vₓ = v₀cosθ, vᵧ = v₀sinθ - gt. 도달 거리 R = v₀²sin2θ/g (최대: θ = 45°).

## 씬 구성
- 화면 좌측 60%: 3D 포물선 궤적 + 속도 벡터 시각화
- 화면 우측 40%: x-t, y-t, vₓ-t, vᵧ-t 그래프 (Canvas 2D)
- 2가지 모드: 수평 투사 / 비스듬히 던지기
- 배경: 0x88ccdd (야외 하늘)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (측면 시점)
- 조명: AmbientLight(0.5) + DirectionalLight(0.7)

## 3D 포물선 환경

### 지면
- PlaneGeometry(30, 20), 0x66aa44 (잔디)
- 격자 눈금: 1m 간격 표시

### 발사대
- 수평 투사: 절벽/테이블 끝 (BoxGeometry, 높이 h, 0x886644)
- 비스듬히 던지기: 바닥에 포신/발사기 (CylinderGeometry 기울어짐, 0x555555)
- 발사 각도 θ: 발사기 기울기로 시각적 표현

### 투사체
- SphereGeometry(r=0.2), 0xff4444 (빨간 공)
- 발사 후 포물선 궤적을 따라 이동
- 잔상(trail): 일정 간격으로 작은 반투명 구를 남김 → 포물선 궤적 시각화
- 착지 지점에 빨간 원 마커

### 속도 벡터 (핵심 시각화!)
- vₓ (수평 속도): 파란 화살표 → 항상 일정 (등속)
- vᵧ (수직 속도): 빨간 화살표 ↓ 점점 증가 (등가속도)
- v (합성 속도): 초록 화살표 (대각선, vₓ와 vᵧ의 합성)
- 각 화살표가 투사체를 따라다니며 실시간 변화
- vₓ가 일정한 것과 vᵧ가 증가하는 것이 시각적으로 명확

### 수평/수직 운동 분리 시각화
- 투사체 옆에 "수평만" 이동하는 파란 구 (x축에 투영)
- 투사체 옆에 "수직만" 이동하는 빨간 구 (y축에 투영)
- 투사체 = 수평 등속 + 수직 자유 낙하의 합성

## 모드 1: 수평 투사 (절벽에서 수평으로 던지기)

### 물리
- 초기: 높이 h에서 수평 속도 v₀로 발사
- 수평: x = v₀t (등속)
- 수직: y = h - ½gt² (자유 낙하)
- 낙하 시간: t = √(2h/g)
- 도달 거리: R = v₀√(2h/g)

### 변수 조절
- 초기 수평 속도 v₀: 슬라이더 1~20m/s
- 절벽 높이 h: 슬라이더 2~20m

## 모드 2: 비스듬히 던지기

### 물리
- 초기: 지면에서 속도 v₀, 각도 θ로 발사
- 수평: x = v₀cosθ·t
- 수직: y = v₀sinθ·t - ½gt²
- 최고점 높이: H = v₀²sin²θ/(2g)
- 체공 시간: T = 2v₀sinθ/g
- 수평 도달 거리: R = v₀²sin2θ/g
- **최대 거리: θ = 45°** ← 강조

### 각도별 비교
- "각도 비교" 버튼: 15°, 30°, 45°, 60°, 75° 5개 궤적을 동시에 표시 (서로 다른 색상)
- 45°가 가장 먼 거리 도달 확인
- 보각 관계: θ와 (90°-θ)는 같은 거리 (예: 30°와 60°)

## 실시간 그래프 (우측 Canvas 2D)
- 4개 그래프 (탭으로 전환):
  - x-t: 직선 (수평 등속)
  - y-t: 포물선 (수직 등가속도)
  - vₓ-t: 수평선 (일정)
  - vᵧ-t: 직선 감소 (등가속도, 최고점에서 0 통과)
- 현재 시각에 세로 점선 마커

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 수평 투사 / 비스듬히 던지기
- "초기 속도 v₀" 슬라이더: 1~30m/s (기본값 15, 1 단계)
- "발사 각도 θ" 슬라이더 (모드 2): 5~85° (기본값 45°, 5 단계)
- "절벽 높이 h" 슬라이더 (모드 1): 2~20m (기본값 10)
- "발사" 버튼
- "각도 비교" 버튼 (모드 2): 5개 각도 동시 발사
- "속도 벡터 표시" 토글: ON/OFF (기본값 ON)
- "수평/수직 분리" 토글: ON/OFF (기본값 OFF)
- "궤적 표시" 토글: ON/OFF (기본값 ON)
- "그래프" 드롭다운: x-t / y-t / vₓ-t / vᵧ-t
- "초기화" 버튼

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "포물선 운동은 수평 방향 등속 운동과 수직 방향 등가속도 운동(자유 낙하)이 동시에 일어나는 2차원 운동입니다. 비스듬히 던질 때 도달 거리는 발사 각도 45°에서 최대입니다."
- 탐구 질문:
  1. "수평 투사에서 초기 수평 속도를 2배로 늘리면 도달 거리는 어떻게 변하나요?"
  2. "비스듬히 던질 때 30°와 60°에서 도달 거리가 같은 까닭을 수식으로 설명해 보세요."
  3. "포물선 운동하는 물체의 최고점에서 속도는 0인가요? (힌트: 수평 속도)"
- 실생활 예시: 농구 슛, 포탄 발사, 분수대 물줄기, 스키 점프, 야구 투구
- 교과서 연계: "역학과 에너지 I편 p16~25 — 뉴턴 운동 법칙과 포물선 운동"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 발사 시 이전 궤적 dispose() + 새 궤적 생성 (또는 잔상 유지 모드)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 잔상 최대 200개
- 파일명: mech_04_projectile_motion.html
