너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 03. 열역학 제1법칙
- 토픽: 열역학 제1법칙 (Q = ΔU + W)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 열역학 제1법칙은 에너지 보존 법칙의 열역학적 표현이다. 계에 가해진 열 Q는 계의 내부 에너지 변화량 ΔU와 계가 외부에 한 일 W의 합이다. Q = ΔU + W. 내부 에너지는 기체 분자들의 운동 에너지 총합이며, 온도에 비례한다. 일 W = PΔV (등압 과정에서)이며 P-V 그래프 아래 면적과 같다.

## 씬 구성
- 화면 좌측: 3D 실린더+피스톤+기체 + 에너지 흐름 화살표
- 화면 우측: 에너지 막대그래프 (Q, ΔU, W) + P-V 다이어그램
- 4가지 과정에서 Q, ΔU, W 비교
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 실린더 + 에너지 흐름

### 실린더+피스톤 (mech_17과 유사)
- 실린더: 투명 CylinderGeometry
- 피스톤: CylinderGeometry, 이동 가능
- 기체 분자: SphereGeometry × 30개, 랜덤 이동

### 에너지 흐름 화살표 (핵심 시각화!)
- **Q (열)**: 빨간 화살표, 외부 → 계 (또는 계 → 외부)
  - 히터에서 실린더로 열이 들어옴: 빨간 파티클 유입
  - 또는 열이 빠져나감: 빨간 파티클 유출
- **ΔU (내부 에너지 변화)**: 노란 화살표, 기체 내부
  - 분자 속도 변화로 시각화 (빨라지면 ΔU > 0, 느려지면 ΔU < 0)
- **W (일)**: 초록 화살표, 피스톤 이동 방향
  - 피스톤이 올라감 (팽창): W > 0 (기체가 외부에 일)
  - 피스톤이 내려감 (압축): W < 0 (외부가 기체에 일)
- **Q = ΔU + W** 관계를 화살표 크기로 시각적 확인

## 4가지 과정별 Q, ΔU, W

### 등온 과정 (T 일정 → ΔU = 0)
- Q = W (열이 모두 일로 전환)
- ΔU = 0 (온도 변화 없으므로)
- 분자 속도 변화 없음, 피스톤만 이동
- 막대그래프: Q = W, ΔU = 0

### 등적 과정 (V 일정 → W = 0)
- Q = ΔU (열이 모두 내부 에너지로)
- W = 0 (부피 변화 없으므로)
- 피스톤 고정, 분자 속도 증가 (온도↑)
- 막대그래프: Q = ΔU, W = 0

### 등압 과정 (P 일정)
- Q = ΔU + W = ΔU + PΔV
- 열의 일부는 내부 에너지로, 일부는 일로
- 피스톤이 올라감 + 분자 속도 증가
- 막대그래프: Q > ΔU > 0, W > 0

### 단열 과정 (Q = 0)
- 0 = ΔU + W → ΔU = -W
- 압축: W < 0, ΔU > 0 (온도↑)
- 팽창: W > 0, ΔU < 0 (온도↓)
- 막대그래프: Q = 0, ΔU = -W

## 에너지 막대그래프 (우측 상단)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "과정" 드롭다운: 등온 / 등적 / 등압 / 단열
- "열 Q 입력" 슬라이더: -500~500J (기본값 200)
- "과정 실행" 버튼
- "에너지 흐름 화살표" 토글: ON/OFF (기본값 ON)
- "막대그래프" 토글: ON/OFF (기본값 ON)
- "P-V 다이어그램" 토글: ON/OFF (기본값 ON)
- "분자 표시" 토글: ON/OFF
- "초기화" 버튼
