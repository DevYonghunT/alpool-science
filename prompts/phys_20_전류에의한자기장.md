너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 2. 전기와 자기의 상호작용 > 전류에 의한 자기장과 에너지 전환
- 토픽: 직선 전류, 원형 전류, 솔레노이드에 의한 자기장
- 대상: 고등학교 (선택과목)
- 핵심 개념: 전류가 흐르는 도선 주위에는 자기장이 형성된다. 직선 전류에 의한 자기장은 동심원 형태이며 오른손 법칙으로 방향을 결정한다. 원형 전류 중심의 자기장은 코일 면에 수직이다. 솔레노이드(긴 코일) 내부에는 균일한 자기장이 형성되며, 전류 세기와 단위 길이당 감은 수에 비례한다.

## 씬 구성
- 3가지 모드: 직선 전류 / 원형 전류 / 솔레노이드
- 각 모드에서 전류 방향, 자기장 방향, 자기력선을 3D로 시각화
- 오른손 법칙 안내 포함
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(0.5)

## 모드 1: 직선 전류에 의한 자기장

### 3D
- 직선 도선: CylinderGeometry(0.08, 0.08, 10), 0xcc6633 (구리색), 수직 배치
- 전류 방향: 노란 화살표 파티클이 도선을 따라 이동 (위→아래 또는 아래→위)
- 전류 방향 표시: "I" + 화살표

### 자기장 시각화
- 동심원 형태 자기력선: TorusGeometry(r, 0.02, 8, 64) × 5~8개 (반지름 다르게)
  - 반지름 r = 1, 2, 3, 4, 5
  - 색상: 0x44ff88 (초록) 또는 세기에 따라 그라데이션
  - 방향: 오른손 법칙 — 엄지=전류방향, 나머지 손가락=자기장 방향
- 자기장 방향 화살표: 각 원 위에 작은 화살촉 (ConeGeometry)
- 자기장 세기: 도선에 가까울수록 강함 (원의 선 두께/색상으로 표현)
  - B ∝ I/r (직선 전류)

### 오른손 법칙 안내
- 3D 손 모형(간략): 엄지 위(전류방향), 나머지 감아쥠(자기장방향)
- 또는 오른손 아이콘 + 설명 텍스트

## 모드 2: 원형 전류에 의한 자기장

### 3D
- 원형 코일: TorusGeometry(2, 0.08, 8, 64), 0xcc6633
- 전류 방향: 코일을 따라 노란 파티클 이동
- 코일 면에 수직 방향으로 자기장 (빨간 화살표, 코일 중심에서 위로)

### 자기장 시각화
- 코일 중심: 강한 자기장 (큰 화살표)
- 코일 외부: 약한 자기장
- 자기력선: TubeGeometry 곡선, 코일 중심에서 나와 바깥으로 돌아옴 (막대 자석과 유사)
- 오른손 법칙: 손가락=전류방향으로 감아쥐면, 엄지=자기장방향(코일 면에 수직)

## 모드 3: 솔레노이드 (긴 코일)

### 3D
- 솔레노이드: 여러 원형 코일(TorusGeometry) 나란히 배치 (10~20개)
  - 코일 간격 조절 가능
  - 전체 길이 L, 감은 수 N
- 전류 방향: 코일을 따라 노란 파티클 이동

### 자기장 시각화
- **내부**: 균일한 자기장 — 평행한 화살표 격자 (초록, 모두 같은 방향·크기)
  - B = μ₀nI (n=N/L: 단위 길이당 감은 수, I: 전류)
- **외부**: 막대 자석과 유사한 패턴 — TubeGeometry 곡선 자기력선
- N극/S극 표시: 전류 방향에 따라 오른손 법칙으로 결정
- "솔레노이드 내부 자기장은 균일하다" 강조 라벨

### 전자석 응용
- 솔레노이드 내부에 철심(강자성체) 삽입 → 자기장 크게 증가
- "전자석" 라벨, 철심 토글

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 직선 전류 / 원형 전류 / 솔레노이드
- "전류 세기 I" 슬라이더: 0.5~5A (기본값 2, 0.5 단계)
- "전류 방향" 토글: 정방향 / 역방향
- "코일 감은 수 N" 슬라이더 (솔레노이드): 5~30 (기본값 15)
- "철심 삽입" 토글 (솔레노이드): ON/OFF
- "자기력선 표시" 토글: ON/OFF (기본값 ON)
- "오른손 법칙 안내" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧲 전류에 의한 자기장
━━━━━━━━━━━━━━━
모드: [직선/원형/솔레노이드]
전류 I = [값]A
방향: [정방향/역방향]

직선 전류: B = μ₀I/(2πr)
  동심원 형태, 오른손 법칙

원형 전류: B = μ₀I/(2R) (중심)
  코일 면에 수직

솔레노이드: B = μ₀nI
  n = N/L = [값]/[값] = [값]
  내부: 균일한 자기장
  외부: 막대 자석과 유사

오른손 법칙:
  엄지 = 전류 방향
  나머지 = 자기장 방향 (감아쥠)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "전류가 흐르는 도선 주위에는 자기장이 형성됩니다. 직선 전류는 동심원 형태, 원형 전류는 코일 면에 수직, 솔레노이드는 내부에 균일한 자기장을 만듭니다. 자기장 방향은 오른손 법칙으로 결정합니다."
- 탐구 질문:
  1. "직선 전류 주위 자기장의 방향을 오른손 법칙으로 어떻게 결정하나요?"
  2. "솔레노이드 내부의 자기장이 균일한 것이 실생활에서 어떻게 유용한가요?"
  3. "전자석의 세기를 강하게 하려면 어떻게 해야 하나요? (힌트: 전류, 감은 수, 철심)"
- 실생활 예시: 전자석(고철 분리), 전자 잠금장치, MRI(강력한 솔레노이드), 전자 벨
- 교과서 연계: "물리학 II편 p104~109 — 직선/원형/솔레노이드 전류의 자기장, 전자석"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 100개
- 파일명: phys_20_current_magnetic_field.html
