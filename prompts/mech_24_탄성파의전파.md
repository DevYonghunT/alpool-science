너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 02. 탄성파
- 토픽: 탄성파의 전파 (종파와 횡파)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 탄성파는 매질의 탄성에 의해 전파되는 파동이다. 횡파는 매질의 진동 방향이 파동의 진행 방향에 수직이고, 종파는 매질의 진동 방향이 파동의 진행 방향과 나란하다. 파동의 속력 v = fλ = λ/T (f: 진동수, λ: 파장, T: 주기). 음파는 종파이며, 지진파의 P파는 종파, S파는 횡파이다.

## 씬 구성
- 화면 상단 50%: 3D 횡파 시각화 (용수철/입자 모형)
- 화면 하단 50%: 3D 종파 시각화 (용수철/입자 모형)
- 또는 탭으로 전환
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 횡파 시각화

### 3D 입자 모형
- 입자 30개: SphereGeometry(r=0.12) × 30, 0x4488ff (파랑), x축 방향으로 등간격 배열
- 각 입자가 y축 방향(수직)으로 진동: y_i = Asin(kx_i - ωt)
  - k = 2π/λ (파수), ω = 2πf (각진동수)
- 파동 진행 방향: x축 (오른쪽 →)
- 진동 방향: y축 (위아래 ↑↓)
- **진행 방향 ⊥ 진동 방향** = 횡파

### 시각화 요소
- 입자 사이 연결선: CylinderGeometry 또는 Line, 0xaaaacc (용수철 비유)
- 파형 곡선: 입자들의 y좌표를 이은 사인 곡선 (Line, 0x44ff88)
- 파장 λ: 양방향 화살표로 표시 (한 파장 길이)
- 진폭 A: 양방향 화살표 (최대 변위)
- 한 입자의 진동 궤적: 수직 양방향 화살표 (빨강)
- 파동 진행 방향 화살표: 큰 수평 화살표 (초록)

### 입자의 운동 강조
- 특정 입자 1개를 빨간색으로 강조
- 이 입자의 위치를 추적: y-t 그래프에 실시간 표시
- "입자는 y방향으로만 진동, 파동은 x방향으로 전파" 라벨

## 종파 시각화

### 3D 입자 모형
- 입자 30개: SphereGeometry(r=0.12) × 30, 0xff6644 (주황), x축 방향 배열
- 각 입자가 x축 방향(수평)으로 진동: x_i = x₀_i + Asin(kx₀_i - ωt)
  - 진동 방향 = 진행 방향 (나란)
- **진행 방향 ∥ 진동 방향** = 종파

### 시각화 요소
- 밀한 부분(compression): 입자가 모여있는 영역 → 밀도 높음, 압력 높음
  - 배경을 연한 빨강으로 표시
- 소한 부분(rarefaction): 입자가 떨어져있는 영역 → 밀도 낮음, 압력 낮음
  - 배경을 연한 파랑으로 표시
- 파장 λ: 밀한 부분~다음 밀한 부분 사이 거리
- 특정 입자 강조: x방향 진동 궤적 표시

### 입자의 운동 강조
- "입자는 x방향으로 진동, 파동도 x방향으로 전파" 라벨
- y-t 그래프 대신 x-t 그래프 (입자의 x방향 변위)

## 파동의 물리량
- v = fλ = λ/T
- 진동수 f 슬라이더: 파장 λ가 자동 변화 (v 일정 시)
- 파장 λ 슬라이더: 진동수 f가 자동 변화
- 진폭 A 슬라이더: 진동 크기 변화 (속력/파장 무관)
- 파동 속력 v 슬라이더: 매질에 따라 (v = √(장력/선밀도) 등)

## 실생활 예시
- 횡파: 기타줄 진동, 전자기파(빛), 지진 S파
- 종파: 음파(소리), 용수철 파동, 지진 P파
- 지진파: P파(종파, 빠름) → S파(횡파, 느림) 순서로 도착
  - 지진계 기록: P파 먼저 도착 → S파 나중 도착

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "파동 종류" 드롭다운: 횡파 / 종파 / 비교 (상하 분할)
- "진동수 f" 슬라이더: 0.5~5Hz (기본값 1, 0.5 단계)
- "파장 λ" 슬라이더: 2~10 (기본값 5)
- "진폭 A" 슬라이더: 0.1~1.5 (기본값 0.5)
- "파동 속력 v" 슬라이더: 1~10 (기본값 5)
- "입자 추적" 토글: 특정 입자 강조 ON/OFF
- "파형 곡선" 토글: ON/OFF (기본값 ON)
- "밀/소 표시" 토글 (종파): ON/OFF (기본값 ON)
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌊 탄성파의 전파 (종파와 횡파)
━━━━━━━━━━━━━━━━━
파동: [횡파/종파]

횡파: 진동 방향 ⊥ 진행 방향
  예: 기타줄, 전자기파, S파
종파: 진동 방향 ∥ 진행 방향
  예: 음파, 용수철파, P파

파동 속력: v = fλ = λ/T
  f = [값]Hz, λ = [값]m
  v = [값]m/s
  T = 1/f = [값]s
  A = [값]m (진폭)

v = fλ에서:
  v 일정 시 f↑ → λ↓
```

- 탐구 질문:
  1. "횡파와 종파에서 매질의 입자는 어떤 방향으로 진동하나요?"
  2. "종파에서 밀한 부분과 소한 부분은 무엇을 의미하나요?"
  3. "지진 발생 시 P파가 S파보다 먼저 감지되는 까닭은 무엇인가요?"
- 실생활 예시: 음파(공기 종파), 기타줄(횡파), 지진파(P파/S파), 초음파 검사
- 교과서 연계: "역학과 에너지 III편 p100~107 — 탄성파, 종파와 횡파"
