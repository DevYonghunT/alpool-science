너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 03. 파동 함수와 터널링
- 토픽: 터널 효과와 확률의 세계
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 양자 역학에서 입자는 파동 함수 ψ(x)로 기술되며, |ψ(x)|²은 입자의 위치 확률 밀도를 나타낸다. 고전 역학에서 입자가 넘을 수 없는 퍼텐셜 장벽도 양자 역학에서는 파동 함수가 장벽 내부에서 지수적으로 감소하지만 완전히 0이 되지 않아, 장벽을 투과(터널링)할 확률이 존재한다. 투과 확률은 장벽의 높이, 폭, 입자 에너지에 의존한다. STM(주사 터널링 현미경)은 이 원리를 이용하여 원자 수준의 표면을 관찰한다.

## 씬 구성
- 화면 좌측 60%: 3D 퍼텐셜 장벽 + 파동 함수 시각화
- 화면 우측 40%: 투과 확률 그래프 + STM 원리 다이어그램 (Canvas 2D / HTML)
- 배경: 0x0a0a1a
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 퍼텐셜 장벽과 파동 함수

### 퍼텐셜 장벽
- BoxGeometry(폭 d, 높이 V₀, 깊이 3), MeshStandardMaterial, 0xff4444 opacity 0.3
- 장벽 높이 V₀: 슬라이더로 조절 (1~10 eV)
- 장벽 폭 d: 슬라이더로 조절 (0.5~5 nm)
- 장벽 양쪽: 바닥 평면(PlaneGeometry, 0x222233)으로 퍼텐셜 = 0 영역 표시

### 파동 함수 ψ(x) 시각화
- TubeGeometry 또는 Line으로 파동 함수 그래프를 3D 공간에 표현
- 색상: 0x44aaff (실수부), 0xff44aa (허수부), 0xffdd44 (|ψ|² 확률 밀도)
- **영역 I (장벽 왼쪽):** 입사파 + 반사파 → 진동하는 정현파
  - ψ₁ = Ae^(ikx) + Be^(-ikx)
  - k = √(2mE)/ℏ
- **영역 II (장벽 내부):** 지수 감소
  - ψ₂ = Ce^(-κx) + De^(κx)
  - κ = √(2m(V₀-E))/ℏ
  - 파동 함수가 0이 아님! → 존재 확률 있음
- **영역 III (장벽 오른쪽):** 투과파 → 진폭이 줄어든 정현파
  - ψ₃ = Fe^(ikx)
  - 투과 확률 T = |F/A|²

### 입자 파티클 시각화
- SphereGeometry(r=0.15), 0x44ddff + 글로우 효과
- 입자가 왼쪽에서 오른쪽으로 이동
- 장벽에 도달하면:
  - 확률 T로 투과 → 장벽 통과 후 진행 (초록 플래시)
  - 확률 (1-T)로 반사 → 되돌아감 (빨강 플래시)
- 연속 발사 모드: 여러 입자를 발사하여 통계적 투과/반사 비율 확인
- 투과 카운터 / 반사 카운터 표시

### 에너지 vs 장벽 높이
- 입자 에너지 E < V₀: 고전적으로 불가능하지만 양자적으로 투과 가능!
  - "고전: 통과 불가 ❌ / 양자: 확률적 투과 ✅" 라벨
- 입자 에너지 E > V₀: 고전적으로도 통과, 양자에서는 부분 반사도 있음
- 입자 에너지 E 슬라이더: 0.5~15 eV

## 투과 확률 그래프 (우측 Canvas 2D)

### T vs 장벽 폭 d
- X축: 장벽 폭 d (nm)
- Y축: 투과 확률 T (0~1)
- 곡선: T ≈ e^(-2κd) (지수 감소)
- 현재 설정값에 빨간 마커
- "폭이 넓어지면 투과 확률 급감"

### T vs 입자 에너지 E
- X축: 입자 에너지 E (eV)
- Y축: 투과 확률 T
- E < V₀: T가 0에서 점점 증가
- E = V₀: T ≈ 0.5 부근
- E > V₀: T → 1에 수렴 (진동하며 접근)

## STM(주사 터널링 현미경) 시각화

### 3D STM 원리
- 탐침(tip): ConeGeometry, 0xcccccc metallic, 위에서 아래로 뾰족한 바늘
- 시료 표면: 원자들이 격자 배열 (SphereGeometry r=0.2, 0x88aacc)
  - 원자 높낮이가 다름 → 표면 울퉁불퉁
- 탐침-시료 간격: ~1nm (터널링 거리)
- 전자 터널링 전류: 간격 d에 따라 지수적 변화
  - I ∝ e^(-2κd)
  - 간격 0.1nm 변화 → 전류 10배 변화! → 원자 수준 해상도

### STM 스캔 애니메이션
- 탐침이 시료 위를 x방향으로 스캔
- 일정 전류 모드: 탐침 높이가 원자 위에서 올라가고 사이에서 내려감
- 스캔 결과: 표면 높낮이 맵 → 원자 배열 이미지
- "원자를 직접 볼 수 있다!" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 터널링 / STM
- "장벽 높이 V₀" 슬라이더: 1~10 eV (기본값 5)
- "장벽 폭 d" 슬라이더: 0.5~5 nm (기본값 1)
- "입자 에너지 E" 슬라이더: 0.5~15 eV (기본값 3)
- "파동 함수 표시" 토글: |ψ|² / Re(ψ) / Im(ψ)
- "입자 발사" 버튼: 단일 입자 발사
- "연속 발사" 토글: ON/OFF (자동 발사)
- "STM 스캔" 버튼: 스캔 애니메이션 시작
- "그래프" 드롭다운: T-d / T-E
- "초기화" 버튼
