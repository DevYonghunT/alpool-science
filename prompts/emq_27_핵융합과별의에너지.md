너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 05. 핵반응과 에너지
- 토픽: 핵융합과 별의 에너지
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 핵융합은 가벼운 원자핵들이 합쳐져 더 무거운 원자핵을 만드는 반응으로, 질량 결손에 의해 E=mc²에 따른 막대한 에너지가 방출된다. 태양은 pp체인(양성자-양성자 연쇄 반응)으로 수소를 헬륨으로 융합하여 에너지를 생산한다. 무거운 별에서는 CNO 순환이 주요 에너지원이다. 핵융합은 쿨롱 장벽을 넘어야 하므로 수천만 K 이상의 초고온이 필요하다.

## 씬 구성
- 화면 좌측 60%: 3D 핵융합 반응 시각화
- 화면 우측 40%: 에너지 다이어그램 + 결합 에너지 곡선 (Canvas 2D / HTML)
- 2가지 보기: pp체인 / CNO 순환
- 배경: 0x0a0510 (어두운 우주)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) + PointLight(2.0, 원점, 0xffdd88)

## 보기 1: pp체인 (양성자-양성자 연쇄 반응)

### 3D 핵 입자
- 양성자(p): SphereGeometry(r=0.25), 0xff4444 emissive 0xff2222, "p" 라벨
- 중성자(n): SphereGeometry(r=0.25), 0x4488ff emissive 0x2266ff, "n" 라벨
- 양전자(e⁺): SphereGeometry(r=0.08), 0xffaa44
- 중성미자(ν): SphereGeometry(r=0.06), 0xaaddff opacity 0.5
- 감마선(γ): 작은 파동 효과, 0xffff44
- 헬륨-4(⁴He): 양성자 2개 + 중성자 2개 결합체

### pp체인 단계별 애니메이션
- **단계 1:** p + p → d + e⁺ + νₑ
  - 양성자 2개가 접근 → 쿨롱 반발 표시(빨간 화살표)
  - 충분한 에너지로 접근 → 융합!
  - 중수소(d: p+n 결합체) + 양전자 + 중성미자 방출
  - 양전자 즉시 소멸: e⁺ + e⁻ → 2γ

- **단계 2:** d + p → ³He + γ
  - 중수소 + 양성자 → 헬륨-3(양성자2+중성자1) + 감마선

- **단계 3:** ³He + ³He → ⁴He + 2p
  - 헬륨-3 두 개 → 헬륨-4 + 양성자 2개 방출!
  - 최종: 4p → ⁴He + 2e⁺ + 2νₑ + 에너지(26.7 MeV)

- 단계 전환: "다음 단계" 버튼 또는 자동 재생
- 각 단계에서 질량 결손 Δm과 방출 에너지 E=Δmc² 표시

### 쿨롱 장벽 시각화
- 퍼텐셜 에너지 곡선: 양성자 간 거리 vs 에너지
  - 먼 거리: 쿨롱 반발 ∝ 1/r (양의 에너지 장벽)
  - 가까운 거리: 핵력 인력 (음의 에너지 우물)
- 터널링: 열에너지만으로는 장벽 못 넘지만, 양자 터널링으로 가능!
- 온도 슬라이더: 1000만 K ~ 1억 K → 입자 운동 에너지 변화

### 태양 내부 단면
- 구형 태양: SphereGeometry(r=4), 0xffdd44 반투명
- 내부 핵(core): r=0~0.25R, 0xff6644, 1500만 K, pp체인 발생 영역
- 복사층: 0.25R~0.7R, 0xffaa44
- 대류층: 0.7R~R, 0xffdd88
- "태양 질량의 0.7%가 에너지로 전환" 라벨

## 보기 2: CNO 순환

### 3D CNO 순환 다이어그램
- 원형 배치: ¹²C → ¹³N → ¹³C → ¹⁴N → ¹⁵O → ¹⁵N → ¹²C (순환!)
- 각 핵종: SphereGeometry, 원소별 색상
  - C(탄소): 0x444444
  - N(질소): 0x4488ff
  - O(산소): 0xff4444
- 화살표: 각 반응 사이 (ArrowHelper)
- 입력: p(양성자) 4개 흡수
- 출력: ⁴He + 2e⁺ + 2νₑ + 에너지(25.0 MeV)
- 촉매: ¹²C (반응 전후 불변!)
- "탄소는 촉매 — 순환 후에도 그대로!" 라벨
- "무거운 별(태양 질량 1.3배 이상)에서 지배적" 라벨

## 에너지 다이어그램 (우측)

### 질량-에너지 등가 (E=mc²)
- 반응 전 질량: 4 × m_p = [값] u
- 반응 후 질량: m(⁴He) = [값] u
- 질량 결손: Δm = [값] u = [값] kg
- 방출 에너지: E = Δmc² = 26.7 MeV
- 에너지 막대: 반응 전(높음) → 반응 후(낮음), 차이 = 방출 에너지

### 핵자당 결합 에너지 곡선
- X축: 질량수 A
- Y축: 핵자당 결합 에너지 (MeV)
- 곡선: 수소(0) → He(7.1) → ... → Fe(8.8, 최대) → ... → U(7.6)
- 융합 영역: 가벼운 쪽(왼쪽) → 올라감 → 에너지 방출!
- 분열 영역: 무거운 쪽(오른쪽) → 올라감 → 에너지 방출!
- Fe(철): 정점 → 가장 안정 → 융합도 분열도 에너지 불리
- 현재 반응 위치 하이라이트

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: pp체인 / CNO 순환
- "단계" 슬라이더: 1~3 (pp체인) 또는 1~6 (CNO)
- "자동 재생" 토글: ON/OFF
- "속도" 슬라이더: 0.5x~3x (애니메이션 속도)
- "온도" 슬라이더: 10⁶~10⁸ K (기본값 1.5×10⁷ = 태양 핵)
- "태양 단면" 토글: ON/OFF
- "결합 에너지 곡선" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼
