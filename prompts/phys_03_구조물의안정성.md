너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 평형과 구조물의 안정성
- 토픽: 구조물의 안정성 (무게중심)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 무게중심이란 물체에 작용하는 중력의 작용점으로, 물체의 질량이 모두 그 점에 집중되어 있는 것처럼 행동하는 점이다. 구조물이 안정적으로 서 있으려면 무게중심에서 내린 수직선이 바닥면(지지면) 안에 있어야 한다. 바닥이 넓고 무게중심이 낮을수록 구조물은 더 안정적이다.

## 씬 구성
- 화면 중앙: 3D 구조물(블록/탑) + 무게중심 표시 + 지지면 표시
- 기울기 슬라이더로 구조물을 기울여 안정성을 시험
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 구조물

### 블록 구조물 (기본)
- BoxGeometry (폭/높이/깊이 조절 가능)
- 기본: 폭 2, 높이 4, 깊이 2, MeshStandardMaterial(0x886644 나무색, roughness 0.7)
- 바닥면(지지면): PlaneGeometry, 0x44aa44 반투명 초록 (구조물 아래)
- 바닥 격자: GridHelper(10, 10)

### 무게중심 표시
- SphereGeometry(r=0.15), 0xff4444 (빨간 점)
- 무게중심 위치 = 구조물 기하학적 중심 (균일 밀도 가정)
- 무게중심에서 아래로 수직선: CylinderGeometry 가느다란 빨간 점선
- 수직선이 지지면 안에 있으면: 초록 (안정)
- 수직선이 지지면 밖으로 나가면: 빨강 (불안정 → 넘어짐)

### 중력 화살표
- 무게중심에서 아래 방향 화살표: 0xff4444, "W=mg"

## 안정성 시험

### 기울기 시뮬레이션
- 기울기 슬라이더: 0°~45° (구조물을 한쪽으로 기울임)
- 기울일수록 무게중심의 수직선이 지지면 가장자리로 이동
- 임계각에 도달하면: 수직선이 지지면 밖 → 구조물 넘어지는 애니메이션 (rotation 빠르게 증가)
- 임계각 계산: tan(θ) = (바닥 폭/2) / 무게중심 높이

### 구조물 형태 비교 (3가지 프리셋)
1. **좁고 높은 구조물**: 폭 1, 높이 6 → 쉽게 넘어짐 (작은 임계각)
2. **넓고 낮은 구조물**: 폭 4, 높이 2 → 안정적 (큰 임계각)
3. **피라미드 형태**: 아래 넓고 위 좁음 → 매우 안정적

### 무게중심 위치 변경
- 구조물 상단/하단에 추가 블록(질량)을 붙여 무게중심 위치 변경
- 상단에 질량 추가: 무게중심 올라감 → 불안정해짐
- 하단에 질량 추가: 무게중심 내려감 → 안정적

## 실생활 구조물 예시 (교과서 그림 I-7~9)

### 에펠탑 미니어처
- BoxGeometry 조합으로 간단한 에펠탑 형태 (아래 넓고 위 좁음)
- "넓은 바닥 + 낮은 무게중심 = 안정적"

### 오뚝이
- SphereGeometry 반구(하단, 무거움) + ConeGeometry(상단, 가벼움)
- 기울여도 스스로 복원되는 애니메이션
- "무게중심이 매우 낮아 안정적"

### 탑/빌딩
- BoxGeometry 높은 직사각형
- 바람(수평 힘) 적용 시 기울어지는 정도 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "구조물 형태" 드롭다운: 직육면체 / 피라미드 / 에펠탑 / 오뚝이 / 탑
- "폭" 슬라이더: 0.5~6 (기본값 2)
- "높이" 슬라이더: 1~8 (기본값 4)
- "기울기" 슬라이더: 0~45° (기본값 0°)
- "상단 질량 추가" 슬라이더: 0~5kg (기본값 0)
- "하단 질량 추가" 슬라이더: 0~5kg (기본값 0)
- "무게중심 표시" 토글: ON/OFF (기본값 ON)
- "수직선 표시" 토글: ON/OFF (기본값 ON)
- "지지면 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널

```

구조물: [형태]
폭: [값]m / 높이: [값]m
무게중심 높이: [값]m
기울기: [값]°
임계각: [값]°
상태: [안정 / 불안정 / 넘어짐]

안정 조건:
  무게중심의 수직선이
  지지면(바닥면) 안에 있어야 함

안정적 구조물:
  • 바닥이 넓을수록 안정
  • 무게중심이 낮을수록 안정
  임계각 θ: tan(θ) = (바닥폭/2) / 무게중심높이
```

화면 상단에 토글 가능한 "📖 학습 가이드" 패널:

- 탐구 질문:
  1. "버스에서 서 있을 때 발을 넓게 벌리면 더 안정적인 까닭을 지지면과 무게중심으로 설명해 보세요."
  2. "피라미드 형태의 구조물이 직육면체보다 안정적인 까닭은 무엇인가요?"
  3. "트럭이 짐을 싣을 때 무거운 짐을 아래에 싣는 것이 좋은 까닭을 설명해 보세요."
- 실생활 예시: 에펠탑 구조, 피라미드, 오뚝이 장난감, 대형 선박의 평형수, 자동차 무게중심
- 교과서 연계: "물리학 I편 p18~21 — 무게중심과 안정성, 구조물의 안정성 (그림 I-7~9)"
