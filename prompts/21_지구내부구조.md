너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 1. 지구시스템 > 지구시스템의 구성과 상호작용
- 토픽: 지구 내부 구조
- 대상: 고등학교 1학년
- 핵심 개념: 지권은 지각, 맨틀, 외핵, 내핵으로 구분한다. 지각은 대륙 지각과 해양 지각으로 나뉜다. 맨틀은 지구 전체 부피의 대부분을 차지하며 고체이지만 유동성이 있다. 핵은 대부분 철과 니켈로 이루어져 있으며 액체 상태의 외핵과 고체 상태의 내핵으로 구분한다.

## 씬 구성
- 화면 중앙에 3D 지구 단면도를 렌더링한다.
- ClippingPlane을 사용하여 지구를 1/4 절단 (쐐기 형태로 내부 노출)
- 배경: 0x000011 (우주)
- 별 파티클: Points(800개)
- 조명: AmbientLight(0.25) + DirectionalLight(0.6, 좌상단) + PointLight(내핵 위치, 0xff4400, 세기 2.0)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls

## 지구 내부 구조 3D 상세 (교과서 그림 III-3, p106)

### 4개 층 (동심원 SphereGeometry)
| 층 | 실제 깊이 | 3D 반지름 | 색상 | 재질 | 상태 |
|---|----------|----------|------|------|------|
| 지각 | 0~35km(대륙)/0~7km(해양) | r=3.0 (외부) | 0x886644 갈색 | MeshStandardMaterial(roughness 0.8) | 고체 |
| 맨틀 | 35~2900km | r=2.85 | 0xcc5533 붉은 갈색 | MeshStandardMaterial(roughness 0.6) | 고체(유동성) |
| 외핵 | 2900~5100km | r=1.7 | 0xff6622 주황 | MeshStandardMaterial(emissive 0x441100, metalness 0.7) | 액체 |
| 내핵 | 5100~6371km | r=0.8 | 0xffaa44 밝은 금속 노랑 | MeshStandardMaterial(emissive 0x663300, metalness 0.9, roughness 0.2) | 고체 |

### 지각 상세 (교과서 그림 III-4, p106)
- 대륙 지각: 두께 ~35km, BoxGeometry 패치로 표면에 돌출 (약간 두꺼움)
  - 색상: 0x886644 갈색 + 0x447744 녹색(산/대륙)
  - 라벨: "대륙 지각 (~35km)"
- 해양 지각: 두께 ~7km, 대륙 사이 얇은 영역
  - 색상: 0x334466 짙은 파랑
  - 라벨: "해양 지각 (~7km)"
  - 위에 바다(반투명 파랑 SphereGeometry r=3.02, opacity 0.2)

### 맨틀 대류 시각화
- 맨틀 내부에 대류 순환 화살표:
  - 뜨거운 물질이 핵 근처에서 올라오는 빨간 화살표 (↑, 0xff4422)
  - 차가운 물질이 지각 근처에서 내려가는 파란 화살표 (↓, 0x4466ff)
  - 4~6개 대류 셀 표시
  - 느린 순환 애니메이션 (화살표 점선이 방향으로 이동)
- 라벨: "맨틀 대류 — 고체이지만 오랜 시간에 걸쳐 서서히 움직인다"

### 외핵 액체 효과
- 외핵 표면에 미세한 파동/소용돌이 패턴 (ShaderMaterial noise 기반)
- 느리게 흐르는 금속 액체 느낌
- 라벨: "외핵 — 액체 상태(철, 니켈)"

### 내핵 고체 효과
- 내핵 표면에 금속 광택 (metalness 0.9, roughness 0.2)
- 강한 내부 발광 (PointLight)
- 라벨: "내핵 — 고체 상태(철, 니켈), 약 6000°C"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "절단 각도" 슬라이더: 0° ~ 180° (기본값 90°) — ClippingPlane 각도 조절
  - 0°: 절단 없음 (외부만 보임)
  - 90°: 1/4 절단 (기본)
  - 180°: 반구 절단 (내부 전체 노출)
- "층 선택" 드롭다운: 전체 / 지각 / 맨틀 / 외핵 / 내핵
  - 선택된 층만 불투명, 나머지 반투명(opacity 0.1)으로 페이드
  - 선택된 층의 상세 정보가 하단 패널에 표시
- "맨틀 대류 표시" 토글: ON/OFF (기본값 ON)
- "지각 구분 표시" 토글: ON/OFF (기본값 ON) — 대륙 지각/해양 지각 구분
- "깊이 탐색" 슬라이더: 0 ~ 6371 (기본값 0), 단위 km
  - 슬라이더 이동 시 해당 깊이에 수평 원반(빨간 반투명) 표시
  - 깊이에 따른 온도/압력/밀도/구성 물질 정보 업데이트
- "지구 회전" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js CDN: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형, 모바일 터치 지원
- ClippingPlane: renderer.localClippingEnabled = true, 2개 평면으로 쐐기 절단
- 내부 층: SphereGeometry 4겹, 각각 clippingPlanes 설정
- 맨틀 대류 화살표: BufferGeometry + Points 또는 ArrowHelper
- 외핵 액체 효과: ShaderMaterial(noise + 시간 오프셋)
- ClippingPlane 쐐기 절단: 2개 Plane — Plane(normal(1,0,0), constant 0) + Plane(normal(0,0,1), constant 0) → 1/4 절단
  - renderer.localClippingEnabled = true
  - 각 층 SphereGeometry의 material.clippingPlanes = [plane1, plane2]
  - 절단 각도 슬라이더: plane2의 normal을 회전하여 0°~180° 제어
- 내부 층 단면 색상: 절단면에 색상 표시를 위해 side: DoubleSide 설정 (내부면도 렌더링)
- 맨틀 대류 화살표: ArrowHelper 4~6개를 맨틀 영역(r=0.7~2.85)에 배치
  - 상승류(핵 근처→지각): 빨강 0xff4422, 길이 1.5
  - 하강류(지각→핵 방향): 파랑 0x4466ff, 길이 1.5
  - 매 프레임 화살표의 opacity를 sin(time)으로 부드럽게 깜빡여 흐름 표현
- 외핵 액체 효과: ShaderMaterial — uniform time, noise(position + time×0.5) 기반 색상 변동 (0xff6622↔0xff8844), 느리게 유동하는 느낌
- 깊이 슬라이더 연동: RingGeometry(r=5.0-깊이/6371×5, tube=0.02, 0xff0000) 수평 원반, 매 프레임 위치 업데이트
- 깊이별 정보: 깊이값 → 배열에서 해당 층 검색 → 온도/밀도/구성 물질 텍스트 업데이트
- 층 선택 시: 선택 층 material.opacity = 원래값, 비선택 층 material.opacity = 0.05 (lerp 0.5초)
- dispose(): 층 선택 시 mesh를 재생성하지 않고 opacity만 변경 (성능 최적화)
- 성능: requestAnimationFrame 루프, SphereGeometry 4개 + 화살표 6개 + 지표 장식 ~10개 = 총 ~20 mesh
- 모바일: lil-gui 기본 접힘, OrbitControls enableDamping=true
- 파일명: earth_internal_structure_sim.html
