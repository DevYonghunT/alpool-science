너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 우주 초기에 생성된 원소
- 토픽: 빅뱅과 우주 진화
- 대상: 고등학교 1학년
- 핵심 개념: 빅뱅 우주론에 따르면 약 138억 년 전 대폭발로 우주가 탄생했다. 빅뱅 직후 쿼크와 전자가 만들어졌고, 이후 양성자와 중성자가 생성되었으며, 빅뱅 후 약 3분 이내에 수소 원자핵과 헬륨 원자핵이 질량비 3:1로 형성되었다. 약 38만 년 후 원자가 형성되면서 우주 배경 복사가 방출되었다.

## 씬 구성
- 전체 화면을 3D 우주 공간으로 렌더링한다.
- 중앙에 "빅뱅 포인트"를 배치하고, 타임라인에 따라 우주가 팽창하는 애니메이션을 재생한다.
- 배경: 검은 우주 0x000005
- 별 파티클: Points(BufferGeometry, 2000개), 랜덤 분포, 크기 0.5~2.0, 흰색~연노랑, 타임라인 후반부에서만 서서히 등장
- 카메라: PerspectiveCamera(FOV 60) + OrbitControls (우주 공간을 마우스로 회전 관찰)
- 조명: AmbientLight(0.1) + PointLight(빅뱅 중심, 세기는 타임라인에 따라 변화)

## 타임라인 6단계 상세

### 단계 1: 빅뱅 (t = 0)
- 3D: 화면 중앙에 극도로 밝은 점(PointLight 세기 5.0 + SphereGeometry r=0.1, emissive 0xffffff)
- 폭발 효과: 중심에서 바깥으로 파티클(Points 500개)이 방사형으로 퍼져나감
  - 파티클 색상: 핵심부 0xffffff(흰색) → 외곽 0xffaa00(주황) → 0xff4400(빨강)
  - 파티클 속도: 초기에 매우 빠르게, 점점 감속
- 배경: 순간적으로 밝은 백색 플래시 → 서서히 어두워짐
- 온도 표시: "~10³²K" (극고온)
- 교과서: "모든 물질과 에너지가 모인 한 점에서 대폭발(빅뱅)이 일어나 우주가 시작되었다" (p51)

### 단계 2: 쿼크와 전자 생성 (t = 10⁻⁶초)
- 3D: 팽창하는 구름(반투명 SphereGeometry, opacity 0.3, 0xff6644 뜨거운 주황) 내부에 작은 파티클들이 생성
- 쿼크 파티클: SphereGeometry(r=0.05), 3가지 색상(빨강 0xff0000, 초록 0x00ff00, 파랑 0x0000ff) — 쿼크의 3가지 "색전하" 표현
- 전자 파티클: SphereGeometry(r=0.03), 노란색 0xffff00, 빠르게 움직임
- 각 파티클에 이름 라벨: "쿼크", "전자"
- 온도: "~10¹²K"
- 교과서: "빅뱅 직후 우주가 팽창하기 시작하면서 최초로 기본 입자인 쿼크와 전자가 만들어졌다" (p52)

### 단계 3: 양성자와 중성자 생성 (t = 10⁻⁴초)
- 3D: 쿼크 3개가 모여 결합하는 애니메이션 → 양성자(빨강 구) 또는 중성자(파랑 구) 형성
- 양성자: SphereGeometry(r=0.12), 빨강 0xff4444, "p⁺" 라벨
- 중성자: SphereGeometry(r=0.12), 파랑 0x4444ff, "n" 라벨
- 결합 애니메이션: 3개의 작은 쿼크 구가 회전하며 중심으로 모여 하나의 큰 구로 합쳐짐 (0.5초)
- 배경 구름 색상: 주황 → 노란색으로 식어감
- 온도: "~10¹⁰K"
- 교과서: "쿼크가 결합해 양성자와 중성자가 만들어졌다" (p52)
- 교과서 그림 II-7: 빅뱅 이후 우주의 진화 과정 도해 (p52)

### 단계 4: 헬륨 원자핵 생성 — 핵융합 (t = 약 3분)
- 3D: 양성자 2개 + 중성자 2개가 결합하여 헬륨 원자핵(He-4) 형성
- 핵융합 애니메이션:
  1. 양성자(빨강 구) 2개와 중성자(파랑 구) 2개가 화면에 배치
  2. 4개가 서로에게 끌려가며 충돌 → 결합 (0.8초)
  3. 결합 순간 에너지 방출 효과: 밝은 노란 플래시 + 작은 파티클 방사
  4. 결합된 헬륨 원자핵: 큰 구(r=0.2) 내부에 빨강 2개 + 파랑 2개 구가 밀착, "He²⁺" 라벨
- 질량비 시각화 (교과서 핵심):
  - 화면 하단에 막대그래프: 수소(양성자) 75% vs 헬륨 25% (질량비 약 3:1)
  - 빨간 막대(수소) : 노란 막대(헬륨) = 3 : 1
  - "수소 원자핵과 헬륨 원자핵의 질량비는 약 3:1" 라벨
- 온도: "~10⁹K"
- 교과서: "양성자(수소 원자핵)와 중성자의 핵융합 반응이 일어나 헬륨 원자핵이 만들어졌는데, 이 과정은 빅뱅 이후부터 3분 사이에 일어났다. 수소 원자핵과 헬륨 원자핵의 질량비는 약 3:1이 되었다." (p52)

### 단계 5: 원자 형성 + 우주 배경 복사 (t = 약 38만 년)
- 3D: 원자핵 주위에 전자가 궤도를 돌며 결합하는 애니메이션
  - 수소 원자: 중심 양성자(빨강 r=0.1) + 전자(노랑 r=0.03) 1개 궤도 회전
  - 헬륨 원자: 중심 헬륨핵(빨강2+파랑2, r=0.15) + 전자(노랑 r=0.03) 2개 궤도 회전
- 우주 배경 복사:
  - 원자 형성 순간, 화면 전체에 파동 형태의 빛이 사방으로 퍼져나감 (RingGeometry 확장 애니메이션, 반복)
  - 색상: 미세파 대역 표현 → 연한 노란-주황 0xffddaa (실제 CMB 온도 ~3000K에 해당)
  - "우주 배경 복사" 라벨 + 교과서 그림 II-6 스타일 전천 CMB 맵(타원형, 미세 온도 차이 색상) HTML overlay
- 온도: "~3000K"
- 교과서: "원자핵이 전자와 결합해 원자가 만들어졌다. 원자가 만들어진 뒤에는 빛이 전기를 띤 입자의 방해를 받지 않고 자유롭게 우주 공간으로 퍼져 나갈 수 있게 되었다. 이때의 빛이 현재 우주 배경 복사로 관측된다." (p52)

### 단계 6: 별과 은하 형성 → 현재 우주 (t = 138억 년)
- 3D: 원자(수소, 헬륨)들이 모여 성운 형성 → 수축 → 별 탄생 → 은하 형성
  - 성운: Points(500개) 구형 분포, 반투명 파랑-보라
  - 별: SphereGeometry(r=0.1~0.3) 발광(emissive), 다양한 크기와 밝기
  - 은하: Points(3000개) 나선 분포, 중심부 밝게
- 현재 우주 상태:
  - 여러 은하가 3D 공간에 분포, 서로 멀어지는 애니메이션 (우주 팽창)
  - "우주는 현재도 계속 팽창하고 있다" 라벨
- 교과서: "이후 우주는 계속 팽창하면서 별과 은하를 형성했고 약 138억 년이 지나 현재의 우주가 되었다" (p53)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "타임라인" 슬라이더: 0 ~ 5 (6단계, 정수), 각 단계명 라벨 표시
  - 0: 빅뱅 / 1: 쿼크·전자 / 2: 양성자·중성자 / 3: 헬륨 핵융합 / 4: 원자·CMB / 5: 별·은하
- "자동 재생" 버튼: 단계 0부터 5까지 자동 진행 (각 단계 5초, 전환 2초)
  - 재생 중 "일시정지" 버튼으로 변경
- "재생 속도" 슬라이더: 0.5x ~ 3x (기본값 1x), 0.5 단계
- "핵융합 상세" 버튼 (단계 3에서만 활성): 양성자+중성자 → 헬륨 핵융합 과정을 슬로모션으로 반복 재생
- "초기화" 버튼: 단계 0, 기본 카메라 위치로 리셋

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌌 빅뱅과 우주 진화
━━━━━━━━━━━━━━━
현재 단계:    [단계명]
경과 시간:    [t 값]
우주 온도:    [온도값]
우주 크기:    [상대적 크기]

수소 : 헬륨 질량비 = 3 : 1 (약 75% : 25%)

빅뱅 핵합성 (t ≈ 3분):
  p + n → ²H (중수소)
  ²H + p → ³He
  ³He + ³He → ⁴He + 2p
  (양성자와 중성자의 직접 결합)
```
타임라인 변경 시 실시간 업데이트.

  1. "빅뱅 후 3분 이내에 헬륨 원자핵이 만들어진 까닭은 무엇인가요? (힌트: 온도와 핵융합)"
  2. "우주 배경 복사는 빅뱅 우주론을 뒷받침하는 증거입니다. 그 까닭을 설명해 보세요."
  3. "현재 우주에서 수소와 헬륨이 가장 많은 원소인 까닭은 무엇인가요?"
- 실생활 예시: COBE/WMAP/Planck 위성(CMB 관측), 허블 우주 망원경(우주 팽창 관측), CERN 입자가속기(초기 우주 재현)
- 교과서 연계: "미래엔 통합과학1 II편 p51~53 — 빅뱅 우주론, 초기 우주의 원소 생성, 우주 배경 복사"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 파티클 시스템: BufferGeometry + Points (성능 최적화)
- 단계 전환 시 이전 단계 객체 dispose() + 새 단계 객체 생성
- 성능: requestAnimationFrame 루프, 파티클 최대 3000개, InstancedMesh 활용 고려
- 파일명: bigbang_universe_evolution_sim.html
