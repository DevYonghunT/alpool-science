너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 02. 양자 기술과 미래
- 토픽: 양자 얽힘과 양자 정보
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 양자 얽힘은 두 입자가 양자적으로 연결되어 한 입자를 측정하면 다른 입자의 상태가 즉시 결정되는 현상이다. 거리에 무관하게 작용하며 아인슈타인은 이를 "유령 같은 원격 작용"이라 불렀다. 양자 얽힘은 양자 암호, 양자 텔레포테이션, 양자 컴퓨팅 등에 활용된다.

## 씬 구성
- 화면 중앙: 3D 양자 얽힘 입자 쌍 + 측정 시각화
- 3가지 시나리오: 양자 얽힘 기본 / 양자 암호(QKD) / 양자 텔레포테이션
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.4)

## 시나리오 1: 양자 얽힘 기본

### 3D 얽힌 입자 쌍
- 입자 A: SphereGeometry(r=0.3), 0xff6644 + 파동 효과(글로우)
  - "앨리스(Alice)" 관측자 옆에 배치
- 입자 B: SphereGeometry(r=0.3), 0x4466ff + 파동 효과
  - "밥(Bob)" 관측자 옆에 배치, 먼 거리에
- 얽힘 연결: 두 입자 사이 빛나는 선(CylinderGeometry, 0xffdd44 emissive) — 양자 상관

### 스핀 상태 (↑/↓)
- 측정 전: 두 입자 모두 "?" 표시 (중첩 상태)
  - 입자가 미세하게 진동/깜빡이는 애니메이션 (불확정)
- 앨리스가 입자 A를 측정 ("측정" 버튼 클릭):
  - 입자 A: ↑ 또는 ↓ 중 하나로 확정 (랜덤)
  - **동시에** 입자 B: 반대 스핀으로 즉시 확정 (↓ 또는 ↑)
  - 측정 순간 플래시 효과 + 얽힘 선 사라짐
  - "앨리스가 ↑를 측정하면 밥은 반드시 ↓!" 라벨
- 거리 무관: 두 입자가 아무리 멀어도 즉시 상관
- "정보 전달 아님! 측정 결과는 랜덤, 통신 불가" 주의 라벨

## 시나리오 2: 양자 암호 (QKD, BB84 프로토콜 간략)

### 3D
- 앨리스(송신): BoxGeometry 컴퓨터, 0xff6644
- 밥(수신): BoxGeometry 컴퓨터, 0x4466ff
- 광섬유 채널: TubeGeometry, 0xdddddd
- 도청자(이브): BoxGeometry, 0xff4444 (빨간 해커 아이콘), 채널 중간에

### 양자 암호 원리
- 앨리스가 양자 상태(광자 편광)로 비밀 키 전송
- 이브가 도청하면: 양자 상태 붕괴 → 오류 발생 → 앨리스와 밥이 도청 감지!
- "양자 역학 원리: 측정하면 상태가 변한다 → 도청 감지 가능" 라벨
- 도청 없을 때: 키 공유 성공 ✅
- 도청 있을 때: 오류율↑ → 도청 감지 → 키 폐기 ❌

## 시나리오 3: 양자 컴퓨팅 (큐비트 기본)

### 3D 큐비트
- 블로흐 구(Bloch sphere): SphereGeometry(r=2), wireframe, 0x4488ff
  - 북극 |0⟩, 남극 |1⟩
  - 큐비트 상태: 구 위의 한 점 (화살표 벡터)
- 고전 비트: 0 또는 1 (2가지)
- 큐비트: |ψ⟩ = α|0⟩ + β|1⟩ (중첩 상태, 구의 어디든)
  - 측정 전: 0과 1의 중첩
  - 측정 후: 0 또는 1로 확정 (확률적)

### 양자 게이트
- NOT 게이트(X): |0⟩↔|1⟩ (블로흐 구에서 반전)
- 아다마르(H): |0⟩→(|0⟩+|1⟩)/√2 (중첩 생성)
- 간단한 양자 회로 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 양자 얽힘 / 양자 암호 / 큐비트
- "측정" 버튼 (얽힘): 앨리스가 입자 A 측정
- "리셋" 버튼: 얽힘 상태로 복원
- "도청자" 토글 (암호): ON/OFF
- "키 전송" 버튼 (암호): 키 교환 시뮬레이션
- "양자 게이트" 드롭다운 (큐비트): X / H / Z
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔗 양자 얽힘과 양자 정보
━━━━━━━━━━━━━━━━
시나리오: [얽힘/암호/큐비트]

양자 얽힘:
  두 입자가 양자적으로 연결
  한쪽 측정 → 다른 쪽 즉시 확정
  거리 무관, 정보 전달 아님

큐비트:
  |ψ⟩ = α|0⟩ + β|1⟩
  |α|² + |β|² = 1
  측정: |0⟩확률 = |α|², |1⟩확률 = |β|²

양자 암호:
  도청 → 양자 상태 붕괴 → 오류 감지
  → 도청 불가능한 완벽한 보안!
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "양자 얽힘에서 한 입자를 측정하면 다른 입자의 상태가 즉시 결정됩니다. 이는 양자 암호, 양자 컴퓨팅 등 양자 정보 기술의 기반입니다."
- 탐구 질문:
  1. "양자 얽힘으로 초광속 통신이 가능한가요? 그 까닭은?"
  2. "양자 암호가 기존 암호보다 안전한 까닭은 무엇인가요?"
  3. "고전 비트와 큐비트의 차이를 블로흐 구로 설명해 보세요."
- 실생활 예시: 양자 암호 통신(중국 묵자 위성), 양자 컴퓨터(구글 Sycamore, IBM), 양자 센서
- 교과서 연계: "전자기와 양자 III편 p104~107 — 양자 기술과 미래"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: emq_22_quantum_entanglement.html
