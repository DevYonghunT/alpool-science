너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 2. 자연의 구성 물질 > 지각과 생명체를 구성하는 물질
- 토픽: 규산염 사면체와 광물 구조
- 대상: 고등학교 1학년
- 핵심 개념: 지각을 구성하는 암석은 광물로 이루어져 있으며, 광물 대부분은 산소와 규소가 결합한 규산염 사면체(SiO₄)를 기본 단위체로 한다. 규산염 사면체는 중심에 규소 1개, 꼭짓점에 산소 4개가 위치한다. 사면체가 산소를 공유하며 결합하는 방식에 따라 감람석(독립형), 휘석(단사슬), 각섬석(이중사슬), 흑운모(판상), 석영(입체 그물) 등 다양한 규산염 광물이 형성된다.

## 씬 구성
- 화면 중앙에 3D 규산염 사면체 및 광물 구조를 렌더링한다.
- 배경: 0x1a1a2e (짙은 남색)
- 조명: AmbientLight(0.35) + DirectionalLight(0.7, 좌상단에서) + 그림자 활성화
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (구조를 360도 회전 관찰)

## 규산염 사면체 기본 단위 (교과서 그림 II-27, p80)
- 중심: 규소(Si) 원자 — SphereGeometry(r=0.3, 32, 32), 0x8888cc (연한 보라-회색)
- 꼭짓점 4개: 산소(O) 원자 — SphereGeometry(r=0.4, 32, 32), 0xff4444 (빨강)
  - 정사면체 꼭짓점 좌표:
    - O1: (1, 1, 1) × 0.8
    - O2: (1, -1, -1) × 0.8
    - O3: (-1, 1, -1) × 0.8
    - O4: (-1, -1, 1) × 0.8
- Si-O 결합: CylinderGeometry(r=0.05, h=결합길이) 4개, 0xaaaaaa (회색)
- 사면체 면: MeshBasicMaterial(0x6688aa, opacity 0.15, side: DoubleSide) — 반투명 삼각면 4개
- 사면체 모서리: LineSegments(0x4466aa, opacity 0.4) — 6개 모서리 선
- 라벨: 중심 "Si", 꼭짓점 "O", 전체 "SiO₄ 규산염 사면체"

## 5가지 광물 구조 상세 (교과서 그림 II-28, p81)

### (1) 감람석 — 독립형
- 구조: 규산염 사면체가 개별적으로 존재 (산소 공유 없음)
- 3D: 사면체 4~6개를 일정 간격(2.5)으로 분산 배치, 사면체 사이에 양이온(Mg²⁺, Fe²⁺) 표시
  - 양이온: SphereGeometry(r=0.2), 0x44aa44 (녹색)
  - 양이온이 사면체를 연결하는 역할
- 색상 테마: 올리브 녹색 계열
- 광물 사진 대체: 녹색 불투명 구체(r=1.5, 0x667744)를 배경에 배치
- 라벨: "감람석 — 독립형: 사면체가 개별 존재"

### (2) 휘석 — 단사슬형
- 구조: 사면체가 한 줄로 길게 연결 (이웃한 사면체끼리 산소 1개 공유)
- 3D: 사면체 8~10개를 X축 방향으로 일렬 연결
  - 공유 산소: 이웃한 두 사면체 사이의 산소 1개가 겹침 (빨간 구 1개 공유)
  - 공유 산소 강조: 크기 r=0.45, 노란색 글로우(emissive 0xffdd44)
  - 비공유 산소: 일반 빨간 구
- 기둥 모양 결정 형태: 전체 사슬을 감싸는 반투명 기둥(CylinderGeometry, 0x556644, opacity 0.1)
- 라벨: "휘석 — 단사슬: 사면체가 한 줄로 연결"

### (3) 각섬석 — 이중사슬형
- 구조: 사면체가 두 줄로 길게 연결 (두 사슬이 산소를 공유하며 결합)
- 3D: 사면체 16~20개를 2줄로 배치
  - 같은 줄: 산소 1개 공유 (휘석과 동일)
  - 줄 사이: 추가 산소 공유로 두 줄 연결
  - 공유 산소 강조: 노란 글로우
- 기둥 모양이지만 휘석보다 두꺼운 반투명 기둥
- 라벨: "각섬석 — 이중사슬: 사면체가 두 줄로 연결"

### (4) 흑운모 — 판상형
- 구조: 사면체가 평면으로 연결되어 얇은 판 형성 (산소 3개 공유)
- 3D: 사면체 25~30개를 XZ 평면에 그리드로 배치 (5×5 또는 5×6)
  - 각 사면체의 3개 산소가 이웃과 공유
  - 1개 산소만 위로 돌출 (비공유)
  - 판 여러 장을 Y축으로 간격 두고 쌓기 (3~4장, 간격 1.5)
- 판 형태 강조: 각 판을 PlaneGeometry(반투명 0x443322, opacity 0.08)로 감쌈
- 라벨: "흑운모 — 판상형: 사면체가 평면으로 연결"
- 판을 분리하는 애니메이션: "판 분리" 버튼 클릭 시 판 간격이 벌어짐 (1.5 → 4.0)

### (5) 석영 — 입체 그물형
- 구조: 사면체가 3D 모든 방향으로 연결 (산소 4개 모두 공유)
- 3D: 사면체 30~40개를 3D 격자로 배치
  - 모든 산소가 이웃 사면체와 공유
  - 규칙적인 3D 네트워크 형성
  - 공유 산소: 노란 글로우
- 투명한 결정 형태: SphereGeometry 또는 육각기둥(석영 결정 형태) 반투명 0xeeeeff, opacity 0.1
- 라벨: "석영 — 입체 그물형: 모든 방향으로 연결"
- 화학식: SiO₂ (모든 산소가 공유되므로 Si:O = 1:2)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "광물 선택" 드롭다운: 기본 사면체 / 감람석(독립) / 휘석(단사슬) / 각섬석(이중사슬) / 흑운모(판상) / 석영(입체 그물)
- "공유 산소 강조" 토글: ON/OFF (기본값 ON)
  - ON: 공유 산소를 노란색으로 강조
  - OFF: 모든 산소 동일 빨간색
- "사면체 면 표시" 토글: ON/OFF (기본값 ON) — 반투명 삼각면 표시
- "원자 라벨" 토글: ON/OFF (기본값 ON) — Si, O 라벨 표시
- "판 분리" 버튼 (흑운모에서만 활성): 판 간격 벌어짐 애니메이션
- "구조 비교" 체크박스: ON 시 현재 광물과 기본 사면체를 좌우로 나란히 배치
- "자동 회전" 토글: ON/OFF (기본값 ON) — Y축 느린 자동 회전
- "초기화" 버튼

## 시각화 규칙

### 공유 산소 수에 따른 비교 표 (하단 HTML overlay)
```
📊 규산염 광물의 산소 공유 비교
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
광물       공유 O 수   결합 형태    Si:O 비
감람석     0개        독립형       1:4
휘석       1개        단사슬       1:3
각섬석     1.5개      이중사슬     4:11
흑운모     3개        판상형       2:5
석영       4개        입체 그물    1:2
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
공유 산소↑ → 결합 강해짐 → 구조 안정
```
- 현재 선택된 광물의 행이 노란색으로 강조

### 광물→광물 전환 애니메이션
- 사면체들이 재배치되는 모프(morph) 애니메이션 (1.5초)
- 독립형 → 한 줄 → 두 줄 → 평면 → 입체로 점진적 변환

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
💎 규산염 사면체와 광물
━━━━━━━━━━━━━━━━━━
선택 광물:      [광물명]
결합 형태:      [독립 / 단사슬 / 이중사슬 / 판상 / 입체 그물]
공유 산소 수:   [0~4]개 / 사면체

기본 단위체: SiO₄⁴⁻
  중심: Si (규소) 1개
  꼭짓점: O (산소) 4개
  형태: 정사면체

지각 구성 원소 비율:
  산소 ~46%, 규소 ~28% (1위, 2위)
  → 규산염 광물이 지각 대부분 차지
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "지각을 구성하는 암석의 대부분은 규산염 광물로 이루어져 있습니다. 규산염 사면체(SiO₄)가 산소를 공유하며 결합하는 방식에 따라 다양한 광물이 형성됩니다."
- 탐구 질문:
  1. "규산염 사면체끼리 연결할 때 어느 원자를 공유하나요? (힌트: 꼭짓점)" (답: 산소)
  2. "석영은 사면체의 산소 4개를 모두 공유합니다. 이때 Si:O 비율이 1:2가 되는 까닭은?"
  3. "흑운모가 얇게 벗겨지는 성질은 어떤 결합 구조와 관련이 있나요?" (답: 판상형)
- 실생활 예시: 화강암(석영+장석+운모), 현무암(휘석+감람석), 석영 시계(석영 진동), 운모판(절연체)
- 교과서 연계: "미래엔 통합과학1 II편 p80~81 — 규산염 광물, 그림 II-27(사면체), 그림 II-28(광물 종류)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원
- 사면체: 정사면체 꼭짓점 좌표 계산, SphereGeometry + CylinderGeometry 결합
- 광물 구조: 사면체 함수로 생성, 배열/반복으로 구조 조립
- InstancedMesh: 대량 사면체(석영 30~40개) 효율 렌더링
- 광물 전환: 사면체 위치 Tween 애니메이션 (자체 lerp 보간 (GSAP 미사용))
- 성능: requestAnimationFrame 루프, 사면체 최대 40개(원자 최대 200개)
- 파일명: silicate_tetrahedra_mineral_sim.html
