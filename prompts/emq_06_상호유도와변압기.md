너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 04. 전자기 유도의 활용
- 토픽: 상호 유도와 변압기
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 변압기는 상호 유도를 이용하여 교류 전압을 변환하는 장치이다. 1차 코일과 2차 코일이 철심에 감겨 있으며, 전압비 V₁/V₂ = N₁/N₂ (N: 감은 수). 이상적인 변압기에서 전력이 보존되므로 V₁I₁ = V₂I₂이다. 송전 시 승압하면 전류가 줄어 송전선 열 손실(P=I²R)을 줄일 수 있다.

## 씬 구성
- 화면 좌측: 3D 변압기 내부 구조 (1차/2차 코일 + 철심)
- 화면 우측: 전압/전류 비교 + 송전 시스템 다이어그램
- 2가지 모드: 변압기 원리 / 송전 시스템
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 모드 1: 변압기 원리

### 3D 변압기 내부
- 철심: BoxGeometry로 'ㅁ'자 형태 (상/하/좌/우 막대), 0x555555 (짙은 회색)
- 1차 코일(입력): TorusGeometry × N₁겹, 0xff6644 (주황), 철심 좌측에 감김
  - "N₁ = [값]" 라벨
  - 교류 전류: 노란 파티클이 코일을 따라 왕복(교류 표현)
- 2차 코일(출력): TorusGeometry × N₂겹, 0x4466ff (파랑), 철심 우측에 감김
  - "N₂ = [값]" 라벨
  - 유도 전류: 파란 파티클 이동

### 자기 선속
- 철심 내부: 자기력선(초록 화살표) 순환
- 1차 코일 전류 변화 → 자기장 변화 → 철심 통해 2차 코일에 전달 → 유도 기전력
- "교류만 변압 가능 (직류는 자기장 변화 없으므로 ❌)" 라벨

### 전압/전류 관계
- V₁/V₂ = N₁/N₂
- N₁ > N₂: 강압 변압기 (전압↓, 전류↑)
- N₁ < N₂: 승압 변압기 (전압↑, 전류↓)
- V₁I₁ = V₂I₂ (전력 보존, 이상적)
- N₁, N₂ 슬라이더 → V₂, I₂ 실시간 계산

### 전구 밝기 시각화
- 2차 코일에 전구 연결
- N₂/N₁ 비에 따라 전구 밝기 변화 (V₂ 변화)
- 승압: 전구 밝아짐 / 강압: 전구 어두워짐

## 모드 2: 송전 시스템

### 3D 송전 다이어그램
- 발전소: BoxGeometry, 0x888888, "V_발전 = 22kV" 라벨
- 승압 변압기: 변압기 3D 모형, "22kV → 345kV"
- 송전선: CylinderGeometry 긴 선, 0x333333
  - 송전선 저항 R 표시
  - 전류 I 파티클 이동
- 강압 변압기: "345kV → 220V"
- 가정: BoxGeometry(집), "220V" 전구 점등

### 송전 손실 비교
- 저압 송전(22kV): 큰 전류 I₁ → 열 손실 P = I₁²R 큼 (빨간 글로우 강함)
- 고압 송전(345kV): 작은 전류 I₂ → 열 손실 P = I₂²R 작음 (글로우 약함)
- 전압을 10배 올리면 → 전류 1/10 → 손실 1/100!
- 손실 비교 막대그래프: 저압 vs 고압

### 에너지 흐름
- 발전소 → 승압 → 송전(손실 일부) → 강압 → 가정
- Sankey 다이어그램으로 에너지 흐름 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 변압기 원리 / 송전 시스템
- "1차 감은 수 N₁" 슬라이더: 5~50 (기본값 20)
- "2차 감은 수 N₂" 슬라이더: 5~50 (기본값 10)
- "1차 전압 V₁" 슬라이더: 10~500V (기본값 100)
- "송전선 저항 R" 슬라이더 (모드 2): 1~20Ω (기본값 5)
- "자기 선속 표시" 토글: ON/OFF (기본값 ON)
- "손실 비교" 토글 (모드 2): ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔌 상호 유도와 변압기
━━━━━━━━━━━━━━━
모드: [변압기/송전]
N₁ = [값], N₂ = [값]
V₁ = [값]V

변압기 원리:
  V₁/V₂ = N₁/N₂
  V₂ = V₁ × N₂/N₁ = [값]V
  전력 보존: V₁I₁ = V₂I₂
  I₂ = I₁ × N₁/N₂ = [값]A

N₂ > N₁: 승압 (전압↑, 전류↓)
N₂ < N₁: 강압 (전압↓, 전류↑)

송전 손실:
  P_손실 = I²R
  전압 10배↑ → 전류 1/10 → 손실 1/100
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "변압기는 상호 유도를 이용해 교류 전압을 변환합니다. V₁/V₂=N₁/N₂. 송전 시 승압하면 전류가 줄어 열 손실(P=I²R)이 크게 감소합니다."
- 탐구 질문:
  1. "변압기가 교류에서만 작동하고 직류에서는 작동하지 않는 까닭은 무엇인가요?"
  2. "송전 전압을 10배 높이면 송전선에서의 열 손실은 몇 분의 1로 줄어드나요?"
  3. "가정에서 사용하는 220V는 어떤 변압 과정을 거쳐 도달하나요?"
- 실생활 예시: 충전기(강압 변압기), 전봇대 변압기, 초고압 송전선(345kV/765kV)
- 교과서 연계: "전자기와 양자 I편 p26~31 — 전자기 유도의 활용, 변압기, 송전"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 50개
- 파일명: emq_06_transformer.html
