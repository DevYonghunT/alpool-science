너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 04. 광전 효과
- 토픽: 광전 효과와 광자
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 금속 표면에 빛을 비추면 전자가 방출되는 광전 효과에서, 빛의 진동수가 문턱 진동수(f₀) 이상일 때만 전자가 방출된다. 광자 에너지 E = hf. 광전자의 최대 운동 에너지 Ek = hf - W (W: 일함수). 정지 전압 V₀에서 eV₀ = hf - W이며, 이를 이용해 플랑크 상수 h를 측정할 수 있다.

## 씬 구성
- 화면 좌측 50%: 3D 광전 효과 실험 장치 (금속판 + 광원 + 전자 방출)
- 화면 우측 50%: Ek-f 그래프 + 정지 전압 실험 (Canvas 2D)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2)

## 3D 광전 효과 실험

### 금속판 (음극)
- PlaneGeometry(3, 3), MeshStandardMaterial(metalness 0.9, roughness 0.15), 0xcccccc
- 표면에 전자(SphereGeometry r=0.06, 0xffff44) 격자 배열

### 광원
- 빛: 광자 파티클(SphereGeometry r=0.04)이 금속판 방향으로 발사
  - 색상: 진동수에 따라 빨강(낮은 f)→보라(높은 f)→자외선(매우 높은 f)
  - 세기: 광자 수/초 (많으면 밝고 적으면 어두움)
- 빛의 진동수 f 슬라이더: 3~10 (×10¹⁴Hz)
- 빛의 세기 슬라이더: 1~10 (광자 수)

### 광전자 방출
- f ≥ f₀: 광자 금속 충돌 → 전자 방출!
  - 전자(0xffff44)가 표면에서 튀어나와 양극 방향으로 이동
  - 전자 속도 ∝ √(Ek) = √(hf - W)
  - 진동수↑ → 전자 속도↑ (운동 에너지↑)
  - 세기↑ → 전자 수↑ (광전류↑), 개별 전자 속도는 변하지 않음!
- f < f₀: 광자 충돌해도 전자 방출 안 됨 ❌
  - 빨간 "×" 표시 + "에너지 부족" 라벨
  - **세기를 아무리 높여도 안 됨!**

### 양극 (전자 수집판)
- PlaneGeometry, 0xddaa44, 금속판 반대편
- 전압 V 인가 가능 (정지 전압 실험용)

## 에너지 다이어그램 (옵션)
- 수직 에너지 축:
  - 바닥: 금속 내부 전자
  - 일함수 W: 탈출 필요 최소 에너지
  - 광자 에너지 hf: 화살표
  - Ek = hf - W: 남은 에너지
- hf < W: 화살표가 W에 못 미침 → 방출 불가

## 정지 전압 실험
- 양극에 역전압(-V₀) 인가 → 전자를 되돌림
- V₀ 슬라이더: 0~5V
- eV₀ = Ek_max = hf - W → V₀ = (hf - W)/e
- V₀에서 광전류 = 0 (모든 전자 되돌림)
- I-V 그래프: 역전압↑ → 전류↓, V₀에서 I = 0

## Ek-f 그래프 (핵심!, 우측 Canvas 2D)
- X축: 빛의 진동수 f (×10¹⁴Hz)
- Y축: 광전자 최대 운동 에너지 Ek (eV)
- 직선: Ek = hf - W → 기울기 = h, y절편 = -W
- f < f₀: Ek = 0 (방출 안 됨, x축 아래)
- f = f₀: Ek = 0 (방출 시작, x절편)
- f₀ = W/h (문턱 진동수)
- **기울기 = h (플랑크 상수)** ← 금속 종류에 무관!
- 다른 금속: 같은 기울기, 다른 y절편(-W) → 평행 이동

### 금속별 비교
- 세슘(W = 2.1eV, f₀ ≈ 5.1×10¹⁴Hz): 파란 직선
- 나트륨(W = 2.3eV): 빨간 직선
- 아연(W = 4.3eV): 초록 직선
- 모두 같은 기울기(h) → 평행!

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "금속" 드롭다운: 세슘 / 나트륨 / 아연 / 금
- "빛의 진동수 f" 슬라이더: 3~10 (×10¹⁴Hz, 기본값 6)
- "빛의 세기" 슬라이더: 1~10 (광자 수/초, 기본값 5)
- "역전압 V₀" 슬라이더: 0~5V (정지 전압, 기본값 0)
- "광자 발사" 버튼
- "에너지 다이어그램" 토글: ON/OFF
- "Ek-f 그래프" 토글: ON/OFF (기본값 ON)
- "금속 비교" 토글: ON/OFF (여러 금속 직선 동시)
- "초기화" 버튼

```

금속: [이름] (W = [값]eV)
f = [값]×10¹⁴Hz, f₀ = [값]×10¹⁴Hz

광자 에너지: E = hf = [값]eV
  h = 6.63×10⁻³⁴ J·s = 4.14×10⁻¹⁵ eV·s

f ≥ f₀: 광전자 방출 ✅
  Ek_max = hf - W = [값]eV
  정지 전압: V₀ = Ek/e = [값]V

f < f₀: 방출 불가 ❌
  세기(광자 수) 무관!
```
