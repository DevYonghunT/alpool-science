너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 05. 핵반응과 에너지
- 토픽: 방사선과 반감기
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 불안정한 원자핵은 방사선을 방출하며 붕괴한다. 알파(α) 붕괴는 헬륨-4 핵(양성자2+중성자2)을 방출하고, 베타(β) 붕괴는 중성자가 양성자로 전환되며 전자와 반중성미자를 방출하고, 감마(γ) 붕괴는 들뜬 핵이 고에너지 광자를 방출한다. 방사성 붕괴는 확률적이며, 반감기(t₁/₂)는 방사성 물질의 양이 절반으로 줄어드는 데 걸리는 시간이다. N(t)=N₀(1/2)^(t/t₁/₂).

## 씬 구성
- 화면 좌측 50%: 3D 방사선 방출 + 붕괴 시각화
- 화면 우측 50%: 붕괴 곡선 그래프 + 방사선 비교표 (Canvas 2D / HTML)
- 3가지 보기: α/β/γ 붕괴 / 반감기 / 의학 활용
- 배경: 0x0a0a1a
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.4)

## 보기 1: α/β/γ 방사선 붕괴

### 3D 원자핵과 방사선

#### 알파(α) 붕괴
- 모핵: SphereGeometry(r=1.0), 양성자(빨강 0xff4444) + 중성자(파랑 0x4488ff) 표면 배열
  - 예: ²³⁸U (우라늄-238)
- α 입자: SphereGeometry(r=0.3), 0xffaa44, 양성자2+중성자2 결합체
  - 핵에서 방출되는 애니메이션 (바깥으로 튀어나감)
- 딸핵: ²³⁴Th (토륨-234), 크기 약간 줄어듦
- 반응식: ²³⁸U → ²³⁴Th + ⁴He (α)
- "질량수 4↓, 원자번호 2↓" 라벨
- 투과력: 종이 한 장으로 차단 (종이 아이콘)

#### 베타(β⁻) 붕괴
- 모핵: ¹⁴C (탄소-14)
- 핵 내부: 중성자 하나가 양성자로 전환 (색상 변화 애니메이션)
  - n → p + e⁻ + ν̄ₑ
- β⁻ 입자(전자): SphereGeometry(r=0.06), 0xffff44, 빠르게 방출
- 반중성미자(ν̄ₑ): SphereGeometry(r=0.04), 0xaaddff opacity 0.3
- 딸핵: ¹⁴N (질소-14)
- "질량수 불변, 원자번호 1↑" 라벨
- 투과력: 알루미늄 판으로 차단

#### 감마(γ) 붕괴
- 들뜬 핵(*): SphereGeometry, 0xff6644, 진동/글로우 효과 (높은 에너지 상태)
- γ선 방출: 파동 형태 이펙트 (사인파 + 0xffff44 빛)
- 바닥 상태 핵: 안정, 글로우 사라짐
- "질량수 불변, 원자번호 불변, 에너지만 방출" 라벨
- 투과력: 두꺼운 납/콘크리트로 차단

### 방사선 비교 (우측 표)
| | α | β | γ |
|---|---|---|---|
| 정체 | ⁴He 핵 | 전자 | 광자 |
| 전하 | +2 | -1 | 0 |
| 질량 | 큼 | 작음 | 0 |
| 속도 | 느림 | 빠름 | 광속 |
| 투과력 | 종이 | 알루미늄 | 납/콘크리트 |
| 전리 능력 | 강 | 중 | 약 |

### 투과력 시각화
- 왼쪽에서 오른쪽으로 방사선 발사
- 차단체: 종이(0xeeddcc) → 알루미늄(0xcccccc) → 납(0x666666) → 콘크리트(0x999988)
- α: 종이에서 멈춤
- β: 알루미늄에서 멈춤
- γ: 납/콘크리트에서 감쇠
- 각 방사선의 궤적 표시 (α: 직선/짧음, β: 구불구불, γ: 직선/김)

## 보기 2: 반감기

### 3D 방사성 원자 시각화
- 방사성 원자: SphereGeometry(r=0.2), 0x44cc44 (미붕괴 = 초록)
- 안정 원자: SphereGeometry(r=0.2), 0x888888 (붕괴 후 = 회색)
- 초기: N₀=100개, 모두 초록
- 시간 진행:
  - t = 0: 100개 초록
  - t = t₁/₂: ~50개 초록, ~50개 회색
  - t = 2t₁/₂: ~25개 초록, ~75개 회색
  - t = 3t₁/₂: ~12개 초록, ~88개 회색
- 각 원자의 붕괴는 확률적 (랜덤!) → 매 시행마다 다른 원자가 붕괴
- "같은 반감기여도 어떤 원자가 붕괴할지는 모른다 — 확률적!" 라벨

### 붕괴 곡선 그래프 (Canvas 2D)
- X축: 시간 (t/t₁/₂ 단위)
- Y축: 남은 원자 수 N(t)
- 곡선: N(t) = N₀ × (1/2)^(t/t₁/₂) — 지수 감소
- 이론 곡선(부드러운 선) + 실험 데이터(점, 약간의 통계적 요동)
- 현재 시간에 수직 마커
- 반감기 선: 수평 점선 (N₀/2, N₀/4, N₀/8...)

### 방사성 동위 원소 예시
- ¹⁴C: t₁/₂ = 5730년 (탄소 연대 측정)
- ¹³¹I: t₁/₂ = 8일 (갑상선 치료)
- ²³⁸U: t₁/₂ = 45억 년 (지구 나이 측정)
- ⁹⁹ᵐTc: t₁/₂ = 6시간 (의학 영상)
- 동위 원소 선택 → 반감기 자동 변경

## 보기 3: 의학 활용

### 3D 방사선 의학
- 인체 실루엣: CylinderGeometry 몸통 + SphereGeometry 머리
- 갑상선: SphereGeometry, 목 부위, 0xff6644
  - ¹³¹I 흡수 → 갑상선암 치료 (β선으로 암세포 파괴)
- PET 스캔: 양전자(e⁺) 방출 핵종 주입
  - e⁺ + e⁻ → 2γ (반대 방향 511 keV)
  - 감마선 검출기 링으로 위치 결정
- 방사선 치료: 외부에서 γ선 조사 → 종양 파괴
  - 다방향에서 집중 조사 → 종양만 고선량
- "방사선: 위험하지만 잘 쓰면 생명을 살린다" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: α/β/γ 붕괴 / 반감기 / 의학 활용
- "방사선 종류" 드롭다운 (붕괴): α / β / γ
- "투과력 데모" 버튼: 방사선 발사 + 차단 시각화
- "동위 원소" 드롭다운 (반감기): ¹⁴C / ¹³¹I / ²³⁸U / ⁹⁹ᵐTc
- "시간 진행" 슬라이더: 0~5 반감기 (기본값 0)
- "속도" 슬라이더: 0.5x~3x
- "자동 재생" 토글: ON/OFF
- "붕괴 곡선" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
☢️ 방사선과 반감기
━━━━━━━━━━━━━━━
보기: [붕괴/반감기/의학]

방사성 붕괴:
  α: ᴬ_ZX → ᴬ⁻⁴_(Z-2)Y + ⁴₂He
  β⁻: ᴬ_ZX → ᴬ_(Z+1)Y + e⁻ + ν̄ₑ
  γ: X* → X + γ (에너지만 방출)

반감기:
  N(t) = N₀ × (1/2)^(t/t₁/₂)
  동위 원소: [이름]
  t₁/₂ = [값]

현재:
  경과 시간: [값] (=[값]×t₁/₂)
  남은 원자: [값]/[N₀]개 ([값]%)

투과력:
  α: 종이 | β: 알루미늄 | γ: 납
```

  3. "탄소-14의 반감기(5730년)를 이용하여 고대 유물의 나이를 어떻게 측정하나요?"
