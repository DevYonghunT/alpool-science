너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 01. 거울과 렌즈
- 토픽: 렌즈의 굴절과 상 (볼록/오목)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 볼록 렌즈(수렴 렌즈)는 빛을 모아 초점에 집중시킨다. 물체가 초점 바깥에 있으면 실상, 안쪽이면 허상이 맺힌다. 오목 렌즈(발산 렌즈)는 빛을 퍼뜨려 항상 축소된 정립 허상이 맺힌다. 렌즈 방정식: 1/f = 1/a + 1/b, 배율 m = -b/a. 볼록 렌즈 f > 0, 오목 렌즈 f < 0.

## 씬 구성
- 화면 중앙: 3D 렌즈 + 물체 + 광선 추적 시각화
- 2가지 렌즈: 볼록(수렴) / 오목(발산)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D 렌즈와 광선 추적

### 렌즈
- 볼록 렌즈: 양쪽 볼록한 형태 — SphereGeometry 2개를 겹쳐 렌즈 형태
  - MeshPhysicalMaterial(transmission 0.9, roughness 0.05, ior 1.5), 0xeeffff
  - 가운데 두껍고 가장자리 얇음
- 오목 렌즈: 양쪽 오목한 형태
  - 가운데 얇고 가장자리 두꺼움
- 렌즈 중심 "O", 초점 "F"(양쪽), "2F" 표시
- 주축: 수평 점선, 0xaaaaaa

### 물체 (화살표)
- 수직 화살표: CylinderGeometry + ConeGeometry, 0x44ff44 (초록)
- 물체 거리 a: 슬라이더로 주축을 따라 이동
- "물체" 라벨

### 광선 추적 (3개 주요 광선)
- **광선 1 (평행 광선)**: 주축에 평행 입사 → 렌즈 통과 후 초점 F를 통과 (볼록) / F에서 발산 (오목)
  - 0xff4444 (빨강)
- **광선 2 (초점 광선)**: 초점 F를 통과하여 입사 → 렌즈 통과 후 주축에 평행
  - 0x4444ff (파랑)
- **광선 3 (중심 광선)**: 렌즈 중심 O를 통과 → 직진 (꺾이지 않음)
  - 0x44ff44 (초록)
- 3개 광선이 만나는 점 = 상의 위치

### 상
- 실상: 광선이 실제로 만남 → 실선 화살표, 0xff8844
- 허상: 광선 연장선이 만남 → 점선 화살표
- 크기/방향: 배율 m에 따라

## 볼록 렌즈: 물체 위치별 상 (오목 거울과 유사)
- a > 2f: 실상, 도립, 축소
- a = 2f: 실상, 도립, 같은 크기
- f < a < 2f: 실상, 도립, 확대
- a = f: 상 무한대
- a < f: 허상, 정립, 확대 (돋보기!)

## 오목 렌즈
- 항상: 허상, 정립, 축소 (렌즈 f < 0)
- 근시 교정 안경에 사용

## 실생활 응용 (교과서 연계)
- 돋보기(볼록, a < f): 확대 허상
- 카메라(볼록, a > 2f): 축소 실상 → 센서에 맺힘
- 눈(볼록, 수정체): 망막에 실상
- 근시 교정: 오목 렌즈 → 초점을 뒤로 이동
- 원시 교정: 볼록 렌즈 → 초점을 앞으로 이동

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "렌즈" 드롭다운: 볼록(수렴) / 오목(발산)
- "초점 거리 f" 슬라이더: 1~8 (기본값 3)
- "물체 거리 a" 슬라이더: 1~15 (기본값 6)
- "물체 높이" 슬라이더: 0.5~3 (기본값 1.5)
- "광선 표시" 토글: ON/OFF (기본값 ON)
- "상 표시" 토글: ON/OFF (기본값 ON)
- "응용" 드롭다운: 없음 / 돋보기 / 카메라 / 근시 교정 / 원시 교정
- "렌즈 방정식" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔍 렌즈의 굴절과 상
━━━━━━━━━━━━━━
렌즈: [볼록/오목]
f = [값], a = [값]

렌즈 방정식: 1/f = 1/a + 1/b
  b = af/(a-f) = [값]
  배율 m = -b/a = [값]

상의 종류:
  [실상/허상], [정립/도립], [확대/축소]

볼록 렌즈:
  a > 2f → 실상, 도립, 축소
  f < a < 2f → 실상, 도립, 확대
  a < f → 허상, 정립, 확대 (돋보기)

오목 렌즈: 항상 허상, 정립, 축소
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "볼록 렌즈는 빛을 모아 초점에 집중시키고, 물체 위치에 따라 실상/허상이 맺힙니다. 오목 렌즈는 항상 축소된 허상을 만듭니다. 1/f=1/a+1/b로 상의 위치와 크기를 계산합니다."
- 탐구 질문:
  1. "돋보기로 글씨가 확대되어 보이려면 물체를 어디에 놓아야 하나요?"
  2. "카메라로 먼 풍경을 찍으면 센서에 맺히는 상은 어떤 종류인가요?"
  3. "근시인 사람에게 오목 렌즈가 필요한 까닭을 초점과 관련지어 설명해 보세요."
- 실생활 예시: 돋보기, 카메라 렌즈, 안경(근시/원시), 현미경, 망원경
- 교과서 연계: "전자기와 양자 II편 p56~63 — 거울과 렌즈, 렌즈 방정식"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 렌즈 종류 전환 시 이전 객체 dispose() + 새 객체 생성
- 물체 위치 변경 시 광선/상 실시간 재계산
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: emq_14_lens_refraction.html
