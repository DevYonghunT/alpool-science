너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 01. 힘의 합성과 운동 예측
- 토픽: 벡터의 합성과 분해
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 힘, 속도, 가속도 등은 크기와 방향을 모두 가지는 벡터량이다. 두 벡터의 합성은 평행사변형 법칙 또는 삼각형 법칙으로 구한다. 벡터를 직교 좌표계의 x성분과 y성분으로 분해할 수 있으며, Fx = Fcosθ, Fy = Fsinθ이다. 합력의 크기 |F| = √(Fx²+Fy²), 방향 tanα = Fy/Fx이다.

## 씬 구성
- 화면 중앙: 3D 좌표계 + 벡터 화살표 (합성/분해 시각화)
- 2가지 모드: 벡터 합성 (두 벡터 → 합벡터) / 벡터 분해 (한 벡터 → x,y 성분)
- 배경: 0xeef0f5 (밝은 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (위에서 약간 비스듬히)
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 좌표계

### 좌표축
- X축: CylinderGeometry(0.02, 0.02, 12), 0xff4444 (빨강), "x" 라벨
- Y축: CylinderGeometry(0.02, 0.02, 12), 0x44ff44 (초록), "y" 라벨
- 화살촉: ConeGeometry 각 축 끝에
- 격자: GridHelper(10, 10), 0xdddddd
- 원점 "O" 라벨

### 벡터 화살표
- 벡터 A: ArrowHelper 또는 CylinderGeometry + ConeGeometry
  - 색상: 0xff4444 (빨강)
  - 크기: 길이 = 벡터 크기에 비례
  - 방향: θ_A 각도 슬라이더
  - 라벨: "A = [크기] N, θ = [각도]°"
- 벡터 B: 0x4444ff (파랑)
  - 크기/방향 독립 조절
- 합벡터 R = A + B: 0x44cc44 (초록), 점선 스타일 또는 두꺼운 화살표
  - 라벨: "R = [크기] N, α = [각도]°"

## 모드 1: 벡터 합성

### 평행사변형 법칙
- 벡터 A와 B를 원점에서 시작
- A의 끝에서 B와 평행한 점선, B의 끝에서 A와 평행한 점선
- 점선들이 만나는 점 = 합벡터 R의 끝점
- 평행사변형 면을 반투명 색상(0xffcc44 opacity 0.15)으로 채움
- 대각선 = 합벡터 R

### 삼각형 법칙
- 벡터 A를 먼저 그리고, A의 끝에서 B를 이어붙임
- 원점에서 B의 끝까지 = 합벡터 R
- 토글로 평행사변형/삼각형 전환

### 성분별 합성
- Ax + Bx = Rx (x성분 막대그래프)
- Ay + By = Ry (y성분 막대그래프)
- R = √(Rx²+Ry²), tanα = Ry/Rx
- 우측에 성분별 계산 과정 표시

## 모드 2: 벡터 분해

### 하나의 벡터를 x,y 성분으로 분해
- 벡터 F: 0xff8844 (주황), 크기 F, 각도 θ
- x성분 Fx = Fcosθ: 빨간 점선 화살표 (x축 방향)
- y성분 Fy = Fsinθ: 초록 점선 화살표 (y축 방향)
- Fx, Fy가 직각삼각형을 형성 (직각 표시)
- 직각삼각형 면을 반투명으로 채움

### 빗면 위 물체 응용
- 빗면(경사각 θ): BoxGeometry 기울어진 면, 0xaa9988
- 물체: BoxGeometry, 0x4488aa
- 중력 W = mg: 아래 방향 화살표
- 빗면 방향 성분: Wsinθ (빗면을 따라 아래)
- 빗면 수직 성분: Wcosθ (빗면에 수직)
- "중력을 빗면 방향과 수직 방향으로 분해" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 벡터 합성 / 벡터 분해 / 빗면 응용
- "벡터 A 크기" 슬라이더: 1~10 (기본값 5, 0.5 단계)
- "벡터 A 각도" 슬라이더: 0~360° (기본값 30°, 5 단계)
- "벡터 B 크기" 슬라이더: 1~10 (기본값 3)
- "벡터 B 각도" 슬라이더: 0~360° (기본값 120°)
- "표시 방법" 토글: 평행사변형 / 삼각형 (합성 모드)
- "성분 표시" 토글: ON/OFF (기본값 ON)
- "빗면 각도" 슬라이더 (빗면 모드): 10~60° (기본값 30°)
- "초기화" 버튼

  Ry = Ay + By = [값]

  F = [값]N, θ = [값]°

  Fy = Fsinθ = [값]N

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "벡터의 합성은 평행사변형 법칙으로, 벡터의 분해는 직교 좌표 성분(Fx=Fcosθ, Fy=Fsinθ)으로 수행합니다. 벡터의 합성과 분해는 힘, 속도, 가속도 등 물리량을 다룰 때 기본이 됩니다."
- 탐구 질문:
  1. "두 벡터의 사이각이 0°일 때와 180°일 때 합벡터의 크기는 각각 어떻게 되나요?"
  2. "빗면 위의 물체에 작용하는 중력을 빗면 방향과 수직 방향으로 분해해 보세요."
  3. "세 힘이 한 점에 작용하여 평형을 이루려면 어떤 조건이 필요한가요?"
- 실생활 예시: 비행기 이착륙(속도 벡터 분해), 줄다리기(힘의 합성), 보트의 강 건너기(상대 속도)
- 교과서 연계: "역학과 에너지 I편 p8~13 — 벡터의 합성과 분해, 힘의 합성"
