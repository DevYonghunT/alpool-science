너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 3. 생명 시스템 > 생명 시스템에서의 화학 반응
- 토픽: 세포 구조와 기능
- 대상: 고등학교 1학년
- 핵심 개념: 생명 시스템의 기본 단위는 세포이다. 동물과 식물 세포는 핵, 세포막, 세포질로 이루어져 있다. 세포질에는 엽록체(광합성), 미토콘드리아(세포 호흡, 에너지 생성), 리보솜(단백질 합성), 소포체(단백질 운반), 골지체(단백질 분비) 등의 세포소기관이 있다. 식물 세포에는 세포벽과 엽록체가 추가로 있다.

## 씬 구성
- 화면 중앙에 3D 세포 모형을 크게 렌더링한다 (교과서 그림 III-28, p144).
- 동물 세포와 식물 세포를 토글로 전환한다.
- 배경: 0x0a1828 (짙은 생물학 남색)
- 조명: AmbientLight(0.3) + DirectionalLight(0.5) + PointLight(핵 위치, 0xffddaa, 세기 0.8)
- 카메라: PerspectiveCamera(FOV 45) + OrbitControls (세포를 360도 회전 관찰)
- 전체적으로 반투명한 세포를 관통하여 내부를 볼 수 있게

## 동물 세포 3D 모형

### 세포막
- SphereGeometry(r=4.0, 64, 64)
- MeshPhysicalMaterial(0xffddbb, opacity 0.15, transmission 0.8, roughness 0.1)
- 인지질 이중층 패턴: 세포막 표면에 미세한 범프맵(procedural)
- 라벨: "세포막 — 세포 안팎의 물질 출입을 조절"

### 핵
- SphereGeometry(r=1.2, 32, 32)
- MeshStandardMaterial(0x6644aa 보라색, opacity 0.6)
- 핵 내부:
  - DNA: 불규칙한 실타래 형태 TubeGeometry(랜덤 곡선), 0x4488ff 파랑, 여러 가닥
  - 핵소체: SphereGeometry(r=0.3), 0xaa3366 짙은 분홍
- 핵막: SphereGeometry(r=1.25, opacity 0.1), 핵 바깥 얇은 막
- 라벨: "핵 — 유전물질(DNA)이 있으며 세포의 생명 활동을 조절"
- 클릭 시 확대: 핵 내부의 DNA 구조를 볼 수 있음

### 미토콘드리아
- CapsuleGeometry(r=0.25, length=0.8, 16, 8) (Three.js r142+ 지원). 대안: SphereGeometry(r=0.25)를 scale.y=2.0으로 타원체화 + 양쪽 반구 CylinderGeometry 결합
- 내부: 주름진 내막 표현 (CurveGeometry 지그재그 면)
- 색상: 0xff6644 (주황-빨강)
- 위치: 세포질 내 3~5개 분산 배치
- 느린 회전 애니메이션
- 라벨: "미토콘드리아 — 세포 호흡, 에너지(ATP) 생성"
- 클릭 시 확대: 내막 크리스타 구조 표시

### 리보솜
- SphereGeometry(r=0.08, 8, 8) × 30~50개
- 색상: 0x44aa44 (초록)
- 소포체 표면에 붙어있는 것(부착 리보솜) + 자유로운 것(유리 리보솜)
- 라벨: "리보솜 — 아미노산을 연결해 단백질을 합성"

### 소포체 (ER)
- 얇은 막 주머니 형태: PlaneGeometry(여러 장, 곡면) 또는 TubeGeometry(평면적 곡선)
- 조면 소포체: 표면에 리보솜(초록 점) 부착
- 활면 소포체: 매끈한 표면
- 색상: 0x88aacc (연한 파란-회색), opacity 0.3
- 핵 주위에 네트워크 형태로 배치
- 라벨: "소포체 — 리보솜에서 합성한 단백질을 운반하는 통로"

### 골지체
- 납작한 주머니 여러 장 쌓인 형태: CylinderGeometry(r=0.5, h=0.03) × 5~7장, 약간 곡면
- 한쪽에서 소포(작은 구)가 떨어져 나오는 애니메이션 (분비)
- 색상: 0xddaa44 (노란-금색)
- 라벨: "골지체 — 세포에서 합성한 단백질을 세포 밖으로 분비"

### 세포질
- 세포막과 핵 사이 공간
- 반투명 젤리 느낌: 약간의 파티클(Points, 0xffeedd, 50개)이 느리게 브라운 운동

## 식물 세포 추가 요소

### 세포벽
- SphereGeometry(r=4.3, 32, 32) 또는 BoxGeometry(8.6, 8.6, 8.6, 둥근 모서리)
- MeshStandardMaterial(0x88aa66 연초록, opacity 0.2, roughness 0.9)
- 세포막 바깥에 위치
- 라벨: "세포벽 — 세포의 형태를 유지, 보호"
- 식물 세포 직사각형 느낌 (BoxGeometry 사용)

### 엽록체
- CapsuleGeometry(r=0.3, length=0.6) 또는 타원체
- 내부: 틸라코이드 원반(CylinderGeometry r=0.15, h=0.02) × 5~8장 쌓임 (그라나)
- 색상: 0x228b22 (짙은 녹색)
- 위치: 세포질 내 4~6개
- 라벨: "엽록체 — 빛에너지를 흡수해 포도당을 합성하는 광합성이 일어남"
- 클릭 시 확대: 틸라코이드/그라나 구조

### 액포 (대형)
- SphereGeometry(r=1.5), 0x88bbee, opacity 0.2
- 세포 중앙의 큰 부분 차지
- 라벨: "액포 — 물, 영양분 저장"

## 인터랙션 (GUI 패널)
- "세포 유형" 토글: 동물 세포 / 식물 세포
  - 전환 시 세포벽, 엽록체, 액포 추가/제거 + 세포 형태 변화(구→직사각형)
- "세포소기관 선택" 드롭다운: 전체 / 핵 / 미토콘드리아 / 리보솜 / 소포체 / 골지체 / (식물) 엽록체 / 세포막
  - 선택된 소기관만 밝게, 나머지 반투명
  - 정보 패널에 기능 상세 표시
- "내부 보기" 슬라이더: 0~100% (기본값 50%)
  - 0%: 세포막 불투명 (외부만)
  - 50%: 세포막 반투명 (내부 보임)
  - 100%: 세포막 제거 (내부 완전 노출)
- "화학 반응 표시" 토글: ON/OFF (기본값 OFF)
  - ON: 각 소기관에서 일어나는 반응을 화살표로 표시
    - 엽록체: CO₂ + H₂O → 포도당 + O₂
    - 미토콘드리아: 포도당 + O₂ → CO₂ + H₂O + ATP
    - 리보솜: 아미노산 → 단백질
- "자동 회전" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 한국어 주석, 반응형, 모바일 터치 지원
- 세포막: MeshPhysicalMaterial(transmission) 유리 효과
- 세포소기관: 각각 함수로 생성, Group에 추가
- Raycaster: 세포소기관 클릭 감지 → 확대 뷰 + 정보 표시
- 동물→식물 전환: 세포벽/엽록체/액포 추가, 세포 형태 구→직사각형 Tween
- 파일명: cell_structure_function_sim.html
