너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 가속도 법칙과 등가속도 운동
- 토픽: 가속도 법칙 (뉴턴 제2법칙)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 물체의 가속도 크기는 물체에 작용하는 알짜힘의 크기에 비례하고, 물체의 질량에 반비례한다. F = ma (F: 알짜힘, m: 질량, a: 가속도). 힘의 단위 1N은 질량 1kg인 물체에 1m/s²의 가속도를 생기게 하는 힘이다.

## 씬 구성
- 화면 중앙: 3D 레일 위의 수레 + 추 실험 장치 (교과서 실험)
- 화면 우측: 실시간 a-F 그래프 또는 a-m 그래프 (Canvas 2D)
- 2가지 실험 모드: 질량 일정(힘 변화) / 힘 일정(질량 변화)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 실험 장치

### 수평 레일
- BoxGeometry(12, 0.1, 0.5), 0xaaaaaa (금속 회색)
- 레일 위에 눈금 표시 (0, 1, 2, ... 10m)

### 수레
- BoxGeometry(0.8, 0.4, 0.5), 0x4488aa (파란색)
- 바퀴 4개: CylinderGeometry(0.1, 0.1, 0.08) × 4, 0x333333
- 수레 위에 추를 올려 질량 변경 가능
- 추: CylinderGeometry(0.15, 0.15, 0.1), 0xff8844, 최대 5개 쌓기

### 도르래 + 매달린 추
- 레일 끝에 도르래: TorusGeometry(0.1, 0.03), 0x888888
- 실: CylinderGeometry(0.01, 0.01, 가변) — 수레에서 도르래를 지나 아래로
- 매달린 추: CylinderGeometry(0.15, 0.15, 0.15) × N개, 0xff4444
- 매달린 추의 수 조절 → 힘(F = 매달린 추의 무게) 변화

### 운동 애니메이션
- "실험 시작" 클릭: 매달린 추가 내려가고 수레가 가속
- 수레의 가속도 a = F/m (F: 매달린 추 무게, m: 수레+올려진 추 질량)
- 수레 위치: x = ½at²
- 수레 속도: v = at
- 실시간 속도·가속도 표시

## 실험 모드 1: 질량 일정, 힘 변화
- 수레 질량: 고정 (0.5kg)
- 매달린 추 개수: 1~5개 (각 0.1kg, 중력 ~1N)
- F = 1N, 2N, 3N, 4N, 5N
- 결과: a가 F에 비례하여 증가
- 그래프: a-F 직선 (원점 통과)

## 실험 모드 2: 힘 일정, 질량 변화
- 매달린 추: 고정 (3개, F = 3N)
- 수레 위의 추 개수: 0~4개 (각 0.5kg 추가)
- m = 0.5, 1.0, 1.5, 2.0, 2.5 kg
- 결과: a가 m에 반비례하여 감소
- 그래프: a-1/m 직선 (원점 통과) 또는 a-m 반비례 곡선

## 실시간 그래프 (Canvas 2D)
- 모드 1: a(y축) vs F(x축) — 직선, 기울기 = 1/m
- 모드 2: a(y축) vs m(x축) — 반비례 곡선
- 각 실험 결과를 점으로 표시
- 최소 3회 실험 데이터가 쌓이면 추세선 자동 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "실험 모드" 드롭다운: 질량 일정(힘 변화) / 힘 일정(질량 변화)
- "수레 위 추" 슬라이더: 0~4개 (질량 변화)
- "매달린 추" 슬라이더: 1~5개 (힘 변화)
- "실험 시작" 버튼: 수레 가속 애니메이션 재생
- "데이터 기록" 버튼: 현재 F, m, a를 그래프에 점으로 기록
- "그래프 초기화" 버튼: 기록된 데이터 삭제
- "속도 배율" 슬라이더: 0.5x~3x
- "초기화" 버튼: 수레 원위치

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🚗 가속도 법칙 (뉴턴 제2법칙)
━━━━━━━━━━━━━━━━━━
실험 모드: [질량 일정 / 힘 일정]

알짜힘 F = [값]N
질량 m = [값]kg
가속도 a = F/m = [값]m/s²

현재 속도: [값]m/s
이동 거리: [값]m

뉴턴 제2법칙:
  F = ma
  가속도 ∝ 알짜힘
  가속도 ∝ 1/질량

1N = 질량 1kg에 1m/s² 가속도를 주는 힘
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물체의 가속도 크기는 물체에 작용하는 알짜힘의 크기에 비례하고, 물체의 질량에 반비례합니다. 이를 뉴턴 제2법칙(가속도 법칙)이라 하며 F = ma로 나타냅니다."
- 탐구 질문:
  1. "같은 힘으로 무거운 물체와 가벼운 물체를 밀면 어느 쪽이 더 빨리 가속되나요? 그 까닭을 F=ma로 설명해 보세요."
  2. "수레에 매다는 추의 질량을 증가시키면 수레의 가속도는 어떻게 변하나요?"
  3. "야구공과 볼링공에 같은 힘을 가했을 때 가속도가 다른 까닭은 무엇인가요?"
- 실생활 예시: 빈 쇼핑 카트 vs 가득 찬 카트, 스포츠카(가벼움) vs 트럭(무거움), 로켓 추진(연료 소모로 질량 감소)
- 교과서 연계: "물리학 I편 p24~29 — 가속도 법칙, 등가속도 운동, 실험"
