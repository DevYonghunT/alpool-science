너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: III. 과학과 미래 사회 > 2. 과학기술의 발전과 윤리
- 토픽: 과학기술의 양면성과 윤리
- 대상: 고등학교 1학년
- 핵심 개념: 과학기술의 발전은 여러 유용성과 문제점을 동시에 가지는 양면성이 있으므로 건전한 가치 판단에 따라 활용한다. 유전자 가위(CRISPR), 인공지능(AI), 핵기술 등 첨단 과학기술은 인류에게 큰 혜택을 주지만 동시에 윤리적 문제와 위험을 내포한다.

## 씬 구성
- 화면 중앙: 3D 천칭 저울 (양면성 시각화)
  - 좌측: 유용성 (초록 아이콘들)
  - 우측: 문제점 (빨간 아이콘들)
- 3가지 과학기술 사례를 탭으로 전환
- 배경: 0x1a1a2e (어두운 보라-남색)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.6)

## 3D 천칭 저울 (양면성 시각화)

### 저울 구조
- 받침대: CylinderGeometry(0.3, 0.5, 2), 0x888888
- 가로대: BoxGeometry(6, 0.1, 0.1), 0xaaaaaa, 중심 피벗으로 회전 가능
- 좌측 접시: CircleGeometry(r=1.5), 0x44aa44 (초록, 유용성)
- 우측 접시: CircleGeometry(r=1.5), 0xff4444 (빨강, 문제점)
- 접시 위에 아이콘 블록(BoxGeometry 0.3³)이 쌓임
- 블록 수에 따라 저울이 기울어지는 물리 시뮬레이션

### 균형 메시지
- 저울이 기울면: "⚠️ 균형이 깨졌습니다"
- 균형 상태: "✅ 유용성과 문제점을 균형 있게 고려해야 합니다"

## 사례 1: 유전자 가위 (CRISPR)

### 유용성 (초록 블록)
- 3D 아이콘: DNA 이중나선 + 가위 형태
- 질병 치료: 유전 질환 교정 (겸상적혈구빈혈증 등)
- 식량 문제: 병해충 내성 작물 개발
- 약물 개발: 표적 치료제 개발 가속

### 문제점 (빨간 블록)
- 맞춤형 아기(디자이너 베이비): 외모/능력 선택
- 생태계 교란: 유전자 조작 생물 방출
- 불평등: 부유층만 기술 접근 가능

### 3D 시각화
- DNA 이중나선 (TubeGeometry 나선)
- CRISPR 가위: 두 개의 곡선 날 + 가이드 RNA
- DNA의 특정 부분을 잘라내는 애니메이션 (2초)
- 잘린 부분에 새 서열을 삽입하는 애니메이션

## 사례 2: 인공지능 (AI)

### 유용성
- 아이콘: 로봇 머리 형태
- 의료 진단: 영상 분석으로 암 조기 발견
- 자율주행: 교통사고 감소
- 연구 가속: 신약 후보 물질 탐색

### 문제점
- 일자리 대체: 자동화로 인한 실업
- 딥페이크: 허위 정보 생성
- 알고리즘 편향: 차별적 결과
- 프라이버시 침해: 개인 데이터 수집

### 3D 시각화
- 신경망 구조: SphereGeometry 노드 + CylinderGeometry 연결선, 다층 구조
- 입력층 → 은닉층 → 출력층 파티클 흐름 애니메이션
- 학습 과정: 데이터 투입 → 가중치 조절 → 결과 출력

## 사례 3: 핵기술

### 유용성
- 아이콘: 원자 모형 (전자 궤도)
- 핵발전: 대용량 전기 생산, CO₂ 미배출
- 의료: 방사선 치료, 암 진단(PET/CT)
- 연구: 방사성 동위원소를 이용한 연대 측정

### 문제점
- 핵무기: 대량 살상 무기
- 방사성 폐기물: 수만 년 관리 필요
- 사고 위험: 체르노빌, 후쿠시마

### 3D 시각화
- 원자 모형: SphereGeometry 핵 + TorusGeometry 전자 궤도 3개 + 전자 회전
- 핵분열 애니메이션: 큰 핵 → 작은 핵 2개 + 에너지 방출 파티클
- 원자력 발전소 미니어처 vs 핵폭발 버섯 구름 (양면성 대비)

## 윤리적 판단 프레임워크 (하단 패널)
- "과학기술의 발전은 양면성이 있다"
- 체크리스트:
  1. 이 기술이 인류에게 주는 혜택은 무엇인가?
  2. 이 기술의 잠재적 위험은 무엇인가?
  3. 위험을 최소화하면서 혜택을 극대화하려면?
  4. 누가 이 기술의 사용을 감독/규제해야 하는가?

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "과학기술" 드롭다운: 유전자 가위(CRISPR) / 인공지능(AI) / 핵기술
- "유용성 블록 추가" 버튼: 좌측 접시에 블록 추가
- "문제점 블록 추가" 버튼: 우측 접시에 블록 추가
- "상세 보기" 토글: 해당 기술의 3D 시각화 ON/OFF
- "윤리 체크리스트" 토글: ON/OFF
- "초기화" 버튼: 블록 전부 제거, 저울 균형

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 과학기술 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_28_science_technology_ethics.html
