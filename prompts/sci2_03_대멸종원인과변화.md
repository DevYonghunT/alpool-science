너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 지질 시대의 환경과 생물 변화
- 토픽: 대멸종의 원인과 그 이후의 변화
- 대상: 고등학교 1학년
- 핵심 개념: 지구상에 살았던 생물 중 많은 종류가 짧은 기간 동안 한꺼번에 사라지는 사건을 생물 대멸종이라고 한다. 특히 고생대 말과 중생대 말에는 생물의 종류가 크게 감소했다. 급격한 환경 변화는 생물 대멸종의 원인이 되기도 하지만, 같은 환경 변화라도 이에 적응한 생물은 다양한 종으로 진화하며 멸종한 생물을 대체해 번성할 기회를 얻는다.

## 씬 구성
- 화면을 좌/우로 분할:
  - 좌측 60%: 3D 장면 (대멸종 전/후 환경)
  - 우측 40%: 해양 생물 과의 수 변화 그래프 (Canvas 2D, 교과서 그림 I-11)
- 2가지 대멸종 시나리오를 탭으로 전환:
  - 고생대 말 대멸종 (~2.52억 년 전)
  - 중생대 말 대멸종 (~0.66억 년 전)

## 대멸종 시나리오 A: 고생대 말
- 3D 장면 (대멸종 전): 바다에 삼엽충, 완족류 등이 풍부
  - 바닥: 해저 0x556677
  - 삼엽충: BoxGeometry 타원 3개, 0x887766
  - 완족류: SphereGeometry 반구, 0xaaaaaa
  - 산호: CylinderGeometry 분기, 0xff8866
- 대멸종 원인 표시 (3가지 가설):
  - "운석(소행성) 충돌설" — 운석 SphereGeometry가 하늘에서 떨어지는 애니메이션
  - "화산 폭발설" — 화산에서 용암/가스 분출 파티클
  - "수륙 분포 변화설" — 대륙이 합쳐지는(판게아) 애니메이션
- GUI에서 3가지 가설 선택 가능
- 대멸종 후: 생물 대부분 사라지고 황량한 바다
- 그 후: 새로운 생물(공룡 등) 등장 암시

## 대멸종 시나리오 B: 중생대 말
- 3D 장면 (대멸종 전): 육지에 공룡, 바다에 암모나이트
  - 공룡: BoxGeometry 조합 실루엣, 0x668855
  - 암모나이트: TorusGeometry 나선, 0x887766
- 대멸종 원인: 운석 충돌 + 환경 급변
  - 거대 운석(SphereGeometry r=2, 0x666666) 하강 → 충돌 순간 밝은 플래시 + 파티클 1000개
  - 먼지 구름으로 하늘 어두워짐 (배경색 변화)
- 대멸종 후: 공룡·암모나이트 사라짐
- 그 후: 포유류, 속씨식물 등장

## 해양 생물 과의 수 변화 그래프 (교과서 그림 I-11)
- Canvas 2D 그래프:
  - X축: 시간 (5.39억 년 전 → 현재), 고생대/중생대/신생대 구간 표시
  - Y축: 해양 생물 과의 수 (0~900)
  - 그래프 곡선: 전체적으로 증가하되 대멸종 시기에 급감
  - 대멸종 시기를 화살표로 표시: "↓ 대멸종 시기"
  - 고생대 말에 가장 큰 규모의 대멸종 강조
- 현재 선택된 시나리오의 대멸종 시기를 빨간색으로 강조
- 교과서: "고생대가 시작된 이후 일어난 생물 대멸종 중 그 규모가 가장 컸던 시기" 표시 (p19)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 고생대 말 대멸종 / 중생대 말 대멸종
- "대멸종 가설" 드롭다운 (고생대 말에서만 활성): 운석 충돌설 / 화산 폭발설 / 수륙 분포 변화설
- "상태" 드롭다운: 대멸종 전 / 대멸종 진행 / 대멸종 후
- "자동 재생" 버튼: 전→진행→후 자동 전환 (각 5초)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
☄️ 생물 대멸종
━━━━━━━━━━━━━━
현재 시나리오:  [고생대 말 / 중생대 말]
시기:          ~[값]억 년 전
대멸종 가설:   [가설명]
멸종된 생물:   [생물 목록]
이후 번성 생물: [생물 목록]

      한꺼번에 사라지는 사건
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "지구 환경의 급격한 변화로 인한 생물계의 큰 변화는 지질 시대를 구분하는 중요한 기준입니다. 같은 환경 변화라도 이에 적응한 생물은 다양한 종으로 진화하며 번성할 기회를 얻습니다."
- 탐구 질문:
  1. "고생대 말과 중생대 말에 일어난 생물 대멸종의 원인을 설명하는 가설은 각각 무엇인가요?"
  2. "대멸종 이후 생물의 종류가 다시 증가한 까닭은 무엇일까요?"
  3. "현재 지구에서 6번째 대멸종이 진행 중이라는 주장이 있습니다. 그 근거는 무엇일까요?"
- 실생활 예시: 공룡 멸종(운석 충돌 증거 — 이리듐 층), 현재 멸종위기종(기후변화)
- 교과서 연계: "미래엔 통합과학2 I편 p18~19 — 생물 대멸종의 원인과 그 이후의 변화, 그림 I-11"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 1000개
- 파일명: sci2_03_mass_extinction.html
