너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: I. 과학의 기초 > 1. 과학의 기본량 > 시간과 공간
- 토픽: 시간과 길이 측정의 발전
- 대상: 고등학교 1학년
- 핵심 개념: 과학자들의 노력으로 측정할 수 있는 시간과 공간의 규모가 다양해지면서 인간이 경험할 수 있는 자연 세계가 크게 확장되었다. 현대에는 시간은 세슘 원자시계, 길이는 레이저를 이용해 정밀하게 측정한다.

## 씬 구성
- 화면을 수평으로 이등분: 좌측 "시간 측정의 발전", 우측 "길이 측정의 발전"
- 각 영역에 타임라인 형태로 시대별 측정 도구를 3D 모형으로 배치한다.
- 선택된 도구는 화면 중앙에 확대되어 3D로 렌더링, 작동 원리 애니메이션 재생.
- 비선택 도구는 타임라인 위에 축소된 실루엣으로 표시.
- 배경: 시대에 따라 전환 — 고대(파피루스 톤 0xd4c5a0), 중세(어두운 갈색 0x3a2a1a), 현대(깔끔한 회색 0xf0f0f0)
- 바닥: 각 시대에 맞는 텍스처 — 고대(모래/돌), 중세(나무), 현대(깨끗한 타일)
- 조명: AmbientLight(0.4) + DirectionalLight(0.8) + PointLight(선택된 도구에 스포트라이트 효과)
- 카메라: PerspectiveCamera + OrbitControls (선택된 도구를 마우스로 회전 관찰)

## 시간 측정 도구 상세 (교과서 그림 I-2, p16 기반)

### (1) 앙부일구 (조선 시대)
- 3D 모형: 반구형 오목한 그릇(SphereGeometry 반구, 내부 눈금선) + 영침(가는 CylinderGeometry 기둥)
- 색상: 0x8B7355 (청동색), 내부 눈금은 0x333333
- 작동 원리 애니메이션:
  - DirectionalLight(태양)의 위치가 슬라이더로 이동
  - 영침의 그림자가 반구 내부에 실시간으로 투영
  - 그림자 위치에 따라 "오전 10시", "정오", "오후 3시" 등 시각 표시
- 교과서: "조선 시대에는 앙부일구를 이용해 태양의 위치 변화에 따른 그림자의 길이로 시간을 측정했다" (p16)
- 측정 오차: ~수십 분 (흐린 날 사용 불가)

### (2) 괘종시계 (근대)
- 3D 모형: 나무 케이스(BoxGeometry) + 시계 문자판(CircleGeometry + 숫자) + 진자(CylinderGeometry + SphereGeometry)
- 색상: 케이스 0x5c3317 (짙은 나무색), 문자판 0xfff8dc, 진자 0xc0a000 (금색)
- 작동 원리 애니메이션:
  - 진자가 좌우로 왕복 운동 (Math.sin 기반, 주기 1초)
  - 진자 왕복에 맞춰 시계 바늘이 1초씩 이동
  - 진자의 궤적을 반투명 호(Arc)로 시각화
- 측정 오차: ~수 초/일

### (3) 석영 시계 (현대)
- 3D 모형: 손목시계 형태 — 원형 시계(CircleGeometry) + 밴드(CurveGeometry)
- 색상: 0xcccccc (은색 메탈), 문자판 0xffffff
- 작동 원리 애니메이션:
  - 석영 결정(육각기둥 CylinderGeometry, 6각형)이 미세하게 진동하는 애니메이션
  - 진동 → 전기 신호 → 초침 이동 흐름을 화살표로 표시
- 측정 오차: ~0.5초/일

### (4) 세슘 원자시계 (현대 최첨단)
- 3D 모형: 과학 장비 형태 — 큰 금속 박스(BoxGeometry) + 계기판(PlaneGeometry + 디지털 숫자) + 안테나(CylinderGeometry)
- 색상: 0x888888 (금속 회색), 계기판 0x001100 (어두운 녹색, LED 숫자)
- 작동 원리 애니메이션:
  - 세슘-133 원자 모형(SphereGeometry) 주위 전자 궤도 애니메이션
  - "9,192,631,770 Hz" 진동 주파수 카운터 표시 (빠르게 증가하는 숫자)
  - 1초 = 세슘-133 원자가 9,192,631,770번 진동하는 시간
- 교과서: "최근의 세슘 원자시계는 3000만 년이 지나야 1초의 오차가 생길 만큼 정밀해!" (p16 말풍선)
- 측정 오차: 1초 / 3000만 년

## 길이 측정 도구 상세 (교과서 그림 I-2, p16 기반)

### (1) 막대자 / 큐빗 (고대)
- 3D 모형: 나무 막대(BoxGeometry 길쭉한 직육면체) + 눈금 표시
- 색상: 0xa0785a (나무색)
- 보조 표시: 사람 팔 실루엣(팔꿈치~손끝) 옆에 큐빗 막대 비교
- 교과서: "옛날에는 손가락 마디의 길이, 발걸음 폭, 일정한 길이의 막대 등을 이용해 측정" (p16)

### (2) 미터원기 (근대, 1875년)
- 3D 모형: 백금-이리듐 합금 막대(BoxGeometry, X자 단면) + 유리 보호 케이스
- 색상: 0xd0d0d0 (백금색), 케이스 0xccddee (투명 유리)
- 보조 표시: "1m" 양 끝에 마크 표시, "1875년 미터 협약, 17개국" 라벨
- 교과서: "북극에서 적도까지 거리의 1000만분의 1을 1m로 정의" (p29)
- 측정 오차: ~0.1mm

### (3) 레이저 길이 측정기 (현대)
- 3D 모형: 레이저 장치(BoxGeometry 작은 박스) + 레이저 빔(가는 CylinderGeometry, 빨간 발광)
- 색상: 장치 0x333333, 빔 0xff0000 (빨강, emissive)
- 작동 원리 애니메이션:
  - 레이저 빔이 목표물까지 발사 → 반사 → 돌아오는 시간 측정
  - 왕복 시간과 빛의 속도(c = 299,792,458 m/s)로 거리 계산
  - "1m = 빛이 1/299,792,458초 동안 이동한 거리" (1983년 재정의)
- 교과서: "현대에는 정밀하게 길이를 측정하기 위해 레이저로 빛을 쏘아 빛이 왕복한 시간을 이용한다." (p16)
- 측정 오차: ~나노미터 수준

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "분야 선택" 드롭다운: 시간 측정 / 길이 측정
- "시대 선택" 드롭다운: 고대 / 근대 / 현대 / 최첨단
- "작동 원리 재생" 버튼: 선택된 도구의 작동 원리 애니메이션 재생
- "전체 타임라인" 버튼: 고대 → 최첨단 순서로 자동 전환 (각 4초)
- "초기화" 버튼: 현대 시대로 리셋

```

시대:         [고대/근대/현대/최첨단]
측정 원리:    [원리 한 줄 설명]
측정 오차:    [오차값]
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 성능: requestAnimationFrame 루프, 불필요한 재생성 방지
- 파일명: time_length_measurement_sim.html
