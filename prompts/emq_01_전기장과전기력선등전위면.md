너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 01. 전기장
- 토픽: 전기장과 전기력선·등전위면
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 전하 주위에 전기장이 형성되어 다른 전하에 전기력이 작용한다. 전기장의 세기 E = F/q = kQ/r². 전기력선은 (+)전하에서 나와 (-)전하로 들어가며, 조밀할수록 전기장이 강하다. 등전위면은 전위가 같은 점들을 연결한 면으로 전기력선과 항상 수직이다. 등전위면의 간격이 좁을수록 전기장의 세기가 크다.

## 씬 구성
- 화면 중앙: 3D 공간에 점전하 배치 + 전기력선 + 등전위면 시각화
- 4가지 배치: +Q 1개 / -Q 1개 / +Q+Q (같은 부호) / +Q-Q (쌍극자)
- 교과서 그림 I-1~4 (p10~13): 전기장, 전기력선, 등전위면
- 배경: 0x0a0a20 (어두운 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(0.5)

## 3D 전하 모형

### 점전하
- (+)전하: SphereGeometry(r=0.3, 16, 16), 0xff4444 (빨강), "+" 라벨
- (-)전하: SphereGeometry(r=0.3, 16, 16), 0x4444ff (파랑), "−" 라벨
- 전하량 Q: 슬라이더로 조절 (1~5)
- 드래그로 전하 위치 이동 가능 (Raycaster)

### 전기력선
- (+)전하에서 방사형으로 나가는 곡선: TubeGeometry + ConeGeometry 화살촉
- (-)전하로 들어오는 곡선
- 전기력선 수 = 전하량에 비례 (Q=1이면 8개, Q=2이면 16개)
- 색상: 0x44ff88 (초록) 또는 세기 그라데이션(파랑→빨강)
- 쌍극자: (+)에서 나와 곡선을 그리며 (-)로 들어가는 아름다운 패턴
- 같은 부호: 두 전하 사이에서 밀어내는 패턴

### 등전위면 (교과서 그림 I-4, 핵심!)
- 전위 V = kQ/r → 등전위면은 전하를 중심으로 동심구
- 3D: SphereGeometry(r_V, 32, 32) 반투명, 0xffcc44 opacity 0.08
- 여러 반지름의 등전위면 동시 표시 (V = 10V, 20V, 30V 등)
- 각 등전위면에 전위값 라벨: "V = [값]V"
- **등전위면과 전기력선은 항상 수직** ← 강조
- 등전위면 간격 좁을수록 → 전기장 강함 (전하에 가까울수록)
- 쌍극자 등전위면: 전하 사이에 안장점 형태

### 2D 단면 모드
- 3D를 xy평면으로 자른 단면 표시
- 전기력선: 2D 곡선
- 등전위면: 2D 등고선 (마치 지형 등고선)
- "등고선을 보면 주변 지형을 알 수 있듯이, 등전위면으로 전기장을 알 수 있다"

### 평행판 축전기의 전기장
- 두 평행한 금속판(+/-): PlaneGeometry × 2
- 판 사이: 균일한 전기장 (평행한 전기력선)
- 등전위면: 판에 평행한 평면들 (등간격)
- "균일한 전기장에서 등전위면은 등간격"

## 전기장 벡터 필드 (옵션)
- 공간 격자 점에 작은 화살표(ArrowHelper)로 전기장 방향/크기 표시
- 화살표 길이 = 전기장 세기에 비례
- "벡터장 표시" 토글

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "배치" 드롭다운: +Q 1개 / -Q 1개 / +Q+Q / +Q-Q(쌍극자) / 평행판
- "전하량 Q₁" 슬라이더: 1~5 (기본값 2)
- "전하량 Q₂" 슬라이더: 1~5 (2전하 모드)
- "전하 간 거리" 슬라이더: 1~8 (기본값 4)
- "전기력선 표시" 토글: ON/OFF (기본값 ON)
- "등전위면 표시" 토글: ON/OFF (기본값 ON)
- "벡터장 표시" 토글: ON/OFF (기본값 OFF)
- "2D 단면" 토글: ON/OFF
- "시험 전하 놓기" 버튼: 마우스 클릭 위치에 시험 전하 배치 → 전기장 방향으로 이동
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 전기장과 전기력선·등전위면
━━━━━━━━━━━━━━━━━
배치: [+Q / -Q / +Q+Q / 쌍극자 / 평행판]
Q₁ = [값]C, Q₂ = [값]C

전기장: E = kQ/r²
  k = 9.0×10⁹ N·m²/C²

전기력선:
  (+)전하에서 나와 (-)전하로 들어감
  조밀 → 전기장 강함
  전기력선 수 ∝ 전하량

등전위면:
  전위가 같은 점들의 연결면
  전기력선과 항상 수직
  간격 좁을수록 → 전기장 강함

전위: V = kQ/r = [값]V (선택 지점)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "전하 주위에 전기장이 형성되며, 전기력선으로 시각화합니다. 등전위면은 전위가 같은 점들을 연결한 면으로 전기력선과 항상 수직입니다. 등전위면의 간격이 좁을수록 전기장이 강합니다."
- 탐구 질문:
  1. "전기력선과 등전위면이 항상 수직인 까닭을 설명해 보세요."
  2. "등전위면의 간격이 좁을수록 전기장의 세기가 큰 까닭은 무엇인가요?"
  3. "(가)의 A, B에 형성된 전기장의 세기를 등전위면 간격으로 비교해 보세요. (교과서 해 보기)"
- 실생활 예시: 번개(전기장), 정전기, 심전도(전위차), 입자 가속기
- 교과서 연계: "전자기와 양자 I편 p10~13 — 전기장, 전기력선, 등전위면 (그림 I-1~4)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 배치 전환 시 이전 전기력선/등전위면 dispose() + 새로 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 전기력선 최대 32개 × 50세그먼트
- 파일명: emq_01_electric_field_equipotential.html
