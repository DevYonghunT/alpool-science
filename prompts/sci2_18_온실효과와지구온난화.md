너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 1. 생태계와 환경 > 지구 환경의 변화
- 토픽: 온실 효과와 지구 온난화
- 대상: 고등학교 1학년
- 핵심 개념: 지구의 대기 중에는 이산화 탄소, 메테인, 수증기 등의 온실 기체가 존재한다. 온실 기체는 지표면에서 방출되는 적외선 복사 에너지의 일부를 흡수한 뒤 다시 지표면으로 방출하여 지구의 평균 기온을 높이는데, 이를 온실 효과라 한다. 인간의 활동으로 대기 중 이산화 탄소 농도가 증가하면서 지구의 평균 기온이 상승하는 지구 온난화가 진행되고 있다.

## 씬 구성
- 화면 중앙: 3D 지구 단면도 (대기층 + 지표면 + 태양 에너지 흐름)
- 화면 우측: CO₂ 농도 vs 기온 변화 그래프 (Canvas 2D)
- 배경: 0x000020 (우주)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) + PointLight(태양 위치, 1.5)

## 온실 효과 메커니즘 3D

### 지구
- SphereGeometry(r=3, 32, 32), 지구 텍스처 (간단한 컬러맵: 대륙 0x44aa44, 바다 0x2266aa)
- 지구 반을 자른 단면도 모드도 지원

### 대기층
- SphereGeometry(r=3.5, 32, 32), 반투명 0x88aacc opacity 0.1
- 온실 기체 분자: CO₂(탄소+산소2 결합체, 0xff3333), CH₄(탄소+수소4), H₂O(산소+수소2)
- 분자들이 대기 내에서 부유하는 애니메이션
- CO₂ 농도 슬라이더에 따라 분자 수 변화 (10~100개)

### 태양
- SphereGeometry(r=0.5), emissive 0xffdd00, 화면 좌상단에 배치
- 태양 광선: PointLight + 노란 파티클 궤적

### 에너지 흐름 화살표 (핵심!)
1. **태양 복사 에너지 (가시광선)**: 노란 화살표 (태양 → 대기 → 지표면)
   - "태양 에너지가 대기를 통과하여 지표면에 도달" 라벨
   - 파티클: 노란색(0xffdd44) 구슬이 태양→지표 방향으로 이동
2. **지표면 복사 에너지 (적외선)**: 빨간 화살표 (지표면 → 위로)
   - "지표면에서 적외선 복사 에너지 방출" 라벨
   - 파티클: 빨간색(0xff4444) 구슬이 지표→위 방향으로 이동
3. **온실 기체 흡수·재방출**: 빨간 화살표가 CO₂ 분자에 부딪히면 **일부가 다시 지표로 반사**
   - "온실 기체가 적외선을 흡수한 뒤 일부를 다시 지표로 방출" 라벨
   - CO₂ 분자가 빛나는 효과 + 아래 방향 빨간 파티클
4. **우주로 빠져나가는 에너지**: 빨간 화살표 일부가 대기를 통과하여 우주로
   - "일부 에너지는 우주로 빠져나감"

### CO₂ 농도에 따른 변화
- CO₂ 슬라이더 조절 시:
  - CO₂ 분자 수 증가/감소
  - 지표로 돌아오는 적외선 파티클 비율 증가/감소
  - 지구 표면 색상 약간 변화 (온도 느낌: 정상 → 적열)
  - 우측 그래프의 현재 위치 마커 이동

## CO₂ 농도 vs 기온 변화 그래프 (우측, Canvas 2D)
- X축: 연도 (1960~2020)
- Y축 (왼쪽): CO₂ 농도 (ppm, 310~420)
- Y축 (오른쪽): 지구 평균 기온 편차 (°C, -0.2~+1.2)
- 2개 곡선:
  - CO₂ 농도: 파란 실선 (마우나로아 관측소 데이터 근사)
  - 기온 편차: 빨간 점선
  - 두 곡선이 함께 상승하는 추이 강조
- 현재 슬라이더 값에 해당하는 연도에 세로 점선 마커

## 지구 온난화의 영향
- GUI 토글로 "영향 표시":
  - 해수면 상승: 지구 모형에서 바다 면적 확대 애니메이션
  - 극지방 빙하 감소: 북극/남극의 흰색 영역 축소
  - 기상 이변: 태풍 아이콘 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "CO₂ 농도" 슬라이더: 280~500 ppm (기본값 420, 현재 수준)
- "에너지 흐름 표시" 토글: ON/OFF (기본값 ON)
- "온실 기체 표시" 토글: ON/OFF (기본값 ON)
- "영향 표시" 토글: ON/OFF (기본값 OFF)
- "단면도 모드" 토글: ON/OFF
- "자동 재생" 버튼: 280ppm(산업혁명 전)→500ppm(미래 예측) 서서히 증가
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌍 온실 효과와 지구 온난화
━━━━━━━━━━━━━━━━━━
CO₂ 농도:     [N] ppm
기온 편차:    +[N]°C
상태:         [정상 / 온난화 진행 / 심각]

온실 효과:
  온실 기체가 지표에서 방출되는 적외선의
  일부를 흡수 → 다시 지표로 방출
  → 지구 평균 기온 유지

지구 온난화:
  인간 활동(화석 연료 연소)으로 CO₂ 증가
  → 온실 효과 강화 → 기온 상승

온실 기체: CO₂, CH₄, H₂O, N₂O 등
```

화면 상단에 토글 가능한 "📖 학습 가이드" 패널:

  2. "대기 중 이산화 탄소 농도가 증가하면 지구의 평균 기온이 어떻게 변하나요?"
  3. "지구 온난화를 줄이기 위해 우리가 할 수 있는 일은 무엇인가요?"
- 실생활 예시: 파리 기후 협약, 탄소 중립, 전기차, 재생 에너지

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- CO₂ 슬라이더 변경 시 파티클 수/색상 실시간 갱신 (dispose 불필요)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
- 파일명: sci2_18_greenhouse_effect.html
