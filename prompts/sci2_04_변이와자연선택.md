너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 생물의 진화
- 토픽: 변이와 자연선택 — 딱정벌레 시뮬레이션
- 대상: 고등학교 1학년
- 핵심 개념: 같은 생물종의 개체 간에 나타나는 형질의 차이를 변이라고 한다. 다양한 변이가 있는 생물집단의 개체들 중 생존과 번식에 유리한 형질을 가진 개체가 살아남아 더 많은 자손을 남기는 과정을 자연선택이라고 한다. 자연선택은 생물집단이 변화하는 환경에 적응하도록 해 준다.

## 씬 구성
- 화면 중앙에 3D 환경(토양 표면)을 렌더링한다.
- 딱정벌레 집단이 환경(토양 색) 변화에 따라 자연선택되는 과정을 시뮬레이션한다.
- 교과서 그림: 산불로 인해 토양이 검게 변한 환경에서 딱정벌레의 몸 색깔에 따른 자연선택 (p21)
- 배경: 하늘 0x99bbdd
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (위에서 내려다보는 시점)
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 3D 환경
- 바닥(토양): PlaneGeometry(20, 20)
  - 산불 전: 밝은 갈색 0xccbb88 (밝은 토양)
  - 산불 후: 어두운 갈색-검정 0x333322 (검은 토양)
  - 전환: 슬라이더 또는 자동 재생으로 색상 보간
- 풀: 산불 전에는 초록 파티클, 산불 후에는 제거
- 새(포식자): BoxGeometry 조합 새 실루엣 1~2마리, 하늘에서 바닥을 관찰
  - 새의 시야 범위를 반투명 원뿔(ConeGeometry)로 표시

## 딱정벌레 집단 (30마리)
- 각 딱정벌레: BoxGeometry(0.3, 0.15, 0.2) 몸통 + SphereGeometry(0.08) 머리 + 다리 6개(가느다란 CylinderGeometry)
- 몸 색깔 변이: 밝은 갈색(0xccaa77) ~ 검은색(0x222211) 사이 연속 분포 (랜덤)
- 초기 분포: 30마리, 밝은색/중간색/어두운색 각 ~10마리
- 각 딱정벌레는 바닥 위에서 랜덤 방향으로 느리게 이동
- 딱정벌레에 마우스 호버 시 몸 색깔 값 툴팁 표시

## 자연선택 시뮬레이션 메커니즘
### 1단계: 산불 전 (밝은 토양)
- 밝은 토양에서 **어두운 색 딱정벌레**가 눈에 잘 띔
- 새가 어두운 색 딱정벌레를 우선 포식 (자동 애니메이션: 새가 내려와 어두운 딱정벌레 제거)
- 밝은 색 딱정벌레가 더 많이 생존

### 2단계: 산불 발생
- 화면에 불꽃 파티클 효과 (Points 200개, 주황-빨강 0xff6600~0xff2200, 2초간)
- 토양 색상이 밝은 갈색 → 검은색으로 전환 (2초 보간)
- 풀 파티클 사라짐

### 3단계: 산불 후 (검은 토양)
- 검은 토양에서 **밝은 색 딱정벌레**가 눈에 잘 띔
- 새가 밝은 색 딱정벌레를 우선 포식
- 어두운 색 딱정벌레가 더 많이 생존

### 4단계: 세대 반복
- 생존한 딱정벌레가 번식 (개체수 다시 30마리로 복원)
- 번식 시 부모 색상 ± 약간의 변이로 자손 색상 결정
- 여러 세대 반복하면 검은 토양에서는 어두운 색 딱정벌레 비율 증가
- 세대별 색상 분포 히스토그램 (HTML overlay) 표시

## 색상 분포 히스토그램 (우측 패널)
- X축: 몸 색깔 (밝은 → 어두운)
- Y축: 개체 수
- 매 세대마다 히스토그램 갱신
- 산불 전/후 분포 변화가 시각적으로 확인 가능

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "환경" 드롭다운: 밝은 토양 (산불 전) / 검은 토양 (산불 후)
- "세대 수" 슬라이더: 1~10 (기본값 1)
- "다음 세대" 버튼: 자연선택 1세대 진행
- "자동 진행" 버튼: 산불 전 3세대 → 산불 → 산불 후 5세대 자동
- "포식 속도" 슬라이더: 0.5x~3x
- "초기화" 버튼: 30마리 랜덤 색상으로 리셋

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🪲 변이와 자연선택
━━━━━━━━━━━━━━
현재 환경:     [밝은 토양 / 검은 토양]
세대:          [N]세대
총 개체수:     [N]마리
밝은색 비율:   [N]%
어두운색 비율: [N]%
포식된 개체:   [N]마리

자연선택:
  환경에 유리한 형질을 가진 개체가
  살아남아 더 많은 자손을 남긴다.
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "다양한 변이가 있는 생물집단에서 생존과 번식에 유리한 형질을 가진 개체가 살아남아 더 많은 자손을 남기는 과정을 자연선택이라고 합니다. 자연선택은 생물집단이 변화하는 환경에 적응하도록 해 줍니다."
- 탐구 질문:
  1. "산불이 나기 전 딱정벌레 집단에서 시간이 지남에 따라 개체수 비율이 가장 많이 증가하는 딱정벌레는 어떤 몸 색깔을 가지는지 설명해 보세요."
  2. "시간이 지남에 따라 딱정벌레 집단에서 개체수 비율이 증가한 까닭을 자연선택으로 설명해 보세요."
  3. "항생제 내성이 있는 세균이 증가하는 현상을 자연선택과 관련지어 설명해 보세요."
- 실생활 예시: 항생제 내성균 증가, 농약 내성 해충, 살충제 내성 모기
- 교과서 연계: "미래엔 통합과학2 I편 p20~21 — 변이의 발생과 자연선택, 딱정벌레 색깔 변화"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- InstancedMesh로 딱정벌레 30마리 효율적 렌더링
- 세대 전환 시 이전 객체 dispose() 불필요 (속성만 변경)
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_04_natural_selection_beetles.html
