너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 2. 역학 시스템 > 역학 시스템과 안전
- 토픽: 충격량과 안전장치
- 대상: 고등학교 1학년
- 핵심 개념: 충격량 = 힘 × 시간 (단위: N·s). 충격량 = 운동량의 변화량. 충격량이 같을 때 충돌 시간이 길면 물체가 받는 힘이 작아진다. 안전장치(에어백, 안전매트, 범퍼 등)는 충돌 시간을 늘려 힘을 줄이는 원리이다.

## 씬 구성
- 화면을 좌우 분할:
  - 좌측(60%): 3D 달걀 낙하 실험 장면
  - 우측(40%): 힘-시간 그래프 (교과서 그림 III-26, p135)
- 배경: 0xf5f5f5 (밝은 실험실)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6) + 그림자 활성화

## 달걀 낙하 실험 (교과서 p135 그림 III-26)

### 실험 장치
- 달걀: SphereGeometry(r=0.2, 32, 32, 약간 타원형으로 scale.y=1.3), 0xffeecc (베이지)
  - 깨진 상태: 하반구만 남김 + 노른자(SphereGeometry r=0.1, 0xffcc00) + 흰자(불규칙 형태, 0xffffff) 퍼짐
- 바닥 A: 나무판 — BoxGeometry(2, 0.1, 1), 0x8B6914 (짙은 나무색)
  - 라벨: "나무판 (딱딱함)"
- 바닥 B: 방석 — BoxGeometry(2, 0.3, 1), 0xff8888 (분홍색), 표면 부드러운 느낌
  - 라벨: "방석 (부드러움)"
- 두 바닥을 나란히 배치 (간격 3.0)
- 높이 기둥: CylinderGeometry(r=0.03, h=설정 높이) + 달걀 지지대

### 낙하 애니메이션
1. "동시 낙하" 버튼 → 양쪽 달걀이 동시에 같은 높이에서 자유 낙하
2. 나무판 충돌:
   - 충돌 시간 짧음 (0.001초 시뮬레이션 표시)
   - 큰 힘 발생: 빨간 플래시 + 충격파 원(RingGeometry 확장)
   - 달걀 깨짐: 껍데기 파편(BufferGeometry 삼각형 조각 10개, 물리 시뮬레이션으로 흩어짐) + 노른자/흰자 퍼짐
   - "❌ 깨짐! (큰 힘, 짧은 시간)" 빨간 라벨
3. 방석 충돌:
   - 충돌 시간 김 (0.05초 시뮬레이션 표시)
   - 작은 힘 발생: 약한 노란 글로우
   - 방석이 움푹 들어가는 변형 애니메이션 (vertex displacement)
   - 달걀 무사: 방석 위에 안착
   - "✅ 무사! (작은 힘, 긴 시간)" 초록 라벨

## 우측 힘-시간 그래프 (교과서 그림 III-26, p135)
- Canvas 2D 렌더링
- X축: 시간 (0~0.1초), Y축: 힘 (0~10000N)
- 나무판(A) 곡선:
  - 좁고 높은 피크 (짧은 시간, 큰 힘)
  - 색상: 0xff4444 빨강
  - 아래 넓이 = 충격량 (빗금 표시)
- 방석(B) 곡선:
  - 넓고 낮은 곡선 (긴 시간, 작은 힘)
  - 색상: 0x44aa44 초록
  - 아래 넓이 = 충격량 (빗금 표시) — **A와 동일!**
- 핵심 표시: "충격량(넓이) 동일! 시간↑ → 힘↓" 라벨
- 비교 표:
  ```
         나무판 A    방석 B
  충격량   S_A = S_B (동일)
  시간     짧음       긺
  힘       큼 ❌      작음 ✅
  ```

## 안전장치 갤러리 (교과서 그림 III-27, p135)
- "안전장치" 탭으로 전환 시:
- 4가지 안전장치를 3D로 표시:

### (1) 도로 보호 난간
- BoxGeometry(긴 직육면체) + 물결 모양 가드레일
- "자동차 충돌 시 충돌 시간↑ → 힘↓"

### (2) 안전 매트 (높이뛰기)
- BoxGeometry(3, 0.8, 2), 0x4488ff, 표면 부드러운 느낌
- 사람 실루엣이 매트에 착지하는 애니메이션
- "착지 시 충돌 시간↑ → 힘↓"

### (3) 무릎 보호대 + 안전모
- 사람 실루엣 + 보호대(BoxGeometry) + 안전모(SphereGeometry 반구)
- "충돌 시 충돌 시간↑ → 힘↓"

### (4) 에어백 (추가)
- 자동차 내부 운전석 + 에어백이 펴지는 애니메이션
- "충돌 시 에어백이 팽창 → 충돌 시간↑ → 승객이 받는 힘↓"

## 인터랙션 (GUI 패널)
- "실험 모드" 드롭다운: 달걀 낙하 / 안전장치 갤러리
- "낙하 높이" 슬라이더: 0.5 ~ 3.0 (기본값 1.0), 단위 m
- "동시 낙하" 버튼
- "슬로모션" 슬라이더: 0.01x ~ 1x (기본값 0.1x — 충돌 순간을 느리게)
- "충격량 비교" 토글: ON/OFF — 그래프에 넓이 빗금 표시
- "바닥 A 재질" 드롭다운: 나무판 / 콘크리트 / 강철 (경도↑ → 시간↓ → 힘↑)
- "바닥 B 재질" 드롭다운: 방석 / 스펀지 / 에어백 (부드러움↑ → 시간↑ → 힘↓)
- "초기화" 버튼

## 수식 패널
```
💥 충격량과 안전장치
━━━━━━━━━━━━━━━━
낙하 높이:     [h] m
충돌 전 속력:  v = √(2gh) = [값] m/s
운동량 변화:   Δp = mv = [값] kg·m/s

충격량 = 힘 × 시간 = 운동량 변화량
  J = F × Δt = Δp

나무판: Δt 짧음 → F 큼 → 달걀 깨짐! ❌
방석:   Δt 긺   → F 작음 → 달걀 무사! ✅

핵심: 충격량 같을 때
  충돌 시간↑ → 힘↓ (안전장치 원리)
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 한국어 주석, 반응형, 모바일 터치 지원
- ConvexGeometry CDN: https://unpkg.com/three@0.160.0/examples/jsm/geometries/ConvexGeometry.js
- 달걀 깨짐: mesh를 파편(ConvexGeometry 조각)으로 교체 + 물리(중력+초기 속도). ConvexGeometry 사용 불가 시 대안: IcosahedronGeometry(r=0.05~0.1, detail=0)로 불규칙 파편 표현
- 방석 변형: PlaneGeometry vertex Y displacement (충돌 지점 중심 가우시안)
- 힘-시간 그래프: Canvas 2D API, 넓이 = 충격량 빗금 표시
- 파일명: impulse_safety_device_sim.html
