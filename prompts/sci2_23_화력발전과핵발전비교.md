너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 2. 에너지 > 발전과 에너지 전환
- 토픽: 화력 발전과 핵발전 비교
- 대상: 고등학교 1학년
- 핵심 개념: 화력 발전은 석탄, 석유, 천연가스 등의 화석 연료를 태울 때 발생하는 열에너지로 물을 끓여 증기를 만들고, 이 증기로 터빈을 돌려 전기를 생산한다. 핵발전은 우라늄 등의 핵연료가 핵분열할 때 발생하는 열에너지로 물을 끓여 증기를 만든다. 열에너지를 얻는 과정을 제외하면 화력 발전과 핵발전에서 에너지 전환 과정은 비슷하다.

## 씬 구성
- 화면 상단: 화력 발전소 3D / 핵발전소 3D (탭으로 전환 또는 좌/우 비교)
- 화면 하단: 에너지 전환 경로 다이어그램 (HTML overlay)
- 교과서 그림 II-22~24 (p106~107)
- 배경: 0xddddee (밝은 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 화력 발전소 3D (교과서 그림 II-22)

### 구조물
- 보일러: BoxGeometry(2, 3, 2), 0x888888 (금속 회색), 내부에 불꽃 효과
- 석탄 저장소: BoxGeometry(1.5, 1, 1.5), 0x333333 (검정, 석탄)
- 증기관: CylinderGeometry(0.1, 0.1, 3) 여러 개, 0xaaaaaa
- 터빈: CylinderGeometry(0.5, 0.5, 1) + PlaneGeometry 날개, 0x999999
- 발전기: CylinderGeometry(0.8, 0.8, 1.5), 0x666699
- 냉각탑: ConeGeometry(역방향, 넓은 상단), 0xcccccc
- 송전탑: CylinderGeometry 기둥 + 전선
- 굴뚝: CylinderGeometry + 연기 파티클(Points, 0xaaaaaa 회색, 위로 올라감)

### 에너지 흐름 애니메이션
1. 석탄(화석 연료) 투입 → 불꽃(빨강-주황 파티클) = **화학 에너지 → 열에너지**
2. 물 → 증기(흰색 파티클 상승) = **열에너지 → 운동 에너지(증기)**
3. 증기 → 터빈 회전 = **운동 에너지**
4. 터빈 → 발전기 → 전기 = **운동 에너지 → 전기 에너지**
5. 굴뚝에서 CO₂ + 오염물질 배출 (회색 파티클)
- 각 단계 사이에 에너지 전환 화살표

## 핵발전소 3D (교과서 그림 II-23)

### 구조물
- 원자로: SphereGeometry(r=1.5) 돔 형태, 0xcccccc + 0x4444aa 건물
- 핵연료봉: CylinderGeometry(0.05, 0.05, 1) × 여러 개, 0xcccc44 (노랑)
- 증기 발생기: CylinderGeometry(0.8, 0.8, 2), 0x888888
- 터빈: 화력과 동일
- 발전기: 동일
- 냉각탑: 더 큰 ConeGeometry, 수증기 방출
- 방사능 표시: ☢ 심볼 (라벨)

### 에너지 흐름 애니메이션
1. 핵연료(우라늄) 핵분열 → 열에너지 = **핵에너지 → 열에너지**
2. 물 → 증기 = **열에너지 → 운동 에너지(증기)**
3. 증기 → 터빈 → 발전기 → 전기 (화력과 동일)
- CO₂ 배출 없음 (대신 방사성 폐기물 표시)

## 에너지 전환 경로 비교 (교과서 그림 II-24, p107)

### 화력 발전
```
화학 에너지(화석 연료) → 열에너지(증기) → 운동 에너지(터빈) → 전기 에너지(발전기)
```

### 핵발전
```
핵에너지(핵연료) → 열에너지(증기) → 운동 에너지(터빈) → 전기 에너지(발전기)
```

- **공통점 강조**: "열에너지를 얻는 과정을 제외하면 에너지 전환 과정은 비슷하다"
- 차이점: 열에너지의 근원 (화학 에너지 vs 핵에너지)

## 발전소가 인간 생활에 미치는 영향 (교과서 p108~109)
- 화력 발전:
  - 장점: 대량 생산, 경제적, 기술 성숙
  - 단점: CO₂ 배출(온실 효과), 대기 오염(미세먼지), 화석 연료 고갈
- 핵발전:
  - 장점: 대량 생산, CO₂ 미배출, 연료 효율 높음
  - 단점: 방사성 폐기물, 사고 위험(체르노빌, 후쿠시마), 핵폐기물 처리
- GUI 토글로 장단점 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "발전 종류" 드롭다운: 화력 발전 / 핵발전 / 비교 모드
- "에너지 흐름 애니메이션" 버튼: 에너지 전환 과정 재생
- "에너지 전환 경로" 토글: ON/OFF (기본값 ON)
- "장단점 표시" 토글: ON/OFF
- "터빈 속도" 슬라이더: 0~5 RPM (기본값 2)
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 발전 종류 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
