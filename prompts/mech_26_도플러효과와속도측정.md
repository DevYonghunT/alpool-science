너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 03. 도플러 효과와 속도 측정
- 토픽: 도플러 효과와 속도 측정
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 도플러 효과는 파원(음원)과 관측자 사이의 상대적 운동에 의해 관측되는 파동의 진동수가 변하는 현상이다. 음원이 관측자에게 다가오면 진동수가 높아지고(고음), 멀어지면 낮아진다(저음). f' = f × (v ± v_관측자)/(v ∓ v_음원). 스피드건(레이더)은 도플러 효과를 이용하여 물체의 속도를 측정한다.

## 씬 구성
- 화면 중앙: 3D 도로 + 음원(자동차/구급차) + 관측자 + 원형 음파
- 화면 우측: 진동수/파장 변화 그래프 (Canvas 2D)
- 3가지 모드: 음원 이동 / 관측자 이동 / 스피드건
- 배경: 0xddeeff (야외)
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls (위에서 내려다보기)
- 조명: AmbientLight(0.5) + DirectionalLight(0.7)

## 3D 도플러 효과

### 도로
- BoxGeometry(30, 0.05, 4), 0x555555 (아스팔트)
- 중앙선: 노란 점선

### 음원 (구급차/자동차)
- BoxGeometry 조합: 몸통(1.5×0.6×0.7, 0xff4444 빨강 또는 0xffffff 흰색+빨강 십자)
- 바퀴 4개: CylinderGeometry
- 속도 v_s: 슬라이더로 조절 (0~30m/s)
- 도로를 따라 이동하는 애니메이션

### 관측자
- BoxGeometry 조합 사람 실루엣, 0x4488aa
- 도로 옆에 고정 (또는 이동 모드에서 이동)
- 귀 아이콘: SphereGeometry(r=0.1), 0xffcc88

### 원형 음파 (핵심 시각화!)
- 음원에서 방출되는 동심원: RingGeometry × 여러 겹
  - 색상: 0x44ff88 반투명 (opacity 0.2)
  - 일정 시간 간격으로 새 원 생성 → 확장
- **음원이 정지**: 동심원이 균일하게 퍼짐 (모든 방향 같은 파장)
- **음원이 이동**: 
  - 진행 방향(앞): 동심원이 압축 → 파장↓ → 진동수↑ (고음)
  - 반대 방향(뒤): 동심원이 늘어남 → 파장↑ → 진동수↓ (저음)
- 동심원의 중심이 음원 위치를 따라 이동하므로 비대칭 패턴
- **이것이 도플러 효과의 핵심 시각화!**

### 소리 피드백 (옵션)
- Web Audio API로 실제 음을 생성하여 도플러 효과를 청각적으로 체험
- 음원이 다가올 때: 높은 음 → 멀어질 때: 낮은 음
- 구급차 사이렌 효과

## 모드 1: 음원 이동, 관측자 정지
- f' = f × v/(v - v_s) (다가올 때, f' > f)
- f' = f × v/(v + v_s) (멀어질 때, f' < f)
- 관측자 위치에서 관측되는 진동수를 실시간 표시

## 모드 2: 관측자 이동, 음원 정지
- f' = f × (v + v_o)/v (다가갈 때, f' > f)
- f' = f × (v - v_o)/v (멀어질 때, f' < f)

## 모드 3: 스피드건 (레이더)
- 경찰관 + 스피드건(BoxGeometry 작은 장치, 0x333333)
- 전자기파 발사 → 자동차에서 반사 → 돌아온 파동의 진동수 변화 측정
- Δf = 2v_s × f/c (이중 도플러: 보냄+되돌아옴)
- 속도 표시: "측정 속도: [값]km/h"
- 과속 여부: >60km/h 이면 "⚠️ 과속!" 빨간 경고

## 진동수/파장 변화 그래프 (우측 Canvas 2D)
- X축: 관측자와 음원 사이 거리
- Y축: 관측 진동수 f'
- 음원이 지나갈 때: f'가 높은 값 → 순간적으로 f → 낮은 값으로 변화
- 파장 비교: 앞쪽(짧은 파장) vs 뒤쪽(긴 파장) 막대그래프

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 음원 이동 / 관측자 이동 / 스피드건
- "음원 속도 v_s" 슬라이더: 0~30m/s (기본값 15)
- "관측자 속도 v_o" 슬라이더 (모드 2): 0~20m/s
- "음원 진동수 f" 슬라이더: 200~1000Hz (기본값 440)
- "음속 v" 표시: 340m/s (고정)
- "소리 재생" 토글: ON/OFF (Web Audio)
- "시뮬레이션 시작" 버튼
- "음파 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🚑 도플러 효과와 속도 측정
━━━━━━━━━━━━━━━━━
모드: [음원 이동/관측자 이동/스피드건]
v_음원 = [값]m/s, v_관측자 = [값]m/s
f_원래 = [값]Hz, v_음속 = 340m/s

도플러 효과:
  다가올 때: f' = f × v/(v-v_s) = [값]Hz ↑
  멀어질 때: f' = f × v/(v+v_s) = [값]Hz ↓

현재 관측 진동수: f' = [값]Hz
파장 변화: λ' = v/f' = [값]m

스피드건:
  Δf = 2v_s·f/c → v_s = [값]m/s
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "음원과 관측자 사이의 상대적 운동에 의해 관측되는 진동수가 변하는 현상을 도플러 효과라 합니다. 다가오면 진동수↑(고음), 멀어지면 진동수↓(저음)."
- 탐구 질문:
  1. "구급차가 다가올 때 사이렌 소리가 높게 들리는 까닭을 도플러 효과로 설명해 보세요."
  2. "음원의 속도가 음속에 가까워지면 앞쪽 파장은 어떻게 되나요? (힌트: 충격파/소닉붐)"
  3. "천문학에서 별의 적색 편이(red shift)는 무엇을 의미하나요?"
- 실생활 예시: 구급차 사이렌, 스피드건(과속 단속), 우주 팽창(적색 편이), 도플러 초음파(혈류 측정)
- 교과서 연계: "역학과 에너지 III편 p108~113 — 도플러 효과와 속도 측정"

- 성능: requestAnimationFrame 루프, RingGeometry 최대 30개
