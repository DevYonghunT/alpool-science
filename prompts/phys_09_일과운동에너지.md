너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 2. 에너지와 열 > 일과 에너지
- 토픽: 일과 운동 에너지
- 대상: 고등학교 (선택과목)
- 핵심 개념: 물체에 일정한 힘 F를 가해 힘의 방향으로 거리 s만큼 이동시키면 힘이 물체에 한 일 W = Fs이다. 일의 단위는 J(줄)이며, 1J = 1N·m이다. 운동하는 물체가 가지는 에너지를 운동 에너지라 하며, Ek = ½mv²이다. 물체에 한 일은 물체의 운동 에너지 변화량과 같다(일-운동 에너지 정리): W = ΔEk = ½mv² - ½mv₀².

## 씬 구성
- 화면 좌측: 3D 수평면 위의 블록 + 힘 적용
- 화면 우측: 에너지 막대그래프 + 일-에너지 관계식 (HTML overlay)
- 2가지 모드: 수평면에서 밀기 / 빗면에서 끌어올리기
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 모드 1: 수평면에서 밀기

### 3D
- 수평면: BoxGeometry(15, 0.1, 3), 0xaaaaaa
- 블록: BoxGeometry(0.8, 0.6, 0.6), 0x4488aa, 질량 m 표시
- 힘 화살표(F): 빨간 화살표, 블록 뒤에서 밀기 방향
- 이동 거리(s): 초록 점선 + 양쪽 화살표로 표시

### 애니메이션
- "힘 적용" 클릭: 힘 F로 거리 s만큼 밀기
- 블록이 s만큼 이동 (등가속도 → 등속으로 단순화하여 시각화)
- 일 W = Fs 계산하여 표시
- 운동 에너지 변화: Ek = ½mv² 계산
- W = ΔEk 확인

### 마찰 없는 경우 vs 마찰 있는 경우
- 마찰 토글: ON/OFF
- 마찰력: f 화살표 (보라색, 이동 반대 방향)
- 마찰 시: W_net = Fs - fs = ΔEk (순수한 일 = 힘의 일 - 마찰의 일)

## 모드 2: 빗면에서 끌어올리기

### 3D
- 빗면: BoxGeometry 기울어진 면, 0xaa9988
- 블록을 빗면 위로 끌어올리는 힘 F (빗면 방향)
- 높이 h만큼 올라감
- 일 = Fs (빗면 방향), 위치 에너지 변화 = mgh
- 빗면의 이점: 작은 힘으로 높이 올릴 수 있음 (단, 거리 s > h)

## 에너지 막대그래프 (우측)
- 막대 3개:
  - 일(W = Fs): 노란 막대
  - 운동 에너지(Ek = ½mv²): 빨간 막대
  - 위치 에너지(Ep = mgh): 파란 막대 (빗면 모드)
- 슬라이더 조작에 따라 실시간 업데이트
- W = ΔEk (일-운동 에너지 정리) 강조

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 수평면 밀기 / 빗면 끌어올리기
- "힘 F" 슬라이더: 1~20N (기본값 10, 1 단계)
- "이동 거리 s" 슬라이더: 1~10m (기본값 3, 0.5 단계)
- "질량 m" 슬라이더: 1~10kg (기본값 2, 0.5 단계)
- "마찰" 토글: ON/OFF (기본값 OFF)
- "마찰 계수" 슬라이더 (마찰 ON 시): 0.1~0.5 (기본값 0.2)
- "빗면 각도" 슬라이더 (모드 2): 10~60° (기본값 30°)
- "힘 적용" 버튼: 애니메이션 재생
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 일과 운동 에너지
━━━━━━━━━━━━━━
모드: [수평면 / 빗면]
힘 F = [값]N
이동 거리 s = [값]m
질량 m = [값]kg

일: W = Fs = [값]×[값] = [값]J
운동 에너지: Ek = ½mv² = [값]J
속도: v = [값]m/s

일-운동 에너지 정리:
  W = ΔEk = ½mv² - ½mv₀²
  물체에 한 일 = 운동 에너지 변화량

단위: 1J = 1N·m = 1kg·m²/s²
```
