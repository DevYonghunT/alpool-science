너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: III. 과학과 미래 사회 > 1. 과학의 유용성과 빅데이터의 활용 > 빅데이터의 활용
- 토픽: 빅데이터의 활용
- 대상: 고등학교 1학년
- 핵심 개념: 빅데이터란 기존의 방법으로는 수집·저장·분석하기 어려운 규모가 매우 큰 데이터를 뜻한다. 빅데이터 분석을 통해 감염병의 전파 경로를 예측하거나, 기상 정보를 분석하고, 교통 흐름을 최적화하는 등 다양한 분야에서 활용된다. 빅데이터 활용에는 장점과 문제점이 있으며, 개인 정보 보호와 알고리즘 편향 등의 윤리적 문제가 있다.

## 씬 구성
- 화면 중앙: 3D 데이터 시각화 (감염병 전파 네트워크 또는 기상 데이터 지도)
- 3가지 활용 사례를 탭으로 전환
- 배경: 0x0a1020 (어두운 디지털 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) + PointLight(0.5)

## 사례 1: 감염병 전파 추적

### 3D 네트워크 그래프
- 노드(사람): SphereGeometry(r=0.15), 50~100개
  - 건강: 0x44aa44 (초록)
  - 감염: 0xff4444 (빨강)
  - 회복: 0x4488ff (파랑)
- 간선(접촉): CylinderGeometry 가느다란 선, 0x444466, 접촉자 연결
- 3D 공간에 클러스터 형태로 분포 (가족, 직장, 학교 그룹)

### 전파 시뮬레이션
- "시작" 클릭: 최초 감염자(빨강)에서 접촉자로 전파 시작
- 시간 경과에 따라 빨간 노드가 확산
- 전파 경로: 간선이 빨간색으로 변하며 전파 방향 표시
- AI 추적: 전파 패턴 분석하여 "슈퍼 전파자" 노드 강조 (큰 노란 원)
- 격리 조치: 특정 노드 클릭하면 격리(회색, 간선 끊김) → 전파 차단 효과 확인

### 통계 패널
- 감염자 수 곡선 (시간별)
- 격리 조치 전/후 비교

## 사례 2: 기상 데이터 분석

### 3D 지구 기상 데이터
- SphereGeometry(r=3) 지구 + 위도/경도 격자
- 기상 데이터 포인트: SphereGeometry 작은 구, 격자 위에 배치
  - 온도: 파랑(차가움) ~ 빨강(따뜻함) 컬러맵
  - 크기: 데이터 값에 비례
- 구름: 반투명 SphereGeometry(r=3.1), 흰색 패턴
- 태풍 경로: TubeGeometry 곡선, 빨간색
- 시간 슬라이더: 1일~30일간의 기상 변화 재생

### 빅데이터 분석 결과
- "이 데이터에서 AI가 패턴을 발견: 태풍 경로 예측" 라벨
- 예측 경로 vs 실제 경로 비교

## 사례 3: 의약품 개발

### 3D 분자 구조
- 약물 분자: SphereGeometry 원자들 + CylinderGeometry 결합
- 단백질 표적: 큰 복잡한 구조 (여러 구의 조합)
- 도킹 시뮬레이션: 약물 분자가 단백질 활성 부위에 결합하는 애니메이션
- "빅데이터 + AI로 수백만 개의 후보 물질 중 최적 약물을 찾음" 라벨

## 빅데이터의 장점과 문제점 (하단 패널)
### 장점
- 정확한 예측과 의사 결정
- 감염병 조기 대응
- 맞춤형 서비스

### 문제점
- 개인 정보 유출 위험
- 알고리즘 편향 (편견 학습)
- 데이터 독점

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "활용 사례" 드롭다운: 감염병 전파 추적 / 기상 데이터 분석 / 의약품 개발
- "시뮬레이션 시작" 버튼
- "시간" 슬라이더: 사례별로 다른 시간 범위
- "격리 조치" 토글 (사례 1): ON/OFF
- "데이터 밀도" 슬라이더 (사례 2): 50~500 포인트
- "장단점 표시" 토글: ON/OFF
- "초기화" 버튼

  2. "빅데이터 활용의 장점과 문제점을 각각 2가지씩 써 보세요."

- 실생활 예시: 코로나19 역학조사(QR 체크인), 넷플릭스 추천 알고리즘, 구글 독감 트렌드, 자율주행차
- 교과서 연계: "미래엔 통합과학2 III편 p136~139 — 빅데이터의 활용, 장점과 문제점"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 노드 최대 100개 (InstancedMesh)
- 파일명: sci2_27_bigdata_application.html
