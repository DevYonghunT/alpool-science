너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 평형과 구조물의 안정성
- 토픽: 알짜힘과 힘의 평형
- 대상: 고등학교 (선택과목)
- 핵심 개념: 한 물체에 여러 힘이 동시에 작용할 때 이 힘들과 같은 효과를 내는 하나의 힘을 알짜힘이라고 한다. 물체에 작용하는 알짜힘이 0인 상태를 힘의 평형이라고 한다. 힘의 평형일 때 운동하던 물체는 속도가 일정한 운동을 계속하고, 정지해 있던 물체는 계속 정지해 있다. 이를 관성 법칙 또는 뉴턴 제1법칙이라고 한다.

## 씬 구성
- 화면 중앙: 3D 물체(블록)에 여러 힘(화살표)이 작용하는 모습
- 화면 우측: 힘의 벡터 합성 다이어그램 (HTML overlay 또는 3D)
- 3가지 모드: 같은 방향 / 반대 방향 / 임의의 2D 방향
- 배경: 0xeef0f5 (밝은 실험실)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 요소

### 물체 (블록)
- BoxGeometry(1.5, 1, 1), MeshStandardMaterial(0x4488aa, roughness 0.6)
- 격자 바닥(GridHelper) 위에 배치
- 바닥: PlaneGeometry(20, 20), 0xdddddd
- 물체에 "m" 라벨 (질량 표시)

### 힘 화살표
- 각 힘: ArrowHelper 또는 CylinderGeometry(화살대) + ConeGeometry(화살촉)
- 화살표 길이 = 힘의 크기에 비례 (1N = 0.5 단위 길이)
- 색상: F₁ = 0xff4444 (빨강), F₂ = 0x4444ff (파랑), 알짜힘 = 0x44cc44 (초록, 점선)
- 화살표 시작점: 물체 중심
- 화살표 위에 힘의 크기 라벨: "[N]N"

### 알짜힘 화살표
- 점선 스타일 (LineDashedMaterial) 또는 반투명 화살표
- 색상: 0x44cc44 (초록)
- "F = [값]N" 라벨

## 3가지 모드

### 모드 1: 같은 방향 (교과서 그림 I-1(가))
- F₁(빨강, 오른쪽 3N) + F₂(파랑, 오른쪽 5N)
- 알짜힘 F = F₁ + F₂ = 8N (오른쪽)
- 물체가 오른쪽으로 가속

### 모드 2: 반대 방향 (교과서 그림 I-1(나))
- F₁(빨강, 오른쪽 3N) + F₂(파랑, 왼쪽 5N)
- 알짜힘 F = F₂ - F₁ = 2N (왼쪽)
- 물체가 왼쪽으로 가속
- **특수 경우**: F₁ = F₂이면 알짜힘 = 0 → 힘의 평형 → 물체 정지/등속

### 모드 3: 2D 임의 방향
- F₁과 F₂의 방향을 각도 슬라이더로 조절
- 벡터 합성: 평행사변형 법칙으로 알짜힘 계산
- 평행사변형을 반투명 면으로 3D 시각화
- 알짜힘 = 대각선 화살표

## 힘의 평형 시각화
- F₁ = -F₂일 때 (크기 같고 방향 반대):
  - 알짜힘 = 0 표시
  - "⚖️ 힘의 평형!" 라벨
  - 물체 주변에 초록 글로우 효과
  - "물체는 정지 또는 등속 직선 운동" 설명 텍스트
- 관성 법칙 애니메이션:
  - 평형 상태에서 정지한 물체 → 정지 유지
  - 평형 상태에서 운동 중인 물체 → 등속 유지 (물체가 일정 속도로 이동)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 같은 방향 / 반대 방향 / 2D 임의 방향
- "F₁ 크기" 슬라이더: 0~10N (기본값 3, 0.5 단계)
- "F₂ 크기" 슬라이더: 0~10N (기본값 5, 0.5 단계)
- "F₁ 방향" 슬라이더 (모드 3): 0~360° (기본값 0°)
- "F₂ 방향" 슬라이더 (모드 3): 0~360° (기본값 180°)
- "알짜힘 표시" 토글: ON/OFF (기본값 ON)
- "운동 시뮬레이션" 버튼: 알짜힘에 따라 물체가 이동하는 애니메이션 재생
- "평형 만들기" 버튼: F₂를 자동으로 -F₁로 설정하여 평형 상태 시연
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚖️ 알짜힘과 힘의 평형
━━━━━━━━━━━━━━━
F₁ = [값]N ([방향])
F₂ = [값]N ([방향])
━━━━━━━━━
알짜힘 F = [값]N ([방향])

힘의 평형 조건:
  알짜힘 = 0 (F₁ + F₂ = 0)

관성 법칙 (뉴턴 제1법칙):
  알짜힘 = 0일 때
  정지한 물체 → 계속 정지
  운동하는 물체 → 등속 직선 운동
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 화살표 dispose() + 새 화살표 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: phys_01_net_force_equilibrium.html
