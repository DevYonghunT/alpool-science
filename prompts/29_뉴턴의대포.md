너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 2. 역학 시스템 > 중력의 작용
- 토픽: 뉴턴의 대포 (원운동과 중력)
- 대상: 고등학교 1학년
- 핵심 개념: 높은 산꼭대기에서 물체를 수평으로 던질 때, 속력이 느리면 포물선으로 낙하하고, 속력이 빠르면 더 멀리 날아가며, 특정 속력에서는 지구 주위를 원운동한다(인공위성). 달이 지구로 떨어지지 않는 것도 같은 원리이다. 인공위성의 가속도 방향은 지구 중심 방향(중력 방향)이다.

## 씬 구성
- 전체 화면에 3D 지구와 대포를 렌더링한다 (교과서 그림 III-18, p130).
- 지구: SphereGeometry(r=5.0, 64, 64), 표면에 대륙/바다 텍스처
- 산꼭대기: 지구 표면(위도 45°N, 경도 0°) 위에 돌출된 산
  - ConeGeometry(radiusBottom=0.5, radiusTop=0.1, height=0.8, radialSegments=8)
  - MeshStandardMaterial(0x886644 암석 갈색, roughness 0.9)
  - 산 꼭대기에 눈: SphereGeometry(r=0.15), 0xffffff, 산 정상 부착
- 대포: Group으로 조립
  - 포신: CylinderGeometry(r=0.08, h=0.5, 16), MeshStandardMaterial(0x444444 짙은 회색, metalness 0.7, roughness 0.3)
  - 포구: CylinderGeometry(r=0.1, h=0.05, 16), 0x333333, 포신 끝에 부착
  - 포대: BoxGeometry(0.3, 0.15, 0.2), 0x553322 (나무 갈색)
  - 바퀴: CylinderGeometry(r=0.1, h=0.05, 16) × 2, 0x443322
  - 대포 방향: 지구 표면 접선 방향(수평)으로 배치
  - 발사 시 포구에서 연기 파티클(Points 20개, 0xcccccc, 0.5초 후 소멸) 효과
- 포탄: SphereGeometry(r=0.06, 16, 16), MeshStandardMaterial(0xffdd44, metalness 0.5, roughness 0.3, emissive 0x443300)
  - 포탄 뒤에 꼬리(궤적선): Line + BufferGeometry, 색상은 궤적 유형에 따라 변경
- 배경: 우주 0x000011
  - 별 파티클: Points(1500개, BufferGeometry, 랜덤 위치(구형 분포 r=50~100), 크기 0.5~2.0, 0xffffff~0xffffcc)
- 달: SphereGeometry(r=0.8, 32, 32), MeshStandardMaterial(0xcccccc, roughness 0.9, bumpScale 0.05)
  - 크레이터: 절차적 범프맵 또는 noise
  - 궤도: TorusGeometry(r=20, tube=0.01), 0x333333, opacity 0.2
  - 공전 애니메이션: 지구 주위 원운동 (0.001 rad/frame)
  - 지구→달 중력 화살표(토글): ArrowHelper(0xff4444, 길이 2)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls, 초기 위치 (0, 8, 15), lookAt(0, 0, 0)
- 조명: AmbientLight(0.25) + DirectionalLight(0.6, position(10, 10, 5)) + PointLight(지구 내부, 0x224488, 세기 0.3)

## 발사 시뮬레이션 (교과서 그림 III-18)

### 발사 속력에 따른 5가지 궤적
- 속력 슬라이더(0~10 km/s)에 따라 궤적이 변화:

| 속력 범위 | 궤적 | 색상 | 설명 |
|----------|------|------|------|
| 0~3 km/s | 짧은 포물선 | 0xff4444 빨강 | 가까운 곳에 낙하 |
| 3~5 km/s | 긴 포물선 | 0xff8844 주황 | 먼 곳에 낙하 |
| 5~7 km/s | 매우 긴 포물선 | 0xffcc44 노랑 | 지구 반대편 근처까지 |
| ~7.9 km/s | **원 궤도** | 0x44ff44 초록 | 지구 주위 원운동! (인공위성) |
| 8~10 km/s | 타원 궤도 | 0x4488ff 파랑 | 타원형 궤도 |
| >11.2 km/s | 탈출 | 0xaa44ff 보라 | 지구 중력 탈출 |

### 궤적 애니메이션
1. "발사" 버튼 → 대포에서 포탄(SphereGeometry r=0.06, 0xffdd44) 발사
2. 포탄이 궤적을 따라 이동 (속력에 따라 다른 경로)
3. 궤적 뒤에 궤적선(Line, 해당 색상) 그려짐
4. 낙하 시: 지표면 충돌 → 작은 폭발 효과
5. 원 궤도 시: 포탄이 지구 주위를 계속 회전 → "인공위성 궤도!" 라벨
6. 탈출 시: 포탄이 화면 밖으로 사라짐

### 물리 계산
- 중력: F = GMm/r² (지구 중심 방향)
- 수치 적분: Euler method 또는 Verlet integration
  - Δv = -GM/r² × r̂ × Δt
  - Δx = v × Δt
- G × M_earth ≈ 3.986×10¹⁴ m³/s²
- 지구 반지름: 6,371 km

### 중력 방향 표시
- 포탄에서 지구 중심 방향으로 빨간 화살표(ArrowHelper): "중력"
- 포탄의 운동 방향으로 초록 화살표: "운동 방향"
- 원 궤도에서: 중력 방향 = 구심력 방향 (항상 지구 중심) → "가속도 = 중력 방향"
- 교과서: "인공위성의 원운동은 지구 중심 방향의 가속도 운동이다" (p130)

## 달의 운동 연결
- 달의 공전: 지구 주위를 원운동하는 달 표시
- 지구→달 방향 중력 화살표
- "달이 지구로 떨어지지 않는 까닭: 달도 인공위성과 같은 원리로 지구 주위를 원운동"
- 교과서 그림 III-19 (p130): "뉴턴은 달이 지구로 떨어지지 않고 지구 주위를 공전하는 까닭을 중력에 의한 운동으로 설명했다"

## 인터랙션 (GUI 패널)
- "발사 속력" 슬라이더: 0 ~ 12 (기본값 3), 단위 km/s, 0.1 단계
- "발사" 버튼
- "궤적 유형 표시" 라벨: 슬라이더 값에 따라 실시간 표시 — "포물선 낙하" / "원 궤도(인공위성)" / "탈출"
- "중력/속도 화살표" 토글: ON/OFF (기본값 ON)
- "이전 궤적 유지" 토글: ON/OFF (기본값 OFF)
  - ON: 여러 속력으로 발사한 궤적을 모두 남겨서 비교
- "달 표시" 토글: ON/OFF (기본값 ON)
- "시점" 드롭다운: 외부 관찰 / 포탄 추적(카메라가 포탄 따라감)
- "슬로모션" 슬라이더: 0.1x ~ 5x
- "초기화" 버튼

## 수식 패널
```
🚀 뉴턴의 대포
━━━━━━━━━━━━
발사 속력:     [값] km/s
궤적 유형:     [포물선/원 궤도/탈출]

원 궤도 조건: v = √(GM/r)
  = √(3.986×10¹⁴ / 6.371×10⁶)
  ≈ 7.9 km/s (제1우주속도)

탈출 속도: v = √(2GM/r) ≈ 11.2 km/s

인공위성 원운동:
  구심력 = 중력
  mv²/r = GMm/r²
  → 가속도 방향 = 지구 중심 방향
```

- 핵심 원리: "지구가 둥글기 때문에 물체를 특정 속력(약 7.9 km/s)으로 수평 발사하면 지표면에 닿지 않고 지구 주위를 원운동합니다. 이것이 인공위성의 원리입니다."

  3. "발사 속력을 11.2 km/s 이상으로 하면 어떻게 되나요?" (답: 지구 중력 탈출)

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 한국어 주석, 반응형, 모바일 터치 지원
- 궤도 물리: Verlet integration (F = -GMm/r² × r̂), 고정 dt = 0.01초 (requestAnimationFrame deltaTime과 별도로, 물리 안정성 위해 고정 시간 간격 사용. 1프레임당 물리 스텝 10회 반복)
- 궤적선: BufferGeometry + Line, 매 프레임 점 추가. 최대 2000점 도달 시 오래된 점부터 shift()로 제거(링 버퍼 방식). 원 궤도에서는 1바퀴 완성 후 궤적선 유지(더 이상 추가 안 함).
- 충돌 감지: 포탄 위치(Vector3)의 length()와 지구 표면 반지름(5.0) 비교. length() < 5.0이면 충돌 처리(정지 + 폭발 효과). 탈출 감지: length() > 50.0이면 화면 밖 처리.
- 파일명: newtons_cannon_sim.html
