너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 06. 기본 입자와 표준 모형
- 토픽: 표준 모형과 기본 입자
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 표준 모형은 자연의 기본 입자와 이들 사이의 상호작용을 기술하는 이론이다. 물질 입자는 쿼크(6종: u, d, c, s, t, b)와 렙톤(6종: e, μ, τ, νₑ, ν_μ, ν_τ)으로 구성된다. 양성자(uud)와 중성자(udd)는 쿼크로 이루어져 있다. 힘을 매개하는 입자는 게이지 보손(광자, W/Z 보손, 글루온)이며, 힉스 보손은 입자에 질량을 부여한다. 4가지 기본 힘: 중력, 전자기력, 강한 핵력(강력), 약한 핵력(약력).

## 씬 구성
- 화면 좌측 55%: 3D 입자 시각화 (회전하는 입자 모형)
- 화면 우측 45%: 표준 모형 표 + 상호작용 다이어그램 (HTML overlay)
- 3가지 보기: 표준 모형 개요 / 쿼크와 하드론 / 기본 상호작용
- 배경: 0x050515
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(1.0, 카메라 위치)

## 보기 1: 표준 모형 개요

### 3D 입자 배열
- 3세대 × 4입자 격자로 기본 입자 배열 (3D 공간에)
- 각 입자: SphereGeometry(r=0.3~0.5, 질량에 비례), 고유 색상 + 라벨

#### 쿼크 (6종, 윗줄)
- 업(u): r=0.25, 0xff4444, 전하 +2/3
- 다운(d): r=0.27, 0x4488ff, 전하 -1/3
- 참(c): r=0.45, 0xff8844, 전하 +2/3
- 이상(s): r=0.30, 0x44cc88, 전하 -1/3
- 꼭대기(t): r=0.60, 0xff6688, 전하 +2/3 (가장 무거움!)
- 바닥(b): r=0.50, 0x8866ff, 전하 -1/3

#### 렙톤 (6종, 아랫줄)
- 전자(e): r=0.15, 0xffff44, 전하 -1
- 뮤온(μ): r=0.30, 0xffcc44, 전하 -1
- 타우(τ): r=0.40, 0xffaa44, 전하 -1
- 전자 중성미자(νₑ): r=0.08, 0xaaddff opacity 0.5, 전하 0
- 뮤온 중성미자(ν_μ): r=0.08, 0xaaddff opacity 0.5, 전하 0
- 타우 중성미자(ν_τ): r=0.08, 0xaaddff opacity 0.5, 전하 0

#### 게이지 보손 (힘 매개 입자, 별도 영역)
- 광자(γ): 파동 형태 이펙트, 0xffff44 (전자기력)
- 글루온(g): SphereGeometry(r=0.2), 0x44ff44 (강력), 8종
- W⁺/W⁻ 보손: SphereGeometry(r=0.4), 0xff44ff (약력)
- Z⁰ 보손: SphereGeometry(r=0.45), 0xcc44ff (약력)

#### 힉스 보손 (H⁰)
- SphereGeometry(r=0.5), 0xffdd44 emissive 0xffaa00, 특별한 글로우 효과
- 중앙 상단에 배치
- "입자에 질량을 부여하는 메커니즘" 라벨
- 2012년 LHC에서 발견!

### 입자 선택
- 입자를 클릭하면:
  - 해당 입자 확대 + 회전 애니메이션
  - 속성 표시: 이름, 질량, 전하, 스핀, 세대
  - 관련 상호작용 하이라이트

## 보기 2: 쿼크와 하드론

### 양성자 구조 (3D)
- 양성자: 큰 구(SphereGeometry r=1.5, wireframe, 0xff4444 opacity 0.1)
- 내부 쿼크 3개:
  - u(업): 0xff4444, 2개
  - d(다운): 0x4488ff, 1개
  - 서로 글루온(초록 스프링 라인)으로 연결
- 쿼크 사이 글루온 교환 애니메이션 (초록 구가 쿼크 사이를 왕복)
- "양성자 = uud, 전하 = +2/3 +2/3 -1/3 = +1" 라벨
- 쿼크가 내부에서 진동/이동하는 애니메이션

### 중성자 구조 (3D)
- 중성자: SphereGeometry(r=1.5, wireframe, 0x4488ff opacity 0.1)
- 내부: u 1개 + d 2개
- "중성자 = udd, 전하 = +2/3 -1/3 -1/3 = 0" 라벨

### 메손 예시
- 파이온(π⁺): u + d̄ (쿼크 + 반쿼크)
  - SphereGeometry(r=0.8), 내부에 2개 입자
  - "메손 = 쿼크 + 반쿼크" 라벨

### 쿼크 가둠(Confinement)
- 쿼크를 분리하려고 잡아당기면 → 글루온 에너지↑ → 새로운 쿼크-반쿼크 쌍 생성!
- "쿼크는 홀로 존재할 수 없다!" 라벨
- 잡아당기기 슬라이더: 거리 → 글루온 에너지 증가 → 쌍 생성 애니메이션

## 보기 3: 기본 상호작용 (4가지 힘)

### 3D 힘 비교
- 4가지 힘 각각:
  - **강력**: 쿼크 사이, 0x44ff44, 글루온 매개
    - 강도: 1 (기준), 범위: ~10⁻¹⁵ m (핵 내부)
    - 쿼크를 묶어 양성자/중성자 형성
  - **전자기력**: 전하 입자 사이, 0xffff44, 광자 매개
    - 강도: 1/137, 범위: 무한대
    - 원자 구조, 화학 결합, 빛
  - **약력**: 모든 입자, 0xff44ff, W/Z 보손 매개
    - 강도: 10⁻⁶, 범위: ~10⁻¹⁸ m
    - β 붕괴, 핵변환
  - **중력**: 질량 있는 모든 것, 0xaaaaaa, 중력자(?) 매개
    - 강도: 10⁻³⁹ (극도로 약함!), 범위: 무한대
    - 표준 모형에 포함되지 않음 (미해결!)

### 파인만 다이어그램 (간략)
- 전자기: e⁻ + e⁻ → γ 교환 → e⁻ + e⁻
- 약력: n → p + W⁻ → p + e⁻ + ν̄ₑ (β 붕괴)
- 강력: q + q → g 교환 → q + q
- 2D 다이어그램을 우측에 표시 (시간축 수평, 공간축 수직)

### 힘의 통일
- 전자기력 + 약력 = 전약력 (높은 에너지에서 통합)
- 강력 통합? → 대통일 이론 (GUT, 미완)
- 중력 통합? → 만물의 이론 (TOE, 미완)
- "인류 물리학의 미완성 퍼즐!" 라벨

## 표준 모형 표 (우측 HTML overlay)
- 4×4 격자 + 보손 영역
- 세대(I, II, III) 열 + 쿼크/렙톤 행
- 각 칸: 이름, 기호, 질량, 전하
- 현재 선택 입자 하이라이트
- 색상 구분: 쿼크(빨강 계열), 렙톤(노랑 계열), 보손(보라 계열), 힉스(금색)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 표준 모형 개요 / 쿼크와 하드론 / 기본 상호작용
- "입자 선택" 드롭다운: 전체 입자 목록 (기본값: 전자)
- "하드론" 드롭다운 (쿼크 보기): 양성자 / 중성자 / 파이온
- "쿼크 분리 거리" 슬라이더 (가둠): 0~3 fm
- "상호작용" 드롭다운 (힘 보기): 강력 / 전자기력 / 약력 / 중력
- "질량 스케일" 토글: 실제 비율 / 로그 비율
- "표준 모형 표" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔬 표준 모형과 기본 입자
━━━━━━━━━━━━━━━━━━
보기: [개요/쿼크/상호작용]

기본 입자:
  물질: 쿼크 6종 + 렙톤 6종 (3세대)
  힘: 광자(γ) + W±/Z⁰ + 글루온(g)
  질량: 힉스 보손(H⁰)

선택: [입자 이름]
  질량: [값] MeV/c²
  전하: [값]e
  스핀: [값]

양성자 = u + u + d (전하 +1)
중성자 = u + d + d (전하 0)

4가지 기본 힘:
  강력 > 전자기력 > 약력 >> 중력
  (1 : 1/137 : 10⁻⁶ : 10⁻³⁹)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "표준 모형은 우주의 기본 구성 요소를 설명합니다. 물질은 쿼크와 렙톤으로, 힘은 게이지 보손이 매개합니다. 양성자와 중성자는 쿼크로 이루어진 복합 입자입니다."
- 탐구 질문:
  1. "양성자(uud)의 전하가 +1이 되는 까닭을 쿼크의 전하로 계산해 보세요."
  2. "쿼크가 홀로 관측되지 않는 까닭(가둠)을 글루온의 성질로 설명해 보세요."
  3. "힉스 보손이 발견되기 전, 물리학자들이 힉스 보손의 존재를 예측한 까닭은?"
- 실생활 예시: LHC(대형 강입자 충돌기, CERN), PET 스캔(양전자-전자 소멸), 뮤온 검출(우주선), 중성미자 관측소
- 교과서 연계: "전자기와 양자 III편 p118~122 — 기본 입자와 표준 모형"

- 성능: requestAnimationFrame 루프
