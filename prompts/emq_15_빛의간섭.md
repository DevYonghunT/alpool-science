너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 02. 빛의 간섭과 회절
- 토픽: 빛의 간섭 (박막 간섭과 이중 슬릿)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 두 빛이 같은 위상으로 만나면 보강 간섭(밝은 무늬), 반대 위상이면 상쇄 간섭(어두운 무늬)이 일어난다. 이중 슬릿 간섭에서 경로차가 mλ이면 보강, (m+½)λ이면 상쇄이다. 박막 간섭은 얇은 막의 윗면과 아랫면에서 반사된 빛의 간섭으로, 비눗방울이나 기름막의 무지개색을 만든다. 반사 시 밀한 매질에서 위상이 π만큼 변한다.

## 씬 구성
- 2가지 모드: 이중 슬릿 간섭 / 박막 간섭
- 배경: 0x0a0a20 (어두운 실험실)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2)

## 모드 1: 이중 슬릿 간섭 (영의 실험)

### 3D 실험 장치
- 광원: SphereGeometry(r=0.3), emissive, 색상은 파장에 따라
- 이중 슬릿 판: BoxGeometry(0.05, 4, 3), 0x333333, 슬릿 2개 빈 공간
- 스크린: PlaneGeometry(0.1, 4, 3), 0xffffff
- 각 슬릿에서 퍼져나가는 원형파: RingGeometry 확장 애니메이션

### 간섭무늬 (스크린)
- 밝고 어두운 띠 반복 패턴
- 색상: 선택한 빛의 색
- Δy = λL/d (무늬 간격)
- 파장/슬릿 간격 슬라이더 → 무늬 변화

### 경로차 시각화
- 특정 점 P에서: S₁→P, S₂→P 경로를 선으로 표시
- 경로차 |r₁-r₂| 계산 + 간섭 유형 표시
- 보강: 경로차 = mλ → "밝음 ✅"
- 상쇄: 경로차 = (m+½)λ → "어두움 ❌"

### 강도 그래프 (우측)
- X축: 스크린 위치, Y축: 빛의 강도
- I ∝ cos²(πdy/λL) 패턴
- 극대/극소 위치 라벨

## 모드 2: 박막 간섭

### 3D 박막
- 얇은 막: BoxGeometry(4, 0.05~0.5, 3), 반투명 0xaaddff
  - 막 두께 t: 슬라이더로 조절
  - 굴절률 n: 슬라이더로 조절
- 입사광: 노란 화살표, 위에서 비스듬히 입사

### 반사광 간섭 시각화
- 윗면 반사광: 빨간 광선 (밀한 매질 반사 → 위상 π 변화)
- 아랫면 반사광: 파란 광선 (아랫면에서 반사 후 윗면 통과)
- 두 반사광의 경로차: 2nt (수직 입사 시)
- 경로차 + 위상 변화 고려:
  - 보강: 2nt = (m+½)λ → 밝게 반사 (해당 파장)
  - 상쇄: 2nt = mλ → 반사 안 됨
- 막 두께에 따라 보강되는 파장이 달라짐 → 다양한 색!

### 비눗방울 시각화
- SphereGeometry(r=2), 반투명, 두께가 위치마다 다름
- 각 위치에서 보강되는 파장이 달라 → 무지개색 패턴
- "비눗방울의 무지개색 = 박막 간섭" 라벨

### 반사 방지 코팅 응용
- 렌즈 표면에 얇은 막 코팅 → 특정 파장에서 상쇄 간섭 → 반사 감소
- "카메라 렌즈, 안경 렌즈의 반사 방지 코팅" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 이중 슬릿 / 박막 간섭
- "파장 λ" 슬라이더: 400~700nm (기본값 550)
- "슬릿 간격 d" 슬라이더 (이중 슬릿): 0.1~2mm
- "슬릿-스크린 거리 L" 슬라이더: 0.5~3m
- "막 두께 t" 슬라이더 (박막): 100~1000nm
- "굴절률 n" 슬라이더 (박막): 1.0~2.0
- "원형파 표시" 토글: ON/OFF
- "경로차 표시" 토글: ON/OFF
- "강도 그래프" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌈 빛의 간섭
━━━━━━━━━━━━━━
모드: [이중 슬릿 / 박막]

이중 슬릿:
  보강: 경로차 = mλ (m=0,±1,±2...)
  상쇄: 경로차 = (m+½)λ
  무늬 간격: Δy = λL/d = [값]mm
  λ=[값]nm, d=[값]mm, L=[값]m

박막 간섭:
  경로차 = 2nt (수직 입사)
  윗면 반사: 위상 π 변화 (밀한→소한)
  보강: 2nt = (m+½)λ
  상쇄: 2nt = mλ
  t=[값]nm, n=[값]
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "두 빛이 같은 위상으로 만나면 보강, 반대 위상이면 상쇄 간섭이 일어납니다. 이중 슬릿 간섭에서 무늬 간격 Δy=λL/d. 박막 간섭은 비눗방울 무지개색의 원인입니다."
- 탐구 질문:
  1. "이중 슬릿에서 파장이 긴 빨간빛과 짧은 파란빛 중 무늬 간격이 더 넓은 것은?"
  2. "비눗방울의 무지개색이 위치마다 다른 까닭을 박막 두께와 관련지어 설명해 보세요."
  3. "카메라 렌즈의 반사 방지 코팅이 보라색으로 보이는 까닭은?"
- 실생활 예시: 비눗방울 무지개색, 기름막 무지개, 반사 방지 코팅, CD/DVD 무지개
- 교과서 연계: "전자기와 양자 II편 p64~71 — 빛의 간섭과 회절"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 원형파 RingGeometry 최대 20개/슬릿
- 파일명: emq_15_light_interference.html
