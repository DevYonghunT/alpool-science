너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 이온 결합과 공유 결합
- 토픽: 이온 결합 형성
- 대상: 고등학교 1학년
- 핵심 개념: 금속 원소는 전자를 잃어 양이온이 되기 쉽고, 비금속 원소는 전자를 얻어 음이온이 되기 쉽다. 양이온과 음이온 사이에 정전기적 인력이 작용하여 이온 결합이 형성된다. 염화 나트륨(NaCl)은 나트륨 이온(Na⁺)과 염화 이온(Cl⁻)이 규칙적으로 배열된 결정 구조를 가진다.

## 씬 구성
- 화면을 3단계로 구성 (탭 전환):
  - 탭 1: 이온 형성 과정 (전자 이동)
  - 탭 2: 이온 결합 형성 (정전기적 인력)
  - 탭 3: NaCl 결정 구조 (3D 격자)
- 배경: 0x0e0e2a (짙은 남색, 과학 실험 느낌)
- 조명: AmbientLight(0.3) + DirectionalLight(0.7) + PointLight(이온 근처, 따뜻한 빛)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls

## 탭 1: 이온 형성 과정 (교과서 그림 II-19, p68)

### 나트륨 원자 → 나트륨 이온 (좌측)
- 나트륨 원자 보어 모형:
  - 원자핵: SphereGeometry(r=0.3), 0xff4444, "+11" 라벨
  - 1번 껍질(K): TorusGeometry(r=0.8), 전자 2개(파란 구 r=0.06)
  - 2번 껍질(L): TorusGeometry(r=1.4), 전자 8개
  - 3번 껍질(M): TorusGeometry(r=2.0), 전자 1개 ← **이 전자가 핵심** (노란색 강조 0xffdd44, 크기 r=0.08, 깜빡임)
- 전자 이동 애니메이션:
  1. 3번 껍질의 전자 1개가 깜빡이며 강조 (1초)
  2. "전자 잃음" 화살표 라벨 표시
  3. 전자가 궤도를 벗어나 우측(염소 원자)을 향해 날아감 (곡선 경로, 1초)
  4. 나트륨 원자 → 나트륨 이온(Na⁺): 3번 껍질 사라짐, "+11" 옆에 "+" 표시, 전체 테두리 빨간 글로우(양전하)
  5. 라벨 변화: "Na (원자)" → "Na⁺ (양이온)"

### 염소 원자 → 염화 이온 (우측)
- 염소 원자 보어 모형:
  - 원자핵: SphereGeometry(r=0.35), 0x44aa44, "+17" 라벨
  - 1번 껍질(K): TorusGeometry(r=0.8), 전자 2개
  - 2번 껍질(L): TorusGeometry(r=1.4), 전자 8개
  - 3번 껍질(M): TorusGeometry(r=2.0), 전자 7개 ← 빈 자리 1개를 점선 원으로 표시
- 전자 흡수 애니메이션:
  1. 날아온 전자가 3번 껍질의 빈 자리에 합류 (0.5초)
  2. "전자 얻음" 화살표 라벨 표시
  3. 3번 껍질 전자 8개로 채워짐 → 18족(아르곤)과 같은 전자 배치
  4. 염소 원자 → 염화 이온(Cl⁻): "+17" 옆에 "−" 표시, 전체 테두리 파란 글로우(음전하)
  5. 라벨 변화: "Cl (원자)" → "Cl⁻ (음이온)"

### 동시 비교 표시
- 좌측 하단: Na 전자 배치 "2, 8, 1 → 2, 8" (전자 1개 잃음)
- 우측 하단: Cl 전자 배치 "2, 8, 7 → 2, 8, 8" (전자 1개 얻음)
- 중앙: 전자 이동 화살표 + "전자 1개 이동" 라벨

## 탭 2: 이온 결합 형성 (교과서 그림 II-20, p69)

### 결합 애니메이션
- Na⁺ (좌측, 빨간 글로우 구 r=0.8) 와 Cl⁻ (우측, 파란 글로우 구 r=1.0) 배치
  - 초기 간격: 5.0 단위
- 결합 과정:
  1. "정전기적 인력" 라벨 + 서로를 향하는 화살표(점선, 0xff44ff 분홍) 표시
  2. Na⁺와 Cl⁻가 서로에게 끌려가며 접근 (5.0 → 1.8, 2초)
  3. 접근 속도: 처음에 느리다가 가까워질수록 빨라짐 (가속, 정전기적 인력 ∝ 1/r²)
  4. 결합 순간: 밝은 플래시 + "이온 결합 형성!" 라벨
  5. 결합된 상태: Na⁺(작은 빨간 구)와 Cl⁻(큰 파란 구)가 밀착
- 이온 크기 비교:
  - Na⁺: 전자를 잃어 원자보다 작아짐 (r=0.6)
  - Cl⁻: 전자를 얻어 원자보다 커짐 (r=1.0)
  - 크기 차이를 라벨로 표시: "Na⁺ < Na (양이온은 원자보다 작음)", "Cl⁻ > Cl (음이온은 원자보다 큼)"

### 전하 표시
- Na⁺ 위에 "+" 기호 (빨간색, 항상 표시)
- Cl⁻ 위에 "−" 기호 (파란색, 항상 표시)
- 결합 후: "NaCl (염화 나트륨)" 라벨

## 탭 3: NaCl 결정 구조 (교과서 그림 II-21, p69)

### 3D 결정 격자
- 3×3×3 단위 셀 격자 구조:
  - Na⁺: SphereGeometry(r=0.25), 0xff6644 (빨간 주황), 27개
  - Cl⁻: SphereGeometry(r=0.35), 0x44cc66 (초록), 27개
  - 교대 배치: Na⁺과 Cl⁻이 체스판처럼 번갈아 배치
  - 총 54개 이온 (InstancedMesh 사용)
- 격자선: 이웃한 Na⁺—Cl⁻ 사이에 가는 연결선(LineGeometry, 0x888888, opacity 0.3)
- 격자 간격: 1.0 단위
- 결정 구조 라벨: "Na⁺과 Cl⁻이 규칙적으로 배열된 결정 구조"

### 결정 확대/축소
- 마우스 휠: 결정 확대/축소
- 드래그: 결정 전체를 360도 회전
- 특정 이온 클릭(Raycaster): 해당 이온과 주변 6개 이웃 이온을 강조 (밝게) + "각 이온은 6개의 반대 전하 이온에 둘러싸여 있다" 라벨

### 층 분리 애니메이션
- "층 분리" 버튼: 결정을 층별로 분리하여 내부 구조 관찰
  - 각 층이 Y축 방향으로 간격 벌어짐 (1.0 → 2.5)
  - 각 층의 Na⁺, Cl⁻ 교대 배열 패턴 명확히 보임

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 이온 형성 / 이온 결합 / 결정 구조
- "전자 이동 재생" 버튼 (탭 1): 전자 이동 애니메이션 재생
- "결합 재생" 버튼 (탭 2): Na⁺+Cl⁻ 결합 애니메이션 재생
- "결정 크기" 슬라이더 (탭 3): 2×2×2 ~ 5×5×5 (기본값 3×3×3), 정수 단계
- "층 분리" 토글 (탭 3): ON/OFF
- "이온 크기 비교" 토글 (탭 2): ON 시 원자 크기와 이온 크기 비교 표시
- "원소 변경" 드롭다운: NaCl (기본) / MgO / CaF₂ — 다른 이온 결합 물질로 전환
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔗 이온 결합 형성
━━━━━━━━━━━━━━━
현재 모드:     [이온 형성 / 결합 / 결정 구조]
물질:          [NaCl / MgO / CaF₂]

이온 형성:
  Na → Na⁺ + e⁻  (전자 1개 잃음)
  Cl + e⁻ → Cl⁻  (전자 1개 얻음)

이온 결합:
  Na⁺ + Cl⁻ → NaCl
  (정전기적 인력: F = k × q₁q₂/r²)

Na 전자 배치: 2, 8, 1 → 2, 8 (Ne과 동일)
Cl 전자 배치: 2, 8, 7 → 2, 8, 8 (Ar과 동일)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "금속 원소는 전자를 잃어 양이온이 되고, 비금속 원소는 전자를 얻어 음이온이 됩니다. 양이온과 음이온 사이의 정전기적 인력으로 이온 결합이 형성됩니다."
- 탐구 질문:
  1. "나트륨이 전자 1개를 잃으면 어떤 18족 원소와 같은 전자 배치가 되나요?" (답: 네온)
  2. "염소가 전자 1개를 얻으면 어떤 18족 원소와 같은 전자 배치가 되나요?" (답: 아르곤)
  3. "NaCl 결정에서 각 Na⁺ 이온 주변에는 몇 개의 Cl⁻ 이온이 있나요?" (답: 6개)
- 실생활 예시: 소금(NaCl), 석회(CaO), 형석(CaF₂), 산화 마그네슘(MgO, 내화 벽돌)
- 교과서 연계: "미래엔 통합과학1 II편 p68~69 — 이온 결합, 그림 II-19(이온 형성), 그림 II-20(이온 결합), 그림 II-21(NaCl 구조)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 결정 구조: InstancedMesh로 54개 이온 효율 렌더링 (geometry 1개 공유)
- 전자 궤도: TorusGeometry + 전자 SphereGeometry 원운동
- 전자 이동: Bezier 곡선 경로 따라 이동 (CubicBezierCurve3)
- 탭 전환 시 이전 객체 dispose()
- Raycaster: 결정에서 이온 클릭 감지
- 성능: requestAnimationFrame 루프
- 파일명: ionic_bonding_formation_sim.html
