너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 02. 정전기 유도와 유전 분극
- 토픽: 정전기 유도와 유전 분극
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 대전체를 중성인 도체에 가까이 하면 정전기 유도에 의해 도체의 양끝이 서로 다른 종류의 전하를 띤다. 부도체에서는 유전 분극이 일어나 분자 내부에서 전하가 재배열된다. 정전기 유도는 정전 도장, 정전기 감지 장치 등에 활용된다. 정전기 차폐는 도체로 물체를 둘러싸 외부 전기장을 차단하는 것이다.

## 씬 구성
- 3가지 시나리오: 도체의 정전기 유도 / 부도체의 유전 분극 / 정전기 차폐
- 입자 수준 시각화 (전자/양이온의 이동)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 도체의 정전기 유도 (교과서 그림 I-5)

### 3D
- 금속 막대: BoxGeometry(4, 1, 1), 0xcccccc (금속 은색)
- 내부에 자유 전자: SphereGeometry(r=0.05) × 20개, 0xffff44 (노랑), 랜덤 분포
- 양이온(격자): SphereGeometry(r=0.08) × 20개, 0xff6644 (주황), 격자 고정
- 대전체(−): SphereGeometry(r=0.5), 0x4444ff (파랑), "−" 라벨, 금속 막대 옆에

### 정전기 유도 과정
- 대전체 접근 시:
  1. 자유 전자가 (−)대전체의 척력으로 반대쪽으로 이동 (애니메이션 2초)
  2. 대전체 가까운 쪽: (+)전하 노출 (양이온만 남음)
  3. 대전체 먼 쪽: (−)전하 축적 (전자 모임)
- 대전체 거리 슬라이더: 가까이할수록 분극 정도 증가
- 대전체 제거 시: 전자가 원래 위치로 복귀 → 중성으로 돌아감
- "도체: 자유 전자가 이동하여 정전기 유도" 라벨

## 시나리오 2: 부도체의 유전 분극 (교과서 그림 I-6)

### 3D
- 플라스틱 막대: BoxGeometry(4, 1, 1), 0xddcc88 (연한 노랑)
- 내부에 분자: 각 분자 = (+)부분(빨강 작은 구) + (−)부분(파랑 작은 구) 쌍
  - 20개 분자, 랜덤 방향으로 배치
- 대전체(−): SphereGeometry(r=0.5), 0x4444ff

### 유전 분극 과정
- 대전체 접근 시:
  1. 각 분자 내부에서 (+)부분이 대전체 쪽으로, (−)부분이 반대쪽으로 약간 이동
  2. 분자가 정렬 (대전체 방향으로 일렬)
  3. 전자가 이동하는 것이 아님! 분자 내에서 재배열
- "부도체: 자유 전자 없음 → 분자 내 전하 재배열(유전 분극)" 라벨
- 대전체 제거 시: 분자가 원래 무작위 방향으로 복귀

## 시나리오 3: 정전기 차폐 (교과서 그림 I-11)

### 3D
- 도체 상자(패러데이 새장): BoxGeometry(3, 3, 3), wireframe, 0xcccccc
- 외부 전기장: 화살표 격자, 0xff4444 (빨강), 수평 방향
- 도체 상자 내부: 전기장 = 0 (화살표 없음)
- 내부 물체: 전기장 영향 받지 않음 → "보호됨" 라벨
- 원리: 외부 전기장에 의해 도체 표면에서 자유 전자 재배열 → 내부 전기장 상쇄

### 정전기 차폐 포장재
- 반도체 부품을 넣는 정전기 차폐 봉투 3D 모형
- "정전기 차폐: 도체로 둘러싸면 내부 전기장 = 0" 라벨

## 정전기 활용 (교과서 p16)
- 정전 도장: 자동차 표면에 고르게 페인트 칠
  - 자동차(0xdddddd) + 대전된 페인트 입자(작은 파티클) → 정전기 유도로 차 표면에 달라붙음
- 정전기 감지 장치: 알루미늄 공이 정전기 유도로 왕복 운동
- 주유소 정전기 방지: 접촉 방전

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 도체 정전기 유도 / 부도체 유전 분극 / 정전기 차폐
- "대전체 거리" 슬라이더: 1~6 (기본값 4)
- "대전체 전하" 드롭다운: (+)전하 / (-)전하
- "대전체 접근" 버튼: 대전체를 가까이 이동하는 애니메이션
- "대전체 제거" 버튼: 대전체를 멀리 이동
- "입자 표시" 토글: ON/OFF (기본값 ON)
- "전기장 표시" 토글: ON/OFF
- "활용 예시" 드롭다운: 정전 도장 / 감지 장치 / 차폐 포장
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 정전기 유도와 유전 분극
━━━━━━━━━━━━━━━━━
시나리오: [도체/부도체/차폐]
대전체: [+/−], 거리: [값]

정전기 유도 (도체):
  자유 전자가 이동
  → 양끝이 다른 종류의 전하를 띔
  대전체 제거 → 원래대로 복귀

유전 분극 (부도체):
  자유 전자 없음
  → 분자 내 전하 재배열
  대전체 제거 → 복귀

정전기 차폐:
  도체로 둘러싸면 내부 전기장 = 0
  자유 전자 재배열 → 외부 전기장 상쇄
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "도체에서는 자유 전자가 이동하여 정전기 유도가, 부도체에서는 분자 내 전하가 재배열되어 유전 분극이 일어납니다. 정전기 차폐는 도체로 둘러싸 내부 전기장을 0으로 만드는 것입니다."
- 탐구 질문:
  1. "도체의 정전기 유도와 부도체의 유전 분극의 차이점은 무엇인가요?"
  2. "대전체를 도체에 가까이 가져갔다가 제거하면 도체의 전하 상태는 어떻게 되나요?"
  3. "정전기 차폐 포장재로 전자 부품을 보호하는 원리를 설명해 보세요."
- 실생활 예시: 정전 도장(자동차), 정전기 방지 패드, 주유소 접촉 방전, 패러데이 새장
- 교과서 연계: "전자기와 양자 I편 p14~17 — 정전기 유도와 유전 분극 (그림 I-5~11)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 전자/분자 InstancedMesh 최대 40개
- 파일명: emq_02_electrostatic_induction.html
