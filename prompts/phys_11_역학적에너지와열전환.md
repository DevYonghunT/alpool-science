너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 2. 에너지와 열 > 역학적 에너지와 열
- 토픽: 역학적 에너지와 열에너지 전환
- 대상: 고등학교 (선택과목)
- 핵심 개념: 물체가 마찰이나 공기 저항을 받으며 운동할 때 역학적 에너지의 일부가 열에너지로 전환되므로 역학적 에너지가 보존되지 않는다. 줄(Joule)은 추를 떨어뜨려 물통 속의 날개를 회전시키는 실험을 통해 역학적 에너지가 열에너지로 전환됨을 보였다. 열에너지는 역학적 에너지로 전환되기도 한다.

## 씬 구성
- 화면 중앙: 3가지 시나리오 전환
- 화면 우측: 에너지 막대그래프 (역학적E, 열E, 총E) 실시간
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 마찰에 의한 에너지 전환

### 3D
- 수평면: BoxGeometry(12, 0.1, 3), 거친 표면 0xaa9988 (나무)
- 블록: BoxGeometry(0.8, 0.5, 0.5), 0x4488aa
- 초기 속도로 블록을 밀어 출발 → 마찰로 감속 → 정지
- 마찰 구간: 빨간 표시 (열에너지 발생 구간)
- 블록 아래에서 열에너지 파티클 (빨강-주황 Points, 위로 올라감)
- 블록과 바닥 접촉면에 온도 표시: 온도 상승 효과 (색상 변화)

### 에너지 변화
- 출발 시: Ek = ½mv² (최대), 열E = 0
- 이동 중: Ek 감소, 열E 증가
- 정지 시: Ek = 0, 열E = ½mv₀² (초기 운동 에너지 전부)
- 총 에너지 = Ek + 열E = 일정 (에너지 보존)

## 시나리오 2: 줄의 실험 (역사적 실험)

### 3D (교과서 그림 기반)
- 물통: CylinderGeometry(1, 1, 2), 반투명 0x88bbdd (물이 보이게)
- 물: CylinderGeometry(0.95, 0.95, 1.5) 내부, 0x4488cc opacity 0.6
- 날개(패들): BoxGeometry(0.05, 1, 0.4) × 4개, 회전축에 부착, 0x888888
- 도르래: TorusGeometry(0.15, 0.03), 물통 위에
- 실: CylinderGeometry(0.01, 0.01, 가변), 도르래를 통해 아래로
- 추: CylinderGeometry(0.2, 0.2, 0.3) × N개, 0xff4444

### 애니메이션
1. 추가 내려감 (위치 에너지 감소)
2. 실에 연결된 날개가 회전 (역학적 에너지)
3. 날개가 물을 저어 마찰 발생 → 물의 온도 상승 (열에너지)
4. 온도계: CylinderGeometry + 빨간 수은 높이 상승
- 추의 위치 에너지 = 물의 열에너지 증가분
- 4.186 J = 물 1g을 1°C 올리는 데 필요한 에너지 (1cal)

### 에너지 전환 경로
- 추의 위치 에너지 → 날개의 운동 에너지 → 물의 열에너지
- 화살표로 에너지 전환 경로 표시

## 시나리오 3: 열에너지 → 역학적 에너지

### 3D
- 증기 기관 간단 모형: 보일러(CylinderGeometry) + 피스톤(CylinderGeometry 왕복)
- 불꽃(파티클): 보일러 아래 빨간-주황 파티클
- 물 → 증기(흰색 파티클 상승) → 피스톤 밀기 → 바퀴 회전
- 에너지 전환: 열에너지(연료) → 운동 에너지(피스톤/바퀴)

### 에너지 변화
- 열에너지 투입 → 역학적 에너지 출력
- 효율: 출력 역학적 에너지 / 투입 열에너지 < 100% (항상)

## 에너지 막대그래프 (우측)
- 역학적 에너지 (Ek + Ep): 빨간 막대
- 열에너지: 주황 막대
- 총 에너지: 초록 수평선 (일정, 에너지 보존)
- 실시간 업데이트

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 마찰에 의한 전환 / 줄의 실험 / 열→역학
- "초기 속도" 슬라이더 (시나리오 1): 1~10m/s (기본값 5)
- "마찰 계수" 슬라이더 (시나리오 1): 0.1~0.5
- "추 질량" 슬라이더 (시나리오 2): 1~10kg
- "낙하 높이" 슬라이더 (시나리오 2): 1~5m
- "시뮬레이션 시작" 버튼
- "에너지 그래프" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔥 역학적 에너지와 열에너지 전환
━━━━━━━━━━━━━━━━━━━
시나리오: [마찰/줄의실험/열→역학]

역학적 에너지: [값]J
열에너지:     [값]J
총 에너지:    [값]J (보존)

에너지 전환:
  마찰 → 역학적E ↓, 열E ↑
  줄의 실험 → 위치E → 운동E → 열E
  열기관 → 열E → 역학적E

줄의 실험 (1843):
  역학적 에너지와 열에너지는
  서로 전환될 수 있다.
  4.186 J = 1 cal
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물체가 마찰을 받으며 운동할 때 역학적 에너지의 일부가 열에너지로 전환됩니다. 줄은 실험을 통해 역학적 에너지가 열에너지로 전환됨을 보였습니다. 열에너지가 역학적 에너지로 전환되는 것도 가능합니다(열기관)."
- 탐구 질문:
  1. "마찰이 있는 면에서 운동하는 블록의 역학적 에너지가 감소하는 까닭을 설명해 보세요."
  2. "줄의 실험에서 추의 위치 에너지가 물의 열에너지로 전환되는 과정을 설명해 보세요."
  3. "운석이 대기에 진입할 때 빛을 내며 타는 까닭을 에너지 전환으로 설명해 보세요."
- 실생활 예시: 브레이크 패드 과열, 운석(유성), 손 비비면 따뜻, 발전소 열기관
- 교과서 연계: "물리학 I편 p62~67 — 역학적 에너지와 열, 줄의 실험"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)

- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)

- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
