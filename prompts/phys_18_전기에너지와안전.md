너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 1. 전기장과 전기 에너지 > 소비 전력과 전기 안전
- 토픽: 전기 에너지와 안전 (누전/합선/접지)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 전기를 안전하게 사용하기 위해 접지, 누전 차단기, 퓨즈 등의 안전장치를 사용한다. 누전은 전류가 정상 회로를 벗어나 의도하지 않은 경로(인체 등)로 흐르는 것이다. 합선(단락)은 저항 없이 전원에 직접 연결되어 큰 전류가 흐르는 것으로 화재의 원인이 된다. 접지는 전기 기구의 금속 외함을 땅에 연결하여 누전 시 인체를 보호하는 것이다.

## 씬 구성
- 화면 중앙: 3D 가정 전기 회로 모형 (분전반 + 전선 + 가전제품)
- 3가지 시나리오: 정상 / 누전 / 합선
- 배경: 0xf0ede8 (가정집 실내 느낌)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 가정 전기 회로

### 분전반
- BoxGeometry(0.8, 1.0, 0.2), 0xdddddd (흰색 문)
- 내부: 차단기 3개 (BoxGeometry 토글 스위치)
  - 주 차단기 + 분기 차단기 2개
  - 차단기 ON/OFF 상태 표시 (빨강/초록)
- 누전 차단기: 특별 표시 (테스트 버튼 포함)

### 전선 (가정 배선)
- TubeGeometry:
  - 활선(L, live): 0xff4444 (빨강), 전류 흐름
  - 중성선(N, neutral): 0x4444ff (파랑)
  - 접지선(E, earth): 0x44aa44 (초록-노랑)
- 전선 내부 전자 흐름 파티클 (노란 구, 전류 방향)

### 가전제품 (2개)
- 세탁기: BoxGeometry(1, 1.2, 0.8), 0xeeeeff (흰색)
  - 금속 외함 표시
  - 접지선 연결 여부 토글
- 전기 히터: BoxGeometry(0.6, 0.8, 0.3), 0xcc4444 (빨강)
  - 열선 표시 (CylinderGeometry 코일)

### 사람 (인체)
- BoxGeometry 조합 실루엣, 0xddccbb
- 손으로 가전제품을 만지는 포즈

## 3가지 시나리오

### 시나리오 1: 정상 작동
- 전류 경로: 활선 → 가전제품 → 중성선 (정상 회로)
- 전자 파티클이 정상 경로로 흐름
- 차단기: ON (초록)
- 가전제품 정상 작동 (히터 발열, 세탁기 모터 회전)
- 인체 안전 ✅

### 시나리오 2: 누전 (접지 없음 vs 있음)
#### 접지 없음:
- 전선 피복 손상 → 활선이 금속 외함에 접촉
- 전류 경로: 활선 → 금속 외함 → 인체 → 대지 (감전!)
- 인체 빨간 글로우 + "⚡ 감전 위험!" 경고
- 전류 파티클이 인체를 통과

#### 접지 있음:
- 접지선이 금속 외함에 연결
- 전류 경로: 활선 → 금속 외함 → 접지선 → 대지 (인체 우회)
- 누전 차단기 자동 작동: "딸깍" 차단기 OFF (빨강)
- 인체 안전 ✅ "접지가 인체를 보호!"

### 시나리오 3: 합선 (단락)
- 활선과 중성선이 직접 연결 (저항 없이)
- 매우 큰 전류 흐름 → 전선 과열 → 빨간 글로우 + 연기 파티클
- 과전류 차단기 작동: 차단기 OFF
- 차단기 미작동 시: 화재 효과 (주황-빨강 파티클)
- "⚠️ 합선 — 매우 큰 전류 → 화재 위험"

## 전류 경로 시각화
- 정상: 노란 파티클이 정상 회로 경로로 이동 (활선→가전→중성선)
- 누전: 빨간 파티클이 인체 또는 접지선으로 이동
- 합선: 매우 빠르고 많은 빨간 파티클이 단락 경로로 이동

## 안전장치 설명 (하단 패널)
| 장치 | 역할 | 원리 |
|------|------|------|
| 접지 | 누전 시 전류를 대지로 유도 | 인체보다 접지선의 저항이 작으므로 |
| 누전 차단기 | 누전 감지 시 전류 차단 | 활선/중성선 전류 차이 감지 |
| 과전류 차단기 | 과전류 시 전류 차단 | 일정 전류 초과 시 자동 차단 |
| 퓨즈 | 과전류 시 끊어짐 | 퓨즈 금속이 녹아 회로 차단 |

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 정상 작동 / 누전(접지 없음) / 누전(접지 있음) / 합선
- "접지" 토글: ON/OFF (기본값 ON)
- "누전 차단기" 토글: ON/OFF (기본값 ON)
- "과전류 차단기" 토글: ON/OFF (기본값 ON)
- "시뮬레이션 시작" 버튼
- "전류 경로 표시" 토글: ON/OFF (기본값 ON)
- "안전장치 표" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔌 전기 에너지와 안전
━━━━━━━━━━━━━━━
시나리오: [정상/누전/합선]
접지: [ON/OFF]
차단기: [ON/OFF]
상태: [안전/감전위험/화재위험]

전류 경로:
  정상: 활선 → 가전 → 중성선
  누전: 활선 → 외함 → 인체/접지
  합선: 활선 → 직접 → 중성선 (과전류)

안전장치:
  접지: 누전 시 전류를 대지로 유도
  차단기: 이상 전류 감지 시 자동 차단
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "전기를 안전하게 사용하려면 접지, 누전 차단기, 과전류 차단기 등의 안전장치를 사용해야 합니다. 누전은 감전의 원인이 되고, 합선은 화재의 원인이 됩니다."
- 탐구 질문:
  1. "접지가 누전 시 인체를 보호하는 원리를 전류 경로와 저항으로 설명해 보세요."
  2. "문어발 콘센트를 사용하면 위험한 까닭을 합선과 관련지어 설명해 보세요."
  3. "젖은 손으로 전기 기구를 만지면 위험한 까닭은 무엇인가요?"
- 실생활 예시: 콘센트 3핀(접지), 욕실 누전 차단기, 가정 분전반, 퓨즈 교체
- 교과서 연계: "물리학 II편 p86~91 — 전기 안전, 누전, 합선, 접지, 차단기"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 전류 경로/파티클 리셋
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 전자 파티클 최대 50개
- 파일명: phys_18_electrical_safety.html
