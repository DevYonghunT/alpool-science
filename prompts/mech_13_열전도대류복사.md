너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 01. 단열과 열팽창
- 토픽: 열전도, 대류, 복사 — 열 전달의 3가지 방식
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 열은 온도가 높은 곳에서 낮은 곳으로 전달된다. 열전도는 물질을 이루는 입자들의 충돌로 열이 전달되는 방식이다. 대류는 유체(기체/액체)가 순환하면서 열이 전달되는 방식이다. 복사는 매질 없이 전자기파 형태로 열이 전달되는 방식이다. 단열은 열 전달을 차단하는 것이며, 보온병은 3가지 열 전달을 모두 차단한다.

## 씬 구성
- 3가지 탭: 열전도 / 대류 / 복사
- 각 탭에서 해당 열 전달 방식을 3D 입자/파티클로 시각화
- 보온병 단면도: 3가지를 모두 차단하는 구조
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 탭 1: 열전도

### 3D 금속 막대
- BoxGeometry(10, 0.5, 0.5), 0xcccccc (금속 은색)
- 한쪽 끝: 불꽃(빨간 파티클) — 고온 영역
- 반대쪽 끝: 얼음(연파랑 블록) — 저온 영역
- 막대 색상이 시간에 따라 고온(빨강)→저온(파랑) 그라데이션으로 변화

### 입자 모형 (확대 뷰)
- 막대 내부의 원자/분자: SphereGeometry(r=0.1) × 30개, 격자 배열
- 고온 쪽: 원자가 크게 진동 (amplitude 큼, 빨간색)
- 저온 쪽: 원자가 작게 진동 (amplitude 작음, 파란색)
- 진동 에너지가 이웃 원자로 전달: 순차적으로 진동 증가
- "입자 충돌로 열에너지 전달" 라벨
- 열전도율 비교: 금속(빠름) vs 나무(느림) vs 스티로폼(매우 느림)

### 온도 분포 그래프 (우측 Canvas 2D)
- X축: 막대 위치 (0~L)
- Y축: 온도 (0~100°C)
- 시간에 따라 온도 분포 곡선이 변화 (열 확산)

## 탭 2: 대류

### 3D 물이 담긴 비커
- 비커: CylinderGeometry(1.5, 1.5, 3), 투명 유리 MeshPhysicalMaterial
- 물: CylinderGeometry(1.45, 1.45, 2.5), 0x4488cc opacity 0.5
- 아래에 알코올램프: 불꽃 파티클, 0xff6633

### 대류 순환 파티클
- 물 내부 파티클(Points 100개): 순환 경로를 따라 이동
  - 바닥(가열됨): 위로 상승 (빨간 파티클, 빠르게)
  - 측면: 옆으로 이동
  - 상단: 냉각 → 아래로 하강 (파란 파티클, 느리게)
  - 바닥: 다시 가열 → 순환 반복
- 순환 화살표: 큰 타원형 화살표로 대류 경로 표시
- "뜨거운 물은 밀도↓ → 위로, 차가운 물은 밀도↑ → 아래로" 라벨

### 일상 대류 예시
- 에어컨(위쪽 설치 → 차가운 공기 아래로)
- 난방기(아래쪽 설치 → 따뜻한 공기 위로)
- 바닷바람/육지바람

## 탭 3: 복사

### 3D
- 태양: SphereGeometry(r=1), emissive 0xffdd00, PointLight
- 지구: SphereGeometry(r=0.5), 0x2266aa, 태양에서 떨어진 위치
- 태양과 지구 사이: 진공 (매질 없음)
- 적외선/전자기파: 사인파(TubeGeometry 곡선) 또는 파티클이 태양→지구로 이동
- "매질 없이 전자기파로 열이 전달된다" 라벨
- 모든 물체는 복사 방출: 온도 높을수록 많이

### 일상 복사 예시
- 모닥불 옆에서 따뜻함 (직접 접촉 없이)
- 전자레인지 (마이크로파 복사)
- 열화상 카메라 (적외선 복사 감지)

## 보온병 단면도 (종합)
- 보온병 3D 단면:
  - 이중벽: 두 개의 CylinderGeometry (안/바깥)
  - 진공층: 두 벽 사이 (열전도/대류 차단)
  - 은색 코팅: 안쪽 벽 반사면 (복사 차단)
  - 마개: 단열 소재 (열전도 차단)
- 각 차단 방식에 라벨:
  - "진공 → 전도/대류 차단"
  - "은 코팅 → 복사 차단"
  - "단열 마개 → 전도 차단"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "열 전달 방식" 드롭다운: 열전도 / 대류 / 복사 / 보온병(종합)
- "물질" 드롭다운 (열전도): 구리 / 알루미늄 / 유리 / 나무 / 스티로폼
- "온도 차이" 슬라이더: 10~100°C (기본값 50)
- "입자 모형" 토글 (열전도): ON/OFF (기본값 ON)
- "대류 속도" 슬라이더 (대류): 0.5~3x
- "시뮬레이션 시작" 버튼
- "온도 그래프" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔥 열 전달의 3가지 방식
━━━━━━━━━━━━━━━━
현재: [전도/대류/복사/보온병]

열전도: 입자 충돌로 열 전달
  매질 필요, 고체에서 주로
  물질별 열전도율: 금속 > 유리 > 나무

대류: 유체 순환으로 열 전달
  액체/기체에서
  뜨거운 유체↑, 차가운 유체↓

복사: 전자기파로 열 전달
  매질 불필요 (진공 통과 가능)
  모든 물체가 방출, 온도↑ 방출↑

보온병: 3가지 모두 차단
  진공(전도/대류) + 은코팅(복사)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "열은 전도(입자 충돌), 대류(유체 순환), 복사(전자기파) 3가지 방식으로 전달됩니다. 보온병은 진공(전도/대류 차단)과 은 코팅(복사 차단)으로 3가지를 모두 차단합니다."
- 탐구 질문:
  1. "금속 숟가락과 나무 숟가락을 뜨거운 국에 넣으면 어느 것이 먼저 뜨거워지나요? 그 까닭은?"
  2. "에어컨은 왜 방의 위쪽에 설치하는 것이 효율적인가요?"
  3. "우주 공간(진공)에서 태양의 열이 지구까지 전달되는 방식은 무엇인가요?"
- 실생활 예시: 보온병, 에어컨/난방기 위치, 이중창(단열), 요리(열전도/대류/복사 조리)
- 교과서 연계: "역학과 에너지 II편 p64~69 — 단열과 열팽창, 열 전달 방식"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 탭 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 100개
- 파일명: mech_13_heat_transfer.html
