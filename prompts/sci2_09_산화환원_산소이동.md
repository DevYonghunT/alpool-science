너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 산화와 환원
- 토픽: 산화와 환원 — 산소 이동에 의한 반응
- 대상: 고등학교 1학년
- 핵심 개념: 산소가 관여하는 반응이 일어날 때 물질이 산소를 얻는 반응을 산화라 하고, 물질이 산소를 잃는 반응을 환원이라고 한다. 어떤 물질이 산소를 얻을 때 다른 물질은 산소를 잃으므로 산화와 환원은 동시에 일어난다.

## 씬 구성
- 구리판 가열 실험(교과서 p40~41)을 3D 입자 모형으로 시각화한다.
- 화면 좌측: 3D 실험 장치 (알코올램프 + 구리판)
- 화면 우측: 입자 모형 (원자 수준에서 산소 이동 시각화)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(알코올램프 위치, 1.5)

## 실험 1: 구리판을 겉불꽃에 가열 (산화)
### 3D 실험 장치
- 알코올램프: CylinderGeometry(0.3, 0.3, 0.6) 몸통 + 심지 + 불꽃 파티클(겉불꽃 0xff8844, 속불꽃 0x4466ff)
- 구리판: BoxGeometry(2, 0.05, 1), 초기 색상 0xcc6633 (붉은 구리색)
- 가열 시: 불꽃에 닿는 부분부터 0x222222 (검은색, 산화 구리(II))로 색 변화
- 가열 애니메이션: 3초간 서서히 색상 변화

### 입자 모형 (우측)
- 구리 원자(Cu): SphereGeometry(r=0.2), 0xcc6633 (구리색), 여러 개 배열
- 산소 분자(O₂): SphereGeometry(r=0.12) × 2 결합, 0xff3333 (빨강)
- 산화 과정 애니메이션:
  1. O₂가 구리 표면으로 접근
  2. O₂가 분리되어 각 O가 Cu에 결합
  3. Cu + O → CuO (산화 구리(II), 색상 0x222222로 변화)
- 화학 반응식: `2Cu + O₂ → 2CuO`
- "산화" 라벨 + 빨간 화살표

## 실험 2: 산화 구리(II)를 속불꽃에 가열 (환원)
### 3D 실험 장치
- 검은색 구리판(산화 구리): 0x222222
- 속불꽃에 넣으면: 0xcc6633 (붉은 구리색)으로 복원
- 속불꽃 특징: "속불꽃은 일산화 탄소가 많고, 산소가 부족하다" 설명

### 입자 모형 (우측)
- 산화 구리(CuO): Cu(0x222222) + O(0xff3333) 결합 상태
- 일산화 탄소(CO): C(0x444444) + O(0xff3333) 결합
- 환원 과정 애니메이션:
  1. CO가 CuO에 접근
  2. CuO에서 O가 분리 → CO와 결합하여 CO₂
  3. Cu만 남음 (색상 0xcc6633으로 복원)
- 화학 반응식: `CuO + CO → Cu + CO₂`
- ⓐ 산화(CO→CO₂) + ⓑ 환원(CuO→Cu) 동시 표시
- "산화와 환원은 항상 동시에 일어난다" 강조 텍스트

## 우리 주변의 산화·환원 반응 (교과서 p44)
- GUI에서 추가 예시를 선택할 수 있다:
  - 광합성: `6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂` (CO₂의 탄소: 환원)
  - 세포호흡: `C₆H₁₂O₆ + 6O₂ → 6CO₂ + 6H₂O` (포도당의 탄소: 산화)
  - 연소(뷰테인): `2C₄H₁₀ + 13O₂ → 8CO₂ + 10H₂O` (탄소: 산화)
  - 철의 제련: `Fe₂O₃ + 3CO → 2Fe + 3CO₂` (철: 환원)
- 각 반응에서 산화/환원 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "실험" 드롭다운: 구리 산화 (겉불꽃) / 구리 환원 (속불꽃) / 광합성 / 세포호흡 / 연소 / 제련
- "반응 재생" 버튼: 해당 반응 애니메이션 재생
- "산화/환원 라벨" 토글: ON/OFF (기본값 ON)
- "입자 크기" 슬라이더: 0.5x~2x
- "초기화" 버튼

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "산소가 관여하는 반응에서 물질이 산소를 얻는 반응을 산화, 산소를 잃는 반응을 환원이라고 합니다. 어떤 물질이 산소를 얻을 때 다른 물질은 산소를 잃으므로 산화와 환원은 항상 동시에 일어납니다."
- 탐구 질문:
  1. "구리판을 겉불꽃에 넣어 가열하면 검은색으로 변하는 까닭을 산화 반응으로 설명해 보세요."
  2. "검은색의 산화 구리(II)를 속불꽃에 넣으면 다시 붉은색으로 변하는 까닭은 무엇인가요?"
  3. "철로 만든 다리나 손잡이에 페인트를 칠하면 녹스는 것을 막을 수 있는 까닭은 무엇인가요?"
- 실생활 예시: 철이 녹슨다(산화), 사과 갈변(산화), 유화 물감 굳는 과정(산화)
- 교과서 연계: "미래엔 통합과학2 I편 p40~45 — 산소의 이동에 의한 산화·환원 반응"
