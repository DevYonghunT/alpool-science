너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 02. 빛의 간섭과 회절
- 토픽: 빛의 회절과 단일 슬릿
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 빛이 좁은 틈(슬릿)이나 장애물의 가장자리를 지날 때 그림자 영역으로 퍼져나가는 현상을 회절이라 한다. 단일 슬릿 회절에서 어두운 무늬(극소) 조건: asinθ = mλ (a: 슬릿 폭, m = ±1, ±2, ...). 슬릿 폭이 파장에 비해 좁을수록 회절이 뚜렷하다. 회절은 광학 기기의 분해능을 제한한다.

## 씬 구성
- 화면 좌측: 3D 단일 슬릿 + 광원 + 스크린
- 화면 우측: 회절 무늬 + 강도 분포 그래프 (Canvas 2D)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2)

## 3D 단일 슬릿 회절

### 실험 장치
- 광원: SphereGeometry(r=0.3), emissive, 색상은 파장에 따라
- 단일 슬릿 판: BoxGeometry(0.05, 4, 3), 0x333333 (불투명), 중앙에 슬릿 1개
  - 슬릿 폭 a: 슬라이더로 조절 (0.01~1mm)
- 스크린: PlaneGeometry, 0xffffff

### 회절 시각화
- 슬릿에서 빛이 부채꼴로 퍼져나감 (파동의 성질)
- 호이겐스 원리: 슬릿의 각 점에서 원형파(RingGeometry) 발생
  - 여러 원형파가 겹쳐 간섭 → 회절 무늬 형성
- 슬릿 폭이 좁을수록: 빛이 더 넓게 퍼짐 (회절 뚜렷)
- 슬릿 폭이 넓을수록: 빛이 직진에 가까움 (회절 약함)

### 회절 무늬 (스크린)
- 중앙: 밝고 넓은 중앙 극대 (가장 밝음)
- 양쪽: 어두운 극소 + 밝은 부극대 반복 (점점 약해짐)
- 중앙 극대 폭: 2λL/a (슬릿 폭 a에 반비례)
- 극소 위치: y_m = mλL/a (m = ±1, ±2, ...)

## 강도 분포 그래프 (우측 Canvas 2D)
- X축: 스크린 위치 y
- Y축: 빛의 강도 I
- I ∝ (sinα/α)² 패턴 (α = πay/(λL))
- 중앙 극대가 가장 넓고 밝음
- 부극대는 중앙의 ~4.7% 정도 (급격히 감소)
- 극소 위치: asinθ = mλ → 빨간 화살표로 표시

## 슬릿 폭과 파장에 따른 변화
- 슬릿 폭 a↓: 회절 무늬 넓어짐 (중앙 극대 폭↑)
- 슬릿 폭 a↑: 회절 무늬 좁아짐 (직진에 가까움)
- 파장 λ↑: 회절 무늬 넓어짐
- 파장 λ↓: 회절 무늬 좁아짐
- a ≈ λ일 때: 회절이 가장 뚜렷

## 분해능 (Rayleigh 기준)
- 원형 구경(렌즈/망원경)의 회절: 에어리 디스크 패턴
- 두 점광원이 분해 가능한 최소 각도: θ_min ≈ 1.22λ/D (D: 구경 직경)
- 구경 D↑: 분해능↑ (더 세밀하게 구분)
- 파장 λ↓: 분해능↑
- "회절이 광학 기기의 분해능을 제한한다" 라벨
- 망원경/현미경 구경과 분해능 관계 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "파장 λ" 슬라이더: 400~700nm (기본값 550)
- "슬릿 폭 a" 슬라이더: 0.01~1mm (기본값 0.1)
- "스크린 거리 L" 슬라이더: 0.5~3m (기본값 1)
- "호이겐스 원형파" 토글: ON/OFF (기본값 OFF)
- "강도 그래프" 토글: ON/OFF (기본값 ON)
- "분해능 모드" 토글: ON/OFF (원형 구경 + 에어리 디스크)
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔬 빛의 회절과 단일 슬릿
━━━━━━━━━━━━━━━
λ = [값]nm, a = [값]mm, L = [값]m

단일 슬릿 회절:
  극소 조건: asinθ = mλ (m=±1,±2...)
  극소 위치: y_m = mλL/a = [값]mm

중앙 극대 폭:
  W = 2λL/a = [값]mm
  a↓ → W↑ (회절 뚜렷)
  a↑ → W↓ (직진에 가까움)

강도: I ∝ (sinα/α)²
  α = πay/(λL)

분해능 (레일리):
  θ_min ≈ 1.22λ/D
  D↑ → 분해능↑
```

화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "빛이 좁은 슬릿을 통과할 때 그림자 영역으로 퍼지는 회절이 일어납니다. 슬릿 폭이 파장에 비해 좁을수록 회절이 뚜렷합니다. 회절은 광학 기기의 분해능을 제한합니다."
- 탐구 질문:
  1. "슬릿 폭을 줄이면 중앙 극대의 폭은 어떻게 변하나요?"
  2. "문틈으로 소리는 잘 들리지만 빛은 잘 통과하지 못하는 까닭을 파장과 관련지어 설명해 보세요."
  3. "망원경의 구경이 클수록 멀리 있는 별을 더 선명하게 관찰할 수 있는 까닭은?"
- 실생활 예시: CD/DVD 무지개(회절 격자), 레이저 회절 실험, 망원경 분해능, X선 회절(결정 구조)
- 교과서 연계: "전자기와 양자 II편 p64~71 — 빛의 간섭과 회절"
