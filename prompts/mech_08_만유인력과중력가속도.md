너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 04. 케플러 법칙과 중력
- 토픽: 만유인력과 중력 가속도
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 뉴턴의 만유인력 법칙에 의하면 모든 물체 사이에는 서로 끌어당기는 인력이 작용하며, F = GMm/r²이다. 지구 표면에서의 중력 가속도 g = GM/R² ≈ 9.8m/s²이다. 높이 h에서의 중력 가속도 g' = GM/(R+h)²이며, 높이가 올라갈수록 g는 감소한다. 케플러 제3법칙은 만유인력으로부터 유도된다.

## 씬 구성
- 화면 중앙: 3D 지구 + 여러 고도의 인공위성 + 중력 벡터 시각화
- 2가지 모드: 만유인력 기본 / 고도별 중력 가속도 변화
- 배경: 0x000010 (우주)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.15) + PointLight(태양 방향, 1.0)

## 모드 1: 만유인력 기본

### 3D
- 물체 1(M): SphereGeometry(r=2), 0x2266aa (지구, 큰 질량)
  - "M = [값]kg" 라벨
- 물체 2(m): SphereGeometry(r=0.3), 0xff4444 (작은 물체)
  - "m = [값]kg" 라벨
- 거리 r: 두 물체 사이 점선 + 양방향 화살표, "[값]m" 라벨

### 만유인력 화살표
- M → m 방향 인력: 파란 화살표 (m에 작용)
- m → M 방향 인력: 빨간 화살표 (M에 작용, 크기 같음 = 작용 반작용)
- 화살표 크기 = F = GMm/r²에 비례
- 거리 r 슬라이더: 가까이할수록 화살표 커짐 (역제곱 법칙)

### 역제곱 법칙 시각화
- 거리 2배 → 힘 1/4: 화살표 크기 1/4로 감소
- 거리 3배 → 힘 1/9
- F-r 그래프 (우측 Canvas 2D): 반비례 제곱 곡선 (1/r²)

## 모드 2: 고도별 중력 가속도

### 3D 지구 + 고도
- 지구: SphereGeometry(r=3), 0x2266aa
  - 지구 반지름 R 라벨
- 고도 h에서의 물체: SphereGeometry(r=0.15), 0xff4444
  - 고도 h 슬라이더로 위치 조절 (지구 표면~고궤도)
- 중력 가속도 벡터: 빨간 화살표, 지구 중심 방향
  - 화살표 크기 = g' = GM/(R+h)²
  - 지표면에서 가장 크고, 높이 올라갈수록 감소

### 고도별 g 값 비교
| 위치 | 고도 h | g' 값 |
|------|--------|-------|
| 지표면 | 0 km | 9.8 m/s² |
| ISS 궤도 | 400 km | ~8.7 m/s² |
| 정지 궤도 | 35,786 km | ~0.22 m/s² |
| 달 궤도 | 384,400 km | ~0.0027 m/s² |
- 각 위치에 인공위성/달 아이콘 배치
- g'-h 그래프 (Canvas 2D): g가 높이에 따라 감소하는 곡선

### 무중력(미세 중력) 시각화
- ISS 고도에서 g ≈ 8.7m/s² (0이 아님!)
- ISS가 자유 낙하하면서 궤도 운동 → 내부에서 무중력 체험
- "무중력 = 중력이 없는 것이 아니라, 자유 낙하하는 것" 강조

## 케플러 제3법칙 유도
- 등속 원운동하는 인공위성: 구심력 = 만유인력
  - mv²/r = GMm/r²
  - v² = GM/r
  - T = 2πr/v → T² = (4π²/GM)r³
  - → T² ∝ r³ = 케플러 제3법칙!
- 이 유도 과정을 단계별로 보여주는 애니메이션

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 만유인력 기본 / 고도별 중력 가속도
- "질량 M" 슬라이더: 10²⁰~10²⁶kg (로그 스케일, 기본값 지구 질량)
- "질량 m" 슬라이더: 1~1000kg (기본값 100)
- "거리 r" 슬라이더 (모드 1): 1~20 (기본값 5)
- "고도 h" 슬라이더 (모드 2): 0~400,000km (로그 스케일)
- "역제곱 그래프" 토글: ON/OFF
- "케플러 유도" 버튼: 단계별 유도 표시
- "ISS/정지궤도/달" 프리셋 버튼
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌍 만유인력과 중력 가속도
━━━━━━━━━━━━━━━━━
모드: [만유인력/고도별 g]

만유인력:
  F = GMm/r²
  G = 6.674×10⁻¹¹ N·m²/kg²
  M = [값]kg, m = [값]kg
  r = [값]m
  F = [값]N

중력 가속도:
  g = GM/R² ≈ 9.8 m/s² (지표면)
  고도 h = [값]km
  g' = GM/(R+h)² = [값]m/s²

케플러 유도:
  구심력 = 만유인력
  → T² = (4π²/GM)r³ ∝ r³
```
