너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 2. 전기와 자기의 상호작용 > 자성체의 종류와 활용
- 토픽: 자성체의 종류 (강자성체, 상자성체, 반자성체)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 물질은 외부 자기장에 대한 반응에 따라 강자성체, 상자성체, 반자성체로 분류된다. 강자성체(철, 니켈, 코발트)는 외부 자기장에 의해 강하게 자기화되어 자석에 세게 붙고, 자기장을 제거해도 자기화가 유지된다. 상자성체(알루미늄, 백금)는 약하게 자기화되어 자석에 약하게 끌린다. 반자성체(구리, 금, 은, 비스무트)는 자석에 밀려나는 성질이 있다.

## 씬 구성
- 화면 중앙: 3D 자석 + 3가지 자성체 물질 배치
- 외부 자기장(자석) 접근 시 각 물질의 반응 비교
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 3D 자석

### 막대 자석
- BoxGeometry(0.5, 0.5, 3), N극 0xff4444(빨강) / S극 0x4444ff(파랑)
- "N" / "S" 라벨
- 자석을 드래그하여 물질에 접근시킬 수 있음 (마우스/터치)
- 자기력선: TubeGeometry 곡선, N극→S극, 0x44ff88 반투명

## 3가지 자성체 비교

### 강자성체 (철, 니켈, 코발트)
- BoxGeometry(1, 1, 1), 0xaaaaaa (금속 은색)
- 라벨: "철 (Fe) — 강자성체"
- **자석 접근 시 반응:**
  - 강하게 끌려옴 (물체가 자석 방향으로 이동하는 애니메이션)
  - 내부 자기 구역(magnetic domain): 작은 화살표들이 정렬
    - 자석 없을 때: 화살표 방향 무작위 → 자기화 안 됨
    - 자석 접근 시: 화살표 방향이 외부 자기장 방향으로 정렬 → 자기화
    - 자석 제거 후: 정렬 유지 → **영구 자석화 가능**
  - 인력 표시: 초록 화살표 (자석 방향)
  - 크기: 매우 큼

### 상자성체 (알루미늄, 백금)
- BoxGeometry(1, 1, 1), 0xddccaa (밝은 금색)
- 라벨: "알루미늄 (Al) — 상자성체"
- **자석 접근 시 반응:**
  - 약하게 끌려옴 (작은 이동)
  - 내부 화살표: 약간만 정렬 (불완전)
  - 자석 제거 시: 정렬 해제 → 원래대로
  - 인력 표시: 작은 초록 화살표
  - 크기: 매우 작음

### 반자성체 (구리, 금, 비스무트)
- BoxGeometry(1, 1, 1), 0xcc6633 (구리색)
- 라벨: "구리 (Cu) — 반자성체"
- **자석 접근 시 반응:**
  - 약하게 밀려남 (자석 반대 방향으로 작은 이동)
  - 내부 화살표: 외부 자기장 반대 방향으로 약간 정렬
  - 자석 제거 시: 원래대로
  - 척력 표시: 작은 빨간 화살표 (자석 반대 방향)
  - 크기: 매우 작음 (강자성체보다 훨씬 약함)

## 자기 구역(domain) 시각화 (교과서 핵심)
- 강자성체 내부 확대 뷰:
  - 작은 화살표(ArrowHelper) 10×10 격자 = 100개
  - 각 화살표 = 하나의 자기 구역
  - **자기장 없을 때**: 무작위 방향 → 총 자기화 = 0
  - **외부 자기장 적용**: 점차 정렬 (애니메이션 2초)
  - **자기장 제거**: 강자성체는 정렬 유지, 상자성체는 원래로
- 색상: 정렬된 화살표 = 빨강, 미정렬 = 회색

## 자기 부상 열차 (실생활 응용)
- "응용" 토글:
  - 자기 부상 열차 간단 3D 모형
  - 열차(BoxGeometry) + 레일(BoxGeometry)
  - 열차가 레일 위에 떠 있는 모습 (반자성체 부상 원리)
  - 또는 초전도체 위에 자석이 뜨는 마이스너 효과 시연

## 비교 표 (하단 HTML overlay)
| 구분 | 강자성체 | 상자성체 | 반자성체 |
|------|---------|---------|---------|
| 예시 | 철, 니켈, 코발트 | 알루미늄, 백금 | 구리, 금, 은 |
| 자석과 반응 | 강하게 끌림 | 약하게 끌림 | 약하게 밀림 |
| 자기장 제거 후 | 자기화 유지 | 원래대로 | 원래대로 |
| 내부 구역 정렬 | 강한 정렬 | 약한 정렬 | 반대 정렬 |

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "자성체" 드롭다운: 강자성체(철) / 상자성체(알루미늄) / 반자성체(구리) / 전체 비교
- "자석 거리" 슬라이더: 1~8 (기본값 5, 가까워질수록 자기장 강함)
- "자기장 세기" 슬라이더: 0~1T (기본값 0.5)
- "자기 구역 표시" 토글: ON/OFF (기본값 ON)
- "자기력선 표시" 토글: ON/OFF
- "비교 표" 토글: ON/OFF
- "자기 부상 응용" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧲 자성체의 종류
━━━━━━━━━━━━━━
현재 자성체: [강자성/상자성/반자성]
물질: [물질명]
외부 자기장: [값]T
자석 거리: [값]
반응: [강하게 끌림/약하게 끌림/밀림]

강자성체: 자기화 강함, 제거 후 유지
상자성체: 자기화 약함, 제거 후 소멸
반자성체: 반대 방향, 제거 후 소멸

자기 구역(domain):
  강자성체 내부의 미소 자석 영역
  외부 자기장 → 구역 정렬 → 자기화
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물질은 외부 자기장에 대한 반응에 따라 강자성체, 상자성체, 반자성체로 분류됩니다. 강자성체는 자기 구역이 정렬되어 강하게 자기화되고 자기장을 제거해도 유지됩니다."
- 탐구 질문:
  1. "냉장고에 자석이 붙는 까닭을 강자성체와 관련지어 설명해 보세요."
  2. "알루미늄 캔은 자석에 잘 붙지 않는 까닭은 무엇인가요?"
  3. "강자성체를 높은 온도로 가열하면 자기화가 사라지는 까닭은 무엇일까요? (힌트: 퀴리 온도)"
- 실생활 예시: 영구 자석, 전자석, 자기 부상 열차, 하드디스크(자기 기록), MRI
- 교과서 연계: "물리학 II편 p100~103 — 자성체의 종류와 활용"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 자성체 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 자기 구역 화살표 InstancedMesh 100개
- 파일명: phys_19_magnetic_materials.html
