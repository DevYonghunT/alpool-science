너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 2. 에너지 > 태양 에너지의 생성과 전환
- 토픽: 태양 에너지의 생성과 전환
- 대상: 고등학교 1학년
- 핵심 개념: 태양 에너지는 수소 핵융합 반응에 의해 생성된다. 수소 핵융합 반응 후 생성된 헬륨 원자핵 1개의 질량은 반응 전 수소 원자핵 4개의 질량 합보다 작으며, 줄어든 질량은 에너지로 전환된다. 태양 에너지는 지구에서 다양한 형태의 에너지로 전환되어 자연 현상을 일으킨다.

## 씬 구성
- 화면을 좌/우로 분할:
  - 좌측 50%: 태양 내부 핵융합 반응 3D
  - 우측 50%: 태양 에너지 전환 다이어그램 (교과서 그림 II-18)

## 태양 핵융합 3D (좌측, 교과서 그림 II-17)

### 태양 외관
- SphereGeometry(r=3, 32, 32), MeshStandardMaterial(emissive 0xffaa00, emissiveIntensity 0.6)
- 반투명 대기: SphereGeometry(r=3.3), opacity 0.1, 0xffdd44
- 단면도 모드: 반구를 잘라 내부 구조 표시

### 태양 내부 구조
- 핵(중심부): SphereGeometry(r=0.8), 0xffffff (극고온 흰색, emissive 1.0)
- 복사층: RingGeometry(0.8, 2.0), 0xffcc44 opacity 0.3
- 대류층: RingGeometry(2.0, 3.0), 0xffaa33 opacity 0.2
- 표면(광구): 가장 바깥 (약 6000K)
- 각 층에 라벨

### 핵융합 애니메이션 (핵 확대 뷰)
- 수소 원자핵 4개(양성자): SphereGeometry(r=0.08), 0xff4444 (빨강), "¹H" 라벨
- 애니메이션:
  1. 4개의 수소 원자핵이 빠르게 이동 (2초)
  2. 2개씩 접근하여 충돌 → 결합 (1초)
  3. 최종: 헬륨 원자핵 1개(SphereGeometry r=0.15, 0xffcc44 노랑, "⁴He" 라벨)
  4. 에너지 방출: 밝은 노란 파티클(Points 30개) 방사형 퍼짐
- 질량 비교 표시:
  - "수소 4개 질량 합 > 헬륨 1개 질량"
  - "줄어든 질량 → 에너지 (E=mc²)"
- 반복 재생

## 태양 에너지 전환 다이어그램 (우측, 교과서 그림 II-18)

### 중심: 태양 아이콘 (SphereGeometry r=0.5, emissive 0xffdd00)

### 6가지 에너지 전환 경로 (태양에서 방사형으로)
각 경로를 화살표 + 아이콘 + 라벨로 표시:

1. **광합성 → 화학 에너지**
   - 아이콘: 나무 (CylinderGeometry + SphereGeometry 초록)
   - 화살표: 태양 → 나무 (노란색)
   - 라벨: "태양 빛에너지 → 식물 화학 에너지"
   - 추가: "→ 화석 연료 (석탄, 석유, 천연가스)"

2. **바람 → 운동 에너지**
   - 아이콘: 풍차 (회전하는 날개)
   - 화살표: 태양 → 대기 가열 → 바람
   - 라벨: "태양 열에너지 → 대기 가열 → 바람(운동 에너지)"

3. **물 순환 → 위치 에너지**
   - 아이콘: 구름 + 비 + 강
   - 화살표: 태양 → 증발 → 비 → 강
   - 라벨: "태양 열에너지 → 물 증발 → 비 → 물의 위치 에너지"

4. **해류/파도 → 운동 에너지**
   - 아이콘: 파도 (사인파 형태)
   - 라벨: "태양 열에너지 → 해수 가열 → 해류, 파도"

5. **태양광 발전 → 전기 에너지**
   - 아이콘: 태양 전지판 (PlaneGeometry 기울어진)
   - 라벨: "태양 빛에너지 → 전기 에너지 (태양 전지)"

6. **태양열 → 열에너지**
   - 아이콘: 태양열 집열판
   - 라벨: "태양 열에너지 → 난방, 온수"

### 상호작용
- 각 경로 클릭 시 해당 경로 강조 + 상세 설명 팝업
- 전체 에너지 흐름 애니메이션: 태양에서 파티클이 각 경로로 이동

## 여러 가지 에너지 형태 (교과서 그림 II-16, p98)
- 빛에너지, 열에너지, 화학 에너지, 전기 에너지, 역학적 에너지(운동+위치), 핵에너지
- 각 에너지 형태를 아이콘으로 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 핵융합 반응 / 에너지 전환 / 전체
- "핵융합 재생" 버튼: 핵융합 애니메이션 재생
- "단면도" 토글: 태양 내부 구조 표시 ON/OFF
- "에너지 경로 강조" 드롭다운: 전체 / 광합성 / 바람 / 물 순환 / 해류 / 태양광 / 태양열
- "에너지 흐름 애니메이션" 버튼: 파티클 이동 시작
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 보기 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
