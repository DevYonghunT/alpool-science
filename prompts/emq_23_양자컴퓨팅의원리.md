너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 02. 양자 기술과 미래
- 토픽: 양자 컴퓨팅의 원리 (큐비트)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 고전 컴퓨터는 비트(0 또는 1)를 사용하지만, 양자 컴퓨터는 큐비트(|0⟩과 |1⟩의 중첩 상태)를 사용한다. n개의 큐비트는 2ⁿ개의 상태를 동시에 표현할 수 있어 특정 문제에서 고전 컴퓨터보다 기하급수적으로 빠르다. 양자 게이트(X, H, CNOT 등)로 큐비트를 조작하고, 양자 얽힘을 이용하여 연산한다.

## 씬 구성
- 화면 좌측: 3D 블로흐 구 + 큐비트 상태 시각화
- 화면 우측: 양자 회로 다이어그램 + 고전 vs 양자 비교 (Canvas 2D / HTML)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.4)

## 3D 블로흐 구

### 블로흐 구
- SphereGeometry(r=2.5, 32, 32), wireframe, 0x4488ff opacity 0.15
- 좌표축: x, y, z축 화살표
- 북극: |0⟩ 라벨, 남극: |1⟩ 라벨
- 적도: 중첩 상태 영역

### 큐비트 상태 벡터
- 빨간 화살표(ArrowHelper): 구의 중심에서 표면까지
- 상태: |ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩
  - θ: 극각 (0=|0⟩, π=|1⟩)
  - φ: 방위각 (위상)
- θ, φ 슬라이더 → 벡터 위치 변화

### 측정
- "측정" 버튼: 큐비트를 z축으로 사영
  - 북극(|0⟩) 쪽에 가까우면: |0⟩ 확률 높음 → |0⟩으로 붕괴
  - 남극(|1⟩) 쪽에 가까우면: |1⟩ 확률 높음 → |1⟩으로 붕괴
  - 확률: P(0) = cos²(θ/2), P(1) = sin²(θ/2)
- 측정 시 벡터가 북극 또는 남극으로 스냅 (애니메이션)

## 양자 게이트 시각화

### X 게이트 (NOT)
- |0⟩ → |1⟩, |1⟩ → |0⟩
- 블로흐 구에서: x축 주위 180° 회전 (북극↔남극)
- "고전 NOT과 동일" 라벨

### H 게이트 (아다마르)
- |0⟩ → (|0⟩+|1⟩)/√2 (중첩!)
- |1⟩ → (|0⟩-|1⟩)/√2
- 블로흐 구에서: 북극 → 적도(+x), 남극 → 적도(-x)
- "중첩 상태 생성 — 양자 컴퓨팅의 핵심" 라벨

### CNOT 게이트 (제어-NOT)
- 2큐비트 게이트: 제어 큐비트가 |1⟩이면 타겟 큐비트를 반전
- 얽힘 상태 생성에 사용
- 2개 블로흐 구로 시각화

## 양자 회로 다이어그램 (우측)
- 수평 선: 큐비트 와이어
- 사각형: 양자 게이트 (H, X, Z, CNOT)
- 측정 기호: 반원 + 화살표
- 간단한 회로:
  1. H → 측정: 50/50 랜덤 (동전 던지기)
  2. H → CNOT: 벨 상태(얽힘) 생성
  3. 도이치-조사 알고리즘(간략): 양자 병렬성 시연

## 고전 vs 양자 컴퓨터 비교

### 병렬성 시각화
- 고전 n비트: 한 번에 1개 상태만 (2ⁿ 중 1개)
  - 3비트: 000, 001, 010, ... 111 중 하나
- 양자 n큐비트: 2ⁿ개 상태를 동시에! (중첩)
  - 3큐비트: 8개 상태를 한 번에 처리
- 큐비트 수 슬라이더: 1~10
  - n=10: 고전 1024개 상태 vs 양자 동시 1024개!
  - n=30: 고전 ~10⁹ vs 양자 동시 ~10⁹!
- 막대그래프: 고전(1개) vs 양자(2ⁿ개) 동시 처리 상태 수

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 블로흐 구 / 양자 회로 / 고전 vs 양자
- "θ (극각)" 슬라이더: 0~180° (기본값 0 = |0⟩)
- "φ (방위각)" 슬라이더: 0~360° (기본값 0)
- "양자 게이트" 드롭다운: X / H / Z / T
- "게이트 적용" 버튼: 선택한 게이트를 현재 상태에 적용
- "측정" 버튼: 큐비트 측정 (확률적 결과)
- "큐비트 수 n" 슬라이더 (비교): 1~10
- "초기화" 버튼 (|0⟩으로 리셋)

```
큐비트 상태:
  |ψ⟩ = cos(θ/2)|0⟩ + e^(iφ)sin(θ/2)|1⟩
  θ = [값]°, φ = [값]°
  P(0) = cos²(θ/2) = [값]%
  P(1) = sin²(θ/2) = [값]%

양자 게이트:
  X: |0⟩↔|1⟩ (NOT)
  H: 중첩 생성 (|0⟩→(|0⟩+|1⟩)/√2)

양자 병렬성:
  n큐비트 → 2ⁿ상태 동시 처리
  n=[값] → 2^[값] = [값]개 동시!
```
