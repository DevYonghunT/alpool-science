너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 생물다양성과 보전
- 토픽: 생물다양성의 세 가지 요소
- 대상: 고등학교 1학년
- 핵심 개념: 생물다양성은 유전적 다양성, 종다양성, 생태계다양성을 모두 포함한다. 유전적 다양성은 한 생물종이 가지는 유전정보의 다양함을 의미하고, 종다양성은 한 생태계에 살고 있는 생물종의 다양함을 의미하며, 생태계다양성은 숲, 강, 바다 등 생태계의 종류가 다양함을 의미한다.

## 씬 구성
- 화면을 3분할하여 세 가지 다양성을 동시에 또는 전환하며 보여준다.
- 교과서 그림 I-15 (p26): 생물다양성의 세 가지 요소
- 배경: 0xeef4ee (연한 자연 녹색)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 3가지 다양성 시각화

### 유전적 다양성 (교과서 그림 I-16: 헬리코니우스나비)
- 3D: 같은 종의 나비 10마리 나란히 배치
- 나비 모형: PlaneGeometry 날개 2장(좌/우) + CylinderGeometry 몸통
  - 날개 크기: 0.5×0.8
  - 날개 색상: 같은 종이지만 **무늬가 개체마다 다름** (10가지 서로 다른 색상 패턴)
  - 패턴: 검정+주황, 검정+빨강, 검정+노랑, 검정+흰색, 갈색+주황 등
- 나비가 날개를 천천히 펄럭이는 애니메이션 (날개 rotation oscillation)
- 라벨: "유전적 다양성 — 같은 종이지만 개체마다 유전정보가 달라 형질이 다양하다"
- 하단 설명: "유전적 다양성이 높은 생물종은 환경 변화에 적응할 수 있는 유전자를 가진 개체가 있어 멸종할 가능성이 낮다"

### 종다양성
- 3D: 한 생태계(초원) 안에 다양한 생물종 배치
  - 초원 바닥: PlaneGeometry, 0x55aa33
  - 얼룩말: BoxGeometry 조합, 흑백 줄무늬 0x222222 + 0xffffff
  - 기린: BoxGeometry 조합, 긴 목, 0xddbb66
  - 나무: CylinderGeometry + SphereGeometry
  - 풀: 작은 ConeGeometry 여러 개
  - 새: 하늘에 날아다니는 작은 형태
- 종다양성 높은 생태계 vs 낮은 생태계 비교 토글:
  - 높음: 10종 이상의 생물
  - 낮음: 2~3종만 존재
- 라벨: "종다양성 — 한 생태계에 살고 있는 생물종의 다양함"
- 하단 설명: "종다양성이 높은 생태계일수록 더 복잡한 먹이그물이 형성되어 안정적으로 유지될 수 있다"

### 생태계다양성 (교과서 그림 I-17)
- 3D: 여러 종류의 생태계를 미니어처로 나란히 배치
  - 숲: 나무 여러 그루(CylinderGeometry + SphereGeometry), 바닥 진초록
  - 강: 파란 리본 형태(PlaneGeometry 길게), 주변에 갈대
  - 바다: 넓은 파란 면 + 물고기 + 산호
  - 초원: 풀밭 + 얼룩말
  - 사막: 모래색 바닥 + 선인장
- 각 생태계 위에 이름 라벨
- 라벨: "생태계다양성 — 숲, 초원, 사막, 강, 바다 등 다양한 종류의 생태계"
- 하단 설명: "생태계다양성이 높을수록 종다양성도 높게 나타난다"

## 생물다양성의 가치 (교과서 그림 I-18, p28)
- GUI 토글로 "생물자원" 패널 표시:
  - 식량 제공: 쌀, 콩 등의 아이콘
  - 의복 재료: 면화, 누에 등
  - 의약품: 디기탈리스 식물
  - 에너지: 바이오에탄올
  - 휴식처: 자연 경관
  - 산업용 재료: 효모
- 각 항목을 3D 아이콘으로 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "다양성 종류" 드롭다운: 유전적 다양성 / 종다양성 / 생태계다양성 / 전체 보기
- "종다양성 수준" 토글: 높음 / 낮음 (종다양성 모드에서만)
- "생물자원 표시" 토글: ON/OFF
- "자동 전환" 버튼: 3가지 다양성을 순서대로 전환 (각 6초)
- "초기화" 버튼

```

━━━━━━━━━━━━━━━━━━━
현재 보기:      [유전적 / 종 / 생태계]

유전적 다양성:
  한 생물종의 유전정보 다양함

  생태계 종류의 다양함
  → 높을수록 종다양성도 높음
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 다양성 종류 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_07_biodiversity_three_elements.html
