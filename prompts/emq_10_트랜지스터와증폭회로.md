너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 06. 반도체 소자를 활용한 전자 회로
- 토픽: 트랜지스터와 증폭 회로
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 트랜지스터는 3개의 반도체 영역(npn 또는 pnp)으로 구성된 소자로, 작은 베이스 전류로 큰 컬렉터 전류를 제어하는 전류 증폭 기능을 한다. 전류 이득 β = I_C/I_B (보통 100~300). 트랜지스터 증폭 회로는 마이크의 약한 전기 신호를 증폭하여 스피커를 구동하는 데 활용된다.

## 씬 구성
- 화면 좌측 50%: 3D 트랜지스터 내부 구조 + 증폭 회로
- 화면 우측 50%: 입력/출력 파형 비교 + 전류 막대그래프 (Canvas 2D)
- 2가지 모드: 트랜지스터 원리 / 마이크-스피커 증폭
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 모드 1: 트랜지스터 원리 (npn)

### 3D 트랜지스터 내부
- 3개 영역:
  - 이미터(E, n형): BoxGeometry(1, 2, 1), 0x4466ff (파랑), "E" 라벨
  - 베이스(B, p형): BoxGeometry(0.3, 2, 1), 0xff6644 (주황), "B" 라벨 (매우 얇음)
  - 컬렉터(C, n형): BoxGeometry(1, 2, 1), 0x4466ff (파랑), "C" 라벨
- 전자(n형): SphereGeometry(r=0.06) 노란 구, 이미터/컬렉터에 분포
- 양공(p형): RingGeometry(r=0.06) 빨간 원, 베이스에 분포

### 전류 흐름 시각화
- 베이스 전류 I_B (작은, 빨간 파티클): 베이스→이미터 방향 (소량)
- 컬렉터 전류 I_C (큰, 파란 파티클): 컬렉터→이미터 방향 (대량)
  - I_C = βI_B (β ≈ 100~300)
- I_B 슬라이더: 0~0.1mA → I_C = β × I_B = 0~30mA
- 전류 크기를 파티클 수/속도로 시각화
- "작은 I_B로 큰 I_C를 제어 = 전류 증폭" 강조 라벨

### 스위칭 동작
- I_B = 0: 트랜지스터 OFF, I_C = 0 (전류 차단)
- I_B > 0: 트랜지스터 ON, I_C = βI_B (전류 통과)
- LED: I_C에 의해 점등/소등 → 스위치 역할

## 모드 2: 마이크-스피커 증폭 회로

### 3D 회로
- 마이크: CylinderGeometry(0.5, 0.5, 0.8), 0x333333 + 그릴
  - 소리 → 약한 전기 신호 (I_B 역할)
  - 입력 파형: 작은 진폭 사인파
- 트랜지스터 (2N2222): BoxGeometry 3핀 소자, 0x333333
- 전원(V_CC): BoxGeometry 배터리, 0xdddddd
- 스피커: CylinderGeometry(1, 1, 0.3) + ConeGeometry 콘, 0x444444
  - 증폭된 전류 → 스피커 진동 → 큰 소리
  - 출력 파형: 큰 진폭 사인파 (입력의 β배)

### 증폭 시각화
- 입력 파형(마이크): 빨간 사인파, 작은 진폭 A_in
- 출력 파형(스피커): 초록 사인파, 큰 진폭 A_out = β × A_in
- 전압 이득: A_v = V_out/V_in
- 마이크에 소리 세기 슬라이더 → 입력 진폭 변화 → 출력 진폭 비례 변화

### 전류 막대그래프 (우측)
- I_B (빨강, 작은 막대): [값]μA
- I_C (파랑, 큰 막대): [값]mA
- β = I_C/I_B = [값]
- I_C ≈ β × I_B 확인

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 트랜지스터 원리 / 마이크-스피커 증폭
- "베이스 전류 I_B" 슬라이더: 0~100μA (기본값 30)
- "전류 이득 β" 슬라이더: 50~300 (기본값 100)
- "입력 신호 세기" 슬라이더 (모드 2): 0.1~1 (기본값 0.5)
- "전류 흐름 표시" 토글: ON/OFF (기본값 ON)
- "파형 표시" 토글: ON/OFF (기본값 ON)
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

```
📢 트랜지스터와 증폭 회로
━━━━━━━━━━━━━━━━
모드: [원리/증폭]
트랜지스터: npn (2N2222)

전류 이득: β = I_C/I_B = [값]
  I_B = [값]μA (베이스, 작음)
  I_C = βI_B = [값]mA (컬렉터, 큼)
  I_E = I_B + I_C ≈ I_C

작은 베이스 전류로 큰 컬렉터 전류 제어
= 전류 증폭!

증폭 회로:
  마이크(약한 신호) → 트랜지스터
  → 스피커(강한 신호)
  전압 이득 A_v = [값]
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "트랜지스터는 작은 베이스 전류(I_B)로 큰 컬렉터 전류(I_C=βI_B)를 제어합니다. 이 전류 증폭 기능으로 마이크의 약한 신호를 스피커를 구동할 수 있는 강한 신호로 증폭합니다."
- 탐구 질문:
  1. "트랜지스터의 전류 이득 β=200이고 I_B=50μA이면 I_C는 몇 mA인가요?"
  2. "트랜지스터를 스위치로 사용할 때 ON/OFF 상태는 각각 I_B와 어떤 관계인가요?"
  3. "마이크-스피커 증폭 회로에서 트랜지스터를 제거하면 어떻게 되나요?"
- 실생활 예시: 앰프(오디오 증폭), 스마트폰 내부 수억 개 트랜지스터, 컴퓨터 CPU
- 교과서 연계: "전자기와 양자 I편 p38~47 — 반도체 소자, 트랜지스터, 전자 회로"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 전류 파티클 최대 50개
- 파일명: emq_10_transistor_amplifier.html
