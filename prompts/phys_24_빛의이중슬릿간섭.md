너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 1. 빛과 물질의 이중성 > 빛의 파동성
- 토픽: 빛의 이중 슬릿 간섭 (영의 실험)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 이중 슬릿에 빛을 통과시키면 스크린에 밝고 어두운 간섭무늬가 나타난다. 두 슬릿에서 나온 빛의 경로차가 파장의 정수배이면 보강 간섭(밝은 무늬), 반파장의 홀수 배이면 상쇄 간섭(어두운 무늬)이 나타난다. 영의 이중 슬릿 간섭 실험은 빛의 파동성을 입증한 대표적 실험이다.

## 씬 구성
- 화면 좌측: 3D 이중 슬릿 실험 장치 (광원 + 슬릿 + 스크린)
- 화면 우측: 스크린 위 간섭무늬 + 강도 그래프 (Canvas 2D)
- 배경: 0x0a0a20 (어두운 실험실)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2) — 어두운 환경

## 3D 실험 장치

### 광원
- SphereGeometry(r=0.3), emissive, 색상은 빛의 파장에 따라 변화
  - 빨강(650nm): 0xff0000
  - 초록(550nm): 0x00ff00
  - 파랑(450nm): 0x0000ff
  - 흰색(전체): 0xffffff
- PointLight(광원 색상, 1.0)

### 이중 슬릿 판
- BoxGeometry(0.05, 4, 3), 0x333333 (검정 불투명판)
- 슬릿 2개: BoxGeometry(0.06, 슬릿폭, 0.01) × 2 빈 공간, 슬릿 간격 d
  - 슬릿 간격 d: 슬라이더로 조절 (0.1~2mm 스케일)
  - 슬릿 폭 w: 슬라이더로 조절
- 슬릿에서 빛이 통과: 노란/색상 파티클이 슬릿을 통과하여 퍼져나감

### 스크린
- PlaneGeometry(0.1, 4, 3), 0xffffff (흰색 스크린)
- 슬릿에서 거리 L 떨어진 위치
- 스크린 위에 간섭무늬가 프로젝션됨
- 간섭무늬: 밝은/어두운 띠가 반복 (스크린 텍스처 또는 동적 생성)

### 빛의 파동 시각화
- 각 슬릿에서 퍼져나가는 원형파: RingGeometry 확장 애니메이션 × 여러 겹
  - S₁에서: 빨간 원형파
  - S₂에서: 파란 원형파
- 보강 간섭 지점: 두 파동이 같은 위상으로 만남 → 밝은 점 표시
- 상쇄 간섭 지점: 반대 위상 → 어두운 점 표시

## 경로차와 간섭 조건

### 보강 간섭 (밝은 무늬)
- 조건: 경로차 Δr = mλ (m = 0, ±1, ±2, ...)
- 스크린 위 점 P까지 S₁, S₂에서의 거리 차 = 파장의 정수배
- 3D에서: S₁→P, S₂→P 경로를 빨간/파란 선으로 표시, 경로차 라벨

### 상쇄 간섭 (어두운 무늬)
- 조건: 경로차 Δr = (m+½)λ (m = 0, ±1, ±2, ...)
- 반파장의 홀수 배

### 무늬 간격
- Δy = λL/d (λ: 파장, L: 슬릿-스크린 거리, d: 슬릿 간격)
- 파장 길수록, L 클수록, d 작을수록 무늬 간격 넓어짐

## 간섭무늬 + 강도 그래프 (우측)

### 간섭무늬 이미지
- Canvas 2D 또는 텍스처: 밝고 어두운 띠가 반복되는 패턴
- 색상: 선택한 빛의 색상
- 파장/슬릿 간격에 따라 무늬 간격 변화

### 강도 분포 그래프
- X축: 스크린 위치 y
- Y축: 빛의 강도 I
- cos² 패턴: I ∝ cos²(πdy/λL)
- 밝은 무늬(극대)와 어두운 무늬(극소) 위치 표시
- 0차, 1차, 2차 극대 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "빛의 색" 드롭다운: 빨강(650nm) / 초록(550nm) / 파랑(450nm) / 흰색
- "파장 λ" 슬라이더: 400~700nm (기본값 550, 10 단계)
- "슬릿 간격 d" 슬라이더: 0.1~2mm (기본값 0.5, 0.1 단계)
- "슬릿-스크린 거리 L" 슬라이더: 0.5~3m (기본값 1)
- "파동 시각화" 토글: ON/OFF (원형파 표시)
- "경로차 표시" 토글: ON/OFF (특정 점에서 경로차)
- "강도 그래프" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌈 빛의 이중 슬릿 간섭 (영의 실험)
━━━━━━━━━━━━━━━━━━━
파장 λ = [값]nm
슬릿 간격 d = [값]mm
스크린 거리 L = [값]m

보강 간섭 (밝은 무늬):
  경로차 = mλ (m = 0, ±1, ±2, ...)

상쇄 간섭 (어두운 무늬):
  경로차 = (m+½)λ

무늬 간격:
  Δy = λL/d = [값]mm

의의:
  빛이 파동임을 입증한 실험 (영, 1801)
```

화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "이중 슬릿에 빛을 통과시키면 밝고 어두운 간섭무늬가 나타납니다. 경로차가 파장의 정수배이면 보강 간섭, 반파장의 홀수 배이면 상쇄 간섭입니다. 이 실험은 빛의 파동성을 입증했습니다."
- 탐구 질문:
  1. "파장이 긴 빨간빛과 짧은 파란빛 중 어느 것의 간섭무늬 간격이 더 넓은가요?"
  2. "슬릿 간격을 좁히면 간섭무늬는 어떻게 변하나요?"
  3. "영의 이중 슬릿 실험이 빛의 파동성을 입증하는 까닭은 무엇인가요? (힌트: 간섭은 파동의 특성)"
- 실생활 예시: CD/DVD 표면 무지개(빛의 간섭), 비눗방울 색, 기름막 무지개색
- 교과서 연계: "물리학 III편 p138~143 — 빛의 파동성, 이중 슬릿 간섭, 영의 실험 (그림 III-2~5)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 파라미터 변경 시 간섭무늬 실시간 재계산
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 원형파 RingGeometry 최대 20개/슬릿
- 파일명: phys_24_double_slit_interference.html
