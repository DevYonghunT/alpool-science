너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 별의 진화와 원소의 생성
- 토픽: 별의 진화와 원소 생성
- 대상: 고등학교 1학년
- 핵심 개념: 성간 물질이 모여 원시별이 되고, 중심부 온도가 1000만 K 이상이 되면 수소 핵융합 반응이 시작된다. 질량이 태양과 비슷한 별은 탄소까지 생성되고, 태양보다 훨씬 큰 별은 철까지 만들어진다. 초신성 폭발 시 철보다 무거운 금, 은, 우라늄 등이 생성된다.

## 씬 구성
- 화면 중앙에 별의 단면도를 3D로 렌더링한다.
- 화면 좌측에 별의 진화 타임라인을 세로로 배치한다.
- 배경: 우주 공간 0x000008, 별 파티클 Points(1000개)
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls
- 조명: AmbientLight(0.15) + PointLight(별 중심, emissive에 따라 세기 변화)

## 별의 진화 5단계 상세

### 단계 1: 성운 → 원시별
- 3D: 성운 — Points(800개) 불규칙 구형 분포
  - 파티클 색상: 0x6644aa(보라) + 0x4466cc(파랑) 랜덤 혼합, 크기 0.5~2.0
  - 반투명 구름 효과: SphereGeometry(r=5, opacity 0.08, 여러 겹 중첩)
- 수축 애니메이션:
  1. 성운 파티클이 서서히 중심으로 모여듦 (3초)
  2. 중심부가 점점 밝아짐 (PointLight 세기 0→2)
  3. 원시별 형성: SphereGeometry(r=1.5), 붉은 색 0xff4422, 약한 발광
- 교과서: "성간 물질이 모여들어 중력이 커지면 주변의 물질을 더 많이 끌어당겨 원시별이 만들어진다" (p54)
- 교과서 그림 II-8: 별의 탄생 과정 (p54)

### 단계 2: 주계열성 — 수소 핵융합
- 3D: 별 외부 SphereGeometry(r=2, 0xffcc44 노란색, emissive 0.6) + 반투명 대기층(r=2.3, opacity 0.15)
- 별 내부 단면도 (반구를 잘라 내부 표시):
  - 중심부: SphereGeometry(r=0.6), 0xffffff 극고온 흰색, emissive 1.0
  - 중심부 라벨: "수소 → 헬륨 (핵융합)" + "T > 1000만 K"
  - 외곽: 수소 가스층 0xffdd66
- 핵융합 애니메이션 (중심부 확대 뷰):
  - 수소 원자핵(빨강 구 r=0.05) 4개가 빠르게 이동
  - 2개씩 충돌 → 결합 → 헬륨 원자핵(노랑 큰 구 r=0.08) 생성
  - 결합 시 에너지 방출: 밝은 플래시 + 작은 파티클 방사
  - "수소 원자핵 4개 → 헬륨 원자핵 1개 + 에너지" 수식 표시
- 중력-압력 균형 시각화:
  - 중심에서 바깥으로 향하는 빨간 화살표(내부 압력) + 바깥에서 중심으로 향하는 파란 화살표(중력)
  - 화살표 크기 동일 → "힘의 평형" 라벨
- 교과서: "원시별의 중심부 온도가 1000만 K 이상이 되면 수소 핵융합 반응이 시작된다. 별의 중력과 내부 압력에 의한 힘이 평형을 이루어 크기가 일정하게 유지된다." (p55)
- 교과서 그림 II-9: 수소 핵융합 반응 도해 (p55)

### 단계 3: 적색 거성 — 헬륨 핵융합 (태양 질량 별)
- 3D: 별이 크게 팽창 — SphereGeometry(r=5, 0xff4422 붉은색, emissive 0.4)
- 별 내부 양파 구조 (교과서 그림 II-10 좌측, p56):
  - 최외곽: 수소 가스층 0xffdd66 (r=5→r=3)
  - 중간층: 헬륨 층 0xffaa33 (r=3→r=1.5)
  - 핵심부: 탄소 0x888888 (r=1.5→r=0)
  - 각 층 사이 경계선(흰색 점선 RingGeometry)
  - 각 층에 원소명 라벨: "수소", "헬륨", "탄소"
- 헬륨 핵융합 라벨: "T > 1억 K → 헬륨 → 탄소"
- 교과서: "헬륨으로 이루어진 중심부가 계속 수축해 온도가 1억 K 이상이 되면 헬륨 핵융합 반응이 일어나 탄소가 만들어진다. 질량이 태양과 비슷한 별은 중심부에서 탄소까지 생성된다." (p56)

### 단계 4: 초거성 — 연속 핵융합 (대질량 별)
- 3D: 매우 큰 별 SphereGeometry(r=7, 0xff2200 짙은 빨강, emissive 0.3)
- 별 내부 양파 구조 (교과서 그림 II-10 우측, p56):
  - 층 구조 (바깥→안쪽):
    | 층 | 원소 | 색상 | 반지름 범위 |
    |---|------|------|-----------|
    | 1 | 수소 | 0xffdd66 노랑 | r=7→5.5 |
    | 2 | 헬륨 | 0xffaa33 주황 | r=5.5→4.2 |
    | 3 | 탄소, 산소 | 0xcc8844 갈색 | r=4.2→3.0 |
    | 4 | 네온, 마그네슘 | 0xaa6633 짙은 갈색 | r=3.0→2.0 |
    | 5 | 규소, 황 | 0x886622 어두운 갈색 | r=2.0→1.2 |
    | 6 | 철 (핵) | 0x666666 회색 | r=1.2→0 |
  - 각 층에 원소명 + 핵융합 온도 라벨
  - 철 핵에 특별 표시: "⚠️ 철 — 더 이상 에너지 발생 없음"
- 교과서: "태양보다 질량이 훨씬 큰 별은 중심부의 온도가 충분히 높아질 수 있기 때문에 탄소가 만들어진 이후에도 연속적인 핵융합 반응이 일어난다. 그 결과 별의 내부에서는 산소, 마그네슘, 규소, 황 등을 포함해 최종적으로 철까지 만들어진다." (p56)

### 단계 5: 초신성 폭발 + 무거운 원소 생성
- 3D: 초거성이 급격히 수축 → 폭발 애니메이션
  1. 별이 빠르게 수축 (r=7 → r=1, 1초)
  2. 폭발: 중심에서 파티클(Points 1000개)이 사방으로 퍼짐
     - 파티클 색상: 흰색 → 노랑 → 주황 → 빨강 (거리에 따라)
     - 파티클 속도: 매우 빠르게, 화면 밖으로 퍼져나감
  3. 폭발 잔해 속에 무거운 원소 파티클 표시:
     - 금(Au): 노란 구 0xffd700, "Au" 라벨
     - 은(Ag): 은색 구 0xc0c0c0, "Ag" 라벨
     - 우라늄(U): 초록 구 0x44aa44, "U" 라벨
  4. 잔해가 서서히 퍼지며 새로운 성운 형성 → 순환 암시
- 교과서: "별의 중심부가 급격하게 붕괴해 폭발하면서 초신성이 된다. 초신성 폭발이 일어나는 과정에서 엄청난 양의 에너지가 발생해 철보다 무거운 금, 은, 우라늄 등의 원소가 만들어진다. 그 결과 별의 진화 과정에서 만들어진 다양한 원소가 우주 공간으로 흩어져 이후 새로운 별을 만드는 재료가 된다." (p56)
- 교과서 그림 II-11: 초신성 폭발로 남겨진 잔해의 모습 (p56)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "진화 단계" 슬라이더: 0 ~ 4 (5단계, 정수)
  - 0: 성운→원시별 / 1: 주계열성(수소 핵융합) / 2: 적색 거성(헬륨→탄소) / 3: 초거성(양파 구조) / 4: 초신성 폭발
- "별의 질량" 토글: 태양 질량 / 대질량 (기본값: 대질량)
  - 태양 질량 선택 시: 단계 2(적색 거성, 탄소까지)에서 진화 종료, 단계 3~4 비활성화
  - 대질량 선택 시: 단계 4(초신성)까지 진행 가능
- "내부 구조 보기" 토글: ON/OFF (기본값 ON)
  - ON: 별의 반구를 잘라 내부 양파 구조 표시
  - OFF: 별의 외관만 표시
- "핵융합 상세" 버튼 (단계 1에서만 활성):
  - 별 중심부를 확대, 수소 핵융합 과정 슬로모션 반복
- "자동 재생" 버튼: 단계 0→4 자동 진행 (각 단계 6초)
- "초기화" 버튼: 단계 0, 대질량, 기본 카메라

## 시각화 규칙

### 별의 크기 변화
- 단계별 별의 반지름이 변하는 것을 애니메이션으로 표현 (자체 lerp 보간)
- 주계열성(r=2) → 적색 거성(r=5) → 초거성(r=7) → 수축(r=1) → 폭발

### 양파 구조 층 인터랙션
- 내부 구조 ON 상태에서 각 층에 마우스 호버 시:
  - 해당 층이 밝게 강조
  - 툴팁: 원소명, 핵융합 온도, 밀도 정보

### 온도 컬러 바
- 별 옆에 세로 온도 컬러 바 표시 (파랑 0K → 빨강 10¹⁰K)
- 현재 중심부 온도 위치를 화살표로 표시

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⭐ 별의 진화와 원소 생성
━━━━━━━━━━━━━━━━━━
현재 단계:     [단계명]
별의 질량:     [태양 질량 / 대질량]
중심부 온도:   [값] K
중심부 원소:   [현재 핵융합 중인 원소]
생성된 원소:   [지금까지 생성된 원소 목록]

주요 핵융합 반응:
  4¹H → ⁴He + 에너지 (T > 10⁷ K)
  3⁴He → ¹²C + 에너지 (T > 10⁸ K)
  ...
  ²⁸Si → ⁵⁶Fe + 에너지 (T > 3×10⁹ K)
  ⁵⁶Fe 이후: 초신성 폭발로 Au, Ag, U 등 생성
```
단계 변경 시 실시간 업데이트.

- 핵심 원리: "별의 중심부에서 핵융합 반응을 통해 가벼운 원소에서 무거운 원소가 차례로 만들어집니다. 철까지는 별 내부에서, 철보다 무거운 원소는 초신성 폭발 시 만들어집니다."
- 탐구 질문:
  1. "태양과 비슷한 질량의 별과 대질량 별의 최종 생성 원소가 다른 까닭은 무엇인가요?"
  2. "철 이후의 원소가 별 내부 핵융합이 아닌 초신성 폭발에서만 만들어지는 까닭은 무엇인가요? (힌트: 에너지)"
  3. "우리 몸을 구성하는 탄소, 산소, 철 등은 어디에서 왔을까요?"
- 실생활 예시: 금 장신구(초신성 폭발에서 생성된 원소), 철 구조물(별 내부 핵융합 산물), 태양(현재 수소 핵융합 중)
- 교과서 연계: "미래엔 통합과학1 II편 p54~56 — 별의 진화와 원소의 생성, 초신성 폭발"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 양파 구조: 동심원 SphereGeometry 여러 겹 (각각 다른 반지름, 반투명)
- 초신성 폭발 파티클: BufferGeometry + Points (1000개)
- 단계 전환 시 이전 객체 dispose()
- 성능: requestAnimationFrame 루프, InstancedMesh로 핵융합 파티클 최적화
- 파일명: stellar_evolution_elements_sim.html
