너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 1. 지구시스템 > 지구시스템의 구성과 상호작용
- 토픽: 지구시스템의 상호작용
- 대상: 고등학교 1학년
- 핵심 개념: 지구시스템을 구성하는 기권, 수권, 지권, 생물권은 서로 밀접한 관계를 맺고 있으며 끊임없이 상호작용하면서 균형을 이루고 있다. 어느 한 권역에서 일어난 변화는 다른 권역에 연쇄적으로 영향을 줄 수 있다. 지구 온난화나 환경오염은 이 균형을 위협한다.

## 씬 구성
- 화면 중앙에 4개 권역을 원형 다이어그램으로 배치 (교과서 그림 III-9, p110)
- 4개 권역을 큰 원 위에 90° 간격으로 배치:
  - 상단(12시): 기권 — 하늘+구름 아이콘 (SphereGeometry + 파란 반투명)
  - 우측(3시): 수권 — 물방울+파도 아이콘 (SphereGeometry 파랑)
  - 하단(6시): 지권 — 산+암석 아이콘 (ConeGeometry 갈색)
  - 좌측(9시): 생물권 — 나무+동물 아이콘 (SphereGeometry 녹색 + CylinderGeometry)
- 각 권역 사이에 양방향 화살표(6쌍, 12개 화살표) — 상호작용 표시
- 배경: 0x0f1f2f (짙은 남색)
- 조명: AmbientLight(0.4) + PointLight(중앙, 0xffffff, 세기 1.0)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (다이어그램을 회전 관찰)

## 6가지 상호작용 상세 (교과서 그림 III-9, p110)

| # | 권역 쌍 | 상호작용 예시 | 화살표 색 |
|---|---------|-------------|----------|
| 1 | 기권 ↔ 수권 | 강수/증발, 해류(바람에 의해) | 0x4488ff 파랑 |
| 2 | 기권 ↔ 생물권 | 호흡/광합성, 바람에 의한 꽃가루 이동 | 0x44aa44 초록 |
| 3 | 기권 ↔ 지권 | 풍화/침식, 화산 활동(화산 가스) | 0xff6644 주황 |
| 4 | 수권 ↔ 생물권 | 수분 공급, 해양 생태계 | 0x22aacc 청록 |
| 5 | 수권 ↔ 지권 | 풍화/침식(파도), 하천에 의한 퇴적 | 0xaa8844 갈색 |
| 6 | 생물권 ↔ 지권 | 토양 형성, 뿌리에 의한 풍화 | 0x88aa44 연두 |

### 각 상호작용 애니메이션
- 상호작용 경로: 두 권역 사이 곡선(QuadraticBezierCurve3) 화살표
- 파티클(작은 구 r=0.03): 화살표 경로를 따라 이동, 양방향 동시 진행
- 권역 쌍 클릭 시: 해당 상호작용 경로가 밝게 강조(emissive 0.5) + 나머지 경로 페이드(opacity 0.1) + 하단에 상세 설명 팝업(HTML overlay, 배경 rgba(0,0,0,0.8), 200×120px)
  - 팝업 내용: 상호작용 이름, 방향(A→B / B→A), 구체적 예시 2~3개, 교과서 근거 페이지
  - 예: 기권↔수권 클릭 → "강수(기권→수권): 구름에서 비/눈 형태로 이동\n증발(수권→기권): 태양 에너지로 수증기 형태로 이동\n해류(기권→수권): 바람에 의해 발생"
  - 예: 기권↔지권 클릭 → "풍화·침식(기권→지권): 바람에 의한 암석 깎임\n화산 활동(지권→기권): 화산 가스·화산재 분출"

## 균형 깨짐 시나리오 (교과서 p111)
"시나리오" 버튼으로 균형이 깨지는 상황을 시뮬레이션:

### 시나리오 1: 지구 온난화
- 기권의 CO₂ 농도↑ → 기권 영역이 빨갛게 변화(열 효과)
- 연쇄 반응: 기권 온도↑ → 수권(빙하 감소, 해수면↑) → 생물권(서식지 변화) → 지권(영구동토 해빙)
- 각 단계를 순차 애니메이션으로 표시 (2초/단계)
- 권역 간 화살표가 빨갛게 변하며 "불균형!" 경고

### 시나리오 2: 오존층 파괴
- 기권 오존층 약화 → 자외선 증가 → 생물권(피부암, 작물 피해) → 수권(해양 플랑크톤 감소)
- 기권 영역에서 오존층(보라색 링)이 옅어지는 애니메이션

### 정상 복귀
- "균형 복원" 버튼: 모든 권역이 정상 색상으로 돌아옴 + "균형 유지" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "상호작용 선택" 드롭다운: 전체 / 기권↔수권 / 기권↔생물권 / 기권↔지권 / 수권↔생물권 / 수권↔지권 / 생물권↔지권
  - 선택된 상호작용만 밝게 강조, 나머지 페이드
- "시나리오" 드롭다운: 정상 / 지구 온난화 / 오존층 파괴
- "연쇄 반응 재생" 버튼: 시나리오 애니메이션 재생
- "균형 복원" 버튼
- "파티클 흐름 표시" 토글: ON/OFF (기본값 ON) — 상호작용 경로 파티클
- "권역 라벨" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

```

선택 상호작용: [권역 쌍]

6가지 상호작용:
  기권↔수권: 강수/증발, 해류
  기권↔생물권: 호흡/광합성
  기권↔지권: 풍화/침식, 화산 활동
  수권↔생물권: 수분 공급
  수권↔지권: 풍화/침식(파도)
  생물권↔지권: 토양 형성

핵심: 한 권역의 변화 → 다른 권역에 연쇄 영향
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 다이어그램: Group으로 4개 권역 아이콘(각각 SphereGeometry/ConeGeometry/CylinderGeometry 조합) + 12개 화살표(ArrowHelper 또는 CurveGeometry + 삼각형 머리) 조립
  - 4개 권역 아이콘:
    - 기권: SphereGeometry(r=0.8), 0x88ccee, opacity 0.5 + 구름(작은 SphereGeometry 3개 뭉침, 0xffffff)
    - 수권: SphereGeometry(r=0.8), 0x2266aa + 물결(sin 기반 vertex 변형)
    - 지권: ConeGeometry(r=0.6, h=1.0), 0x886644 + 암석(IcosahedronGeometry r=0.1 × 3)
    - 생물권: SphereGeometry(r=0.5, 0x228b22 녹색) + CylinderGeometry(r=0.05, h=0.4, 0x6B4226 줄기) = 나무 형태
  - 권역 아이콘 배치: 큰 원(r=4.0) 위 90° 간격 (12시, 3시, 6시, 9시)
  - 각 아이콘 아래 라벨: Sprite(한글, 20px bold, 흰색)
- 화살표 경로: QuadraticBezierCurve3(시작 권역 위치, 제어점(중앙 약간 바깥), 도착 권역 위치)
  - 양방향: 시작→도착, 도착→시작 각각 별도 곡선 (약간 다른 제어점으로 겹침 방지)
  - 화살표 머리: ConeGeometry(r=0.06, h=0.15), 곡선 끝에 배치, 방향 = 곡선 접선
  - 각 경로 색상: 상호작용 유형별 (6가지 색상, 위 표 참조)
- 파티클 흐름: BufferGeometry + Points, 각 경로에 3~5개 작은 구(r=0.03) 파티클이 곡선 위 curve.getPointAt(t) 경로를 따라 이동
  - t: 0→1 (3초 주기), 도착 후 0으로 리셋 (연속 흐름)
  - 파티클 색상 = 경로 색상
- 시나리오 애니메이션 (지구 온난화 등):
  - 기권 아이콘 색상: lerp(정상색, 0xff4444 빨강, 진행도), 2초
  - 연쇄 반응: setTimeout 체이닝으로 순차 트리거 (2초 간격)
  - 각 권역 변화 시 "→" 화살표 텍스트 라벨로 연쇄 표시
- dispose(): 시나리오 전환 시 mesh를 재생성하지 않고 material.color/emissive 변경
  - 균형 복원 시 원래 색상으로 lerp 복원 (1초)
- 성능: requestAnimationFrame 루프, 권역 아이콘 4개 + 화살표 12개 + 파티클 ~36개 = 총 ~52 mesh
- 모바일: lil-gui 기본 접힘, OrbitControls enableDamping=true
- 파일명: earth_system_interaction_sim.html
