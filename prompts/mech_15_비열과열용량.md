너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 02. 열에 의한 물질의 변화
- 토픽: 비열과 열용량
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 비열은 물질 1kg의 온도를 1K 높이는 데 필요한 열량으로, 단위는 J/(kg·K)이다. Q = cmΔT (Q: 열량, c: 비열, m: 질량, ΔT: 온도 변화). 비열은 물질에 따라 다르며, 물의 비열(4186 J/(kg·K))은 대부분의 금속보다 훨씬 크다. 열용량 = 비열 × 질량 (C = cm)으로 물체의 온도를 1K 높이는 데 필요한 열량이다.

## 씬 구성
- 화면 좌측: 3D 가열 실험 장치 (알코올램프 + 비커 + 물질)
- 화면 우측: 온도-시간 그래프 (Canvas 2D) + 비열 비교 표
- 2가지 모드: 비열 비교 실험 / 열평형
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 모드 1: 비열 비교 실험

### 3D 실험 장치
- 알코올램프 2개: CylinderGeometry + 불꽃 파티클(0xff6633), 동일 열량 공급
- 비커 A: CylinderGeometry(0.8, 0.8, 1.5) 투명, 물 담김 (0x4488cc)
- 비커 B: CylinderGeometry(0.8, 0.8, 1.5) 투명, 식용유 담김 (0xddcc44)
  - 또는 금속(알루미늄 블록, 0xaabbcc)
- 각 비커에 온도계: CylinderGeometry + 빨간 수은 막대
- 동시에 같은 열량 공급 → 온도 변화 비교

### 온도 변화 시뮬레이션
- 같은 열량 Q를 공급할 때:
  - 물(c=4186): 온도 적게 상승 (비열 큼)
  - 식용유(c≈2000): 온도 더 많이 상승
  - 알루미늄(c=900): 온도 훨씬 많이 상승
- ΔT = Q/(cm) → c가 클수록 ΔT 작음
- 온도계 빨간 막대가 실시간 상승 (서로 다른 속도)

### 온도-시간 그래프 (우측)
- X축: 시간 t (0~120초)
- Y축: 온도 T (20~100°C)
- 물질별 곡선 (물: 파랑/느리게 상승, 알루미늄: 빨강/빠르게 상승)
- 같은 열량에서 비열이 클수록 온도 상승 느림

### 비열 비교 표
| 물질 | 비열 J/(kg·K) | 순위 |
|------|-------------|------|
| 물 | 4186 | 매우 큼 |
| 식용유 | ~2000 | 중간 |
| 알루미늄 | 900 | 작음 |
| 구리 | 390 | 매우 작음 |
| 납 | 130 | 극히 작음 |

## 모드 2: 열평형

### 3D
- 단열 용기: BoxGeometry, 0xddddcc (스티로폼)
- 뜨거운 금속 블록(80°C): BoxGeometry, 0xcc4444 → 냉각됨
- 차가운 물(20°C): CylinderGeometry, 0x4488cc → 가열됨
- 금속을 물에 넣으면: 열평형 도달까지 온도 변화 애니메이션

### 열평형 계산
- 열 잃음 = 열 얻음: c_금속 × m_금속 × ΔT_금속 = c_물 × m_물 × ΔT_물
- 최종 온도 T_f 실시간 계산
- 온도-시간 그래프: 금속 온도 하강 + 물 온도 상승 → 같은 값에서 만남

## 물의 높은 비열 효과
- "물의 비열이 큰 것의 영향" 패널:
  - 해안 도시 vs 내륙 도시 기온 차이 (바다의 비열이 커서 온도 변화 완만)
  - 자동차 냉각수로 물 사용 (열을 많이 흡수)
  - 바다 수영장이 천천히 데워지고 천천히 식음

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 비열 비교 실험 / 열평형
- "물질 A" 드롭다운: 물 / 식용유 / 알코올
- "물질 B" 드롭다운: 알루미늄 / 구리 / 철 / 납
- "질량" 슬라이더: 0.1~2kg (기본값 0.5)
- "공급 열량" 슬라이더 (모드 1): 1000~50000J (기본값 10000)
- "금속 초기 온도" 슬라이더 (모드 2): 50~100°C (기본값 80)
- "물 초기 온도" 슬라이더 (모드 2): 10~30°C (기본값 20)
- "시뮬레이션 시작" 버튼
- "비열 표" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌡️ 비열과 열용량
━━━━━━━━━━━━━━
모드: [비열 비교 / 열평형]

비열: c [J/(kg·K)]
  물: 4186 / 알루미늄: 900
  구리: 390 / 납: 130

열량: Q = cmΔT
  Q = [값]J
  c = [값]J/(kg·K)
  m = [값]kg
  ΔT = [값]K → T_최종 = [값]°C

열평형:
  Q_잃음 = Q_얻음
  c₁m₁ΔT₁ = c₂m₂ΔT₂
  T_평형 = [값]°C
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "비열은 물질 1kg의 온도를 1K 높이는 데 필요한 열량입니다. Q=cmΔT. 물의 비열은 대부분의 물질보다 크기 때문에 같은 열량을 받아도 온도가 적게 변합니다."
- 탐구 질문:
  1. "같은 질량의 물과 구리에 같은 열량을 가하면 어느 것의 온도가 더 많이 올라가나요? 그 까닭은?"
  2. "해안 도시가 내륙 도시보다 기온의 일교차가 작은 까닭을 물의 비열과 관련지어 설명해 보세요."
  3. "80°C 구리 100g을 20°C 물 200g에 넣으면 최종 온도는 몇 °C인가요?"
- 실생활 예시: 자동차 냉각수(물), 해안 기후(바다의 큰 비열), 온돌(바닥 열용량)
- 교과서 연계: "역학과 에너지 II편 p70~75 — 열에 의한 물질의 변화, 비열"

- 성능: requestAnimationFrame 루프
