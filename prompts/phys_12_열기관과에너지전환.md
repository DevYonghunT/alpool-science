너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 2. 에너지와 열 > 역학적 에너지와 열
- 토픽: 열기관과 에너지 전환
- 대상: 고등학교 (선택과목)
- 핵심 개념: 열기관은 열에너지를 역학적 에너지로 전환하는 장치이다. 1700년대에 석탄을 태워 얻은 열에너지를 역학적 에너지로 전환할 수 있는 장치인 증기 기관이 발명되었고, 이후 내연 기관이 개발되었다. 열기관에서 공급한 열에너지가 모두 역학적 에너지로 전환되지는 않으며, 일부는 폐열로 방출된다.

## 씬 구성
- 화면 중앙: 3D 열기관 모형 (증기 기관 또는 내연 기관)
- 화면 우측: 에너지 흐름 다이어그램 (Sankey 다이어그램 스타일)
- 2가지 열기관: 증기 기관 / 내연 기관(4행정)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 열기관 1: 증기 기관

### 3D 모형
- 보일러: CylinderGeometry(1.5, 1.5, 2), 0x666666 (금속 회색)
  - 내부에 물(파란색) + 불꽃(빨간 파티클, 하단)
  - 증기(흰색 파티클, 위로 올라감)
- 피스톤: CylinderGeometry(0.4, 0.4, 1) + BoxGeometry 로드, 0xaaaaaa
  - 증기 압력으로 왕복 운동 (좌-우 반복)
- 플라이휠(바퀴): TorusGeometry(1, 0.15), 0x888888
  - 피스톤과 연결, 왕복 → 회전 변환
  - 회전 애니메이션

### 에너지 전환 과정
1. 연료(석탄) 연소 → **열에너지** (불꽃 파티클)
2. 물 가열 → 증기 생성 → 피스톤 밀기 → **운동 에너지** (왕복)
3. 크랭크 → 바퀴 회전 → **역학적 에너지** (회전)
4. 일부 열에너지는 폐열로 방출 (주황 파티클 외부로)
- 각 단계를 화살표 + 라벨로 표시

## 열기관 2: 내연 기관 (4행정)

### 3D 모형
- 실린더: CylinderGeometry(0.5, 0.5, 1.5), 0x777777, 투명(단면)
- 피스톤: CylinderGeometry(0.48, 0.48, 0.3), 0xaaaaaa
- 밸브 2개: CylinderGeometry(0.08, 0.08, 0.3) × 2 (흡기/배기)
- 점화 플러그: CylinderGeometry(0.04, 0.04, 0.2) + 스파크 효과

### 4행정 사이클 애니메이션
1. **흡입 행정**: 피스톤 하강, 흡기 밸브 열림, 연료-공기 혼합물(파란 파티클) 유입
2. **압축 행정**: 밸브 닫힘, 피스톤 상승, 혼합물 압축 (부피 감소, 온도 상승)
3. **폭발(팽창) 행정**: 점화 스파크 → 폭발(빨간-주황 플래시) → 피스톤 하강 (동력)
4. **배기 행정**: 배기 밸브 열림, 피스톤 상승, 배기 가스(회색 파티클) 배출
- 각 행정을 단계별로 재생 (각 1.5초)
- 현재 행정 라벨 표시: "① 흡입 / ② 압축 / ③ 폭발 / ④ 배기"

## 에너지 흐름 다이어그램 (우측)
- Sankey 다이어그램 스타일:
  - 입력: 열에너지 Q_in (넓은 빨간 막대, 100%)
  - 출력 1: 역학적 에너지 W (초록 막대, ~25~40%)
  - 출력 2: 폐열 Q_out (주황 막대, ~60~75%)
- Q_in = W + Q_out (에너지 보존)
- 효율 η = W/Q_in × 100% 표시
- 효율 슬라이더에 따라 비율 변화

## 영구 기관 불가능 (교과서 내용)
- "영구 기관" 설명 패널:
  - 외부에서 에너지를 공급받지 않고 영원히 일을 하는 장치는 불가능
  - 에너지 보존 법칙에 의해 불가능
  - 과거 영구 기관 시도 사례 이미지 (간단한 3D 표현)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "열기관 종류" 드롭다운: 증기 기관 / 내연 기관(4행정)
- "행정 선택" 드롭다운 (내연 기관): 흡입 / 압축 / 폭발 / 배기 / 자동 순환
- "효율" 슬라이더: 10~50% (기본값 30%)
- "열에너지 입력" 슬라이더: 100~1000J (기본값 500)
- "애니메이션 속도" 슬라이더: 0.5x~3x
- "에너지 흐름 표시" 토글: ON/OFF (기본값 ON)
- "영구 기관 설명" 토글: ON/OFF
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 열기관 종류 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 200개
- 파일명: phys_12_heat_engine.html
