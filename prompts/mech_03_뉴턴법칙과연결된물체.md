너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 02. 뉴턴 운동 법칙과 포물선 운동
- 토픽: 뉴턴 운동 법칙과 연결된 물체
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 실로 연결된 두 물체(수레+매달린 추, 양쪽에 매달린 추 등)의 운동을 뉴턴 운동 법칙으로 분석할 수 있다. 연결된 물체 전체를 하나의 계(system)로 보면 계의 가속도 a = F_net/m_total이다. 각 물체에 작용하는 장력 T는 개별 물체의 운동 방정식으로 구한다. 엘리베이터 안에서의 겉보기 무게도 뉴턴 법칙으로 설명한다.

## 씬 구성
- 3가지 시나리오: 수평면 수레+매달린 추 / 앳우드 기계 / 엘리베이터
- 각 시나리오에서 힘 분석 + 운동 시뮬레이션
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 수평면 수레 + 매달린 추

### 3D
- 수평 레일: BoxGeometry(10, 0.1, 0.5), 0xaaaaaa
- 수레(m₁): BoxGeometry(1, 0.5, 0.6), 0x4488aa, 바퀴 4개
  - 위에 추 올리기 가능 (질량 변경)
  - "m₁ = [값]kg" 라벨
- 도르래: TorusGeometry(0.15, 0.03), 레일 끝에
- 실: CylinderGeometry(0.01, 0.01, 가변), 0xdddddd
- 매달린 추(m₂): CylinderGeometry(0.15, 0.15, 0.2) × N개, 0xff4444
  - "m₂ = [값]kg" 라벨

### 힘 분석 (힘 화살표 표시)
- 수레: 장력 T(오른쪽 →), 수직 항력 N(위 ↑), 중력 m₁g(아래 ↓)
- 매달린 추: 중력 m₂g(아래 ↓), 장력 T(위 ↑)
- 알짜힘: 계 전체 F_net = m₂g, m_total = m₁ + m₂
- 가속도 a = m₂g/(m₁+m₂)
- 장력 T = m₁a = m₁m₂g/(m₁+m₂)

### 애니메이션
- "시작" 클릭: 매달린 추 하강 → 수레 가속 → 실시간 속도/위치 표시
- 도르래 회전 애니메이션
- 추의 질량 변경 → 가속도 변화

## 시나리오 2: 앳우드 기계 (Atwood machine)

### 3D
- 도르래: TorusGeometry(0.3, 0.05), 상단에 고정, 0x888888
- 실: 도르래를 지나 양쪽으로 내려옴
- 물체 1(m₁): SphereGeometry(r=0.2), 0xff4444, 좌측에 매달림
- 물체 2(m₂): SphereGeometry(r=0.25), 0x4444ff, 우측에 매달림 (m₂ > m₁)

### 힘 분석
- m₁: 중력 m₁g(↓), 장력 T(↑)
- m₂: 중력 m₂g(↓), 장력 T(↑)
- 계 전체: F_net = (m₂-m₁)g, m_total = m₁+m₂
- a = (m₂-m₁)g/(m₁+m₂)
- T = 2m₁m₂g/(m₁+m₂)
- m₁ = m₂이면: a = 0 (평형)

### 애니메이션
- 무거운 쪽(m₂) 하강, 가벼운 쪽(m₁) 상승
- 질량비에 따른 가속도 변화

## 시나리오 3: 엘리베이터와 겉보기 무게

### 3D
- 엘리베이터 상자: BoxGeometry(2, 3, 2), 0xddddcc (반투명 벽)
- 사람: BoxGeometry 조합 실루엣, 0xddccbb, 엘리베이터 안에 서 있음
- 체중계: BoxGeometry(1, 0.1, 1), 0xcccccc, 사람 아래
  - 디지털 표시: "W' = [값]N" (겉보기 무게)
- 층 표시: 1F, 2F, 3F... 옆에 표시

### 상황별 겉보기 무게
- 정지/등속: W' = mg (실제 무게와 같음)
- 위로 가속: W' = m(g+a) > mg (무거워지는 느낌)
- 아래로 가속: W' = m(g-a) < mg (가벼워지는 느낌)
- 자유 낙하(a=g): W' = 0 (무중력!)
- 각 상황을 애니메이션으로 시연
- 체중계 숫자가 실시간 변화

### 힘 분석
- 사람에 작용: 중력 mg(↓), 수직 항력 N(↑)
- N = m(g±a) = 겉보기 무게
- 엘리베이터 가속도 방향/크기에 따라 N 변화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 수레+추 / 앳우드 기계 / 엘리베이터
- "m₁ 질량" 슬라이더: 0.5~10kg (기본값 2)
- "m₂ 질량" 슬라이더: 0.5~10kg (기본값 3)
- "엘리베이터 상태" 드롭다운 (시나리오 3): 정지 / 위로 가속 / 아래로 가속 / 등속 / 자유낙하
- "가속도" 슬라이더 (시나리오 3): 0~9.8m/s²
- "시뮬레이션 시작" 버튼
- "힘 화살표 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔗 뉴턴 법칙과 연결된 물체
━━━━━━━━━━━━━━━━━
시나리오: [수레+추/앳우드/엘리베이터]
m₁ = [값]kg, m₂ = [값]kg

수레+추:
  a = m₂g/(m₁+m₂) = [값]m/s²
  T = m₁m₂g/(m₁+m₂) = [값]N

앳우드:
  a = (m₂-m₁)g/(m₁+m₂) = [값]m/s²
  T = 2m₁m₂g/(m₁+m₂) = [값]N

엘리베이터:
  상태: [정지/위가속/아래가속/자유낙하]
  겉보기 무게 W' = m(g±a) = [값]N
  실제 무게 W = mg = [값]N
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: mech_03_connected_objects.html
