너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 2. 자연의 구성 물질 > 지각과 생명체를 구성하는 물질
- 토픽: DNA 이중나선 구조
- 대상: 고등학교 1학년
- 핵심 개념: 핵산에는 DNA와 RNA가 있다. 핵산의 기본 단위체는 뉴클레오타이드(인산+당+염기)이다. DNA는 두 가닥의 폴리뉴클레오타이드가 꼬여 있는 이중나선 구조이다. DNA의 염기 규칙: 아데닌(A)은 타이민(T)과, 구아닌(G)은 사이토신(C)과 결합한다. RNA는 단일 가닥이며 타이민 대신 유라실(U)을 가진다. DNA는 유전정보를 저장하며 염기 배열 순서에 따라 서로 다른 유전정보를 가진다.

## 씬 구성
- 화면 중앙에 3D DNA 이중나선 모형을 렌더링한다.
- 배경: 0x0a1020 (짙은 생물학 남색)
- 조명: AmbientLight(0.3) + DirectionalLight(0.6, 좌상단) + SpotLight(DNA 위에서 아래로, 세기 0.5)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (DNA를 360도 회전 관찰)
- DNA는 Y축을 따라 수직으로 배치, 느리게 자동 회전 (Y축, 0.003 rad/frame)

## DNA 이중나선 3D 모형 상세 (교과서 그림 II-30, p84)

### 뉴클레오타이드 기본 구조
- 인산(P): SphereGeometry(r=0.15), 0xff4444 (빨강), "P" 라벨
- 당(디옥시리보스): 오각형 형태 — ExtrudeGeometry(오각형 Shape, depth=0.1), 0x8844aa (보라), "S" 라벨
- 염기: BoxGeometry(너비=0.6~1.0, 높이=0.3, 깊이=0.1), 색상은 염기 종류에 따라:
  | 염기 | 기호 | 색상 | 짝 |
  |------|------|------|-----|
  | 아데닌 | A | 0x44aa44 초록 | T와 결합 |
  | 타이민 | T | 0xff6644 주황 | A와 결합 |
  | 구아닌 | G | 0x4488ff 파랑 | C와 결합 |
  | 사이토신 | C | 0xffcc44 노랑 | G와 결합 |

### 이중나선 구조 생성
- 나선 매개변수:
  - 반지름: R = 1.5 (당-인산 백본의 나선 반지름)
  - 피치: P = 3.4 (1회전당 높이, 실제 DNA는 3.4nm)
  - 회전당 염기쌍: 10개
  - 총 염기쌍 수: 20개 (2회전)
- 두 가닥의 당-인산 백본:
  - 가닥 1: TubeGeometry(나선 곡선 1, r=0.06), 0x8844aa (보라)
    - 나선 곡선: x = R × cos(t), y = t × P/(2π), z = R × sin(t)
  - 가닥 2: TubeGeometry(나선 곡선 2, r=0.06), 0x8844aa
    - 나선 곡선: x = R × cos(t + π), y = t × P/(2π), z = R × sin(t + π)
    - (가닥 1에서 180° 위상차)
  - 각 나선 위에 당(보라 오각형)과 인산(빨간 구)이 교대로 배치
- 염기쌍 연결:
  - 가닥 1의 염기와 가닥 2의 상보적 염기를 CylinderGeometry(r=0.03)로 연결
  - A-T 결합: 연결봉 2개 (이중 수소결합 표현), 연결봉 색상 0xffffff
  - G-C 결합: 연결봉 3개 (삼중 수소결합 표현), 연결봉 색상 0xffffff
  - 각 염기 블록에 기호(A, T, G, C) 라벨
- 기본 염기 서열 (가닥 1, 5'→3'):
  - A-T-G-C-C-A-G-T-A-C | G-T-A-C-G-A-T-C-G-A (20개)
  - 가닥 2(상보적): T-A-C-G-G-T-C-A-T-G | C-A-T-G-C-T-A-G-C-T

### DNA vs RNA 비교 (토글)
- "RNA 보기" 토글 ON 시:
  - DNA 옆에 RNA 단일 가닥을 배치
  - RNA: 가닥 1개만 (나선이 아닌 구불구불한 단일 곡선 TubeGeometry)
  - 당: 리보스(보라보다 약간 밝은 0xaa66cc)
  - 염기: A, G, C는 동일, T 대신 **유라실(U)**: 0xff8866 (연한 주황, T와 구분)
  - 라벨: "RNA — 단일 가닥, U(유라실)이 T(타이민) 대체"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "표시 모드" 드롭다운: 완성 모형 / 조립 과정 / 뉴클레오타이드 상세
- "조립 과정" 모드:
  - 뉴클레오타이드가 하나씩 추가되며 DNA가 조립되는 애니메이션
  - 각 뉴클레오타이드가 위에서 날아와 적절한 위치에 결합 (0.3초/개)
  - 상보적 염기쌍 연결 시 A↔T, G↔C 매칭 효과 (맞으면 초록 플래시, 틀리면 빨간 경고)
- "염기 서열 입력" 텍스트 필드: 사용자가 A, T, G, C 문자열 입력 → 상보적 가닥 자동 생성 + DNA 모형 업데이트
- "RNA 비교" 토글: ON/OFF (기본값 OFF)
- "자동 회전" 토글: ON/OFF (기본값 ON)
- "회전 속도" 슬라이더: 0x ~ 5x (기본값 1x)
- "구조 요소 표시" 체크박스 3개:
  - ☑ 당-인산 백본 (기본 ON)
  - ☑ 염기쌍 (기본 ON)
  - ☑ 수소결합 (기본 ON)
- "풀기" 슬라이더: 0% ~ 100% (기본값 0%)
  - 0%: 정상 이중나선
  - 50%: 반쯤 풀린 사다리 형태
  - 100%: 완전히 풀린 평면 사다리 → 염기쌍 규칙을 한눈에 볼 수 있음
- "초기화" 버튼

## 시각화 규칙

### 염기쌍 규칙 강조
- 특정 염기쌍에 마우스 호버 시:
  - 해당 염기쌍 밝게 강조 (emissive 0.5)
  - 툴팁: "A — T (이중 수소결합)" 또는 "G — C (삼중 수소결합)"
  - 나머지 염기쌍 어둡게 (opacity 0.3)

### 뉴클레오타이드 상세 모드
- 단일 뉴클레오타이드를 크게 확대:
  - 인산(P) + 당(S) + 염기(B)가 어떻게 연결되는지 상세 표시
  - 인산-당 결합: 에스테르 결합
  - 당-염기 결합: 글리코시드 결합
  - 각 결합을 색상이 다른 CylinderGeometry로 표시

### 서열 표시 패널 (하단 HTML overlay)
```
5'— A  T  G  C  C  A  G  T  A  C  G  T  A  C  G  A  T  C  G  A —3'
    |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
3'— T  A  C  G  G  T  C  A  T  G  C  A  T  G  C  T  A  G  C  T —5'
```
- 현재 호버된 염기쌍이 하이라이트
- 사용자 입력 서열이 반영

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧬 DNA 이중나선 구조
━━━━━━━━━━━━━━━━━
기본 단위체:    뉴클레오타이드 (인산 + 당 + 염기)
구조:          이중나선 (두 가닥이 꼬인 형태)
염기쌍 수:     [n]쌍
회전 수:       [n/10]회전

염기쌍 규칙 (샤가프 규칙):
  A ═══ T  (이중 수소결합)
  G ≡≡≡ C  (삼중 수소결합)

DNA: 디옥시리보스(당) + A, T, G, C
RNA: 리보스(당) + A, U, G, C (단일 가닥)

기능: DNA → 유전정보 저장
      RNA → 유전정보 전달, 단백질 합성
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "DNA는 뉴클레오타이드가 반복 결합한 두 가닥이 꼬여 이중나선 구조를 이룹니다. A는 T와, G는 C와 결합하는 규칙이 있어 한 가닥의 서열을 알면 상보적 가닥의 서열을 알 수 있습니다."
- 탐구 질문:
  1. "DNA 한 가닥의 서열이 ATGCCAGT라면 상보적 가닥의 서열은 무엇인가요?" (답: TACGGTCA)
  2. "G-C 결합이 A-T 결합보다 안정한 까닭은 무엇인가요?" (답: 수소결합 3개 vs 2개)
  3. "생명체마다 특성이 다른 까닭은 DNA의 어떤 차이 때문인가요?" (답: 염기 배열 순서)
- 실생활 예시: 유전자 검사(친자 확인), PCR(코로나 검사), 유전공학(GMO), 법의학(범인 식별)
- 교과서 연계: "미래엔 통합과학1 II편 p84~87 — 핵산, DNA 이중나선(그림 II-30), DNA 모형 제작 탐구"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원
- 나선 곡선: CatmullRomCurve3 또는 자체 Helix 함수 (cos/sin + 선형 Y)
- 당-인산 백본: TubeGeometry(나선곡선, tubularSegments=200, r=0.06)
- 뉴클레오타이드: 함수로 생성, 나선 위 위치에 배치 (각도 = i × 2π/10)
- 풀기 애니메이션: 나선 반지름 R과 위상차를 보간(lerp)하여 나선 → 사다리 변환
- Raycaster: 염기쌍 호버 감지
- 성능: requestAnimationFrame 루프, 뉴클레오타이드 최대 40개(20쌍)
- 파일명: dna_double_helix_sim.html
