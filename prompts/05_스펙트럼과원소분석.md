너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 우주 초기에 생성된 원소
- 토픽: 스펙트럼과 원소 분석
- 대상: 고등학교 1학년
- 핵심 개념: 햇빛이 프리즘을 통과하면 여러 가지 색의 띠가 나타나는데 이를 스펙트럼이라고 한다. 원소마다 고유한 위치에 선이 나타나므로 스펙트럼을 관찰해 원소의 종류를 구분할 수 있다. 별의 스펙트럼을 분석하면 별의 대기에 포함된 원소를 알 수 있다.

## 씬 구성
- 화면 좌측(폭 60%): 3D 프리즘 + 광원 + 스펙트럼 실험 장치
- 화면 우측(폭 40%): 스펙트럼 결과 표시 패널 (HTML overlay)
- 3D 프리즘:
  - 형상: 정삼각기둥 — ExtrudeGeometry로 삼각형 단면을 깊이 2로 압출
  - 재질: MeshPhysicalMaterial(투명도 transmission: 0.9, roughness: 0.05, ior: 1.5, 유리 효과)
  - 색상: 약간 연한 청록 0xeeffff
  - 크기: 높이 2, 밑변 2, 깊이 2
  - 위치: 화면 중앙
- 광원 장치 (프리즘 좌측):
  - 광원 모드에 따라 다른 3D 모형:
    (a) 백열등: SphereGeometry(0.3) 발광 + 원뿔형 갓(ConeGeometry)
    (b) 기체 방전관: CylinderGeometry(0.15, 0.15, 1.5) 유리관 + 내부 발광 가스
    (c) 태양: SphereGeometry(0.5) + 강한 emissive 0xffdd00
  - 광원에서 프리즘으로 향하는 빛: CylinderGeometry 가느다란 원뿔(0.02→0.05, 길이 3), 색상은 광원 모드에 따라 다름
- 프리즘 통과 후 분산된 빛:
  - 7가지 색상의 가느다란 원뿔이 프리즘 우측에서 부채꼴로 퍼짐
  - 각 빛 색상: 빨강 0xff0000, 주황 0xff8800, 노랑 0xffff00, 초록 0x00ff00, 파랑 0x0088ff, 남색 0x0000cc, 보라 0x8800ff
  - 각 빛의 굴절 각도가 약간씩 다름 (빨강 가장 적게, 보라 가장 많이)
- 스크린 (프리즘 우측):
  - PlaneGeometry(3, 1), 백색 MeshBasicMaterial
  - 스크린 위에 스펙트럼 결과가 투영됨 (텍스처로 렌더링)
- 바닥: PlaneGeometry(20, 20), 0x222222 어두운 바닥 (실험실 느낌)
- 조명: AmbientLight(0.2, 어두운 실험실) + PointLight(광원 위치, 세기 0.5)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 그림자: 광원에서 프리즘으로 가는 빛이 바닥에 색 그림자 투영 (선택적)

## 스펙트럼 결과 패널 (우측 HTML overlay)
파장 400nm~700nm 가시광선 영역을 가로로 표시:
- 상단: 파장 눈금자 (400, 450, 500, 550, 600, 650, 700 nm)
- 배경: 가시광선 무지개 그라데이션 (보라→빨강)

### 연속 스펙트럼 모드
- 전체 영역이 끊김 없는 무지개 띠로 표시
- 라벨: "연속 스펙트럼 — 백열등과 같은 광원에서 방출한 빛"

### 방출 스펙트럼 모드
- 검은 배경 위에 밝은 선만 특정 위치에 표시
- 원소별 선 위치 (교과서 기반 + 실제 분광 데이터):
  - 수소(H): 656.3nm(빨강 Hα), 486.1nm(청록 Hβ), 434.0nm(파랑 Hγ), 410.2nm(보라 Hδ)
  - 헬륨(He): 587.6nm(노랑), 501.6nm(초록), 471.3nm(파랑), 447.1nm(보라)
  - 나트륨(Na): 589.0nm, 589.6nm (짙은 노란색 이중선) ← 교과서 "나트륨 전등의 빛이 노란색" (p49)
  - 칼슘(Ca): 422.7nm(보라), 612.2nm(주황)
- 각 선에 마우스 호버 시 파장값 + 원소명 툴팁 표시

### 흡수 스펙트럼 모드
- 무지개 연속 배경 위에 검은 선이 특정 위치에 표시
- 검은 선 위치 = 해당 원소의 방출 스펙트럼 밝은 선 위치와 동일
- 프라운호퍼 선(태양 스펙트럼): 수소 + 헬륨 + 나트륨 + 칼슘의 흡수선 모두 표시
- 교과서: "19세기 초 프라운호퍼는 태양의 스펙트럼에서 수많은 검은 선을 발견했다" (p49)

### 별 스펙트럼 매칭 모드
- 상단: 임의의 별의 흡수 스펙트럼 (수소+헬륨 흡수선 포함)
- 하단: 수소 방출 스펙트럼과 헬륨 방출 스펙트럼을 나란히 배치
- "매칭 확인" 버튼 클릭 시:
  - 일치하는 선 위치를 세로 연결선(점선, 밝은 초록)으로 연결
  - "이 별의 대기에 수소와 헬륨이 포함되어 있습니다!" 결론 텍스트 표시
- 교과서 그림 II-3 (p49) 재현

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "광원 모드" 드롭다운: 백열등 (연속) / 기체 방전관 (방출) / 태양+저온 기체 (흡수) / 별 (매칭)
- "원소 선택" 드롭다운 (방출/흡수 모드에서만 활성): 수소(H) / 헬륨(He) / 나트륨(Na) / 칼슘(Ca) / 전체
- "프리즘 회전" 슬라이더: -30° ~ 30° (기본값 0°), 1° 단계
  - 프리즘 각도에 따라 분산 각도 미세 변화
- "빛 세기" 슬라이더: 0.1 ~ 1.0 (기본값 0.7), 0.1 단계
  - 광원 밝기와 스펙트럼 선 밝기 동시 조절
- "매칭 확인" 버튼 (별 매칭 모드에서만 활성): 연결선 표시
- "초기화" 버튼: 백열등 모드, 기본 각도로 리셋

## 시각화 규칙

### 광원 모드 전환 시
- 3D 광원 모형이 페이드아웃/페이드인으로 교체 (0.5초)
- 프리즘 통과 후 빛 색상이 모드에 맞게 변경
- 스펙트럼 패널이 슬라이드 애니메이션으로 전환

### 원소 선택 시
- 방출 모드: 선택된 원소의 밝은 선만 표시, 나머지는 페이드
- 흡수 모드: 선택된 원소의 검은 선만 강조, 나머지는 연한 회색
- 기체 방전관 내부 발광 색상이 원소에 따라 변경:
  - 수소: 분홍-빨강 0xff6688
  - 헬륨: 노란-주황 0xffaa44
  - 나트륨: 진한 노랑 0xffcc00
  - 칼슘: 주황-빨강 0xff5522

### 오로라 보조 표시 (교과서 p48~49)
- 흡수 모드에서 산소(O) 선택 시, 스펙트럼 패널 상단에 오로라 이미지(초록색 글로우 효과)와 함께 표시
- "오로라의 초록색 빛 = 산소의 방출 스펙트럼" 라벨

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌈 스펙트럼 분석
━━━━━━━━━━━━━━
광원 모드:     [연속 / 방출 / 흡수 / 별 매칭]
선택 원소:     [원소명] ([원소 기호])
스펙트럼 유형: [연속 / 선 스펙트럼]

가시광선 영역: 400nm (보라) ~ 700nm (빨강)

선택 원소의 주요 스펙트럼 선:
  [파장1] nm — [색상명]
  [파장2] nm — [색상명]
  ...

원리: 원소마다 고유한 에너지 준위를 가지므로
      고유한 파장의 빛을 방출/흡수한다.
```
모든 값은 드롭다운/슬라이더 변경 시 실시간 업데이트.

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "원소마다 고유한 위치에 스펙트럼 선이 나타나므로, 스펙트럼을 분석하면 물질에 포함된 원소를 알아낼 수 있습니다. 이 원리로 수백만 광년 떨어진 별의 성분도 알 수 있습니다."
- 탐구 질문:
  1. "나트륨 전등의 빛이 노란색인 까닭을 나트륨의 방출 스펙트럼과 관련지어 설명해 보세요."
  2. "태양의 스펙트럼에서 검은 선(프라운호퍼 선)이 나타나는 까닭은 무엇인가요?"
  3. "별의 스펙트럼에서 수소의 흡수선이 발견된다면 이 별에 대해 무엇을 알 수 있나요?"
- 실생활 예시: 불꽃 반응(원소 확인), 네온사인(기체 방전관), 천문학(별의 성분 분석), 형광등(수은 스펙트럼)
- 교과서 연계: "미래엔 통합과학1 II편 p48~50 — 스펙트럼, 프라운호퍼, 별의 스펙트럼 분석"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 프리즘 유리 효과: MeshPhysicalMaterial + transmission
- 스펙트럼 패널은 Canvas 2D API로 렌더링 (픽셀 단위 정밀 제어)
- 모드 전환 시 이전 3D 객체 dispose()
- 성능: requestAnimationFrame 루프
- 파일명: spectrum_element_analysis_sim.html
