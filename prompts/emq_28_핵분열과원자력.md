너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: III. 양자와 미시세계 > 05. 핵반응과 에너지
- 토픽: 핵분열과 원자력
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 핵분열은 무거운 원자핵(우라늄-235 등)이 중성자를 흡수하여 두 개의 중간 크기 핵으로 쪼개지는 반응이다. 이때 질량 결손에 의해 약 200 MeV의 에너지와 2~3개의 중성자가 방출된다. 방출된 중성자가 다른 우라늄 핵의 분열을 유발하여 연쇄 반응이 일어난다. 원자로에서는 감속재와 제어봉으로 연쇄 반응 속도를 제어하여 안정적인 에너지를 생산한다. 임계 질량 이상에서만 연쇄 반응이 지속된다.

## 씬 구성
- 화면 좌측 60%: 3D 핵분열 반응 + 연쇄 반응 시각화
- 화면 우측 40%: 원자로 구조 다이어그램 + 에너지 비교 (Canvas 2D / HTML)
- 2가지 보기: 핵분열 연쇄 반응 / 원자로 구조
- 배경: 0x0a0a1a
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 보기 1: 핵분열과 연쇄 반응

### 3D 핵 입자
- 우라늄-235(²³⁵U): SphereGeometry(r=0.8), 0x44aa44, 표면에 양성자(빨강)/중성자(파랑) 텍스처
  - "²³⁵U" 라벨, 핵자 수 표시
- 중성자(n): SphereGeometry(r=0.12), 0x4488ff emissive 0x2266ff
- 분열 생성물:
  - 바륨-141(¹⁴¹Ba): SphereGeometry(r=0.55), 0xffaa44
  - 크립톤-92(⁹²Kr): SphereGeometry(r=0.45), 0xff6688
  - (또는 다른 분열 생성물 조합)
- 감마선(γ): 0xffff44 파동 효과

### 핵분열 단계별 애니메이션
- **단계 1: 중성자 포획**
  - 느린 중성자(열중성자)가 ²³⁵U에 접근
  - 흡수 → ²³⁶U*(불안정 상태) 형성
  - 핵이 진동/변형하는 애니메이션 (물방울 모형)

- **단계 2: 핵 변형**
  - ²³⁶U*가 아령 형태로 늘어남 (변형 애니메이션)
  - 쿨롱 반발 vs 핵력: 반발이 이김 → 찢어짐!

- **단계 3: 분열**
  - 두 조각으로 쪼개짐 + 중성자 2~3개 방출 + 감마선
  - ²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 3n + ~200 MeV
  - 폭발 효과 (파티클 + 플래시)
  - 질량 결손 → 에너지 방출 표시

### 연쇄 반응 시각화
- 1세대: 1개 ²³⁵U 분열 → 2~3개 중성자 방출
- 2세대: 각 중성자가 새로운 ²³⁵U에 흡수 → 분열 → 더 많은 중성자
- 3세대, 4세대... → 기하급수적 증가!
- 트리 구조로 시각화: 1→3→9→27...
- 중성자 수 증가 그래프 (지수 함수)

### 증배 계수 k
- k < 1 (미임계): 연쇄 반응 소멸 → 반응 줄어듦
- k = 1 (임계): 안정 → 일정한 에너지 생산 (원자로!)
- k > 1 (초임계): 기하급수적 증가 → 폭발! (원자폭탄)
- k 슬라이더: 0.5~2.0 → 연쇄 반응 속도 변화 시각화
- "원자로: k=1 유지가 핵심!" 라벨

### 임계 질량 시각화
- 작은 우라늄 덩어리: 중성자 대부분 표면으로 탈출 → 연쇄 반응 불가
- 큰 우라늄 덩어리: 중성자가 내부에서 다른 핵과 반응 → 연쇄 반응 지속
- 우라늄 크기 슬라이더 → 탈출 vs 반응 중성자 비율 변화

## 보기 2: 원자로 구조

### 3D 원자로 단면
- 원자로 압력 용기: CylinderGeometry, 0x888888, 반투명
- 핵연료봉: CylinderGeometry(가느다란), 0x44aa44, 여러 개 격자 배열
  - 내부: ²³⁵U 포함 UO₂ 펠릿
- 제어봉: CylinderGeometry, 0x222222 (흑연/보론)
  - 위아래 이동 가능 (슬라이더 연동)
  - 삽입 → 중성자 흡수 → 반응 속도↓
  - 인출 → 중성자 증가 → 반응 속도↑
- 감속재(물): 연료봉 사이, 0x4488ff opacity 0.2
  - 빠른 중성자 → 느린 중성자(열중성자)로 감속
- 냉각수 순환: 화살표로 흐름 표시
  - 1차 냉각수 → 증기 발생기 → 2차 냉각수(증기) → 터빈 → 발전기

### 제어봉 인터랙션
- 제어봉 삽입 깊이 슬라이더: 0%(완전 인출)~100%(완전 삽입)
- 0%: k > 1 → 반응 증가 (위험!)
- 50%: k ≈ 1 → 안정 운전
- 100%: k < 1 → 반응 정지 (비상 정지, SCRAM)
- 출력 게이지: MW 단위 표시

## 에너지 비교 (우측)

### 1kg 연료별 에너지
- 막대 그래프:
  - 석탄: ~30 MJ
  - 석유: ~45 MJ
  - 천연가스: ~55 MJ
  - 우라늄(핵분열): ~82,000,000 MJ (82 TJ!)
- "우라늄 1kg = 석탄 약 2700톤!" 라벨
- 로그 스케일 vs 선형 스케일 토글

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 핵분열 연쇄 반응 / 원자로 구조
- "단계" 슬라이더 (핵분열): 1~3 (포획/변형/분열)
- "자동 재생" 토글: ON/OFF
- "증배 계수 k" 슬라이더: 0.5~2.0 (기본값 1.0)
- "세대 수" 슬라이더 (연쇄 반응): 1~6
- "제어봉 삽입" 슬라이더 (원자로): 0~100% (기본값 50%)
- "에너지 비교" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚛️ 핵분열과 원자력
━━━━━━━━━━━━━━━
보기: [연쇄 반응/원자로]

핵분열 반응:
  ²³⁵U + n → ¹⁴¹Ba + ⁹²Kr + 3n + ~200 MeV
  질량 결손: Δm ≈ 0.215 u
  E = Δmc² ≈ 200 MeV

연쇄 반응:
  증배 계수 k = [값]
  n세대 후 중성자 수: N = k^n
  k<1: 소멸 / k=1: 임계 / k>1: 초임계

원자로:
  제어봉: 중성자 흡수 → k 조절
  감속재: 빠른 n → 열중성자
  제어봉 삽입: [값]%
  출력: ~[값] MW

에너지 비교:
  U 1kg ≈ 석탄 2700톤 ≈ 석유 1300톤
```

- 핵심 원리: "핵분열에서 무거운 원자핵이 쪼개지면 질량 결손에 의해 막대한 에너지가 방출됩니다. 연쇄 반응의 증배 계수를 k=1로 유지하는 것이 원자로의 핵심입니다."
- 탐구 질문:
  1. "연쇄 반응에서 증배 계수 k=1.1이면 10세대 후 중성자 수는 처음의 몇 배가 되나요?"
  2. "제어봉을 완전히 삽입하면 원자로에서 어떤 일이 일어나나요?"
  3. "감속재(물)가 없으면 핵분열 연쇄 반응이 어려운 까닭은?"
- 실생활 예시: 원자력 발전소(한국 전력의 ~30%), 핵잠수함, 방사성 폐기물 처리, 핵무기 비확산
- 교과서 연계: "전자기와 양자 III편 p118~122 — 핵반응과 에너지, 핵분열과 원자력"
