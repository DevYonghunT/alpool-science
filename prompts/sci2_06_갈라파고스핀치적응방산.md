너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 생물의 진화
- 토픽: 갈라파고스 핀치의 적응 방산
- 대상: 고등학교 1학년
- 핵심 개념: 생물집단에서 오랜 세월 동안 여러 세대를 거치면서 생물의 특성이 변화하여 원래의 종과는 다른 새로운 종이 생겨나는 과정을 진화라고 한다. 갈라파고스 제도에 처음 정착한 핀치 집단의 개체들 간에는 다양한 변이가 있었다. 오랜 세월 동안 서로 다른 먹이 환경에 적합한 변이를 가진 개체가 자연선택되는 과정을 거쳐 여러 종의 핀치로 진화했다.

## 씬 구성
- 화면 좌측에 갈라파고스 제도 지도(HTML overlay 또는 3D 섬들), 우측에 핀치 부리 비교
- 중앙에 3D 핀치 모형 + 먹이 환경
- 교과서 그림 I-13, I-14 (p24~25): 갈라파고스 핀치의 부리 모양과 자연선택
- 배경: 0x88ccdd (열대 하늘)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 4가지 핀치 종류 (교과서 그림 I-13)
| 핀치 종류 | 부리 특징 | 먹이 환경 | 부리 색상 |
|----------|---------|---------|---------|
| 짧고 단단한 부리 | ConeGeometry(0.08, 0.12) 짧고 둥근 | 곤충이 많은 환경 | 0xdd8844 |
| 뾰족하고 가느다란 부리 | ConeGeometry(0.03, 0.25) 길고 가늘게 | 나무 구멍 속 곤충 | 0xcc7733 |
| 길고 뾰족한 부리 | ConeGeometry(0.04, 0.3) 길고 뾰족 | 선인장이 잘 자라는 환경 | 0xbb6622 |
| 크고 두꺼운 부리 | ConeGeometry(0.1, 0.15) 크고 두꺼운 | 크고 단단한 씨앗 환경 | 0xaa5511 |

## 3D 핀치 모형
- 몸통: SphereGeometry(r=0.3, 16, 16), 0x333333 (검은색, 교과서 그림 기반)
- 머리: SphereGeometry(r=0.18, 16, 16), 0x333333
- 부리: ConeGeometry (종류별로 크기/형태 다름, 위 표 참고)
- 눈: SphereGeometry(r=0.03), 0xffffff + 0x111111 동공
- 꼬리: BoxGeometry 납작하게, 0x444444
- 다리: CylinderGeometry 2개, 0x886644
- 핀치 종류 선택 시 부리 형태가 부드럽게 전환(lerp)

## 먹이 환경 3D
- 핀치 종류에 따라 배경 환경이 변경:
  - 곤충 환경: 작은 벌레(SphereGeometry r=0.02, 0x222200) 여러 마리
  - 나무 구멍 환경: CylinderGeometry 나무 + 구멍 안에 벌레
  - 선인장 환경: 선인장 3D (CylinderGeometry 기둥 + SphereGeometry 가시)
  - 씨앗 환경: 큰 씨앗(SphereGeometry r=0.1, 0x886633) 여러 개

## 진화 과정 타임라인 (교과서 그림 I-14)
- 5단계 시각화:
  1. 남아메리카에서 핀치 일부가 갈라파고스로 날아옴
  2. 많은 수의 핀치가 태어나고, 다양한 부리 변이 발생
  3. 먹이와 서식지를 차지하기 위해 경쟁
  4. 먹이 환경에 적합한 부리를 가진 핀치가 생존에 유리해 더 많은 자손
  5. 몇 세대가 지난 뒤 특정 부리 핀치 집단이 번성
- 각 단계를 3D 애니메이션 + 설명 텍스트로 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "핀치 종류" 드롭다운: 짧고 단단한 부리 / 뾰족하고 가느다란 부리 / 길고 뾰족한 부리 / 크고 두꺼운 부리
- "진화 과정 보기" 버튼: 5단계 타임라인 자동 재생 (각 4초)
- "부리 비교 모드" 토글: 4가지 부리를 나란히 비교 표시
- "회전" 토글: 핀치 모형 자동 회전 ON/OFF
- "초기화" 버튼

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "갈라파고스 제도에 처음 정착한 핀치 집단에서 다양한 부리 변이가 있었습니다. 오랜 세월 동안 서로 다른 먹이 환경에 적합한 변이를 가진 개체가 자연선택되는 과정을 거쳐 여러 종의 핀치로 진화했습니다."
- 탐구 질문:
  1. "같은 조상에서 유래한 핀치가 여러 종으로 분화한 까닭을 먹이 환경과 관련지어 설명해 보세요."
  2. "선인장이 잘 자라는 환경에서 길고 뾰족한 부리를 가진 핀치가 생존에 유리한 까닭은 무엇인가요?"
  3. "변이가 적은(작은) 생물집단은 환경 변화에 적응하기 유리할까요, 불리할까요?"
- 실생활 예시: 다윈의 비글호 탐사(1831~1836), 『종의 기원』(1859), 기린의 목 길이 진화
- 교과서 연계: "미래엔 통합과학2 I편 p24~25 — 진화와 다양한 생물의 출현, 그림 I-13~14"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 핀치 종류 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_06_galapagos_finch.html
