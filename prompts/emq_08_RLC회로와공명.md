너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 05. 저항, 축전기, 인덕터
- 토픽: RLC 회로와 공명
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 교류 회로에서 저항(R), 축전기(C), 인덕터(L)는 각각 다른 방식으로 전류를 제한한다. 축전기의 용량 리액턴스 X_C = 1/(2πfC)는 진동수가 높을수록 작아지고, 인덕터의 유도 리액턴스 X_L = 2πfL은 진동수가 높을수록 커진다. RLC 직렬 회로에서 X_L = X_C인 공명 진동수 f₀ = 1/(2π√LC)에서 임피던스가 최소(=R)가 되어 전류가 최대가 된다.

## 씬 구성
- 화면 좌측 50%: 3D 교류 회로 (R + L + C + 교류 전원)
- 화면 우측 50%: 임피던스-진동수 그래프 + 위상 관계 (Canvas 2D)
- 3가지 모드: 개별 소자(R/C/L) / RLC 직렬 / 공명
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 교류 회로

### 회로 소자
- 저항 R: CylinderGeometry(0.15, 0.15, 0.8), 색띠 패턴, 0xaa8844
- 축전기 C: 2개 평행판(PlaneGeometry) + 간격, 0x4488aa
- 인덕터 L: 코일 형태(TorusGeometry × 5겹), 0xcc6633
- 교류 전원: CircleGeometry + "~" 표시, 0xdddddd
- 전선: TubeGeometry로 소자들 연결
- 전류: 노란 파티클이 전선을 따라 이동 (교류: 왕복)

### 전류/전압 위상 관계
- R: 전압과 전류 동위상
- C: 전류가 전압보다 90° 앞섬 (전류 선행)
- L: 전류가 전압보다 90° 뒤짐 (전류 지연)
- 각 소자 옆에 v-t, i-t 그래프 미니 표시 → 위상 차이 시각화

### 전류 크기
- 개별: I = V/R, I = V/X_C, I = V/X_L
- RLC 직렬: I = V/Z, Z = √(R² + (X_L - X_C)²)
- 진동수 f 슬라이더 → 전류 크기/위상 실시간 변화

## 임피던스-진동수 그래프 (우측)
- X축: 진동수 f (로그 스케일)
- Y축: 임피던스 Z / 전류 I
- 곡선:
  - X_C = 1/(2πfC): 진동수↑ → X_C↓ (파란 하강 곡선)
  - X_L = 2πfL: 진동수↑ → X_L↑ (빨간 상승 곡선)
  - Z = √(R² + (X_L - X_C)²): U자 곡선 (초록)
  - I = V/Z: 역U자 곡선 (주황)
- 공명 진동수 f₀: X_L = X_C → Z 최소 → I 최대!
  - f₀ 위치에 빨간 마커 + 세로 점선
- 현재 진동수에 파란 마커

## RLC 공명 시각화 (핵심!)
- 진동수 슬라이더를 연속으로 변화
- f < f₀: 용량성 (X_C > X_L), 전류 작음
- f = f₀: 공명! 전류 최대, Z = R (최소)
  - 전구가 가장 밝게 빛남
  - "✨ 공명! f₀ = 1/(2π√LC) = [값]Hz" 라벨
- f > f₀: 유도성 (X_L > X_C), 전류 작음
- 전구 밝기가 진동수에 따라 변하는 것으로 공명 확인

## 위상 다이어그램 (페이저)
- 회전하는 벡터(페이저) 시각화:
  - V_R: 빨간 화살표 (전류와 동위상)
  - V_C: 파란 화살표 (전류보다 90° 뒤)
  - V_L: 초록 화살표 (전류보다 90° 앞)
  - V_합: 주황 화살표 (합성 전압)
- 공명 시: V_L = V_C → 상쇄 → V_합 = V_R

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 개별 소자 / RLC 직렬 / 공명 스캔
- "저항 R" 슬라이더: 10~500Ω (기본값 100)
- "축전기 C" 슬라이더: 1~100μF (기본값 10)
- "인덕터 L" 슬라이더: 0.1~10mH (기본값 1)
- "진동수 f" 슬라이더: 100~10000Hz (로그, 기본값 1000)
- "전압 V₀" 슬라이더: 1~12V (기본값 6)
- "공명 주파수로 이동" 버튼: f를 자동으로 f₀로 설정
- "Z-f 그래프" 토글: ON/OFF (기본값 ON)
- "페이저 다이어그램" 토글: ON/OFF
- "초기화" 버튼
