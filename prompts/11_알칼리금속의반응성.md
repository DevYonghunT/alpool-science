너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 원소의 주기성과 결합
- 토픽: 알칼리 금속과 할로젠의 반응성
- 대상: 고등학교 1학년
- 핵심 개념: 같은 족 원소들은 원자가 전자 수가 같아 유사한 화학적 성질을 가진다. 1족 알칼리 금속(Li, Na, K)은 물과 격렬하게 반응하며, 원자 번호가 커질수록 반응성이 증가한다. 17족 할로젠(F, Cl, Br, I)은 실온에서 2개의 원자가 결합한 분자로 존재하며, 반응성이 매우 크다. 알칼리 금속은 산소, 물과 빠르게 반응하므로 석유나 파라핀에 담가 밀폐 보관한다.

## 씬 구성
- 화면을 좌우 분할:
  - 좌측(폭 50%): 알칼리 금속 반응 실험 3D 장면
  - 우측(폭 50%): 할로젠 원소 비교 3D 장면
- 상단 탭: "알칼리 금속 + 물 반응" | "할로젠 비교" | "통합 비교"

### 알칼리 금속 + 물 반응 장면 (교과서 p62 그림 II-15, 탐구 p63)
- 3D 실험 장치:
  - 수조: BoxGeometry(4, 2, 3) 하단부, 반투명 유리(MeshPhysicalMaterial, transmission 0.85, 0xaaccee)
  - 물: BoxGeometry(3.8, 1.0, 2.8), MeshStandardMaterial(0x4488cc, opacity 0.6), 수조 내부 하반부
  - 물 표면: PlaneGeometry(3.8, 2.8), MeshStandardMaterial(0x66aadd, opacity 0.3), 윗면 반사 효과
  - 페트리접시: CylinderGeometry(r=0.8, h=0.05, 32), 0xdddddd, 수조 옆에 배치
  - 알칼리 금속 조각: 페트리접시 위에 작은 큐브(BoxGeometry 0.15×0.15×0.15)
    - Li: 0xcccccc 은백색, "Li" 라벨
    - Na: 0xdddddd 은백색, "Na" 라벨  
    - K: 0xbbbbcc 은청색, "K" 라벨
- 바닥: PlaneGeometry(10, 10), 0xeeeeee (실험실 바닥)
- 조명: AmbientLight(0.4) + DirectionalLight(0.7, 위에서) + 그림자 활성화
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls

### 반응 애니메이션 (원소별 차이 — 핵심)

#### 리튬(Li) + 물
1. Li 조각이 물 위에 떨어짐 (0.5초 낙하)
2. 물 위에서 천천히 움직이며 반응 (3초)
   - 작은 기포 파티클(Points 20개, 0xffffff) 천천히 올라옴 — 수소 기체 발생
   - Li 조각이 서서히 작아짐 (scale 1.0 → 0.0, 3초)
   - 물 표면 파동: 약한 ripple 효과
3. 반응 강도 표시: "🟢 완만한 반응" (초록)

#### 나트륨(Na) + 물
1. Na 조각이 물 위에 떨어짐
2. 물 위에서 빠르게 움직이며 격렬하게 반응 (2초)
   - 많은 기포(Points 60개) 빠르게 올라옴
   - Na 조각이 빠르게 녹으며 구 형태로 변형 (금속이 녹아 둥글어지는 효과)
   - 물 표면: 강한 ripple + 물이 튀는 파티클(Points 15개 방사형)
   - 페놀프탈레인 반응: 물 색상이 투명 → 분홍색(0xff88aa) 점진적 변화 (염기성 확인)
3. 반응 강도: "🟡 격렬한 반응" (노란)

#### 칼륨(K) + 물
1. K 조각이 물 위에 떨어짐
2. 즉시 폭발적 반응 (1초)
   - 매우 많은 기포(Points 150개) 순식간에 발생
   - 밝은 보라색 불꽃 효과(PointLight 순간 세기 5.0, 0xcc44ff)
   - K 조각 즉시 소멸 + 물이 크게 튀는 파티클(Points 30개)
   - 물 표면: 격렬한 파동 + 수증기 파티클(0xffffff, 위로 빠르게 퍼짐)
   - 물 전체가 분홍색(0xff66aa)으로 빠르게 변화
3. 반응 강도: "🔴 폭발적 반응!" (빨간)

### 할로젠 비교 장면 (교과서 p62 그림 II-15)
- 4개 시약병을 나란히 배치:
  | 원소 | 상태(실온) | 시약병 내용물 표현 | 색상 |
  |------|----------|-----------------|------|
  | F₂ | 기체 | 병 내부에 연한 안개 파티클 | 0xeeff88 연한 노랑 |
  | Cl₂ | 기체 | 병 내부에 녹황색 안개 파티클 | 0xaacc00 녹황색 |
  | Br₂ | 액체 | 병 하단부에 액체(BoxGeometry) + 상단 증기 | 0xaa3300 적갈색 |
  | I₂ | 고체 | 병 하단부에 고체 결정(작은 큐브들) + 약간의 증기 | 0x442266 보라색 |
- 시약병: CylinderGeometry(r=0.3, h=1.2), 반투명 유리 MeshPhysicalMaterial
- 각 병 아래에 원소 기호 + 한글명 + 상태 라벨
- "할로젠은 실온에서 2개의 원자가 결합한 분자로 존재한다" 라벨
- 교과서: 할로젠 비교 (p62 그림 II-15)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "실험 탭" 드롭다운: 알칼리 금속 반응 / 할로젠 비교 / 통합 비교
- "금속 선택" 드롭다운 (알칼리 금속 탭): Li / Na / K
- "반응 시작" 버튼: 선택된 금속을 물에 넣는 애니메이션 실행
- "페놀프탈레인 표시" 토글: ON/OFF (기본값 OFF)
  - ON: 물에 페놀프탈레인 용액 추가 → 반응 후 분홍색 변화
- "비교 모드" 체크박스: ON 시 Li, Na, K 3개를 동시에 물에 넣어 반응성 차이 비교
  - 3개 수조를 나란히 배치, 동시에 반응 시작
- "반응 속도" 슬라이더: 0.5x ~ 3x (기본값 1x)
- "초기화" 버튼: 물 상태 초기화, 금속 조각 복원

## 시각화 규칙

### 반응 강도 인디케이터 (화면 하단)
- 3개 막대(Li, Na, K)로 반응 강도를 비교:
  ```
  반응 강도 비교
  Li ██░░░░░░░░ 완만
  Na █████░░░░░ 격렬
  K  █████████░ 폭발적
  ```
- 원자 번호가 커질수록 막대 길어짐 + 색상 초록→노랑→빨강

### 전자 배치 보조 표시 (우측 하단)
- 선택된 금속의 보어 원자 모형(간략화): 원자핵 + 전자 껍질 + 원자가 전자 1개 강조
- "원자가 전자 1개를 잃기 쉬움 → 반응성 큼" 라벨
- 원자 번호가 클수록 원자가 전자가 핵에서 멀리 있어 더 쉽게 잃음 → 반응성 ↑

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧪 알칼리 금속의 반응성
━━━━━━━━━━━━━━━━━━
선택 금속:    [원소명] ([기호])
원자 번호:    [번호]
원자가 전자:  1개
반응 상태:    [대기 / 반응 중 / 완료]

반응식:
  2[M] + 2H₂O → 2[M]OH + H₂↑
  (M = Li, Na, K)

반응 강도: Li < Na < K
  → 원자 번호↑ → 원자 반지름↑
  → 원자가 전자 잃기 쉬움↑ → 반응성↑
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 물 표면 ripple: ShaderMaterial 또는 UV 오프셋 애니메이션
- 기포 파티클: BufferGeometry + Points, Y축 방향 이동
- 불꽃 효과: PointLight 순간 밝기 변화 + 파티클 방사
- 반응 시퀀스: setTimeout 체이닝 또는 자체 timeline 구현
- 성능: requestAnimationFrame 루프, 파티클 최대 150개
- 파일명: alkali_metal_halogen_reactivity_sim.html
