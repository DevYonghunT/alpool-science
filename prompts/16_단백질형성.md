너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 2. 자연의 구성 물질 > 지각과 생명체를 구성하는 물질
- 토픽: 단백질 형성 과정
- 대상: 고등학교 1학년
- 핵심 개념: 단백질의 기본 단위체는 아미노산이다. 이웃한 2개의 아미노산은 펩타이드결합(공유 결합)으로 연결되며, 이때 물 분자 1개가 빠져나온다. 많은 아미노산이 연결되면 폴리펩타이드가 형성된다. 아미노산의 종류(약 20종류)와 배열에 따라 폴리펩타이드 사슬이 접혀 고유한 입체 구조와 기능을 가진 단백질이 형성된다.

## 씬 구성
- 화면을 3단계로 구성 (탭 전환):
  - 탭 1: 아미노산 구조 + 펩타이드결합
  - 탭 2: 폴리펩타이드 형성
  - 탭 3: 단백질 입체 구조 (헤모글로빈 vs 케라틴)
- 배경: 0x0a1628 (짙은 생물학 남색)
- 조명: AmbientLight(0.35) + DirectionalLight(0.65) + 그림자 비활성화
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls

## 탭 1: 아미노산 + 펩타이드결합 (교과서 그림 II-29, p82)

### 아미노산 3D 모형
- 아미노산을 단순화된 블록으로 표현:
  - 중심부(R기): BoxGeometry(0.8, 0.8, 0.8), 색상은 아미노산 종류에 따라 다름 (20가지 색상)
  - 아미노기(-NH₂): SphereGeometry(r=0.25), 0x4488ff (파랑), 왼쪽에 부착, "NH₂" 라벨
  - 카복실기(-COOH): SphereGeometry(r=0.25), 0xff4444 (빨강), 오른쪽에 부착, "COOH" 라벨
  - R기 위에 아미노산 이름 라벨 (예: "글리신", "알라닌", "발린" 등)
- 5가지 대표 아미노산 선택 가능:
  | 아미노산 | R기 색상 | 특성 |
  |---------|---------|------|
  | 글리신(Gly) | 0xcccccc 회색 | 가장 작은 R기 |
  | 알라닌(Ala) | 0x88cc44 연초록 | 비극성 |
  | 세린(Ser) | 0x44aaff 파랑 | 극성(OH기) |
  | 아스파르트산(Asp) | 0xff6644 주황 | 음전하 |
  | 라이신(Lys) | 0xaa44ff 보라 | 양전하 |

### 펩타이드결합 애니메이션
1. 아미노산 A(좌)와 아미노산 B(우) 배치, 간격 3.0
2. A의 카복실기(-COOH)와 B의 아미노기(-NH₂)가 접근 (3.0 → 1.5, 1초)
3. 결합 순간:
   - A의 -COOH에서 OH 분리 + B의 -NH₂에서 H 분리
   - OH + H → H₂O 물 분자 생성: SphereGeometry(r=0.15, 0x44aaff) 위로 날아감
   - A의 C와 B의 N 사이에 새로운 결합 형성: CylinderGeometry(r=0.06, 0xffdd44 노랑) — 펩타이드결합
   - 밝은 플래시 효과
4. "펩타이드결합" 라벨 + "H₂O 방출" 라벨
5. 결합된 디펩타이드: A-B 블록이 연결된 형태
- 교과서: "이웃한 2개의 아미노산은 펩타이드결합으로 연결된다. 2개의 아미노산은 물 분자 1개가 빠져나가면서 결합한다." (p82)

## 탭 2: 폴리펩타이드 형성

### 아미노산 연결 시뮬레이션
- 사용자가 5가지 아미노산 중 선택하여 하나씩 추가하는 인터랙티브 모드
- 하단에 5가지 아미노산 버튼(색상별):
  - 클릭 시 체인 끝에 해당 아미노산이 펩타이드결합으로 추가 (애니메이션 0.5초)
  - 물 분자 방출 카운터: "H₂O 방출: [n]개"
  - 최대 15개까지 연결 가능
- 체인 3D 표현:
  - 아미노산 블록들이 TubeGeometry 곡선(CatmullRomCurve3) 위에 배치
  - 체인이 길어지면 자연스럽게 곡선을 그리며 화면에 맞게 조정
  - 각 블록의 R기 색상으로 아미노산 종류 구분
- "랜덤 생성" 버튼: 10~15개 아미노산을 랜덤으로 빠르게 연결
- 아미노산 서열 표시 (하단 HTML): "Gly - Ala - Ser - Asp - Lys - ..." 텍스트

### 폴리펩타이드 접힘 애니메이션
- "접기" 버튼 클릭 시:
  1. 직선 형태의 폴리펩타이드 사슬이 구부러지고 접히기 시작 (2초)
  2. R기의 특성(극성/비극성/전하)에 따라 접힘 방향이 결정되는 것을 시각적으로 표현
     - 비극성 R기(초록): 안쪽으로 숨음 (소수성 상호작용)
     - 극성/전하 R기(파랑, 주황, 보라): 바깥쪽으로 노출
  3. 접힌 최종 형태: 불규칙한 구형 또는 타원형 덩어리
  4. "아미노산의 종류와 배열에 따라 고유한 입체 구조 형성!" 라벨

## 탭 3: 단백질 입체 구조 비교 (교과서 탐구 p83)

### 헤모글로빈 (좌측)
- 3D: 4개의 서브유닛(각각 다른 색상의 불규칙 구형)이 결합
  - α 서브유닛 2개: 0xff6644 (주황-빨강)
  - β 서브유닛 2개: 0x4466ff (파랑)
  - 각 서브유닛: SphereGeometry 여러 개를 뭉쳐 불규칙 형태 (또는 TubeGeometry로 리본 구조)
  - 중심에 철 이온(Fe): SphereGeometry(r=0.1, 0xaa4400), 4개 서브유닛 각각에 1개
- 라벨: "헤모글로빈 — 온몸의 조직 세포로 산소를 운반한다" (교과서 p83)
- 기능 애니메이션: O₂ 분자(빨간 작은 구 2개)가 Fe에 결합 → 분리되며 조직으로 이동

### 케라틴 (우측)
- 3D: 3개의 나선 구조(α-helix)가 꼬여 있는 밧줄 형태
  - TubeGeometry로 3개의 나선 곡선 (HelixCurve)
  - 색상: 0xddaa44 (금색), 0xcc8833, 0xbb7722 (약간씩 다른 갈색 톤)
  - 나선이 서로 꼬인 슈퍼코일 구조
- 라벨: "케라틴 — 머리카락과 손톱 등 몸의 일부를 이룬다" (교과서 p83)

### 비교 포인트
- 하단: "두 단백질 모두 아미노산으로 이루어져 있지만, 아미노산의 종류와 배열이 달라 서로 다른 입체 구조와 기능을 가진다" 라벨
- 아미노산 서열 차이 → 구조 차이 → 기능 차이 화살표 다이어그램

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 아미노산+펩타이드결합 / 폴리펩타이드 형성 / 단백질 비교
- "펩타이드결합 재생" 버튼 (탭 1): 결합 애니메이션 재생
- "아미노산 추가" 5개 버튼 (탭 2): Gly / Ala / Ser / Asp / Lys
- "랜덤 생성" 버튼 (탭 2): 랜덤 폴리펩타이드 생성
- "접기" 버튼 (탭 2): 폴리펩타이드 접힘 애니메이션
- "단백질 선택" 드롭다운 (탭 3): 헤모글로빈 / 케라틴 / 둘 다
- "자동 회전" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧬 단백질 형성 과정
━━━━━━━━━━━━━━━━━
현재 모드:      [아미노산 / 폴리펩타이드 / 단백질]
아미노산 수:    [n]개
펩타이드결합:   [n-1]개
방출된 H₂O:    [n-1]개

형성 과정:
  아미노산 → (펩타이드결합) → 폴리펩타이드
  → (접힘) → 단백질 (고유한 입체 구조)

아미노산 종류: 약 20가지
다양한 단백질이 형성되는 까닭:
  아미노산의 종류와 배열이 다르기 때문
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "단백질은 아미노산이 펩타이드결합으로 연결되어 형성됩니다. 아미노산은 약 20종류이지만, 종류와 배열의 조합에 따라 수없이 다양한 단백질이 만들어집니다."
- 탐구 질문:
  1. "아미노산 2개가 펩타이드결합으로 연결될 때 빠져나오는 물질은 무엇인가요?" (답: H₂O)
  2. "헤모글로빈과 케라틴의 기능이 다른 까닭은 무엇인가요?" (답: 아미노산 종류와 배열이 다름)
  3. "20가지 아미노산으로 10개짜리 폴리펩타이드를 만들 수 있는 조합은 몇 가지인가요?" (답: 20¹⁰ = 약 10조 가지)
- 실생활 예시: 달걀 흰자(알부민), 근육(액틴+미오신), 효소(촉매 단백질), 항체(면역 단백질)
- 교과서 연계: "미래엔 통합과학1 II편 p82~83 — 단백질의 형성(그림 II-29), 헤모글로빈과 케라틴 비교"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원
- 폴리펩타이드 곡선: CatmullRomCurve3 + TubeGeometry
- 나선 구조(케라틴): 매개변수 방정식 x=r*cos(t), y=t*pitch, z=r*sin(t)
- 아미노산 블록: BoxGeometry 재사용, 색상만 Material.color 변경
- 탭 전환 시 이전 객체 dispose()
- 성능: requestAnimationFrame 루프, 아미노산 블록 최대 15개
- 파일명: protein_formation_sim.html
