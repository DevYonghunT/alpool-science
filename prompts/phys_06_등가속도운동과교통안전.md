너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 가속도 법칙과 등가속도 운동
- 토픽: 등가속도 운동과 교통안전
- 대상: 고등학교 (선택과목)
- 핵심 개념: 자동차를 운전하다가 멈춰야 할 상황을 발견하면 브레이크를 밟는다. 이때 자동차가 정지하기까지 이동한 거리를 정지 거리라 하며, 정지 거리 = 반응 거리 + 제동 거리이다. 반응 거리는 위험을 인지하고 브레이크를 밟을 때까지 등속으로 이동한 거리이고, 제동 거리는 브레이크를 밟은 후 등가속도(감속)로 정지할 때까지 이동한 거리이다. 속력이 2배가 되면 제동 거리는 4배가 된다 (v²=2as에서 s∝v²).

## 씬 구성
- 화면 중앙: 3D 도로 + 자동차 + 보행자(장애물)
- 화면 하단: 반응 거리/제동 거리/정지 거리 막대그래프 (HTML overlay)
- 배경: 0xddeeff (주간 도로 하늘)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (측면 시점)
- 조명: AmbientLight(0.5) + DirectionalLight(0.7)

## 3D 도로 환경

### 도로
- BoxGeometry(40, 0.05, 4), 0x444444 (아스팔트)
- 중앙선: 노란 점선, 양측 차선 구분
- 횡단보도: 흰색 줄무늬 BoxGeometry × 6개, 도로 끝 부분
- 도로 옆 인도: BoxGeometry, 0xbbbbaa

### 자동차
- BoxGeometry 조합: 몸통(1.5×0.5×0.8, 0x3366aa) + 캐빈(1.0×0.35×0.75, 0x2255aa) + 바퀴 4개
- 브레이크 등: 후면에 SphereGeometry(r=0.05) × 2, 제동 시 0xff0000 발광
- 타이어 스키드 마크: 제동 시 도로에 검은 자국 (PlaneGeometry 얇은 줄)

### 보행자 (장애물)
- 횡단보도 위에 사람 실루엣(BoxGeometry 조합), 0xddccbb
- "⚠️ 위험 인지 지점" 라벨이 표시되는 위치

## 정지 거리 시뮬레이션

### 시뮬레이션 단계 (5초)
1. **등속 주행** (0~2초): 자동차가 초기 속도 v₀로 등속 이동
2. **위험 인지** (2초 시점): 보행자 발견 — "⚠️" 표시, 빨간 느낌표
3. **반응 시간** (2~2.7초, 약 0.7초): 운전자가 브레이크를 인지하고 밟기까지
   - 이 동안 자동차는 등속으로 계속 이동
   - **반응 거리** = v₀ × t_반응 (노란 구간으로 도로에 표시)
4. **제동 시작** (2.7초~): 브레이크 밟음 → 브레이크 등 빨강, 타이어 스키드
   - 등가속도 감속 (a = -v₀²/(2×s_제동))
   - **제동 거리**: v² = v₀² + 2as에서 s = v₀²/(2|a|) (주황 구간으로 도로에 표시)
5. **정지** 또는 **충돌**: 정지 거리 내에 보행자가 있으면 충돌 경고

### 도로 위 색상 구간 표시
- 노란색 구간: 반응 거리
- 주황색 구간: 제동 거리
- 빨간색 점선: 보행자 위치
- 초록 체크: 정지 성공 / 빨간 X: 충돌

## 속력별 정지 거리 비교 (핵심!)

### 3가지 속력 동시 비교 (3차선)
- 30km/h (약 8.3m/s): 파란 차
- 60km/h (약 16.7m/s): 노란 차
- 90km/h (약 25.0m/s): 빨간 차
- 3대가 동시에 출발, 같은 지점에서 위험 인지
- 정지 거리 차이를 시각적으로 비교
- **속력 2배 → 제동 거리 4배** 강조 (v²에 비례)

### 정지 거리 막대그래프 (하단)
| 속력 | 반응 거리 | 제동 거리 | 정지 거리 |
|------|---------|---------|---------|
| 30km/h | ~5.8m | ~4.2m | ~10m |
| 60km/h | ~11.7m | ~16.7m | ~28.4m |
| 90km/h | ~17.5m | ~37.5m | ~55m |

## 안전장치 시뮬레이션
- "안전띠" 토글: ON/OFF → 충돌 시 승객 관성 표현 (안전띠 없으면 앞으로 튀어나감)
- "에어백" 토글: ON/OFF → 충돌 시 에어백 팽창 애니메이션
- 안전띠/에어백의 원리: 충돌 시간을 늘려 힘(F=Δp/Δt)을 줄임

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "초기 속력" 슬라이더: 20~120km/h (기본값 60, 10 단계)
- "반응 시간" 슬라이더: 0.3~1.5초 (기본값 0.7, 0.1 단계)
- "도로 상태" 드롭다운: 마른 도로 / 젖은 도로(제동거리 1.5배) / 빙판(제동거리 3배)
- "시뮬레이션 시작" 버튼
- "3차선 비교" 버튼: 30/60/90km/h 동시 비교
- "안전띠" 토글: ON/OFF
- "에어백" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🚗 등가속도 운동과 교통안전
━━━━━━━━━━━━━━━━━━
초기 속력: [값]km/h ([값]m/s)
반응 시간: [값]s
도로 상태: [마른/젖은/빙판]

반응 거리 = v₀ × t반응 = [값]m
제동 거리 = v₀²/(2|a|) = [값]m
정지 거리 = 반응 + 제동 = [값]m

핵심:
  속력 2배 → 제동 거리 4배 (v²에 비례)
  v² = v₀² + 2as → s = v₀²/(2|a|)
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "정지 거리 = 반응 거리 + 제동 거리입니다. 속력이 2배가 되면 제동 거리는 4배가 됩니다(v²에 비례). 이 때문에 과속은 매우 위험하며, 안전 거리를 확보하는 것이 중요합니다."
- 탐구 질문:
  1. "속력이 60km/h에서 120km/h로 2배가 되면 정지 거리는 몇 배가 되나요?"
  2. "비가 올 때 정지 거리가 길어지는 까닭을 마찰력과 관련지어 설명해 보세요."
  3. "안전띠와 에어백이 충돌 시 승객을 보호하는 원리를 F=Δp/Δt로 설명해 보세요."
- 실생활 예시: 도로 속도 제한(스쿨존 30km/h), 안전거리 규정, 빗길/빙판 사고
- 교과서 연계: "물리학 I편 p36~37 — 등가속도 운동과 교통안전 사고 예방 (그림 I-18)"
