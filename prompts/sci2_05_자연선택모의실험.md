너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 생물의 진화
- 토픽: 자연선택 모의실험 (초콜릿 실험)
- 대상: 고등학교 1학년
- 핵심 개념: 모의실험을 통해 변이의 발생과 자연선택 과정을 이해한다. 노란색 도화지 위에 빨간색, 노란색, 초록색, 파란색 초콜릿을 놓고, 눈에 먼저 띄는 것을 집어내는 과정이 자연선택과 유사하다. 초록색 초콜릿을 노란색으로 바꾸는 과정은 돌연변이(변이의 발생)를 나타낸다.

## 씬 구성
- 화면 중앙에 노란색 도화지(PlaneGeometry)를 3D로 렌더링한다.
- 도화지 위에 4가지 색 초콜릿(SphereGeometry)을 배치한다.
- 교과서 탐구 (p22~23): 자연선택 과정에 대한 모의실험
- 배경: 0xf5f0e0 (밝은 크림)
- 카메라: PerspectiveCamera(FOV 45) + OrbitControls (위에서 내려다보는 시점)
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 모의실험 규칙 (교과서 탐구 정확 반영)
1. 노란색 도화지 위에 빨간색, 노란색, 초록색, 파란색 초콜릿을 **각각 10개씩** 골고루 섞어서 놓는다. (총 40개)
2. 3명이 눈을 감았다가 뜨자마자 **눈에 먼저 띄는 초콜릿을 5개씩** 손으로 집어낸다. (총 15개 제거)
3. 남은 초콜릿을 색깔별로 세고, **같은 색 초콜릿을 남은 수만큼 추가**해 골고루 섞는다. (번식 = 개체수 2배)
4. 과정 2~3을 반복한다. (최대 5회)
5. **3회 반복 후**: 초록색 초콜릿 3개를 노란색 초콜릿 3개로 바꾼다. (돌연변이)
6. 과정 2~3을 3회 더 반복한다.

## 3D 요소
### 도화지
- PlaneGeometry(8, 8), MeshStandardMaterial(0xeedd66 노란색, roughness 0.8)
- 살짝 두께감: BoxGeometry(8, 0.05, 8)

### 초콜릿 (각 색상별)
- SphereGeometry(r=0.15, 16, 16)
- 빨간색: 0xff3333
- 노란색: 0xffdd33
- 초록색: 0x33aa33
- 파란색: 0x3366ff
- 랜덤 위치로 도화지 위에 분산 배치
- 약간의 광택(metalness 0.2, roughness 0.4)

### 손 (포식자 역할)
- 초콜릿 제거 시 반투명 손 모양이 내려와 초콜릿을 집는 애니메이션
- 또는 초콜릿이 위로 빠르게 사라지는 효과

## 시각화 요소
### 색상별 개체수 막대그래프 (우측 패널, HTML overlay)
- 4가지 색상 막대
- 매 라운드마다 갱신
- 노란색 초콜릿 개수가 증가하는 추이를 명확히 보여줌

### 라운드 진행 표시
- "라운드 [N]/6" 표시
- 라운드 3 → "🧬 돌연변이 발생! 초록색 3개 → 노란색 3개" 알림
- 초콜릿이 집어내지는 것: 눈에 잘 띄는 색 = 생존에 불리한 형질
- 남은 초콜릿이 번식하는 것: 생존한 개체가 자손을 남김

### 비유 설명 (하단 텍스트)
- "초콜릿 색깔이 여러 종류인 것 = 변이"
- "눈에 띄는 것을 집어내는 과정 = 자연선택 (포식)"
- "같은 색 초콜릿을 추가하는 과정 = 번식"
- "초록색을 노란색으로 바꾸는 과정 = 돌연변이"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "다음 라운드" 버튼: 한 라운드 진행 (자동으로 집어내기 → 번식)
- "자동 진행" 버튼: 6라운드 자동 (각 라운드 3초)
- "라운드" 표시: 현재 라운드 / 6
- "제거 수" 슬라이더: 5~20 (기본값 15, 한 라운드에 제거되는 수)
- "돌연변이 라운드" 슬라이더: 1~5 (기본값 3, 돌연변이가 발생하는 라운드)
- "초기화" 버튼: 각 10개씩 40개로 리셋

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🍫 자연선택 모의실험
━━━━━━━━━━━━━━━━
현재 라운드:  [N] / 6
총 초콜릿:   [N]개

색깔별 개수:
  🔴 빨간색: [N]개 ([N]%)
  🟡 노란색: [N]개 ([N]%)
  🟢 초록색: [N]개 ([N]%)
  🔵 파란색: [N]개 ([N]%)

비유:
  초콜릿 색깔 = 생물의 형질 (변이)
  눈에 띄어 집힘 = 포식 (자연선택)
  같은 색 추가 = 번식
  색깔 변경 = 돌연변이
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "이 모의실험에서 노란색 도화지 위에서 노란색 초콜릿은 눈에 잘 띄지 않아 집어내지 않게 됩니다. 이는 환경(배경)에 적합한 형질(색깔)을 가진 개체가 생존에 유리한 자연선택 과정과 같습니다."
- 탐구 질문:
  1. "모의실험에서 노란색 도화지 위의 초콜릿 개수의 비율은 어떻게 변화했나요?"
  2. "변이의 발생과 자연선택 과정을 모의실험 결과와 관련지어 설명해 보세요."
  3. "도화지 색을 초록색으로 바꾸면 결과가 어떻게 달라질까요?"
- 실생활 예시: 항생제 내성균, 나방의 공업 암화(산업혁명 후 어두운 나방 증가)
- 교과서 연계: "미래엔 통합과학2 I편 p22~23 — 탐구: 자연선택 과정에 대한 모의실험"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- InstancedMesh로 초콜릿 40개 효율적 렌더링
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_05_natural_selection_experiment.html
