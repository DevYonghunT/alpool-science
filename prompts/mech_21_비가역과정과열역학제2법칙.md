너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 05. 비가역 현상과 엔트로피
- 토픽: 비가역 과정과 열역학 제2법칙
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 자연에서 일어나는 많은 과정은 한 방향으로만 자발적으로 진행되며 역과정은 저절로 일어나지 않는다. 이를 비가역 과정이라 한다. 열역학 제2법칙: 열은 고온에서 저온으로 자발적으로 흐르지만, 저온에서 고온으로 저절로 흐르지 않는다(클라우지우스 표현). 또한 열을 모두 일로 전환하는 것은 불가능하다(켈빈-플랑크 표현).

## 씬 구성
- 화면 중앙: 3D 비가역 과정 시각화 (4가지 사례)
- 화면 우측: 열역학 제2법칙 설명 패널 (HTML overlay)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 비가역 과정 사례 (4가지)

### 사례 1: 열의 이동 방향
- 3D: 두 금속 블록 접촉
  - 고온 블록(80°C): BoxGeometry(1.5, 1, 1), 0xff4444 (빨강)
  - 저온 블록(20°C): BoxGeometry(1.5, 1, 1), 0x4444ff (파랑)
- 접촉 시: 열이 고온→저온으로 자발적으로 이동
  - 빨간 파티클이 고온→저온으로 이동
  - 고온 블록 색상: 빨강→주황
  - 저온 블록 색상: 파랑→보라→연보라
  - 최종: 두 블록 같은 온도(50°C, 연보라)
- 역과정: 저온→고온으로 열이 자발적으로 이동? → ❌ 불가!
  - "역 재생" 버튼 → "❌ 이 과정은 저절로 일어나지 않습니다" 경고
- 클라우지우스 표현: "열은 저온→고온으로 저절로 흐르지 않는다"

### 사례 2: 기체의 자유 팽창
- 3D: 상자 2개 연결 (한쪽에 기체, 한쪽 진공)
  - 왼쪽 상자: 기체 분자(SphereGeometry × 30개, 0x44cc88) 가득
  - 오른쪽 상자: 비어있음 (진공)
  - 중간 칸막이: BoxGeometry 얇은 판
- 칸막이 제거 → 기체가 양쪽으로 퍼짐 (자발적)
  - 분자들이 양쪽에 균일하게 분포
- 역과정: 양쪽에 퍼진 기체가 한쪽으로 저절로 모인다? → ❌ 불가!

### 사례 3: 잉크의 확산
- 3D: 투명 비커에 물 + 한 방울의 잉크
  - 잉크: 0xff0088 (진한 분홍) 파티클, 처음에 한 점에 집중
- 시간 경과 → 잉크가 물 전체로 퍼짐 (확산, 자발적)
  - 파티클이 랜덤 워크(브라운 운동)로 퍼져나감
- 역과정: 퍼진 잉크가 한 점으로 모인다? → ❌ 불가!

### 사례 4: 깨진 달걀
- 3D: 탁자 위 달걀 → 바닥에 떨어져 깨짐
  - 달걀: SphereGeometry(r=0.3), 0xeedd99 (노른자색)
  - 깨진 상태: 조각(ConvexGeometry) 여러 개 + 내용물 흘러나옴
- 역과정: 깨진 달걀이 저절로 원래대로 복원? → ❌ 불가!

## 열역학 제2법칙 표현 (우측 패널)

### 클라우지우스 표현
- "열은 저온 물체에서 고온 물체로 저절로 흐르지 않는다"
- 화살표 다이어그램: 고온→저온 (✅) / 저온→고온 (❌)

### 켈빈-플랑크 표현
- "열을 모두 일로 전환하는 것(효율 100%)은 불가능하다"
- 열기관 다이어그램: Q_H → W + Q_L (Q_L > 0 항상)
- 효율 100% 열기관은 ❌

### 방향성 강조
- "자연에서 자발적으로 일어나는 과정은 한 방향으로만 진행된다"
- 시간의 화살: 과거→미래 (비가역)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "사례" 드롭다운: 열 이동 / 자유 팽창 / 잉크 확산 / 깨진 달걀
- "정방향 재생" 버튼: 자연스러운 과정 재생
- "역방향 재생" 버튼: 역과정 시도 → "❌ 불가!" 표시
- "재생 속도" 슬라이더: 0.5x~3x
- "열역학 제2법칙" 토글: 클라우지우스 / 켈빈-플랑크 표현 전환
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔄 비가역 과정과 열역학 제2법칙
━━━━━━━━━━━━━━━━━━━
사례: [열 이동/자유 팽창/잉크/달걀]
방향: [정방향(자발적) / 역방향(불가)]

비가역 과정:
  자발적으로 한 방향으로만 진행
  역과정은 저절로 일어나지 않음

열역학 제2법칙:
  클라우지우스: 열은 저온→고온으로
    저절로 흐르지 않는다
  켈빈-플랑크: 열을 100% 일로
    전환하는 것은 불가능하다
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "자연에서 자발적으로 일어나는 많은 과정은 한 방향으로만 진행되며 역과정은 저절로 일어나지 않습니다(비가역 과정). 열역학 제2법칙은 이 방향성을 설명합니다."
- 탐구 질문:
  1. "커피에 설탕을 넣으면 저절로 녹아 퍼지지만, 설탕이 저절로 다시 모이지는 않습니다. 이 현상을 비가역 과정으로 설명해 보세요."
  2. "냉장고는 저온→고온으로 열을 이동시킵니다. 이것이 열역학 제2법칙에 위배되지 않는 까닭은? (힌트: 외부에서 일 투입)"
  3. "열효율 100%인 열기관이 불가능한 까닭을 켈빈-플랑크 표현으로 설명해 보세요."
- 실생활 예시: 열이 전도되는 방향, 잉크 확산, 얼음 녹음, 배터리 방전
- 교과서 연계: "역학과 에너지 II편 p92~97 — 비가역 현상과 엔트로피, 열역학 제2법칙"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 100개
- 파일명: mech_21_irreversible_second_law.html
