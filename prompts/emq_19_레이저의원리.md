너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: II. 빛과 정보 통신 > 05. 레이저
- 토픽: 레이저의 원리 (유도 방출)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 레이저(LASER: Light Amplification by Stimulated Emission of Radiation)는 유도 방출에 의해 증폭된 빛이다. 자발 방출은 들뜬 상태의 원자가 저절로 빛을 내는 것이고, 유도 방출은 외부 광자에 의해 같은 진동수·위상·방향의 빛이 방출되는 것이다. 밀도 반전(들뜬 상태 원자 > 바닥 상태 원자)을 만들고 공진기(거울 2개)로 빛을 왕복 증폭시키면 레이저가 된다.

## 씬 구성
- 화면 좌측 50%: 3D 레이저 내부 구조 (원자 에너지 준위 + 자발/유도 방출 + 공진기)
- 화면 우측 50%: 에너지 준위 다이어그램 + 레이저 특성 비교 (Canvas 2D)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.2)

## 3D 원자의 에너지 전이

### 원자 모형
- 원자핵: SphereGeometry(r=0.2), 0xff4444
- 전자: SphereGeometry(r=0.08), 0xffff44
- 에너지 준위: 수평 링(TorusGeometry)
  - 바닥 상태 E₁: 0x4444ff (파랑), 반지름 r₁
  - 들뜬 상태 E₂: 0xff8844 (주황), 반지름 r₂ > r₁
  - 준안정 상태 E₃: 0x44ff44 (초록), 반지름 r₃

### 3가지 전이 과정
1. **흡수**: 광자(입사) → 전자가 E₁→E₂로 올라감
   - 입사 광자(SphereGeometry r=0.04, 0xffdd44): 원자에 흡수되어 사라짐
   - 전자가 낮은 궤도 → 높은 궤도로 점프 (애니메이션 0.5초)
   - "흡수: 광자 에너지 = E₂ - E₁" 라벨

2. **자발 방출**: 전자가 E₂→E₁로 저절로 내려감 → 광자 방출
   - 전자가 높은 궤도 → 낮은 궤도 점프
   - 광자(SphereGeometry r=0.04) 랜덤 방향으로 방출
   - "자발 방출: 랜덤 방향, 랜덤 위상" 라벨
   - 방출된 광자들이 여러 방향으로 퍼져나감 (비간섭 빛)

3. **유도 방출**: 외부 광자가 들뜬 원자를 자극 → 같은 광자 2개 방출!
   - 입사 광자(0xffdd44) 1개 → 들뜬 원자 통과
   - 전자가 E₂→E₁로 전이 + 광자 2개 방출 (입사 광자와 동일!)
   - **2개 광자: 같은 진동수, 같은 위상, 같은 방향!**
   - "유도 방출: 같은 진동수·위상·방향 → 간섭성 빛" 라벨
   - 2개 광자가 나란히 같은 방향으로 진행

## 레이저 공진기 시각화

### 3D 레이저 구조
- 매질(활성 매체): CylinderGeometry(0.8, 0.8, 6), 0xff4466 반투명 (루비/가스)
  - 내부에 원자 여러 개 (SphereGeometry × 20)
  - 바닥 상태(파랑) / 들뜬 상태(주황) 비율 표시
- 펌핑 장치: BoxGeometry, 0x44ff44 (플래시 램프/전류)
  - 펌핑 → 원자들을 들뜬 상태로 올림 → 밀도 반전
- 거울 1 (완전 반사): PlaneGeometry, 0xddddee (100% 반사)
- 거울 2 (부분 반사): PlaneGeometry, 0xddddee 반투명 (99% 반사, 1% 투과)
- 레이저 빔 출력: 거울 2에서 나오는 빨간 빔(CylinderGeometry, emissive 0xff0000)

### 레이저 작동 과정 (단계별 애니메이션)
1. **펌핑**: 외부 에너지 투입 → 원자들이 들뜬 상태로 (파랑→주황)
   - 밀도 반전 달성: 들뜬 상태 > 바닥 상태
2. **자발 방출 시작**: 일부 원자가 자발 방출 → 광자 랜덤 방향
3. **유도 방출 연쇄**: 축 방향 광자가 다른 원자를 유도 방출
   - 1개 → 2개 → 4개 → 8개 → ... (기하급수적 증폭!)
   - 축 방향 광자만 거울 사이에서 왕복 → 더 증폭
   - 비축 방향 광자는 매질 밖으로 빠져나감
4. **레이저 빔 출력**: 부분 반사 거울에서 일부 빛이 외부로 방출
   - 단색, 간섭성, 방향성, 높은 세기의 빛!

## 레이저 빛 vs 일반 빛 비교 (우측)
| 특성 | 레이저 빛 | 일반 빛(백열등) |
|------|---------|------------|
| 단색성 | 단일 파장 | 여러 파장 혼합 |
| 간섭성 | 같은 위상 | 무작위 위상 |
| 방향성 | 한 방향 (좁은 빔) | 모든 방향 |
| 세기 | 매우 높음 | 낮음 |
- 3D에서 레이저 빔(직선, 좁음) vs 전구 빛(퍼짐) 비교

## 에너지 준위 다이어그램 (우측 Canvas 2D)
- 3준위 또는 4준위 시스템
- 흡수(↑), 자발 방출(↓ 물결), 유도 방출(↓↓ 같은 방향)
- 펌핑, 밀도 반전, 레이징 과정 단계별 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 3가지 전이(흡수/자발/유도) / 레이저 공진기 / 빛 비교
- "전이 종류" 드롭다운 (전이 모드): 흡수 / 자발 방출 / 유도 방출
- "펌핑 세기" 슬라이더 (공진기): 0~10 (기본값 5)
- "레이저 작동 시작" 버튼: 단계별 애니메이션
- "레이저 종류" 드롭다운: He-Ne(빨강 632.8nm) / Nd:YAG(초록 532nm) / CO₂(적외선)
- "일반 빛 비교" 토글: ON/OFF
- "에너지 준위" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔴 레이저의 원리 (유도 방출)
━━━━━━━━━━━━━━━━
보기: [전이/공진기/비교]

LASER:
  Light Amplification by
  Stimulated Emission of Radiation

3가지 전이:
  흡수: 광자 → E₁→E₂ (광자 흡수)
  자발 방출: E₂→E₁ → 광자 (랜덤)
  유도 방출: 광자+E₂→E₁ → 2광자 (동일!)

레이저 조건:
  1. 밀도 반전 (들뜬 > 바닥)
  2. 유도 방출에 의한 증폭
  3. 공진기(거울)로 왕복 증폭

레이저 빛 특성:
  단색, 간섭성, 방향성, 고세기
```

  2. "레이저에서 밀도 반전이 필요한 까닭은 무엇인가요?"

- 실생활 예시: 레이저 포인터, 바코드 리더, 레이저 수술, 광통신, 레이저 가공, 홀로그램
- 교과서 연계: "전자기와 양자 II편 p84~89 — 레이저, 유도 방출, 광 다이오드"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 보기 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 광자 파티클 최대 100개
