너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 1. 빛과 물질의 이중성 > 빛과 물질의 이중성
- 토픽: 물질파와 전자 회절
- 대상: 고등학교 (선택과목)
- 핵심 개념: 드브로이(de Broglie)는 빛이 파동과 입자의 이중성을 가지듯이 전자와 같은 물질 입자도 파동성을 가진다고 제안했다. 물질파의 파장 λ = h/mv (h: 플랑크 상수, m: 질량, v: 속력)이다. 전자를 얇은 결정에 통과시키면 회절 무늬가 나타나며, 이는 전자의 파동성을 입증한다. 빛과 물질 모두 파동성과 입자성을 가지는 이중성이 있다.

## 씬 구성
- 화면을 좌/우로 분할:
  - 좌측: 빛의 이중성 (파동성 + 입자성) 시각화
  - 우측: 물질(전자)의 이중성 시각화
- 하단: 비교 표
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3)

## 좌측: 빛의 이중성

### 빛의 파동성
- 이중 슬릿 간섭무늬 (phys_24 간략 버전)
- 파동 표현: 사인파(TubeGeometry 곡선), 0xffdd44
- "간섭 → 파동의 특성" 라벨

### 빛의 입자성
- 광전 효과 (phys_26 간략 버전)
- 광자 파티클: SphereGeometry, 0xffdd44
- "광전 효과 → 입자의 특성" 라벨

### 비유
- 빛 = 파동 + 입자 (이중성)

## 우측: 물질(전자)의 이중성

### 전자의 입자성
- 전자총에서 전자 발사: SphereGeometry(r=0.05), 0x44ddff (파란 구)
- 전자가 직선으로 이동 (입자처럼)
- 형광판에 점으로 충돌

### 전자의 파동성 — 전자 회절 (핵심!)
- 전자총 → 얇은 결정(격자) → 스크린
- 결정(회절 격자): BoxGeometry 얇은 판, 0x888888, 격자 무늬
- **전자를 하나씩 발사**:
  - 처음: 랜덤한 위치에 점이 찍힘 (입자처럼)
  - 많이 쌓이면: **회절 무늬**(동심원 패턴)가 나타남!
  - "전자가 파동성을 가짐을 입증" 라벨
- 회절 무늬: 밝은 원형 띠(동심원) — 빛의 회절 무늬와 유사
- 전자 수 슬라이더: 10 → 100 → 1000 → 10000개
  - 10개: 무작위 점 (패턴 안 보임)
  - 1000개: 희미한 패턴
  - 10000개: 명확한 동심원 패턴

## 물질파 파장 계산

### 드브로이 파장
- λ = h/mv = h/p
  - h = 6.63×10⁻³⁴ J·s
  - m: 입자 질량
  - v: 입자 속력
  - p = mv: 운동량
- 전자(m = 9.1×10⁻³¹kg):
  - v = 10⁶m/s → λ ≈ 0.73nm (원자 크기 수준 → 회절 관찰 가능)
- 야구공(m = 0.15kg):
  - v = 40m/s → λ ≈ 1.1×10⁻³⁴m (관찰 불가능하게 작음)
- **결론**: 일상적 물체의 물질파 파장은 너무 작아 관찰 불가능하지만, 전자처럼 질량이 매우 작은 입자는 파동성이 관측 가능

### 파장 비교 시각화
- 전자 물질파 파장 vs 야구공 물질파 파장 막대그래프
- 스케일 차이를 로그 스케일로 표시
- "질량이 작을수록, 속력이 느릴수록 파장이 길다" 강조

## 빛과 물질의 이중성 비교 (하단 표)
| 구분 | 빛 | 물질(전자) |
|------|---|----------|
| 파동성 입증 | 이중 슬릿 간섭 | 전자 회절 |
| 입자성 입증 | 광전 효과 | 구름 상자 궤적 |
| 파장 | λ = c/f | λ = h/mv |
| 에너지 | E = hf | Ek = ½mv² |

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 빛의 이중성 / 전자 회절 / 물질파 파장 비교 / 전체
- "전자 수" 슬라이더: 10~10000 (로그 스케일, 기본값 100)
- "전자 속력" 슬라이더: 10⁵~10⁷m/s (기본값 10⁶)
- "전자 발사" 버튼: 전자를 하나씩 또는 연속 발사
- "입자 종류" 드롭다운 (파장 비교): 전자 / 양성자 / 야구공
- "비교 표" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

- 단일 HTML 파일로 작성 (외부 파일 없음)

- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js

- 성능: requestAnimationFrame 루프
