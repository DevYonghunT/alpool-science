너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 1. 힘과 운동 > 작용 반작용과 운동량 보존
- 토픽: 운동량 보존 법칙
- 대상: 고등학교 (선택과목)
- 핵심 개념: 운동량은 질량과 속도의 곱 p = mv이며, 벡터량이다. 외부에서 힘이 작용하지 않을 때 충돌 전후 두 물체의 운동량의 합은 보존된다. m₁v₁ + m₂v₂ = m₁v₁' + m₂v₂' (충돌 전 = 충돌 후). 운동량 보존 법칙은 로켓 추진, 당구, 교통사고 분석 등에 활용된다.

## 씬 구성
- 화면 중앙: 3D 레일 위 두 수레 충돌 시뮬레이션 (교과서 탐구)
- 화면 하단: 운동량 막대그래프 (충돌 전/후 비교)
- 3가지 충돌 모드: 탄성 충돌 / 완전 비탄성 충돌(합체) / 폭발(분리)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 레일 실험 (교과서 탐구, p42~44)

### 레일
- BoxGeometry(14, 0.1, 0.5), 0xaaaaaa (금속)
- 눈금: 0~14m 표시

### 수레 1
- BoxGeometry(1.0, 0.5, 0.6), 0xff4444 (빨강)
- 바퀴 4개, 0x333333
- 위에 추 올리기 가능 (CylinderGeometry, 0xff8844)
- "m₁ = [값]kg" 라벨
- 속도 벡터 화살표: 빨간색

### 수레 2
- BoxGeometry(1.0, 0.5, 0.6), 0x4444ff (파랑)
- 바퀴 4개
- 추 올리기 가능
- "m₂ = [값]kg" 라벨
- 속도 벡터 화살표: 파란색

### 충돌 부착 장치
- 탄성 충돌: 수레 앞면에 스프링(코일 형태 CylinderGeometry)
- 비탄성 충돌: 수레 앞면에 벨크로/접착면(BoxGeometry, 0x888844)

## 3가지 충돌 모드

### 모드 1: 탄성 충돌
- 두 수레가 스프링으로 튕겨져 분리
- 충돌 전후 운동에너지도 보존
- 충돌 순간 스프링 압축 애니메이션 (0.3초)
- v₁' = ((m₁-m₂)v₁ + 2m₂v₂)/(m₁+m₂)
- v₂' = ((m₂-m₁)v₂ + 2m₁v₁)/(m₁+m₂)

### 모드 2: 완전 비탄성 충돌 (합체)
- 두 수레가 접착면으로 붙어서 함께 이동
- v' = (m₁v₁ + m₂v₂)/(m₁+m₂)
- 합체 순간 약간의 파티클 효과
- 운동에너지는 줄어듦 (일부 열·소리로 전환)

### 모드 3: 폭발 (분리)
- 두 수레가 붙어 정지 상태에서 스프링으로 분리
- 초기 총 운동량 = 0
- 분리 후: m₁v₁' + m₂v₂' = 0 → v₁'/v₂' = -m₂/m₁
- 무거운 수레는 천천히, 가벼운 수레는 빠르게 이동

## 운동량 막대그래프 (하단 HTML overlay)
- 4개 막대:
  - 수레 1 충돌 전 운동량: p₁ = m₁v₁ (빨강)
  - 수레 2 충돌 전 운동량: p₂ = m₂v₂ (파랑)
  - 수레 1 충돌 후 운동량: p₁' = m₁v₁' (연빨강)
  - 수레 2 충돌 후 운동량: p₂' = m₂v₂' (연파랑)
- 총 운동량 합계 표시:
  - 충돌 전: p₁ + p₂ = [값] kg·m/s
  - 충돌 후: p₁' + p₂' = [값] kg·m/s
  - **두 합계가 같음** = 운동량 보존 ✅

## 실생활 예시 (추가 시나리오)
- 당구: 큐 볼이 정지한 공에 충돌
- 스케이트: 두 사람 밀기 (폭발)
- 로켓: 연소 가스 분출 (폭발)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "충돌 모드" 드롭다운: 탄성 충돌 / 완전 비탄성(합체) / 폭발(분리)
- "m₁ 질량" 슬라이더: 0.5~5kg (기본값 1, 0.5 단계)
- "v₁ 속도" 슬라이더: -5~5m/s (기본값 3)
- "m₂ 질량" 슬라이더: 0.5~5kg (기본값 1)
- "v₂ 속도" 슬라이더: -5~5m/s (기본값 0)
- "충돌 시작" 버튼
- "속도 배율" 슬라이더: 0.2x~2x (기본값 1)
- "운동량 그래프" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

```
  총 운동량 = [값] kg·m/s

충돌 후:
  p₁' = m₁v₁' = [값]×[값] = [값] kg·m/s
  p₂' = m₂v₂' = [값]×[값] = [값] kg·m/s
  총 운동량 = [값] kg·m/s

```
