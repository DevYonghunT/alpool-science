너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 01. 힘의 합성과 운동 예측
- 토픽: 알짜힘에 의한 운동 예측
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 물체에 작용하는 여러 힘의 합력(알짜힘)을 벡터 합성으로 구하면, 뉴턴 제2법칙 F=ma로 가속도를 예측할 수 있다. 빗면 위의 물체에 작용하는 알짜힘은 중력의 빗면 방향 성분에서 마찰력을 뺀 것이며, 이를 통해 가속도를 계산할 수 있다.

## 씬 구성
- 화면 중앙: 3D 빗면 + 물체 + 힘 화살표 시각화
- 3가지 시나리오: 빗면(마찰 없음) / 빗면(마찰 있음) / 수평면+경사력
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 빗면 위 물체 (마찰 없음)

### 3D
- 빗면: BoxGeometry, 경사각 θ 조절 가능, 0xaa9988 (나무색)
- 물체: BoxGeometry(0.8, 0.6, 0.6), 0x4488aa
- 바닥: PlaneGeometry, 0xdddddd

### 힘 화살표
- 중력 W = mg: 아래 방향 빨간 화살표 (물체 중심에서)
- 수직 항력 N: 빗면에 수직인 파란 화살표 (위로)
- 중력의 빗면 방향 성분 W∥ = mgsinθ: 주황 점선 화살표 (빗면 아래 방향)
- 중력의 수직 방향 성분 W⊥ = mgcosθ: 보라 점선 화살표 (빗면 수직 안쪽)
- 알짜힘 F_net = mgsinθ: 초록 두꺼운 화살표 (빗면 아래 방향)

### 운동 예측
- a = gsinθ (빗면 방향 가속도)
- "시뮬레이션 시작": 물체가 빗면을 미끄러져 내려감 (등가속도 운동)
- 실시간 속도/위치/시간 표시
- 경사각 θ를 바꾸면 가속도 변화 → 운동 속도 변화

## 시나리오 2: 빗면 위 물체 (마찰 있음)

### 추가 힘
- 운동 마찰력 f = μN = μmgcosθ: 보라 화살표 (빗면 위 방향, 운동 반대)
- 알짜힘 F_net = mgsinθ - μmgcosθ = mg(sinθ - μcosθ)
- a = g(sinθ - μcosθ)
- μ < tanθ: 미끄러짐 (양의 가속도)
- μ = tanθ: 등속 (가속도 0)
- μ > tanθ: 정지 유지 (정지 마찰력이 mgsinθ 상쇄)
- 마찰 계수 μ 슬라이더로 각 경우 시연

## 시나리오 3: 수평면 위 경사력

### 3D
- 수평면: PlaneGeometry, 0xdddddd
- 물체: BoxGeometry, 0x4488aa
- 경사진 힘 F: 수평면과 θ 각도로 작용 (빨간 화살표)
  - x성분: Fcosθ (수평, 운동 방향)
  - y성분: Fsinθ (수직, 위로)

### 힘 분석
- 수직 방향: N + Fsinθ - mg = 0 → N = mg - Fsinθ
- 수평 방향: Fcosθ - f = ma
  - f = μN = μ(mg - Fsinθ)
- a = (Fcosθ - μ(mg - Fsinθ)) / m
- 최적 각도: 가속도를 최대로 하는 θ 찾기 (미분)

## 실시간 그래프 (우측 Canvas 2D)
- X축: 시간 t
- Y축: 가속도 a / 속도 v / 변위 s (3개 그래프 전환)
- 시뮬레이션 진행에 따라 실시간으로 그려짐

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 빗면(마찰 없음) / 빗면(마찰 있음) / 수평면+경사력
- "질량 m" 슬라이더: 1~10kg (기본값 2, 0.5 단계)
- "경사각 θ" 슬라이더: 5~60° (기본값 30°, 1 단계)
- "마찰 계수 μ" 슬라이더 (시나리오 2,3): 0~0.8 (기본값 0.2, 0.05 단계)
- "힘 F" 슬라이더 (시나리오 3): 0~50N (기본값 20)
- "시뮬레이션 시작" 버튼
- "힘 분해 표시" 토글: ON/OFF (기본값 ON)
- "그래프 종류" 드롭다운: a-t / v-t / s-t
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📐 알짜힘에 의한 운동 예측
━━━━━━━━━━━━━━━━━
시나리오: [빗면/수평면]
m = [값]kg, θ = [값]°, μ = [값]

빗면 (마찰 없음):
  W∥ = mgsinθ = [값]N
  a = gsinθ = [값]m/s²

빗면 (마찰 있음):
  f = μmgcosθ = [값]N
  F_net = mg(sinθ - μcosθ) = [값]N
  a = g(sinθ - μcosθ) = [값]m/s²

수평면 + 경사력:
  Fx = Fcosθ = [값]N
  N = mg - Fsinθ = [값]N
  a = (Fcosθ - μN)/m = [값]m/s²
```

화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "물체에 작용하는 여러 힘을 벡터 합성하여 알짜힘을 구하고, F=ma로 가속도를 예측합니다. 빗면 위 물체의 가속도는 a=gsinθ(마찰 없음) 또는 a=g(sinθ-μcosθ)(마찰 있음)입니다."
- 탐구 질문:
  1. "빗면 경사각을 크게 하면 물체의 가속도는 어떻게 변하나요?"
  2. "마찰 계수가 tanθ와 같을 때 물체는 어떤 운동을 하나요?"
  3. "수평면 위의 물체에 비스듬히 힘을 가할 때 수직 항력이 mg보다 작아지는 까닭은 무엇인가요?"
- 실생활 예시: 스키 점프(빗면 가속), 미끄럼틀, 경사로 주차
- 교과서 연계: "역학과 에너지 I편 p8~15 — 힘의 합성과 운동 예측, 빗면 위 물체"
