너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: III. 탄성파와 소리 > 05. 정상파와 악기
- 토픽: 현악기와 관악기의 원리
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 현악기(기타, 바이올린)는 현(줄)의 정상파를 이용하여 소리를 낸다. 현의 기본 진동수 f₁ = (1/2L)√(T/μ) (L: 현 길이, T: 장력, μ: 선밀도)이다. 현 길이를 줄이면 높은 음, 장력을 높이면 높은 음이 난다. 관악기(플루트, 클라리넷)는 관 내부 공기 기둥의 정상파를 이용한다. 열린 관은 모든 배음, 닫힌 관은 홀수 배음만 가진다. 악기의 음색은 기본음과 배음의 조합으로 결정된다.

## 씬 구성
- 화면 좌측 60%: 3D 악기 모형 + 현/관 정상파 시각화
- 화면 우측 40%: 진동수 스펙트럼(배음 구조) + 파형 그래프 (Canvas 2D)
- 2가지 모드: 현악기(기타) / 관악기(플루트/클라리넷)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 모드 1: 현악기 (기타)

### 3D 기타 모형
- 몸통: BoxGeometry(3, 0.4, 2) + SphereGeometry 반 (음공), 0x8B5A2B (나무색)
- 넥: BoxGeometry(5, 0.3, 0.5), 0x7B4B2A
- 현(줄) 6개: CylinderGeometry(0.01, 0.01, L) × 6, 다른 굵기/색상
  - 1번 줄(가장 가늘, 높은 음): 0xdddddd
  - 6번 줄(가장 두꺼운, 낮은 음): 0xccaa66
- 프렛: BoxGeometry(0.02, 0.01, 0.5) × 여러 개, 0xdddddd (금속)
- 브릿지/너트: BoxGeometry, 0xeeeedd (뼈색)

### 현의 진동 시각화
- 선택한 현이 정상파 형태로 진동
  - BufferGeometry의 position을 매 프레임 사인파로 갱신
  - n차 고조파 선택 가능
- 프렛 누르기:
  - 프렛 위치 슬라이더 → 유효 현 길이 L 단축
  - L 줄어듦 → f₁ 증가 (높은 음)
  - 프렛 위치에 손가락 표시 (SphereGeometry)

### 현의 진동수 결정 요인
- f₁ = (1/2L)√(T/μ)
  - L(현 길이): 프렛으로 조절, L↓ → f↑
  - T(장력): 조율 페그로 조절, T↑ → f↑
  - μ(선밀도): 줄 교체로 변경, μ↓ → f↑ (가는 줄 = 높은 음)
- 각 변수에 슬라이더 → 진동수 실시간 변화
- "기타 조율: 줄의 장력 T를 조절" 라벨

### 배음 구조
- 기타줄을 튕기면 기본음 + 2차, 3차, ... 배음이 함께 발생
- 줄의 어느 위치를 튕기는지에 따라 배음 조합 변화
  - 중앙: 기본음 강함, 배음 약함 (부드러운 음색)
  - 브릿지 근처: 배음 강함 (날카로운 음색)
- 배음 슬라이더: 각 배음의 상대적 세기 조절

## 모드 2: 관악기 (플루트/클라리넷)

### 3D 악기 모형
- 플루트 (열린 관): CylinderGeometry(0.15, 0.15, L), 0xcccccc (은색)
  - 키(구멍): CylinderGeometry(0.04, 0.04, 0.02) × 여러 개, 0xdddddd
  - 양쪽 열림 → 모든 배음
- 클라리넷 (닫힌 관): CylinderGeometry(0.1, 0.15, L), 0x333333 (검정)
  - 마우스피스(한쪽 닫힘): ConeGeometry, 0xddaa44
  - 한쪽 닫힘 → 홀수 배음만

### 관 내부 정상파 시각화
- 관을 투명하게 하여 내부 공기 기둥 정상파 표시
- 변위 정상파: 사인파 곡선 (관 내부)
  - 열린 끝 = 배(최대 변위)
  - 닫힌 끝 = 마디(변위 0)
- 압력 정상파: 변위와 90° 위상차
  - 열린 끝 = 마디(압력 변화 0)
  - 닫힌 끝 = 배(최대 압력 변화)
- "변위 정상파 / 압력 정상파" 토글

### 키(구멍) 열기/닫기
- 관의 키를 클릭하여 열기 → 유효 관 길이 변화 → 진동수 변화
- 키를 열면 그 지점이 열린 끝 역할 → 관 길이 짧아짐 → 높은 음
- 실제 악기 연주 원리

### 플루트 vs 클라리넷 배음 비교
- 플루트: 모든 배음 (f₁, 2f₁, 3f₁, 4f₁, ...)
- 클라리넷: 홀수 배음만 (f₁, 3f₁, 5f₁, ...)
- → 음색 차이! "같은 높이의 음이라도 배음 구조가 달라 음색이 다르다"

## 진동수 스펙트럼 (우측 상단 Canvas 2D)
- X축: 진동수 (또는 배음 번호)
- Y축: 세기 (진폭)
- 막대그래프:
  - 기본음 f₁: 가장 큰 막대
  - 2차 배음 2f₁: 그 다음 (기타/플루트)
  - 3차 배음 3f₁: ...
  - 클라리넷: 짝수 배음 없음
- 악기별 스펙트럼 비교 모드

## 파형 그래프 (우측 하단 Canvas 2D)
- 기본음만: 사인파
- 기본음 + 2차 배음: 약간 복잡한 파형
- 기본음 + 여러 배음: 악기 특유의 복잡한 파형
- "음색 = 기본음 + 배음의 조합" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "악기" 드롭다운: 기타(현) / 플루트(열린 관) / 클라리넷(닫힌 관)
- "현/관 길이 L" 슬라이더: 0.3~1.5m (기본값 0.65)
- "장력 T" 슬라이더 (기타): 50~200N (기본값 100)
- "선밀도 μ" 슬라이더 (기타): 0.001~0.01kg/m
- "고조파 n" 슬라이더: 1~5 (기본값 1)
- "배음 세기" 폴더: 2차~5차 배음 각각 0~100%
- "프렛 위치" 슬라이더 (기타): 0~12 (기본값 0)
- "소리 재생" 토글: ON/OFF (Web Audio)
- "변위/압력 정상파" 토글 (관악기)
- "초기화" 버튼

- 탐구 질문:
  1. "기타줄을 세게 조이면(장력↑) 음이 어떻게 변하나요?"

  3. "기타의 12번 프렛을 누르면 현 길이가 반으로 줄어듭니다. 이때 진동수는 어떻게 변하나요? (힌트: 옥타브)"
- 실생활 예시: 기타 조율/프렛, 바이올린 현 두께, 플루트 키, 오르간 파이프 길이
