너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 05. 비가역 현상과 엔트로피
- 토픽: 엔트로피와 무질서도
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 엔트로피는 무질서한 정도를 나타내는 물리량이다. 자발적으로 일어나는 비가역 과정에서 전체 엔트로피는 항상 증가한다(엔트로피 증가 법칙). ΔS = Q/T (가역 과정). 고립계에서 엔트로피는 증가하거나 일정하며, 감소하지 않는다. 우주 전체의 엔트로피는 계속 증가하고 있다.

## 씬 구성
- 화면 좌측 50%: 3D 분자 배열 시각화 (질서→무질서)
- 화면 우측 50%: 엔트로피 막대그래프 + 미시 상태 수 표시 (HTML/Canvas)
- 3가지 시나리오: 기체 확산 / 얼음 녹음 / 카드 섞기 비유
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 시나리오 1: 기체 확산과 엔트로피

### 3D 상자
- BoxGeometry(6, 4, 4), 투명 (wireframe 또는 반투명 면)
- 중앙 칸막이: PlaneGeometry, 0x888888 (제거 가능)
- 왼쪽: 기체 분자 30개 (SphereGeometry r=0.08, 0xff4444 빨강)
- 오른쪽: 비어있음 (진공)

### 단계별 시각화
- **단계 1 (질서)**: 모든 분자가 왼쪽에 정렬
  - 엔트로피: 낮음 (S_low)
  - 미시 상태 수: 적음 (분자 배치 방법 적음)
  - 라벨: "높은 질서 = 낮은 엔트로피"
- **단계 2 (칸막이 제거)**: 분자들이 양쪽으로 퍼짐
  - 분자들의 랜덤 워크 애니메이션
  - 점차 양쪽에 균일하게 분포
- **단계 3 (무질서)**: 분자가 양쪽에 균일 분포
  - 엔트로피: 높음 (S_high)
  - 미시 상태 수: 많음 (분자 배치 방법 많음)
  - 라벨: "높은 무질서 = 높은 엔트로피"
- **역과정 불가**: 양쪽에 퍼진 분자가 자발적으로 한쪽에 모이지 않음
  - ΔS > 0 (엔트로피 증가) → 자발적 과정

### 미시 상태 수 (볼츠만)
- S = k_B ln(Ω) (S: 엔트로피, k_B: 볼츠만 상수, Ω: 미시 상태 수)
- 분자 2개, 칸 2개:
  - 모두 왼쪽: 1가지 (Ω=1)
  - 1개씩: 2가지 (Ω=2)
- 분자 30개:
  - 모두 왼쪽: Ω = 1 (확률 ≈ 10⁻⁹)
  - 균등 분포: Ω = C(30,15) ≈ 10⁸ (가장 높은 확률)
- "미시 상태 수가 가장 큰 배치 = 가장 높은 확률 = 최대 엔트로피" 

## 시나리오 2: 얼음 녹음과 엔트로피

### 3D
- 얼음: BoxGeometry, 0xaaddff, 내부에 분자 격자 배열 (질서)
  - 분자: SphereGeometry × 20, 격자에 고정, 진동만 함
- 물: CylinderGeometry, 0x4488cc, 내부에 분자 자유 이동 (무질서)
  - 분자: SphereGeometry × 20, 랜덤 이동

### 엔트로피 변화
- 얼음(질서, 낮은 S) → 물(무질서, 높은 S): ΔS > 0
- 열 흡수: Q_융해 = L_f × m
- ΔS = Q/T = L_f × m / T
- 슬라이더로 온도 변화 → 얼음↔물 전환 + 엔트로피 변화

## 시나리오 3: 카드 섞기 비유

### 3D
- 카드 52장: PlaneGeometry(0.6, 0.9) × 52, 앞면에 숫자/무늬
- **정렬 상태**: 스페이드 A~K, 하트 A~K, ... 순서대로 쌓임
  - 미시 상태: 1가지
  - 엔트로피: 최소
- **섞은 상태**: 무작위 순서
  - 미시 상태: 52! ≈ 8×10⁶⁷ 가지
  - 엔트로피: 최대
- "섞기" 버튼: 카드가 공중에서 무작위로 재배열되는 애니메이션
- "정렬로 돌아올 확률 = 1/52! ≈ 0" 라벨
- "자발적으로 정렬 상태로 돌아가지 않는다 = 엔트로피 증가"

## 엔트로피 막대그래프 (우측)
- 과정 전/후 엔트로피 비교
- S_전 (파란 막대) < S_후 (빨간 막대)
- ΔS = S_후 - S_전 > 0 (항상, 자발적 과정)
- "고립계에서 엔트로피는 항상 증가한다" 강조

## 우주의 엔트로피 (보너스)
- "우주의 열죽음(heat death)": 우주 전체 엔트로피가 최대에 도달하면 모든 과정이 정지
- 간단한 다이어그램: 빅뱅(낮은 S) → 현재 → 미래(최대 S)
- "우주의 엔트로피는 계속 증가하고 있다"

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 기체 확산 / 얼음 녹음 / 카드 섞기
- "과정 시작" 버튼: 비가역 과정 재생
- "역과정 시도" 버튼: → "❌ 자발적으로 불가!" 표시
- "분자 수" 슬라이더 (기체): 10~50 (기본값 30)
- "미시 상태 수 표시" 토글: ON/OFF (기본값 ON)
- "엔트로피 막대" 토글: ON/OFF (기본값 ON)
- "우주 열죽음" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
📊 엔트로피와 무질서도
━━━━━━━━━━━━━━━━
시나리오: [기체/얼음/카드]

엔트로피:
  S = k_B ln(Ω)
  k_B = 1.38×10⁻²³ J/K
  Ω: 미시 상태 수

변화:
  S_전 = [값], S_후 = [값]
  ΔS = S_후 - S_전 = [값] > 0

엔트로피 증가 법칙:
  고립계에서 ΔS ≥ 0
  자발적 과정 → ΔS > 0
  가역 과정 → ΔS = 0

열 교환: ΔS = Q/T
```
