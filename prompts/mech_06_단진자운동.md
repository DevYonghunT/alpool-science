너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 03. 여러 가지 운동
- 토픽: 단진자 운동
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 실에 매달린 추가 작은 각도로 왕복 운동하는 것을 단진자 운동이라 한다. 단진자의 주기 T = 2π√(L/g)이며, 추의 질량과 진폭(작은 각도)에는 무관하고 실의 길이 L과 중력 가속도 g에만 의존한다(등시성). 단진자는 단진동의 예이다.

## 씬 구성
- 화면 좌측 60%: 3D 단진자 (고정점 + 실 + 추) 왕복 운동
- 화면 우측 40%: θ-t, ω-t 그래프 + 주기 계산 (Canvas 2D)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 단진자

### 고정점
- SphereGeometry(r=0.08), 0x444444, 상단 중앙에 고정
- 지지대: CylinderGeometry(0.05, 0.05, 0.3), 0x888888

### 실
- CylinderGeometry(0.015, 0.015, L), 0xdddddd
- 실 길이 L: 슬라이더로 조절 (0.5~3m)
- 고정점에서 추까지 연결, 추의 운동에 따라 기울기 변화

### 추
- SphereGeometry(r=0.2), 0xff4444 (빨간 공)
- 또는 CylinderGeometry(0.15, 0.15, 0.3) 원통형 추
- 질량 m: 라벨 표시, 슬라이더로 조절
- 추의 운동: θ(t) = θ₀cos(ωt), ω = √(g/L)

### 운동 시각화
- 추가 좌우로 왕복: sin/cos 함수에 따라
- 최대 변위(θ₀)에서 정지 → 최저점에서 최대 속도 → 반대쪽 최대 변위에서 정지
- 속도 벡터: 초록 화살표, 접선 방향, 크기 변화
- 가속도 벡터: 빨간 화살표, 항상 평형 위치(최저점) 방향
- 복원력: 파란 화살표, 평형 위치 방향 (F = -mgsinθ ≈ -mgθ for small θ)

### 잔상/궤적
- 추의 궤적을 반투명 호(arc)로 표시
- 최대 변위에서 멈추는 순간을 강조 (속도 0)

## 주기와 변수 관계 (핵심!)

### 실험 1: 실의 길이 L 변화
- L을 0.5, 1.0, 1.5, 2.0, 2.5m으로 변경
- T = 2π√(L/g) → L이 길수록 주기 길어짐
- T²∝L (비례 관계)
- 여러 길이의 진자를 동시에 비교하는 모드

### 실험 2: 질량 m 변화
- m을 0.5, 1.0, 2.0, 3.0kg으로 변경
- **주기는 질량에 무관!** T = 2π√(L/g)에 m이 없음
- 질량이 다른 진자를 동시에 흔들어 주기가 같음을 확인

### 실험 3: 진폭(초기 각도) θ₀ 변화
- θ₀를 5°, 10°, 15°, 20°로 변경
- **작은 각도에서 주기는 진폭에 거의 무관** (등시성)
- 큰 각도(>30°)에서는 주기 약간 증가 (선택적으로 보여주기)

### 실험 4: 중력 가속도 g 변화
- 지구(g=9.8), 달(g=1.6), 목성(g=24.8)
- g가 작으면 주기 길어짐 (달에서 진자가 느리게)

## 실시간 그래프 (우측 Canvas 2D)
- θ-t 그래프: 코사인 곡선, 주기 T 표시
- ω(각속도)-t 그래프: 사인 곡선 (θ와 90° 위상차)
- 주기 T 측정: 자동으로 한 왕복 시간 측정하여 표시
- T = 2π√(L/g) = [값]s (이론값과 비교)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "실 길이 L" 슬라이더: 0.5~3m (기본값 1, 0.1 단계)
- "질량 m" 슬라이더: 0.5~5kg (기본값 1)
- "초기 각도 θ₀" 슬라이더: 5~45° (기본값 15, 1 단계)
- "중력" 드롭다운: 지구(9.8) / 달(1.6) / 화성(3.7) / 목성(24.8) / 사용자 지정
- "시뮬레이션 시작" 버튼
- "비교 모드" 버튼: 2개 진자를 다른 L로 동시에 (주기 비교)
- "속도/가속도 벡터" 토글: ON/OFF
- "에너지 막대그래프" 토글: ON/OFF (Ek+Ep 실시간)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🕰️ 단진자 운동
━━━━━━━━━━━━━━
L = [값]m, m = [값]kg, θ₀ = [값]°
중력: [행성] (g = [값]m/s²)

주기: T = 2π√(L/g)
     = 2π√([값]/[값])
     = [값]s

각진동수: ω = √(g/L) = [값]rad/s
진동수: f = 1/T = [값]Hz

등시성:
  T는 질량 m에 무관
  T는 진폭 θ₀에 거의 무관 (작은 각도)
  T는 L과 g에만 의존
  T ∝ √L, T ∝ 1/√g
```
