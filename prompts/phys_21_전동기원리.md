너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: II. 전기와 자기 > 2. 전기와 자기의 상호작용 > 전류에 의한 자기장과 에너지 전환
- 토픽: 전동기 원리 (전류가 자기장에서 받는 힘)
- 대상: 고등학교 (선택과목)
- 핵심 개념: 자기장 속에서 전류가 흐르는 도선은 힘을 받는다. 이 힘의 방향은 전류의 방향과 자기장의 방향 모두에 수직이며, 왼손 법칙(플레밍)으로 결정한다. 전동기(모터)는 자기장 속에서 전류가 흐르는 코일이 힘을 받아 회전하는 원리를 이용하여 전기 에너지를 운동 에너지(역학적 에너지)로 전환하는 장치이다.

## 씬 구성
- 화면 중앙: 3D 전동기 내부 구조 (자석 + 코일 + 정류자 + 브러시)
- 2가지 모드: 직선 도선에 작용하는 힘 / 전동기(DC 모터)
- 배경: 0x1a1a2e
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 모드 1: 자기장에서 전류가 받는 힘

### 3D
- 균일 자기장: 두 큰 자석(BoxGeometry) N극/S극 마주보기, 자기장 화살표(파란) 수평
- 직선 도선: CylinderGeometry(0.05, 0.05, 4), 0xcc6633, 자기장에 수직으로 배치
- 전류 방향: 노란 파티클이 도선을 따라 이동
- 힘 방향: 빨간 화살표, 전류와 자기장 모두에 수직 (플레밍 왼손 법칙)

### 플레밍 왼손 법칙 시각화
- 3D 왼손 모형:
  - 검지: 자기장 B 방향 (파랑)
  - 중지: 전류 I 방향 (노랑)
  - 엄지: 힘 F 방향 (빨강)
- 전류/자기장 방향 변경 시 힘 방향 자동 갱신

### 힘의 크기
- F = BIL (B: 자기장, I: 전류, L: 도선 길이)
- 슬라이더로 B, I 조절 → 힘의 크기(화살표 길이) 변화
- 도선이 힘 방향으로 이동하는 애니메이션

## 모드 2: 전동기 (DC 모터) 구조와 작동

### 3D 전동기 내부 구조
- 영구 자석: 2개의 곡면 자석(BoxGeometry), N극(빨강)/S극(파랑), 고정
- 회전 코일(전기자):
  - 코일: TorusGeometry 직사각형 형태 (4변), 0xcc6633 (구리색)
  - 코일 축: CylinderGeometry(0.05, 0.05, 3), 0x888888
  - 코일이 축을 중심으로 회전
- 정류자(commutator):
  - 원통형, 2개의 반원 금속 조각 (CylinderGeometry 반원 × 2), 0xddaa44
  - 코일 축 끝에 부착, 코일과 함께 회전
- 브러시(brush):
  - BoxGeometry(0.1, 0.3, 0.1) × 2, 0x444444 (탄소 검정)
  - 정류자에 접촉, 고정된 위치
  - 전원(전지)과 연결

### 작동 원리 애니메이션 (4단계)
1. **전류 흐름**: 전지 → 브러시 → 정류자 → 코일 (노란 파티클)
2. **힘 작용**: 자기장 속 코일의 양쪽 변에 반대 방향 힘 → 돌림힘 발생
   - 한쪽 변: 위로 힘 (빨간 화살표)
   - 반대쪽 변: 아래로 힘 (빨간 화살표)
   - → 코일 회전!
3. **정류자 역할**: 반 바퀴 회전할 때마다 전류 방향 바뀜 → 계속 같은 방향으로 회전
   - 정류자가 브러시와 접촉면이 바뀌는 순간 강조 (플래시 효과)
4. **연속 회전**: 코일이 계속 같은 방향으로 회전

### 에너지 전환 경로
- 전기 에너지(전지) → 운동 에너지(코일 회전) + 열에너지(마찰)
- 화살표로 에너지 전환 경로 표시

## 실생활 전동기 응용
- "응용" 토글:
  - 선풍기: 전동기 + 날개 → 바람
  - 전기차: 전동기 → 바퀴 회전
  - 세탁기: 전동기 → 드럼 회전
  - 간단한 3D 아이콘으로 표시

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 직선 도선의 힘 / 전동기(DC모터)
- "전류 I" 슬라이더: 0.5~5A (기본값 2)
- "자기장 B" 슬라이더: 0.1~1T (기본값 0.5)
- "전류 방향" 토글: 정방향 / 역방향 (모터 회전 방향 변경)
- "재생 속도" 슬라이더: 0.5x~3x
- "왼손 법칙 안내" 토글: ON/OFF
- "에너지 전환 표시" 토글: ON/OFF
- "실생활 응용" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚙️ 전동기 원리
━━━━━━━━━━━━━━
모드: [직선 도선 / 전동기]
전류 I = [값]A
자기장 B = [값]T

전류가 자기장에서 받는 힘:
  F = BIL = [값]N
  방향: 플레밍 왼손 법칙

전동기:
  자기장 속 코일에 전류 → 돌림힘 → 회전
  정류자: 반 바퀴마다 전류 방향 전환
  → 계속 같은 방향으로 회전

에너지 전환:
  전기 에너지 → 운동 에너지(회전)
```

  2. "전동기의 회전 속도를 빠르게 하려면 어떤 물리량을 변화시켜야 하나요?"
  3. "전동기와 발전기는 에너지 전환 방향이 반대입니다. 이를 비교해 설명해 보세요."
- 실생활 예시: 선풍기, 전기차, 세탁기, 엘리베이터, 전동 드릴
- 교과서 연계: "물리학 II편 p104~111 — 전류가 자기장에서 받는 힘, 전동기 원리"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
