너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 3. 생명 시스템 > 생명 시스템에서의 화학 반응
- 토픽: 물질대사와 효소
- 대상: 고등학교 1학년
- 핵심 개념: 생명 시스템을 유지하기 위해 세포에서 다양한 화학 반응(물질대사)이 일어난다. 효소는 화학 반응의 속도를 빠르게 하는 생체 촉매로, 기질의 활성 부위에 결합하여 반응을 촉진한다(자물쇠-열쇠 모형). 효소는 특정 기질에만 작용하며(기질 특이성), 반응 후 효소 자체는 변하지 않아 재사용된다. 효소의 활성은 온도와 pH에 의해 영향을 받으며, 최적 조건에서 활성이 최대이다.

## 씬 구성
- 2탭: 자물쇠-열쇠 모형 (3D) / 온도·pH에 따른 효소 활성 (그래프)
- 배경: 0x0a1828 (짙은 생물학 남색)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.35) + DirectionalLight(0.6) + PointLight(효소 근처)

## 탭 1: 자물쇠-열쇠 모형 3D

### 효소 3D 모형
- 효소: 불규칙한 큰 구형 덩어리
  - 기본 형태: SphereGeometry(r=2.0, 16, 16), 약간 변형(각 vertex를 noise로 미세 변형)
  - 색상: 0x4488cc (파랑), opacity 0.7
  - 활성 부위: 표면에 오목한 홈(V자 또는 반구형 홈)
    - 홈 색상: 0x44ccaa (밝은 청록), emissive 0x113333
    - 홈 형태: 기질과 딱 맞는 모양 (자물쇠 구멍 역할)
  - 라벨: "효소" + 활성 부위에 "활성 부위" 화살표 라벨

### 기질 3D 모형
- 기질: 활성 부위와 정확히 맞는 형태 (열쇠 역할)
  - 형태: 활성 부위 홈의 역상(볼록한 돌출부)
  - 색상: 0xff8844 (주황), emissive 0.2
  - 크기: 활성 부위에 딱 맞는 크기
  - 라벨: "기질"

### 기질-효소 결합 애니메이션
1. 기질이 효소를 향해 접근 (3초, 곡선 경로)
2. 기질이 활성 부위에 정확히 맞아들어감 (자물쇠-열쇠)
   - 결합 순간: 초록 글로우 + "결합!" 라벨
   - 활성 부위 색상이 밝아짐
3. 화학 반응 진행: 기질 내부에서 빛 효과 (PointLight 깜빡임, 1초)
4. 생성물 방출: 기질이 2개의 작은 조각(생성물 A, B)으로 분리
   - 생성물 A: SphereGeometry(r=0.4), 0xffcc44 (노랑)
   - 생성물 B: BoxGeometry(0.4, 0.4, 0.4), 0x44ff44 (초록)
   - 두 생성물이 효소에서 떨어져 나감 (1.5초)
5. 효소 복원: 효소가 원래 상태로 돌아옴 (색상/형태 복원)
   - "효소는 반응 후 변하지 않아 재사용됩니다!" 라벨
6. 새로운 기질이 접근 → 반복 가능

### 기질 특이성 시연
- "다른 기질" 버튼: 활성 부위와 맞지 않는 형태의 기질(삼각형 등)이 접근
  - 활성 부위에 맞지 않아 결합 실패 → 튕겨 나감
  - "❌ 기질 특이성: 효소는 특정 기질에만 작용합니다" 빨간 라벨

## 탭 2: 온도·pH에 따른 효소 활성

### 반응 속도-온도 그래프
- Canvas 2D 렌더링
- X축: 온도 (0°C ~ 80°C)
- Y축: 반응 속도 (상대값, 0~100%)
- 곡선 형태: 종형 곡선 (정규분포와 유사)
  - 0°C: 활성 낮음 (~10%)
  - 최적 온도(~37°C): 활성 최대 (100%)
  - 40°C 이상: 활성 급격히 감소 (효소 변성)
  - 60°C 이상: 활성 거의 0 (비가역적 변성)
- 최적 온도 표시: 빨간 점선 + "최적 온도 ~37°C" 라벨
- 변성 영역 표시: 40°C 이상 빗금 영역 + "효소 변성(입체 구조 변화)" 라벨

### 반응 속도-pH 그래프
- X축: pH (0 ~ 14)
- Y축: 반응 속도 (0~100%)
- 효소 종류별 다른 곡선:
  - 펩신(위): 최적 pH ~2 (강산성)
  - 아밀레이스(침): 최적 pH ~7 (중성)
  - 트립신(소장): 최적 pH ~8 (약염기성)
- 3개 곡선을 색상 다르게 동시 표시
- 각 최적 pH에 수직 점선 + 효소명 라벨

### 온도 슬라이더와 3D 연동
- 온도 슬라이더를 조절하면:
  - 그래프 위에 현재 온도 위치 빨간 수직선 표시
  - 탭 1의 3D 효소에도 영향:
    - 최적 온도: 효소 정상 형태, 활발한 반응
    - 고온: 효소가 변형(vertex 노이즈 증가, 활성 부위가 뒤틀림) → 기질이 결합 못함
    - 변성 상태에서 기질 접근 시: 맞지 않아 결합 실패 시연

## 인터랙션 (GUI 패널)
- "보기 모드" 드롭다운: 효소-기질 결합 (3D) / 온도·pH 그래프
- 탭 1:
  - "반응 재생" 버튼: 기질 접근→결합→반응→생성물 방출 애니메이션
  - "다른 기질 시도" 버튼: 맞지 않는 기질 접근 → 실패
  - "반응 속도" 슬라이더: 0.5x ~ 3x
  - "효소 투명도" 슬라이더: 0.3 ~ 1.0 (내부 활성 부위 관찰)
- 탭 2:
  - "온도" 슬라이더: 0 ~ 80 (기본값 37), 단위 °C
  - "pH" 슬라이더: 0 ~ 14 (기본값 7)
  - "효소 선택" 드롭다운: 펩신 / 아밀레이스 / 트립신
  - "변성 시각화" 토글: ON/OFF — 고온 시 효소 변형 3D 표시
- "초기화" 버튼

## 수식 패널
```
🧫 물질대사와 효소
━━━━━━━━━━━━━━━
선택 효소:     [이름]
온도:          [값]°C
pH:            [값]
현재 활성:     [값]%

효소 작용 원리 (자물쇠-열쇠 모형):
  기질 + 효소 → [효소-기질 복합체] → 생성물 + 효소
  
  • 기질 특이성: 특정 기질에만 작용
  • 효소 재사용: 반응 후 효소 변하지 않음
  • 생체 촉매: 반응 활성화 에너지↓ → 반응 속도↑

최적 조건:
  온도: ~37°C (인체 효소 기준)
  pH: 효소마다 다름 (펩신 pH2, 아밀레이스 pH7)
  → 최적 조건에서 활성 최대
  → 고온/극단 pH → 효소 변성(입체 구조 파괴)
```

## 교육적 요소
- 핵심 원리: "효소는 생체 촉매로, 기질의 활성 부위에 자물쇠-열쇠처럼 결합하여 화학 반응을 촉진합니다. 온도나 pH가 최적 조건에서 벗어나면 효소의 입체 구조가 변하여 활성이 감소합니다."
- 탐구 질문:
  1. "효소가 특정 기질에만 작용하는 까닭은?" (답: 활성 부위의 형태가 특정 기질과만 맞음)
  2. "달걀을 삶으면 흰자가 굳는 것은 어떤 현상과 관련이 있나요?" (답: 단백질 변성)
  3. "위에서 분비되는 펩신의 최적 pH가 2인 까닭은?" (답: 위산이 강산성)
- 교과서 연계: "미래엔 통합과학1 III편 p144~148 — 물질대사, 효소의 작용 원리"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 한국어 주석, 반응형, 모바일 터치 지원
- 효소 변형: SphereGeometry vertex를 noise로 변형 (BufferAttribute 직접 수정)
- 기질 이동: CubicBezierCurve3 경로 따라 이동
- 결합 판정: 기질 형태와 활성 부위 형태 비교 (하드코딩)
- 그래프: Canvas 2D API, 종형 곡선 = gaussian(x, μ, σ)
- 변성 효과: vertex noise amplitude를 온도에 비례하여 증가
- 파일명: metabolism_enzyme_sim.html
