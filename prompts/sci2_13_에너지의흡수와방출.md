너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 2. 우리 주변의 변화 > 에너지의 흡수와 방출
- 토픽: 에너지의 흡수와 방출
- 대상: 고등학교 1학년
- 핵심 개념: 우리 주변에서 물질이 변화할 때 에너지를 흡수하거나 방출하는 현상을 볼 수 있다. 물이 수증기로 증발하거나, 수산화 바륨과 염화 암모늄이 반응할 때 주변으로부터 에너지를 흡수해 주변의 온도가 낮아진다. 또 수증기가 물로 변하거나, 염화 칼슘이 물에 녹을 때 주변으로 에너지를 방출해 주변의 온도가 높아진다.

## 씬 구성
- 화면을 좌(에너지 흡수)/우(에너지 방출)로 분할하여 비교한다.
- 각 영역에 3D 실험 장치 + 열화상 카메라 뷰(온도 색상 변화)
- 교과서 그림 I-29 (p55): 에너지를 흡수하거나 방출하는 현상
- 배경: 0xf5f0e8
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.6)

## 에너지 흡수 현상 (좌측)
### 현상 1: 수산화 바륨 + 염화 암모늄 (교과서 p54)
- 3D: 비커(CylinderGeometry 투명) + 수산화 바륨 분말(흰색 파티클) + 염화 암모늄 분말(흰색 파티클)
- 반응 시: 온도 하강 표시 (디지털 온도계 25°C → 16°C)
- 비커 주변에 서리/물방울 효과 (파란 파티클)
- 열화상 색상: 따뜻한 색(주황) → 차가운 색(파랑)으로 변화

### 현상 2: 얼음이 녹음
- 3D: 컵에 얼음 조각(BoxGeometry, 반투명 0xaaddff) + 물
- 얼음이 녹으면서 주변 온도 하강
- 얼음 크기 서서히 감소 애니메이션

### 현상 3: 알코올 증발 (손소독제)
- 3D: 손 모형 위에 투명 액체 방울
- 액체가 증발(파티클 위로 퍼짐)하면서 주변 온도 하강
- "기화열 흡수" 라벨

### 공통 특징
- "주변으로부터 에너지를 흡수 → 주변 온도 낮아짐" 텍스트
- 에너지 화살표: 주변 → 반응 방향 (파란색 화살표)

## 에너지 방출 현상 (우측)
### 현상 1: 염화 칼슘 + 물 (교과서 p54)
- 3D: 비커 + 염화 칼슘 알갱이(0xdddddd) + 물
- 반응 시: 온도 상승 표시 (25°C → 35°C)
- 비커 주변에 열기 효과 (빨간 파티클 상승)
- 열화상 색상: 차가운 색(파랑) → 따뜻한 색(빨강)

### 현상 2: 연료 연소 (뷰테인 가스)
- 3D: 가스 버너 + 불꽃 파티클(0xff4400)
- 열에너지 방출, 주변 온도 상승

### 현상 3: 철이 녹슬면서 에너지 방출 (손난로)
- 3D: 손난로 패키지(BoxGeometry) + 내부에 철 가루 + 산소
- 흔들면 발열 반응 시작
- "산화 반응으로 열에너지 방출" 라벨

### 공통 특징
- "주변으로 에너지를 방출 → 주변 온도 높아짐" 텍스트
- 에너지 화살표: 반응 → 주변 방향 (빨간색 화살표)

## 실생활 이용 (교과서 그림 I-30, p56)
- 냉찜질 팩: 에너지 흡수 이용 (질산 암모늄이 물에 녹으면서 에너지 흡수)
- 손난로: 에너지 방출 이용 (철 가루가 산소와 반응하면서 에너지 방출)
- 3D 모형으로 나란히 비교

## 자연에서의 에너지 흡수/방출 (교과서 그림 I-31, p57)
- 물 증발 → 수증기 → 구름: 에너지 흡수 후 방출
- 식물의 광합성: 에너지 흡수
- 생명체의 세포호흡: 에너지 방출

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기 모드" 드롭다운: 흡수 vs 방출 비교 / 흡수 현상만 / 방출 현상만 / 실생활 이용 / 자연 현상
- "흡수 예시" 드롭다운: 수산화바륨+염화암모늄 / 얼음 녹음 / 알코올 증발
- "방출 예시" 드롭다운: 염화칼슘+물 / 연료 연소 / 손난로
- "반응 시작" 버튼
- "열화상 뷰" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌡️ 에너지의 흡수와 방출
━━━━━━━━━━━━━━━━
현재 보기:    [비교 / 흡수 / 방출]
현재 예시:    [예시명]
온도 변화:    [N]°C → [N]°C

에너지 흡수:
  주변으로부터 에너지를 흡수
  → 주변 온도 낮아짐
  예: 증발, 광합성 등

에너지 방출:
  주변으로 에너지를 방출
  → 주변 온도 높아짐
  예: 연소, 세포호흡 등
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "우리 주변에서 물질이 변화할 때 에너지를 흡수하거나 방출하는 현상을 볼 수 있습니다. 에너지를 흡수하면 주변 온도가 낮아지고, 에너지를 방출하면 주변 온도가 높아집니다. 이러한 현상은 일상생활이나 자연에서 다양하게 일어나며 그 과정에서 출입하는 에너지를 이용할 수 있습니다."
- 탐구 질문:
  1. "냉찜질 팩은 에너지를 흡수하는 현상을 이용한 것입니다. 그 까닭을 설명해 보세요."
  2. "식물이 광합성을 할 때 에너지를 흡수하고, 생명체의 세포호흡이 일어날 때 에너지를 방출합니다. (O, X)"
  3. "냉장고 없이 음료수나 신선 식품을 차갑게 보관하려면 에너지를 흡수하거나 방출하는 현상 중에서 어떤 것을 이용해야 할까요?"
- 실생활 예시: 냉찜질팩, 손난로, 쿨링 포그(도심 버스 정류장), 자가 발열 도시락
- 교과서 연계: "미래엔 통합과학2 I편 p54~59 — 에너지의 흡수와 방출, 그림 I-29~31"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 보기 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_13_energy_absorption_release.html
