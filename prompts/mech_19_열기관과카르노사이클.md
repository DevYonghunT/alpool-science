너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: II. 열과 에너지 > 04. 열기관과 열효율
- 토픽: 열기관과 카르노 사이클
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 열기관은 고온 열원에서 열(Q_H)을 흡수하여 일(W)을 하고, 저온 열원으로 폐열(Q_L)을 방출하는 장치이다. 열효율 η = W/Q_H = 1 - Q_L/Q_H. 카르노 사이클은 이상적인 열기관 사이클로 등온 팽창→단열 팽창→등온 압축→단열 압축의 4단계로 구성된다. 카르노 효율 η_C = 1 - T_L/T_H로 고온과 저온의 온도 차이가 클수록 효율이 높다. 카르노 효율은 해당 온도 사이에서 달성 가능한 최대 효율이다.

## 씬 구성
- 화면 좌측 50%: 3D 열기관 에너지 흐름 다이어그램
- 화면 우측 50%: P-V 다이어그램에 카르노 사이클 (Canvas 2D)
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 3D 열기관 에너지 흐름

### 에너지 흐름 블록 다이어그램
- 고온 열원(T_H): BoxGeometry(3, 1, 1), 0xff4444 (빨강), "T_H = [값]K" 라벨, 상단
- 열기관: BoxGeometry(3, 2, 1.5), 0x888888 (회색, 기어 무늬), "열기관" 라벨, 중앙
  - 내부에 피스톤/기어 회전 애니메이션
- 저온 열원(T_L): BoxGeometry(3, 1, 1), 0x4444ff (파랑), "T_L = [값]K" 라벨, 하단

### 에너지 화살표 (파티클 흐름)
- Q_H: 고온 열원 → 열기관 (빨간 파티클 하향, 큰 화살표)
  - "Q_H = [값]J" 라벨
- W: 열기관 → 외부 (초록 파티클 옆으로, 일 출력)
  - "W = [값]J" 라벨
  - 바퀴/발전기 회전으로 시각화
- Q_L: 열기관 → 저온 열원 (파란 파티클 하향, 폐열)
  - "Q_L = [값]J" 라벨
- Q_H = W + Q_L (에너지 보존)
- 화살표 두께 = 에너지 크기에 비례

### 열효율 표시
- η = W/Q_H = (Q_H - Q_L)/Q_H = 1 - Q_L/Q_H
- 효율 게이지: 원형 게이지(0~100%), 현재 효율 표시
- η < η_카르노 (항상) ← 강조

## 카르노 사이클 P-V 다이어그램 (우측)

### 4단계 사이클
1. **A→B: 등온 팽창** (T = T_H, 일정)
   - 고온 열원에서 Q_H 흡수
   - 기체 팽창, 피스톤 올라감
   - P-V: 쌍곡선 (빨간 선)
2. **B→C: 단열 팽창** (Q = 0)
   - 열 교환 없음
   - 기체 팽창, 온도 T_H → T_L 하강
   - P-V: 등온보다 급한 곡선 (초록 선)
3. **C→D: 등온 압축** (T = T_L, 일정)
   - 저온 열원으로 Q_L 방출
   - 기체 압축, 피스톤 내려감
   - P-V: 쌍곡선 (파란 선)
4. **D→A: 단열 압축** (Q = 0)
   - 열 교환 없음
   - 기체 압축, 온도 T_L → T_H 상승
   - P-V: 급한 곡선 (보라 선)

### P-V 그래프
- 4개 곡선이 닫힌 사이클 형성
- **사이클 내부 면적 = 순 일 W = Q_H - Q_L**
- 면적을 노란 음영으로 표시
- 각 꼭짓점 A, B, C, D에 라벨 + (P, V, T) 값
- 현재 위치에 빨간 마커 (애니메이션 진행에 따라)

### 카르노 효율
- η_C = 1 - T_L/T_H
- T_H, T_L 슬라이더 → 효율 실시간 변화
- T_L/T_H 비가 작을수록(온도 차이 클수록) 효율 높음
- T_L = 0K (절대 0도): η = 100% (이론적 최대, 실현 불가)
- T_L = T_H: η = 0% (온도 차이 없으면 일 불가)

## 실제 열기관 효율 비교
| 열기관 | 실제 효율 | 카르노 효율 |
|--------|---------|----------|
| 증기 기관 | ~10% | ~40% |
| 가솔린 엔진 | ~25% | ~55% |
| 디젤 엔진 | ~35% | ~65% |
| 화력 발전소 | ~40% | ~60% |
- "실제 효율 < 카르노 효율 (항상)" 강조

## 냉장고/에어컨 (역카르노)
- 열기관의 역과정: 일(W)을 투입하여 저온→고온으로 열 이동
- 냉장고: 내부(저온)에서 열 흡수 → 외부(고온)로 방출
- 성능 계수(COP) = Q_L/W
- 간단한 3D 블록 다이어그램으로 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "보기" 드롭다운: 열기관 에너지 흐름 / 카르노 사이클 P-V / 냉장고(역카르노)
- "고온 T_H" 슬라이더: 400~1200K (기본값 600)
- "저온 T_L" 슬라이더: 200~500K (기본값 300)
- "Q_H" 슬라이더: 100~1000J (기본값 500)
- "사이클 재생" 버튼: 카르노 사이클 4단계 순서대로 애니메이션
- "실제 효율 비교" 토글: ON/OFF
- "에너지 흐름 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔧 열기관과 카르노 사이클
━━━━━━━━━━━━━━━━
T_H = [값]K, T_L = [값]K
Q_H = [값]J

열효율:
  η = W/Q_H = 1 - Q_L/Q_H
  W = Q_H - Q_L = [값]J
  Q_L = [값]J
  η = [값]%

카르노 효율 (최대):
  η_C = 1 - T_L/T_H
  = 1 - [값]/[값] = [값]%

카르노 사이클:
  등온팽창 → 단열팽창
  → 등온압축 → 단열압축
  P-V 내부 면적 = W
```
