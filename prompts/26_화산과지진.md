너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 1. 지구시스템 > 지권의 변화와 판 구조론
- 토픽: 화산과 지진
- 대상: 고등학교 1학년
- 핵심 개념: 지진과 화산 활동은 지구 내부 에너지에 의해 일어나는 격렬한 지각 변동이다. 화산 활동 시 화산 쇄설물(화산암괴, 화산력, 화산재), 화산 가스, 용암이 분출된다. 화산재는 햇빛을 가려 기후를 변화시키고, 화산 가스는 산성비를 내리게 한다. 화산 분출물이 풍화를 받으면 비옥한 토양을 만들며, 화산 지대는 온천이나 독특한 지형을 형성한다. 지진파를 분석하면 지구 내부 구조와 구성 물질에 대한 정보를 얻을 수 있다.

## 씬 구성
- 2탭 구성: 화산 분출 시뮬레이션 / 지진파 전파 시뮬레이션
- 배경: 0x1a1a2a (짙은 남색)
- 조명: 탭별로 다르게 설정 (아래 상세)
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls

## 탭 1: 화산 분출 시뮬레이션

### 3D 화산 모형 상세
- 화산체: ConeGeometry(radiusBottom=4, radiusTop=1.5, height=5, radialSegments=32)
  - 재질: MeshStandardMaterial(0x664433 짙은 갈색, roughness 0.9, bumpScale 0.3)
  - 화산 표면 질감: 절차적 범프맵 또는 noise 기반 vertex displacement로 거친 암석 표현
  - 화산 측면 균열: LineGeometry 3~5줄, 0xff4400 주황, 용암 누출 경로 표현
- 분화구: CylinderGeometry(r=1.0, h=0.5, radialSegments=32, openEnded=true)
  - 분화구 내부: MeshBasicMaterial(0xff4400 주황-빨강, emissive 0xff2200)
  - 분화구 테두리: TorusGeometry(r=1.0, tube=0.1), 0x554433 (암석 갈색)
  - 용암 풀: CircleGeometry(r=0.8) 분화구 바닥에 배치, ShaderMaterial로 용암 유동 패턴 (noise + 시간 오프셋, 0xff4400↔0xffcc00 사이 변동)
- 마그마 관: CylinderGeometry(r=0.3, h=5), 화산 내부 중심축
  - MeshBasicMaterial(0xff3300, opacity 0.25, transparent true)
  - 마그마 상승 파티클: Points(30개, 0xffaa00), 아래→위 이동, 분출 전조 시 속도 증가
- 주변 지형:
  - PlaneGeometry(25, 25, 50, 50), vertex displacement로 완만한 언덕 표현
  - 재질: MeshStandardMaterial(0x447744 풀밭 녹색, roughness 0.8)
  - 화산 기슭에서 점진적으로 평탄해지는 경사
  - 나무 5~8그루: CylinderGeometry(줄기 r=0.08, h=0.8, 0x6B4226) + SphereGeometry(잎 r=0.3, 0x228b22), 화산 기슭~평지에 분산 배치
  - 건물 2~3채: BoxGeometry(0.5, 0.4, 0.3), 0xddccbb (베이지), 평지에 배치
- 하늘: 반구 SphereGeometry(r=30, 내부면), 그라데이션 텍스처 (0x87CEEB 하늘색 → 0x334466 지평선)
- 조명: AmbientLight(0.35) + DirectionalLight(0.6, 좌상단) + PointLight(분화구 내부, 0xff4400, 세기 2.0, 깜빡임)
- 그림자: DirectionalLight에서 활성화

### 분출 애니메이션 상세 ("분출 시작" 버튼)

#### 1단계: 전조 현상 (0~2초)
- 화산체 미세 진동: mesh.position에 Math.random() × 0.02 오프셋 (매 프레임)
- 분화구에서 연기 파티클: Points(50개, 0x666666 회색), 분화구 중심에서 위로 천천히 상승
  - 파티클 크기: 0.1 → 0.3 (올라가면서 확대)
  - 파티클 opacity: 0.6 → 0.0 (올라가면서 투명)
  - 높이: 0 → 8 (4초에 걸쳐)
- 마그마 관 내부 파티클 속도 2배 증가
- PointLight(분화구) 세기: 2.0 → 4.0 (깜빡임 빈도 증가)
- "⚠️ 전조 현상: 지진 감지, 화산 가스 분출" 라벨

#### 2단계: 폭발적 분출 (2~4초)
- 용암 파티클: Points(300개, BufferGeometry)
  - 색상: 0xff4400(빨강) ~ 0xffcc00(노랑) 랜덤
  - 초기 위치: 분화구 중심 (x=0, y=5, z=0) ± 반지름 0.5 랜덤
  - 초기 속도: Y방향 8~15 (위로 강하게), X/Z방향 -3~3 (옆으로 퍼짐)
  - 중력: y_velocity -= 9.8 × dt (포물선 궤적)
  - 크기: 0.05~0.2 랜덤
  - 수명: 3~5초 후 제거
- 화산재 파티클: Points(500개, BufferGeometry)
  - 색상: 0x444444 ~ 0x888888 (어두운 회색)
  - 초기 속도: Y방향 5~10, X/Z방향 -5~5 (넓게 퍼짐)
  - 올라간 후 수평으로 확산 (Y속도 감소 후 X/Z 속도 유지)
  - 크기: 0.02~0.08
  - 올라간 높이에서 수평 구름층 형성 (Y > 10에서 Y속도 0으로 감쇠)
- 화산탄: SphereGeometry(r=0.08~0.15) × 10개
  - MeshStandardMaterial(0xff2200, emissive 0x661100)
  - 포물선 궤적: 위로 솟았다가 주변에 착지
  - 착지 시: 작은 폭발 효과 (PointLight 순간 세기 3.0, 0.1초)
- 화산 가스: Points(150개, 0xaaaa88)
  - opacity 0.3, 빠르게 위로 퍼짐
- 폭발 순간 효과:
  - PointLight(분화구) 세기: 4.0 → 10.0 (0.2초) → 5.0
  - 카메라 흔들림: camera.position에 Math.random() × 0.05 오프셋 (1초)
  - 폭발음 대체: 화면 테두리가 빨갛게 깜빡 (CSS box-shadow 0 0 30px red, 0.5초)

#### 3단계: 용암 흐름 (4~8초)
- 용암 파티클: Points(150개, 0xff4400)
  - 분화구에서 화산 경사면을 따라 아래로 흘러내림
  - 경사면 경로: CatmullRomCurve3(분화구 → 기슭 → 평지), 3~4개 경로
  - 용암이 지나간 자리: 나무 mesh를 검은색(0x222222)으로 변경 (소실 표현)
  - 건물에 도달하면: 건물 mesh를 반투명 검정으로 변경 (파괴 표현)
- 교과서: "지표로 분출된 용암은 주변의 건물과 숲을 뒤덮어 파괴한다" (p117 그림 III-13)

#### 4단계: 기후 영향 (6~10초)
- 화산재가 상공에 쌓이면:
  - DirectionalLight 세기: 0.6 → 0.15 (하늘이 어두워짐, 4초에 걸쳐)
  - 하늘 색상: 0x87CEEB → 0x555566 (칙칙해짐)
  - "화산재가 햇빛을 가려 기후를 변화시킨다" 라벨 (교과서 p117)
- 긍정적 효과 표시 (토글):
  - "화산 분출물이 풍화 → 비옥한 토양" (바닥 일부가 짙은 갈색 → 진한 녹색으로 변화)
  - "화산 지대 → 온천 형성" (화산 기슭에 작은 파란 원 + 수증기 파티클)

### VEI(화산 폭발 지수) 슬라이더에 따른 차이
| VEI | 분출물 양 | 용암 파티클 | 화산재 파티클 | 기후 영향 | 높이 |
|-----|----------|-----------|-------------|----------|------|
| 1 | 소량 | 50개 | 100개 | 없음 | ~2km |
| 3 | 중간 | 200개 | 300개 | 약간 어두워짐 | ~8km |
| 5 | 대량 | 400개 | 600개 | 매우 어두워짐 | ~25km(성층권) |

## 탭 2: 지진파 전파 시뮬레이션

### 3D 지구 단면 모형
- 지구: SphereGeometry(r=5.0, 64, 64), ClippingPlane으로 반구 절단
- 내부 층 (동심원, 교과서 p106 기반):
  | 층 | 반지름 | 색상 | 상태 |
  |---|--------|------|------|
  | 지각 | r=5.0 외부 | 0x886644 | 고체 |
  | 맨틀 | r=4.85 | 0xcc5533 | 고체(유동성) |
  | 외핵 | r=2.7 | 0xff6622 (emissive 0x441100) | **액체** |
  | 내핵 | r=1.3 | 0xffaa44 (emissive 0x663300, metalness 0.8) | 고체 |
- 각 층 경계에 흰색 점선 원(LineDashedMaterial)
- 각 층 라벨 + 상태(고체/액체)
- 진원: SphereGeometry(r=0.12, 16, 16), MeshBasicMaterial(0xff0000, emissive 0xff0000)
  - 위치: 지각 내부(r=4.7, 위도 약 30° 위치)
  - 깜빡임 애니메이션: opacity 0.5↔1.0 (0.5초 주기)
  - "진원" 라벨
- 진앙: 진원 바로 위 지표면에 별표 마커(✱), 0xff4444
  - "진앙" 라벨
- 배경: 0x000011 (우주)
- 조명: AmbientLight(0.2) + PointLight(내핵, 0xff4400, 세기 1.5)

### 지진파 전파 애니메이션 ("지진 발생" 버튼)

#### P파 (종파, Primary wave)
- 표현: RingGeometry(innerR, outerR=innerR+0.1), MeshBasicMaterial(0x4488ff 파랑, opacity 0.4)
- 진원에서 동심원으로 확장: innerRadius 0 → 10 (3초에 걸쳐)
- 확장하면서 opacity 0.4 → 0.0 (페이드아웃)
- 동시에 3~4개 파동이 0.5초 간격으로 생성 (연속 파동 느낌)
- **외핵(액체)을 통과!**: 파동이 외핵 경계를 지나도 계속 진행
- 속도: 빠름 (전체 지구를 3초에 통과)
- 라벨: "P파 (종파) — 빠름, 고체+액체 모두 통과"

#### S파 (횡파, Secondary wave)
- 표현: RingGeometry 동일 구조, MeshBasicMaterial(0xff6644 주황, opacity 0.4)
- 진원에서 동심원으로 확장: P파보다 느리게 (5초에 걸쳐)
- **외핵(액체)에서 차단!**: 파동이 외핵 경계(r=2.7)에 도달하면 사라짐
  - 차단 순간: 빨간 플래시 + "✗ 차단!" 라벨
  - 외핵 경계가 잠시 빨갛게 강조 (emissive 0xff0000, 1초)
- S파 암영대 시각화:
  - 진원 반대편의 부채꼴 영역을 반투명 회색(0x888888, opacity 0.15)으로 채움
  - "S파 암영대 — S파가 도달하지 못하는 영역" 라벨
- 라벨: "S파 (횡파) — 느림, 고체만 통과, 액체 ✗"

#### P파와 S파 비교 표 (하단 HTML overlay)
```
📊 지진파 비교
━━━━━━━━━━━━━━━━━━━━━━
        P파 (종파)     S파 (횡파)
속도     빠름           느림
통과     고체+액체       고체만
외핵     통과 ✓         차단 ✗
결론: S파가 외핵에서 차단 → 외핵은 액체!
```

#### 지진 규모 슬라이더 (리히터 1~9)
- 규모에 따라:
  - 파동 진폭(Ring 두께): 규모 1 → 0.05, 규모 9 → 0.3
  - 지표면 흔들림: 카메라 진동 진폭 (규모 1: 0.01, 규모 9: 0.1)
  - 파동 수: 규모 1 → 2개, 규모 9 → 8개

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "실험 탭" 드롭다운: 화산 분출 / 지진파 전파
- 화산 탭:
  - "분출 시작" 버튼: 4단계 분출 애니메이션 재생
  - "분출 강도(VEI)" 슬라이더: 1 ~ 5 (기본값 3), 정수 단계
  - "분출 단계" 슬라이더: 0~3 (전조/폭발/용암/기후), 특정 단계만 관찰
  - "긍정적 효과 표시" 토글: ON/OFF (기본값 OFF) — 비옥한 토양, 온천
  - "기후 영향 표시" 토글: ON/OFF (기본값 ON)
  - "슬로모션" 슬라이더: 0.1x ~ 3x (기본값 0.5x)
- 지진 탭:
  - "지진 발생" 버튼: P파+S파 전파 애니메이션 시작
  - "지진 규모" 슬라이더: 1 ~ 9 (기본값 5), 정수 단계
  - "P파 표시" 토글: ON/OFF (기본값 ON)
  - "S파 표시" 토글: ON/OFF (기본값 ON)
  - "S파 암영대 표시" 토글: ON/OFF (기본값 OFF)
  - "내부 층 라벨" 토글: ON/OFF (기본값 ON)
  - "슬로모션" 슬라이더: 0.1x ~ 3x (기본값 1x)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널(배경 rgba(0,0,0,0.7), 둥근 모서리 12px, padding 20px)로 표시:

```
🌋 화산과 지진
━━━━━━━━━━━━

[화산 탭]
분출 강도(VEI): [값]
분출 단계:      [전조/폭발/용암/기후]
분출물:
  용암 — 고온 액체 암석, 경사면 따라 흐름
  화산재 — 미세 입자, 기권으로 확산, 기후 영향
  화산 가스 — SO₂, CO₂ 등, 산성비 원인
  화산탄 — 큰 암석 조각, 포물선 궤적 낙하

피해: 건물·숲 파괴, 기후 변화, 산성비
이점: 비옥한 토양, 온천, 관광 자원

[지진 탭]
지진 규모: [값]
P파 속도: ~6~8 km/s (맨틀 기준)
S파 속도: ~3.5~4.5 km/s

P파 (종파): 고체+액체 통과
S파 (횡파): 고체만 통과
→ S파가 외핵에서 차단
→ 외핵이 액체임을 알 수 있다!
```
모든 값 실시간 업데이트.

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "지진과 화산 활동은 지구 내부 에너지에 의해 일어납니다. 화산 분출물은 기후를 변화시키기도 하고 비옥한 토양을 만들기도 합니다. 지진파 분석으로 지구 내부 구조를 파악할 수 있습니다."
- 탐구 질문:
  1. "화산 활동이 지구시스템에 미치는 긍정적 영향과 부정적 영향을 각각 설명해 보세요."

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원), 모바일에서 lil-gui 기본 접힘 상태
- 파티클 시스템: BufferGeometry + Points, position/velocity attribute로 물리 시뮬레이션
  - 용암/화산재/화산가스 각각 별도 BufferGeometry (독립 제어)
  - 파티클 수명 관리: age > maxAge이면 초기 위치로 리셋 (오브젝트 풀링)
- 지진파: RingGeometry, 매 프레임 innerRadius 증가 + opacity 감소, geometry dispose() 후 재생성 또는 scale 변경
- 화산 진동: mesh.position.x/y/z에 Math.random() 오프셋 (진동 강도 GUI 연동)
- 카메라 흔들림: camera.position에 오프셋 추가 후 원래 위치로 lerp 복원
- ClippingPlane (지진 탭): renderer.localClippingEnabled = true
- 탭 전환 시 이전 씬 dispose() + 새 씬 구성
- 성능: requestAnimationFrame 루프, 파티클 총 최대 800개 (VEI 5 기준), PointsMaterial.size 속성으로 파티클 크기 제어
- 파일명: volcano_earthquake_sim.html
