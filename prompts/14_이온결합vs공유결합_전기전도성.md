너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: II. 물질과 규칙성 > 1. 원소의 생성과 규칙성 > 이온 결합과 공유 결합
- 토픽: 이온 결합 vs 공유 결합 전기 전도성
- 대상: 고등학교 1학년
- 핵심 개념: 이온 결합 물질은 고체 상태에서 전류가 흐르지 않지만, 수용액에서는 양이온과 음이온이 자유롭게 이동하여 전류가 흐른다. 공유 결합 물질은 수용액에서도 이온으로 나눠지지 않으므로 대부분 전류가 흐르지 않는다.

## 씬 구성
- 화면에 4개 실험 비커를 나란히 배치한다 (교과서 탐구 p72).
- 각 비커에 물질을 넣고 전기 전도성을 측정하는 회로를 연결한다.
- 배경: 0xf0f0f0 (밝은 실험실)
- 바닥: PlaneGeometry(15, 10), 0xdddddd (실험대)
- 조명: AmbientLight(0.5) + DirectionalLight(0.6, 위에서) + 그림자 활성화
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls

## 4개 실험 비커 상세

### 비커 구조 (공통)
- 비커: CylinderGeometry(r상=1.0, r하=0.8, h=2.0, 32, 1, true), MeshPhysicalMaterial(transmission 0.85, 0xccddee 유리)
- 전극 2개: BoxGeometry(0.05, 1.5, 0.3), 0x888888 (금속 회색), 비커 양쪽에 삽입
  - 좌측 전극: "(−)극" 라벨
  - 우측 전극: "(+)극" 라벨
- 전선: TubeGeometry로 전극에서 전구까지 연결
- 전구: SphereGeometry(r=0.2) + 필라멘트(가는 CylinderGeometry 꼬인 형태)
  - 켜진 상태: emissive 0xffee44, PointLight(세기 2.0)
  - 꺼진 상태: 0x444444, PointLight 없음
- 전지: BoxGeometry(0.5, 0.3, 0.2), 0xdd4444(+극) + 0x4444dd(−극)

### 비커 1: 고체 염화 나트륨 (NaCl)
- 비커 내부: 흰색 입방체 결정 여러 개(BoxGeometry 0.1×0.1×0.1, 0xffffff) 쌓여있음
- 이온 상태: Na⁺(빨간 작은 구)와 Cl⁻(초록 작은 구)가 격자에 고정되어 움직이지 않음
- 전구: **꺼짐** ❌
- 라벨: "고체 NaCl — 이온이 고정 → 전류 ✗"
- 교과서: "이온 결합 물질은 고체 상태에서 양이온과 음이온이 강하게 결합하고 있으므로 전류가 흐르지 않는다" (p73)

### 비커 2: 염화 나트륨 수용액 (NaCl 수용액)
- 비커 내부: 물(BoxGeometry, 0x4488cc, opacity 0.4) 채움
- 이온 애니메이션 (교과서 그림 II-25, p73):
  - Na⁺(빨간 구 r=0.06, "+" 라벨) 여러 개: (−)극 방향으로 이동
  - Cl⁻(초록 구 r=0.08, "−" 라벨) 여러 개: (+)극 방향으로 이동
  - 이온들이 전극 사이를 왔다갔다하며 이동 (sin 기반 왕복, 속도 다양)
  - 이동 경로를 연한 점선 화살표로 표시
- 전구: **켜짐** ✅ (밝게 발광)
- 전류 화살표: 전선 위에 전류 방향 화살표 파티클(작은 삼각형)이 흐름
- 라벨: "NaCl 수용액 — 이온이 자유롭게 이동 → 전류 ✓"

### 비커 3: 고체 설탕 (C₁₂H₂₂O₁₁)
- 비커 내부: 흰색 작은 구(SphereGeometry r=0.08, 0xffffff) 여러 개 — 설탕 결정
- 분자 고정: 움직이지 않음
- 전구: **꺼짐** ❌
- 라벨: "고체 설탕 — 분자 상태(이온 없음) → 전류 ✗"

### 비커 4: 설탕 수용액 (교과서 그림 II-26, p73)
- 비커 내부: 물(0x4488cc, opacity 0.4) + 설탕 분자(작은 흰 구 r=0.06) 여러 개
- 분자 상태: 설탕 분자가 물에 녹았지만 이온으로 분리되지 않음
  - 분자들이 브라운 운동(랜덤 미세 이동)하지만 전극 방향으로 이동하지 않음
- 전구: **꺼짐** ❌
- 라벨: "설탕 수용액 — 이온으로 분리 안 됨 → 전류 ✗"
- 교과서: "공유 결합 물질은 이온 결합 물질과 달리 물에 녹아도 대부분 이온으로 나눠지지 않는다" (p73)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "전원 연결" 토글: ON/OFF (기본값 OFF)
  - OFF: 전지 미연결, 모든 전구 꺼짐, 이온 이동 없음
  - ON: 전지 연결, 전류 흐르는 비커만 전구 켜짐 + 이온 이동 시작
- "물에 녹이기" 버튼:
  - 비커 1(고체 NaCl) → 비커 2(NaCl 수용액)로 전환 애니메이션
    - 물이 비커에 채워지면서 NaCl 결정이 분해 → Na⁺와 Cl⁻ 이온으로 분리
  - 비커 3(고체 설탕) → 비커 4(설탕 수용액)로 전환 애니메이션
    - 물이 채워지면서 설탕 결정이 녹지만 분자 형태 유지 (이온 분리 없음)
- "이온 속도" 슬라이더: 0x ~ 5x (기본값 1x) — NaCl 수용액 이온 이동 속도
- "이온 라벨" 토글: ON/OFF (기본값 ON) — Na⁺, Cl⁻ 라벨 표시
- "증류수 전도성" 체크박스: ON 시 5번째 비커 추가 (순수 증류수 — 전류 안 흐름)
- "초기화" 버튼

## 시각화 규칙

### 전구 밝기에 따른 전도성 표시
| 물질 | 고체 | 수용액 | 전구 |
|------|------|--------|------|
| NaCl (이온 결합) | ❌ 꺼짐 | ✅ 밝게 켜짐 | 🔴→🟢 |
| 설탕 (공유 결합) | ❌ 꺼짐 | ❌ 꺼짐 | 🔴→🔴 |

### 결론 패널 (하단 HTML overlay)
```
📊 전기 전도성 비교
━━━━━━━━━━━━━━━━━━━━━━━━━━━
             고체    수용액
이온 결합(NaCl)  ❌      ✅ ← 이온 자유 이동
공유 결합(설탕)  ❌      ❌ ← 이온 없음

결론: 이온 결합 물질의 수용액은 전류가 흐르고,
      공유 결합 물질의 수용액은 전류가 흐르지 않는다.
```

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚡ 전기 전도성과 결합
━━━━━━━━━━━━━━━━━━
전원 상태:     [연결 / 미연결]

이온 결합 물질 (NaCl):
  고체: Na⁺, Cl⁻ 고정 → 전류 ✗
  수용액: NaCl → Na⁺(aq) + Cl⁻(aq)
         이온 자유 이동 → 전류 ✓
  Na⁺ → (−)극 방향
  Cl⁻ → (+)극 방향

공유 결합 물질 (설탕 C₁₂H₂₂O₁₁):
  고체: 분자 고정 → 전류 ✗
  수용액: 분자 유지(이온 분리 ✗) → 전류 ✗
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "이온 결합 물질은 수용액에서 양이온과 음이온으로 나뉘어 자유롭게 이동하므로 전류가 흐릅니다. 공유 결합 물질은 수용액에서도 이온으로 분리되지 않으므로 전류가 흐르지 않습니다."
- 탐구 질문:
  1. "고체 NaCl에 전류가 흐르지 않는 까닭은 무엇인가요?"
  2. "NaCl 수용액에서 Na⁺은 어느 극으로 이동하나요? 그 까닭은?"
  3. "순수한 물(증류수)에는 전류가 흐를까요? 그 까닭은?"
- 실생활 예시: 소금물 전기분해(나트륨+염소 생성), 감전 위험(물에 젖은 손), 배터리(전해질 용액)
- 교과서 연계: "미래엔 통합과학1 II편 p72~73 — 이온 결합 물질과 공유 결합 물질의 성질, 그림 II-25(이온 결합 전도성), 그림 II-26(공유 결합 전도성)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원
- 비커 유리: MeshPhysicalMaterial(transmission)
- 이온 이동: requestAnimationFrame에서 position 업데이트 (sin 기반 왕복)
- 전구 발광: emissive + PointLight ON/OFF
- 전류 화살표: 작은 삼각형 파티클(BufferGeometry) 전선 경로 따라 이동
- 성능: 이온 최대 30개(InstancedMesh), requestAnimationFrame 루프
- 파일명: ionic_covalent_conductivity_sim.html
