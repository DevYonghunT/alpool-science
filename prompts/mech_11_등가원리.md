너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 06. 등가 원리와 시공간의 휨
- 토픽: 등가 원리 (관성력과 중력의 등가)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 아인슈타인의 등가 원리에 따르면 관성력과 중력은 구별할 수 없다. 밀폐된 엘리베이터 안에서 위로 가속하는 것과 중력이 작용하는 것은 동일한 효과를 낸다. 자유 낙하하는 엘리베이터 안에서는 무중력 상태가 된다. 등가 원리는 일반 상대성 이론의 기초가 되며, 중력에 의한 빛의 휨과 시간 지연 등을 예측한다.

## 씬 구성
- 화면 중앙: 3D 밀폐된 엘리베이터(상자) + 내부의 사람/물체
- 3가지 시나리오: 정지 중력장 / 가속하는 우주선 / 자유 낙하
- 배경: 0x222233 (어두운 배경)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.5)

## 3D 밀폐된 상자 (엘리베이터/우주선)

### 상자
- BoxGeometry(3, 4, 3), 0xddddcc 반투명 (내부가 보이게, opacity 0.3)
- 바닥: PlaneGeometry, 0x888888 (불투명)
- 벽에 "A" 또는 "B" 라벨 (두 시나리오 비교용)

### 사람 (내부)
- BoxGeometry 조합 실루엣, 0xddccbb
- 체중계 위에 서 있음: BoxGeometry(1, 0.1, 1), 0xaaaaaa
- 체중계 표시: "W = [값]N"

### 공 (내부 실험용)
- SphereGeometry(r=0.15), 0xff4444
- 사람이 손에서 공을 놓는 실험
- 공의 궤적으로 관성력/중력 구별 시도

## 시나리오 A: 지구 표면에 정지 (중력)

### 설정
- 상자가 지구 표면에 정지
- 중력 g = 9.8m/s² 아래로 작용
- 사람 체중계: W = mg

### 실험
- 공을 놓으면: 아래로 떨어짐 (자유 낙하)
- 사람: 바닥에 서 있음, 체중 느낌
- 외부 시점: 지구 위에 상자가 정지

## 시나리오 B: 무중력 공간에서 위로 가속 (관성력)

### 설정
- 상자가 우주 공간에서 위로 가속도 a = g로 가속
- 중력 없음, 대신 가속에 의한 관성력
- 사람 체중계: W = ma = mg (시나리오 A와 동일!)

### 실험
- 공을 놓으면: 상자 바닥이 위로 올라오므로 공이 아래로 떨어지는 것처럼 보임
- 사람: 바닥에 눌리는 느낌, 체중 동일
- 외부 시점: 상자가 위로 가속, 공은 제자리

### 핵심 비교
- **내부 관측자**: A와 B를 구별할 수 없다! (등가 원리)
- **외부 관측자**: A는 중력, B는 가속 → 원인이 다르지만 효과 동일
- "내부에서는 중력과 관성력을 구별할 수 없다" 강조 라벨

## 시나리오 C: 자유 낙하 (무중력)

### 설정
- 상자가 지구 중력장에서 자유 낙하 (a = g)
- 내부: 무중력 상태!
- 사람 체중계: W = 0

### 실험
- 공을 놓으면: 공이 그 자리에 떠있음 (상자와 함께 낙하)
- 사람: 둥둥 떠다님
- 외부 시점: 상자, 사람, 공 모두 같은 가속도로 낙하
- "자유 낙하 = 무중력 체험" (ISS, 무중력 비행)

## 등가 원리의 결과: 빛의 휨 (보너스)

### 시각화
- 가속하는 상자 안에서 수평으로 빛(레이저) 발사
- 외부: 빛은 직선, 상자가 올라감 → 내부에서 빛은 아래로 휜 것처럼 보임
- 등가 원리: 가속 = 중력이므로 → 중력장에서 빛도 휜다!
- 노란 레이저 빔(CylinderGeometry 가느다란)이 곡선으로 휘는 애니메이션
- "중력에 의해 빛이 휜다 → 일반 상대성 이론" 라벨

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: A. 지구 표면(중력) / B. 우주 가속(관성력) / C. 자유 낙하(무중력)
- "가속도" 슬라이더: 0~15m/s² (기본값 9.8)
- "관점" 토글: 내부 관측자 / 외부 관측자
- "공 놓기" 버튼: 공을 놓는 실험
- "빛의 휨" 토글: ON/OFF (레이저 실험)
- "체중계 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
⚖️ 등가 원리
━━━━━━━━━━━━━━
시나리오: [A/B/C]
가속도: [값]m/s²
체중계: W = [값]N

등가 원리 (아인슈타인):
  관성력과 중력은 구별할 수 없다

A. 중력장 정지: W = mg = [값]N
B. 무중력 가속: W = ma = [값]N
   → A = B (내부에서 구별 불가!)
C. 자유 낙하: W = 0 (무중력)

결과:
  중력 = 시공간의 휨
  중력장에서 빛도 휜다
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시나리오 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: mech_11_equivalence_principle.html
