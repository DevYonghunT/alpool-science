너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: II. 환경과 에너지 > 1. 생태계와 환경 > 지구 환경의 변화
- 토픽: 기후 변화와 생태계 영향
- 대상: 고등학교 1학년
- 핵심 개념: 생태계는 크고 급격한 환경 변화를 겪으면 평형이 깨질 수 있으며, 오랜 시간이 지나면 새로운 생태계로 변할 수도 있다. 알프스 산맥의 빙하가 녹으면서 생태계가 변하고, 아마존 열대우림의 파괴가 전 지구적 기후에 영향을 미치는 등 환경 변화가 생태계에 큰 영향을 준다.

## 씬 구성
- 2가지 사례를 탭으로 전환:
  - 사례 A: 알프스 빙하 후퇴와 생태계 변화
  - 사례 B: 아마존 열대우림 파괴와 기후 영향
- 각 사례에서 "과거"와 "현재/미래" 비교
- 배경: 사례별로 다름
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 사례 A: 알프스 빙하 (교과서 p80)

### 과거 (빙하 존재)
- 3D: 산(ConeGeometry 대형, 0x887766) + 빙하(SphereGeometry 상단, 0xddeeff 흰색-연파랑)
- 빙하가 산 상단부를 덮고 있는 모습
- 바닥: 회색 바위 0x999999
- 생물: 거의 없음 (황량)
- 하늘: 맑은 파랑 0x88bbee

### 현재 (빙하 후퇴)
- 빙하 크기 축소 (scale 감소 애니메이션)
- 빙하가 녹은 자리에 새로운 생태계 형성:
  - 이끼/지의류: 작은 녹색 패치 (PlaneGeometry, 0x55aa44)
  - 풀: ConeGeometry 여러 개, 0x66bb33
  - 작은 나무: CylinderGeometry + SphereGeometry
  - 곤충: 작은 파티클
- 물줄기: CylinderGeometry 파란색 0x4488cc (빙하 녹은 물)
- 라벨: "빙하가 녹은 지역에서 새로운 생태계가 형성됨"

### 시간 슬라이더
- 1900년 → 2024년 → 2100년(예측)
- 빙하 크기가 시간에 따라 줄어듦
- 생태계 다양성이 시간에 따라 증가

## 사례 B: 아마존 열대우림 (교과서 p80~81)

### 건강한 열대우림
- 3D: 울창한 나무들 (CylinderGeometry 줄기 + SphereGeometry 잎, 30~50그루)
- 다양한 크기/색상의 나무: 0x228b22, 0x338822, 0x447733
- 바닥: 진한 갈색 토양 0x553322
- 동물: 새(하늘), 원숭이(나무), 재규어(바닥)
- 비 파티클: Points 200개, 위에서 아래로
- CO₂ 흡수 화살표: 파란 화살표 하늘→나무 (광합성으로 CO₂ 흡수)
- 라벨: "열대우림은 지구의 허파 — CO₂ 흡수, O₂ 방출"

### 파괴된 열대우림
- 나무가 제거된 빈 땅 (베어진 그루터기만 남음)
- 바닥: 마른 갈색 0x886644
- 동물: 사라짐
- 불꽃 파티클: 산불 표현 (빨강-주황 Points)
- CO₂ 방출 화살표: 빨간 화살표 땅→하늘 (나무가 사라져 CO₂ 방출)
- 라벨: "열대우림 파괴 → CO₂ 흡수 능력 감소 → 기후 변화 가속"

### 파괴 비율 슬라이더
- 0%(건강)~100%(완전 파괴)
- 슬라이더에 따라 나무가 하나씩 사라지는 애니메이션
- CO₂ 흡수량 그래프 연동

## 영향 비교 그래프 (우측 패널)
- Canvas 2D:
  - 사례 A: 빙하 면적 vs 생물종 수 (반비례 관계)
  - 사례 B: 삼림 면적 vs CO₂ 흡수량 (비례 관계)

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "사례" 드롭다운: 알프스 빙하 / 아마존 열대우림
- "시간" 슬라이더 (사례 A): 1900~2100 (기본값 2024)
- "파괴 비율" 슬라이더 (사례 B): 0~100% (기본값 30%)
- "CO₂ 흐름 표시" 토글: ON/OFF
- "생물 표시" 토글: ON/OFF
- "자동 재생" 버튼
- "초기화" 버튼

화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌿 기후 변화와 생태계 영향
━━━━━━━━━━━━━━━━━
현재 사례:    [알프스/아마존]
현재 상태:    [상태 설명]

사례 A (알프스):
  빙하 면적: [N] km² (1900년 대비 [N]% 감소)
  새로운 종 수: [N]종

사례 B (아마존):
  삼림 면적: [N]% 남음
  CO₂ 흡수량: 연간 [N] 억 톤

핵심:
  급격한 환경 변화 → 생태계평형 깨짐
  → 새로운 생태계 형성 또는 생태계 파괴
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "생태계는 크고 급격한 환경 변화를 겪으면 평형이 깨질 수 있으며, 오랜 시간이 지나면 새로운 생태계로 변할 수도 있습니다. 인간의 활동에 의한 환경 변화는 전 지구적 규모로 생태계에 영향을 미칩니다."
- 탐구 질문:
  1. "알프스 빙하가 녹은 지역에 새로운 생태계가 형성되는 과정을 설명해 보세요."
  2. "아마존 열대우림의 파괴가 전 지구적 기후에 미치는 영향은 무엇인가요?"
  3. "인간의 활동에 의해 생태계가 파괴된 사례를 찾아보고, 이를 막기 위한 방법을 토의해 보세요."
- 실생활 예시: 극지방 빙하 감소(북극곰 서식지), 산호초 백화(해수 온도 상승), 사막화
- 교과서 연계: "미래엔 통합과학2 II편 p80~81 — 환경 변화와 생태계, 알프스/아마존 사례"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_20_climate_change_ecosystem.html
