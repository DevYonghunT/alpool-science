너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 전자기와 양자
- 단원: I. 전자기적 상호작용 > 03. 자기장과 로런츠 힘
- 토픽: 전류에 의한 자기장 (직선/원형/솔레노이드)
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 전류가 흐르는 도선 주위에는 자기장이 형성된다. 직선 전류에 의한 자기장 B = kI/r (동심원 형태), 원형 전류 중심 자기장 B = k'I/r, 솔레노이드 내부 자기장 B = k''nI (균일). 자기장의 방향은 오른손 법칙으로 결정한다. 자기력선은 자석의 N극에서 나와 S극으로 들어간다.

## 씬 구성
- 3가지 모드: 직선 전류 / 원형 전류 / 솔레노이드
- 각 모드에서 전류 방향, 자기장 방향, 자기력선을 3D로 시각화
- 교과서 그림 I-12~14 (p18~20)
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + PointLight(0.5)

## 모드 1: 직선 전류

### 3D
- 직선 도선: CylinderGeometry(0.06, 0.06, 10), 0xcc6633 (구리색), 수직
- 전류 방향: 노란 파티클이 도선을 따라 이동
- 자기력선: TorusGeometry(r, 0.015, 8, 64) × 5~8개 동심원
  - 반지름 r = 1, 2, 3, 4, 5
  - 색상: 0x44ff88 (초록), 세기에 따라 그라데이션
  - 방향 화살촉: ConeGeometry, 오른손 법칙
- 자기장 세기: B = kI/r → 가까울수록 강함 (원의 선 두께/밝기)
- 나침반: 도선 주위에 작은 나침반(화살표) 여러 개 → 자기장 방향 가리킴

### 오른손 법칙
- 3D 손 아이콘: 엄지=전류방향, 나머지=자기장방향(감아쥠)
- 전류 방향 토글 → 자기장 방향 반전

## 모드 2: 원형 전류

### 3D
- 원형 코일: TorusGeometry(2, 0.06, 8, 64), 0xcc6633
- 전류 방향: 코일을 따라 노란 파티클
- 코일 중심 자기장: 코일 면에 수직 (큰 빨간 화살표)
- 자기력선: TubeGeometry 곡선, 코일 중심에서 나와 바깥으로 돌아옴
- B = k'I/r (중심), r 클수록 B 작음
- 오른손 법칙: 손가락=전류방향 감아쥐면 엄지=자기장방향

## 모드 3: 솔레노이드

### 3D
- 솔레노이드: TorusGeometry × 15~20개 나란히 배치
- 전류 방향: 코일을 따라 파티클
- 내부: 균일한 자기장 → 평행한 초록 화살표 격자
  - B = k''nI (n: 단위 길이당 감은 수)
- 외부: 막대 자석과 유사한 자기력선 패턴
- N극/S극 표시
- 오른손 법칙: 네 손가락=전류방향, 엄지=자기장(N극)

### 두 직선 전류 사이의 힘 (보너스)
- 같은 방향 전류: 인력 (끌어당김)
- 반대 방향 전류: 척력 (밀어냄)
- 자기력선 패턴으로 시각화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "모드" 드롭다운: 직선 전류 / 원형 전류 / 솔레노이드 / 두 직선 전류
- "전류 I" 슬라이더: 0.5~5A (기본값 2, 0.5 단계)
- "전류 방향" 토글: 정방향 / 역방향
- "코일 감은 수" 슬라이더 (솔레노이드): 5~30
- "자기력선 표시" 토글: ON/OFF (기본값 ON)
- "나침반 표시" 토글: ON/OFF
- "오른손 법칙" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🧲 전류에 의한 자기장
━━━━━━━━━━━━━━━
모드: [직선/원형/솔레노이드]
전류 I = [값]A

직선 전류: B = kI/r
  k = 2×10⁻⁷ T·m/A
  동심원, 오른손 법칙

원형 전류: B = k'I/r (중심)
  k' = 2π×10⁻⁷ T·m/A
  코일 면에 수직

솔레노이드: B = k''nI (내부 균일)
  k'' = 4π×10⁻⁷ T·m/A
  n = N/L = [값]
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "전류가 흐르는 도선 주위에 자기장이 형성됩니다. 직선 전류는 동심원 형태, 원형 전류는 코일 면에 수직, 솔레노이드는 내부 균일한 자기장을 만듭니다."
- 탐구 질문:
  1. "직선 전류 주위의 자기장 방향을 오른손 법칙으로 결정하는 방법을 설명해 보세요."
  2. "솔레노이드 내부의 자기장이 균일한 것은 어떤 장점이 있나요?"
  3. "같은 방향으로 전류가 흐르는 두 직선 도선 사이에 인력이 작용하는 까닭을 자기력선으로 설명해 보세요."
- 실생활 예시: 전자석(솔레노이드+철심), MRI(강력한 자기장), 전자 벨, 자기 부상 열차
- 교과서 연계: "전자기와 양자 I편 p18~21 — 자기장과 로런츠 힘 (그림 I-12~14)"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 모드 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 파티클 최대 100개
- 파일명: emq_03_current_magnetic_field.html
