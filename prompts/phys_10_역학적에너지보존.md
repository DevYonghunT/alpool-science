너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: I. 힘과 에너지 > 2. 에너지와 열 > 일과 에너지
- 토픽: 역학적 에너지 보존
- 대상: 고등학교 (선택과목)
- 핵심 개념: 중력만 작용하는 경우 물체의 운동 에너지와 위치 에너지의 합인 역학적 에너지는 일정하게 보존된다. ½mv² + mgh = 일정. 높은 곳에서 낮은 곳으로 내려올 때 위치 에너지가 감소하고 운동 에너지가 증가하며, 올라갈 때는 반대이다. 탄성력에 의한 역학적 에너지 보존도 성립한다.

## 씬 구성
- 화면 좌측 60%: 3D 롤러코스터/진자/자유낙하 시뮬레이션
- 화면 우측 40%: 에너지 막대그래프 (Ek, Ep, E_total) 실시간
- 3가지 시나리오: 자유 낙하 / 진자 운동 / 롤러코스터
- 배경: 0xddeeff
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls
- 조명: AmbientLight(0.4) + DirectionalLight(0.7)

## 시나리오 1: 자유 낙하

### 3D
- 높은 위치에 공: SphereGeometry(r=0.3), 0xff4444
- 높이 기준선: 바닥 PlaneGeometry, 0x888888
- 높이 눈금: 0, 2, 4, 6, 8, 10m 표시 (세로 막대)
- 공이 자유 낙하: h 감소, v 증가

### 에너지 변화
- 최고점: Ep = mgh(최대), Ek = 0
- 중간: Ep 감소, Ek 증가
- 바닥: Ep = 0, Ek = mgh(최대)
- 총 역학적 에너지 E = Ek + Ep = 일정 (초록 수평선)
- 잔상(trail): 공의 경로를 점으로 표시, 색상은 속도에 따라 변화(파랑→빨강)

## 시나리오 2: 진자 운동

### 3D
- 천장 고정점: SphereGeometry(r=0.05), 0x444444
- 실: CylinderGeometry(0.02, 0.02, L), 0x888888 (실 길이 L 조절 가능)
- 추: SphereGeometry(r=0.2), 0x4488aa
- 진자가 좌우로 왕복하는 애니메이션
- 최고점(좌/우)에서 정지 → 최저점에서 최대 속도

### 에너지 변화
- 최고점: Ep 최대, Ek = 0, v = 0
- 최저점: Ep = 0(기준점), Ek 최대, v 최대
- 역학적 에너지 보존: 어느 위치에서나 Ek + Ep = 일정
- 속도 벡터: 추에 접선 방향 화살표 (크기 = 속도에 비례)

## 시나리오 3: 롤러코스터

### 3D
- 롤러코스터 트랙: TubeGeometry로 곡선 경로
  - 출발점: 높이 10m
  - 내리막: 0m
  - 루프: 원형 루프 (반지름 3m)
  - 올라가는 언덕: 높이 7m
  - 최종 내리막
- 트랙 색상: 0x888888, 지지대(CylinderGeometry) 여러 개
- 카트: BoxGeometry(0.5, 0.3, 0.3), 0xff4444, 트랙 위를 이동
- 바퀴: CylinderGeometry × 4

### 에너지 변화
- 출발점(높이 10m): Ep 최대, Ek = 0 (정지에서 출발)
- 내리막: Ep → Ek (속도 증가)
- 루프 꼭대기: Ep 증가, Ek 감소 (속도 감소, 최소 속도 필요)
- 올라가는 언덕: Ep 증가, Ek 감소
- 각 지점에서 v = √(2g(h₀-h)) (마찰 무시)

### 루프 통과 조건
- 루프 꼭대기에서 원심력 ≥ 중력: v² ≥ gr
- 출발 높이가 충분해야 루프 통과 가능
- 출발 높이 슬라이더로 조절 → 루프 통과 성공/실패 시각화

## 에너지 막대그래프 (우측, 실시간)
- 3개 막대 (세로 방향):
  - 운동 에너지 Ek = ½mv² : 빨간 막대
  - 위치 에너지 Ep = mgh : 파란 막대
  - 총 역학적 에너지 E = Ek + Ep : 초록 수평 점선 (일정)
- 물체 위치에 따라 막대 높이가 실시간으로 변화
- Ek + Ep = E (일정) 확인 가능

## 마찰이 있는 경우 (옵션)
- 마찰 토글 ON: 역학적 에너지가 서서히 감소
- 감소한 에너지 = 열에너지로 전환 (주황 막대 추가)
- "역학적 에너지의 일부가 열에너지로 전환" 설명

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 자유 낙하 / 진자 운동 / 롤러코스터
- "초기 높이" 슬라이더: 2~15m (기본값 10)
- "질량" 슬라이더: 0.5~5kg (기본값 1)
- "실 길이" 슬라이더 (진자): 1~5m (기본값 2)
- "초기 각도" 슬라이더 (진자): 10~80° (기본값 45°)
- "마찰" 토글: ON/OFF (기본값 OFF)
- "시뮬레이션 시작" 버튼
- "잔상 표시" 토글: ON/OFF
- "초기화" 버튼
