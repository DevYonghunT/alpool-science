너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 05. 역학적 에너지 보존과 탈출 속도
- 토픽: 중력에 의한 역학적 에너지 보존
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 중력장에서 물체의 역학적 에너지(운동 에너지 + 중력 위치 에너지)는 보존된다. 지표면 근처에서 Ep = mgh이지만, 만유인력에 의한 중력 위치 에너지는 Ep = -GMm/r (r: 중심까지 거리, 무한대에서 0 기준)이다. 인공위성의 역학적 에너지 E = ½mv² - GMm/r = -GMm/(2r) (원궤도)로 일정하며, E < 0이면 속박 궤도(타원/원), E = 0이면 포물선 궤도(탈출), E > 0이면 쌍곡선 궤도이다.

## 씬 구성
- 화면 좌측 60%: 3D 지구 + 인공위성/발사체 궤도
- 화면 우측 40%: 에너지 막대그래프 (Ek, Ep, E_total) 실시간 + 궤도 에너지 다이어그램
- 3가지 시나리오: 지표면 근처 / 인공위성 원궤도 / 발사체 궤도 변화
- 배경: 0x000010 (우주)
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.15) + PointLight(1.0)

## 시나리오 1: 지표면 근처 역학적 에너지 보존

### 3D
- 지표면: PlaneGeometry(20, 20), 0x66aa44 (잔디)
- 경사면/언덕: BoxGeometry 기울어진 면, 0xaa9988
- 물체(공): SphereGeometry(r=0.2), 0xff4444
- 경사면 위에서 출발 → 내려오면서 속도 증가 → 평지 통과 → 반대쪽 언덕 올라감

### 에너지 변화
- 높은 곳: Ep = mgh (최대), Ek = 0 (정지)
- 중간(평지): Ep = 0, Ek = mgh (최대 속도)
- 반대편 높이 h': Ep = mgh', Ek = mg(h-h')
- E_total = Ek + Ep = mgh = 일정
- 롤러코스터 형태 트랙에서도 시연 가능

## 시나리오 2: 인공위성 원궤도

### 3D
- 지구: SphereGeometry(r=3), 0x2266aa
- 인공위성: BoxGeometry(0.3, 0.1, 0.3), 0xdddddd
- 원형 궤도: RingGeometry, 0x444488 점선
- 궤도 반지름 r: 슬라이더로 조절

### 에너지 (만유인력)
- 운동 에너지: Ek = ½mv² = GMm/(2r)
- 위치 에너지: Ep = -GMm/r
- 역학적 에너지: E = Ek + Ep = -GMm/(2r) (항상 음수)
- 궤도 반지름 r 증가 시: Ek 감소, Ep 증가(덜 음수), E 증가(덜 음수)
- "높은 궤도 = 느린 속도 + 더 큰(덜 음수) 에너지"

### 에너지 막대그래프
- Ek (빨강, 양수 막대)
- Ep (파랑, 음수 막대, 아래로)
- E_total (초록 수평선)
- |Ep| = 2Ek (원궤도에서 항상) 관계 확인

## 시나리오 3: 발사체 궤도와 에너지

### 3D
- 지구 표면에서 발사체 발사
- 초기 속도 v₀에 따라 궤도가 달라짐:
  - v₀ < v_원궤도: 지구에 다시 떨어짐 (타원, 지구와 교차)
  - v₀ = v_원궤도 = √(GM/R): 원궤도
  - v_원궤도 < v₀ < v_탈출: 타원 궤도 (지구 속박)
  - v₀ = v_탈출 = √(2GM/R): 포물선 궤도 (E = 0, 무한히 멀어짐)
  - v₀ > v_탈출: 쌍곡선 궤도 (E > 0)

### 궤도 시각화
- 발사 속도 슬라이더: 0 ~ 1.5×v_탈출
- 궤도가 실시간으로 변화:
  - 느리면: 지구 표면에 떨어짐 (빨간 궤적)
  - 원궤도 속도: 원형 (파란 궤적)
  - 탈출 속도: 포물선 (초록 궤적, 돌아오지 않음)
- 에너지 다이어그램: E vs r 곡선에서 현재 상태 마커

## 에너지-거리 다이어그램 (우측 Canvas 2D)
- X축: 거리 r (지구 중심~무한대)
- Y축: 에너지
- 곡선: Ep = -GMm/r (파랑, 음수 곡선)
- 수평선: E_total (초록)
- Ek = E_total - Ep (빨강, E_total과 Ep의 차이)
- E < 0: 속박 궤도 (타원/원)
- E = 0: 탈출 (포물선)
- E > 0: 쌍곡선

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "시나리오" 드롭다운: 지표면 근처 / 인공위성 원궤도 / 발사체 궤도
- "초기 높이 h" 슬라이더 (시나리오 1): 1~20m
- "궤도 반지름 r" 슬라이더 (시나리오 2): R~5R (R=지구 반지름)
- "발사 속도" 슬라이더 (시나리오 3): 0~15km/s (v_탈출 ≈ 11.2km/s)
- "에너지 막대그래프" 토글: ON/OFF (기본값 ON)
- "E-r 다이어그램" 토글: ON/OFF
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

```

━━━━━━━━━━━━━━━━━━━
시나리오: [지표면/원궤도/발사체]

지표면 근처:
  Ek = ½mv², Ep = mgh
  E = Ek + Ep = mgh = 일정

만유인력:
  Ep = -GMm/r
  Ek = ½mv²
  E = Ek + Ep = -GMm/(2r) (원궤도)

인공위성 (r=[값]R):
  v = √(GM/r) = [값]km/s
  Ek = [값]J, Ep = [값]J
  E = [값]J

궤도 분류:
  E < 0: 속박 (타원/원)
  E = 0: 탈출 (포물선)
  E > 0: 쌍곡선
```
