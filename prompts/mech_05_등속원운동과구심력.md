너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 역학과 에너지
- 단원: I. 시공간과 운동 > 03. 여러 가지 운동
- 토픽: 등속 원운동과 구심력
- 대상: 고등학교 (진로 선택과목)
- 핵심 개념: 물체가 일정한 빠르기로 원을 따라 도는 운동을 등속 원운동이라 한다. 등속 원운동하는 물체에는 원의 중심 방향으로 구심 가속도가 작용하며 a = v²/r = rω²이다. 이 가속도를 일으키는 힘을 구심력이라 하며 F = mv²/r = mrω²이다. 구심력은 실의 장력, 중력, 마찰력 등 실제 힘이 제공한다.

## 씬 구성
- 화면 중앙: 3D 등속 원운동 시각화 (원 궤도 + 물체 + 벡터)
- 3가지 사례: 실에 매달린 공 / 자동차 곡선 도로 / 인공위성
- 배경: 0xeef0f5
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (위에서 비스듬히)
- 조명: AmbientLight(0.5) + DirectionalLight(0.6)

## 사례 1: 실에 매달린 공 (수평 원운동)

### 3D
- 고정점: SphereGeometry(r=0.05), 상단 중앙
- 실: CylinderGeometry(0.01, 0.01, L), 고정점→공 연결, 0xdddddd
- 공: SphereGeometry(r=0.2), 0xff4444
- 원 궤도: RingGeometry(r, r+0.02, 64), 0xaaaaaa 반투명 점선

### 속도/가속도 벡터 (핵심!)
- 속도 v: 초록 화살표, 원의 접선 방향 (항상 운동 방향)
- 구심 가속도 a: 빨간 화살표, 원의 중심 방향 (항상 안쪽)
- 구심력 F: 파란 화살표, 중심 방향 = 실의 장력 성분
- **속도 방향이 계속 변하지만 크기는 일정** → 등속 원운동
- **가속도는 항상 중심을 향함** → 구심 가속도

### 운동 애니메이션
- 공이 원 궤도를 따라 등속으로 회전
- 속도 벡터가 접선 방향으로 계속 회전 (방향만 변화)
- 가속도 벡터가 항상 중심을 가리킴
- 주기 T = 2πr/v, 각속도 ω = v/r = 2π/T

### 잔상 표시
- 일정 시간 간격으로 속도 벡터 잔상 남김
- 모든 잔상이 접선 방향인 것 확인 → "속도 방향이 계속 변한다"

## 사례 2: 자동차 곡선 도로

### 3D
- 원형 도로: TorusGeometry(r=5, tube=0.5, 16, 64), 0x555555 (아스팔트)
- 자동차: BoxGeometry 조합, 0xff4444
- 바깥쪽 가드레일, 안쪽 풀밭

### 구심력 = 마찰력
- 구심력: 타이어와 도로 사이 마찰력 f = mv²/r
- 속도를 올리면: 필요 구심력 증가 → μmg 초과 시 미끄러짐!
- 최대 속도: v_max = √(μgr)
- 속도 슬라이더로 조절: v < v_max → 안전 (초록), v > v_max → 미끄러짐 (빨간 경고)

## 사례 3: 인공위성 궤도 운동

### 3D
- 지구: SphereGeometry(r=3), 0x2266aa (파랑+녹색)
- 인공위성: BoxGeometry(0.3, 0.1, 0.3) + 태양 전지판, 0xdddddd
- 원형 궤도: RingGeometry, 0xaaaaaa 점선
- 구심력 = 만유인력 F = GMm/r²
- 궤도 속도: v = √(GM/r)
- 높이에 따라 궤도 속도/주기 변화

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "사례" 드롭다운: 실에 매달린 공 / 곡선 도로 / 인공위성
- "반지름 r" 슬라이더: 1~10m (기본값 3)
- "속력 v" 슬라이더: 1~20m/s (기본값 5)
- "질량 m" 슬라이더: 0.5~5kg (기본값 1)
- "마찰 계수 μ" 슬라이더 (사례 2): 0.1~1.0 (기본값 0.5)
- "속도 벡터 표시" 토글: ON/OFF (기본값 ON)
- "가속도 벡터 표시" 토글: ON/OFF (기본값 ON)
- "잔상 표시" 토글: ON/OFF
- "시뮬레이션 시작" 버튼
- "초기화" 버튼

화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🔄 등속 원운동과 구심력
━━━━━━━━━━━━━━━
사례: [실+공 / 곡선도로 / 인공위성]
r = [값]m, v = [값]m/s, m = [값]kg

구심 가속도: a = v²/r = [값]m/s²
구심력: F = mv²/r = [값]N
각속도: ω = v/r = [값]rad/s
주기: T = 2πr/v = [값]s

구심력의 원인:
  실+공: 실의 장력
  곡선도로: 마찰력 (v_max = √(μgr))
  인공위성: 만유인력

핵심:
  속력 일정, 속도 방향 변화
  가속도는 항상 원의 중심 방향
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "등속 원운동에서 속력은 일정하지만 속도의 방향이 계속 변하므로 가속도가 존재합니다. 이 구심 가속도는 항상 원의 중심을 향하며, a=v²/r입니다. 구심력은 실의 장력, 마찰력, 중력 등 실제 힘이 제공합니다."
- 탐구 질문:
  1. "등속 원운동하는 물체의 속력이 일정한데 왜 가속도가 0이 아닌가요?"
  2. "곡선 도로에서 자동차의 속력을 2배로 높이면 필요한 구심력은 몇 배가 되나요?"
  3. "인공위성의 궤도 반지름이 커지면 궤도 속도는 어떻게 변하나요?"
- 실생활 예시: 세탁기 탈수(원심 효과), 놀이공원 회전 놀이기구, GPS 위성, 원심 분리기
- 교과서 연계: "역학과 에너지 I편 p26~31 — 여러 가지 운동, 등속 원운동, 구심력"

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 사례 전환 시 이전 객체 dispose() + 새 객체 생성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프, 잔상 최대 100개
- 파일명: mech_05_circular_motion.html
