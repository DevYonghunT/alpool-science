너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학2
- 단원: I. 변화와 다양성 > 1. 환경 변화와 생물다양성 > 지질 시대의 환경과 생물 변화
- 토픽: 지질 시대 타임라인과 생물 변화
- 대상: 고등학교 1학년
- 핵심 개념: 지구가 탄생한 약 46억 년 전부터 현재까지를 지질 시대라고 한다. 지질 시대는 선캄브리아시대, 고생대, 중생대, 신생대로 구분한다. 각 시대마다 대표적인 환경과 생물이 달랐으며, 지구 환경의 급격한 변화로 인한 생물계의 큰 변화(대멸종)는 지질 시대를 구분하는 중요한 기준으로 사용된다.

## 씬 구성
- 화면 중앙에 3D 풍경을 렌더링하고, 4개의 지질 시대를 전환하며 각 시대의 환경과 대표 생물을 보여준다.
- 화면 하단에 지질 시대 타임라인 바(HTML overlay)를 배치한다.
- 배경: 시대별 하늘색 변화
- 카메라: PerspectiveCamera(FOV 55) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.8) + 시대별 색조 변화

## 타임라인 바 (하단 HTML overlay)
- 전체 46억 년을 가로 바로 표시
- 4개 시대를 상대적 길이로 구분:
  - 선캄브리아시대: 88.2% (약 40억 년), 색상 0x665544 (갈색)
  - 고생대: 6.3% (약 2.9억 년), 색상 0x558855 (초록)
  - 중생대: 4.1% (약 1.9억 년), 색상 0x885544 (적갈색)
  - 신생대: 1.4% (약 0.66억 년), 색상 0x5577aa (청색)
- 각 시대 클릭 시 해당 시대로 전환
- 현재 선택된 시대 강조 표시
- 주요 연대 표시: 46억 년 전, 5.39억 년 전, 2.52억 년 전, 0.66억 년 전, 현재

## 4개 지질 시대 상세

### 선캄브리아시대 (46억~5.39억 년 전)
- 3D 풍경: 황량한 바위 지형, 얕은 바다
  - 바닥: PlaneGeometry(30, 30), 갈색-회색 0x887766
  - 하늘: SphereGeometry 반구, 주황-갈색 0xaa7744 (산소 부족한 대기)
  - 얕은 바다: PlaneGeometry, 반투명 파란색 0x335577 opacity 0.4
- 대표 생물:
  - 남세균(스트로마톨라이트): CylinderGeometry 층층이 쌓인 구조물, 0x667755 녹갈색, 바다 바닥에 3~5개 배치
  - 에디아카라 생물군: 납작한 원형 생물(CircleGeometry + 약간 볼록), 0x998877, 바닥에 2~3개
- 라벨: "선캄브리아시대 — 최초의 생물 출현, 남세균의 광합성으로 산소 증가"
- 교과서: "최초의 생물이 출현한 이후 광합성을 하는 남세균의 등장으로 바다와 대기 중에 산소가 많아졌다" (p16)

### 고생대 (5.39억~2.52억 년 전)
- 3D 풍경: 얕은 바다와 초기 육지
  - 바닥: 초록빛 육지 0x557744 + 파란 바다
  - 하늘: 밝은 파란색 0x88bbdd (오존층 형성)
  - 나무: 양치식물 — CylinderGeometry 줄기 + SphereGeometry 잎 여러 개, 0x336622
- 대표 생물:
  - 삼엽충: BoxGeometry 타원형 몸체(세그먼트 표현), 0x887766 갈색, 바닥에 3개
  - 완족류: SphereGeometry 조개 형태, 0xaaaaaa
  - 어류: 물고기 형태(BoxGeometry 조합), 0x6688aa, 물속에서 이동
  - 양서류: 도마뱀 형태(BoxGeometry 조합), 0x668844, 물가에 배치
- 라벨: "고생대 — 삼엽충, 어류, 양서류 번성. 오존층 형성으로 육상 진출"
- 교과서: "고생대에는 단단한 껍데기와 뼈를 가진 해양 생물들이 많아졌다. 오존층이 형성되어 생물의 서식지가 바다에서 육지로 확장되었다" (p16)
- 대멸종 표시: "고생대 말 — 판게아 형성, 기후 급변으로 생물 종류 크게 감소" (빨간 경고 아이콘)

### 중생대 (2.52억~0.66억 년 전)
- 3D 풍경: 따뜻하고 울창한 숲
  - 바닥: 진초록 0x448833
  - 하늘: 따뜻한 하늘색 0x99ccee
  - 나무: 겉씨식물(은행나무 형태) — CylinderGeometry 줄기 + ConeGeometry 잎, 0x557733
- 대표 생물:
  - 공룡: BoxGeometry 조합으로 간단한 공룡 실루엣 (머리/목/몸통/꼬리/다리), 0x668855 녹갈색, 크기 3~4
  - 암모나이트: TorusGeometry 나선형, 0x887766, 바다에 배치
  - 해양 파충류: 물속에서 움직이는 형태
- 수륙 분포: 그림 I-8 스타일 대륙 분리 표시
- 라벨: "중생대 — 공룡, 암모나이트 번성. 화산 활동으로 온난한 기후"
- 교과서: "중생대에는 판게아가 분리되며 생물 서식지의 환경이 다양해졌다. 육지에서는 공룡이 번성했고, 바다에서는 암모나이트가 번성했다" (p17)
- 대멸종 표시: "중생대 말 — 지구 환경 급격히 변해 공룡과 암모나이트 등 멸종" (빨간 경고 아이콘)

### 신생대 (0.66억 년 전~현재)
- 3D 풍경: 현대적 초원과 숲
  - 바닥: 밝은 초록 0x66aa44
  - 하늘: 맑은 파란색 0x87CEEB
  - 나무: 속씨식물(활엽수) — CylinderGeometry 줄기 + SphereGeometry 둥근 잎, 0x338822
- 대표 생물:
  - 매머드: BoxGeometry 조합 (코끼리 형태 + 긴 엄니), 0x8b7355 갈색
  - 포유류: 작은 동물 형태
  - 인류: 간단한 사람 실루엣(BoxGeometry 조합), 0xddccbb
  - 화폐석: 바다에 SphereGeometry 납작한 형태, 0xccbb99
- 라벨: "신생대 — 포유류, 속씨식물 번성. 빙하기와 간빙기 반복. 인류 출현"
- 교과서: "신생대 중기까지는 대체로 온난한 기후가 지속되었으나, 이후 기온이 점차 낮아져 빙하기와 간빙기가 여러 차례 반복되었다" (p17)

  - 고생대 말: "판게아 형성, 기후 급변 → 삼엽충 등 다수 생물 멸종"
  - 중생대 말: "운석 충돌설, 화산 폭발설 → 공룡, 암모나이트 멸종"
- 교과서 p18: "생물 대멸종을 설명하는 가설: 운석(소행성) 충돌설, 화산 폭발설, 수륙 분포 변화설"

## 해양 생물 과의 수 변화 그래프 (교과서 그림 I-11)
- 화면 우측에 작은 그래프 패널(HTML overlay):
  - X축: 시간 (5.39억 년 전 → 현재)
  - Y축: 해양 생물 과의 수 (0~900)
  - 대멸종 시기를 화살표로 표시
  - 현재 선택된 시대 구간을 강조

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "지질 시대" 드롭다운: 선캄브리아시대 / 고생대 / 중생대 / 신생대
- "자동 재생" 버튼: 선캄브리아→신생대 자동 전환 (각 단계 6초)
- "대멸종 표시" 토글: ON/OFF (기본값 ON)
- "생물 이름 표시" 토글: ON/OFF (기본값 ON)
- "초기화" 버튼

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 주석은 한국어
- 반응형: window resize 이벤트 처리
- 모바일 터치 지원 (OrbitControls 기본 지원)
- 시대 전환 시 이전 씬 객체 dispose() + 새 씬 구성
- gui.controllersRecursive().forEach(c => c.updateDisplay()) 동기화
- 성능: requestAnimationFrame 루프
- 파일명: sci2_01_geological_timeline.html
