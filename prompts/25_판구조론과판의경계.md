너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 통합과학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 통합과학1
- 단원: III. 시스템과 상호작용 > 1. 지구시스템 > 지권의 변화와 판 구조론
- 토픽: 판 구조론과 판의 경계
- 대상: 고등학교 1학년
- 핵심 개념: 지권의 표면은 크고 작은 10여 개의 판으로 이루어져 있다. 판의 경계는 발산형(새 지각 생성, 해령/열곡대), 수렴형(판 충돌/섭입, 해구/습곡 산맥/호상열도), 보존형(어긋나게 이동, 변환 단층)으로 구분한다. 암석권 아래 연약권에서 맨틀 대류가 일어나며, 이것이 판 운동의 원동력이다. 지진대와 화산대는 판의 경계를 따라 띠 모양으로 분포한다.

## 씬 구성
- 화면을 2단으로 구성:
  - 상단(높이 60%): 3D 지구 + 판 경계 시각화
  - 하단(높이 40%): 판 경계 단면도 3D 시각화
- 상단 3D 지구:
  - SphereGeometry(r=4.0, 128, 128)
  - 표면 텍스처: 대륙(0x447744 녹색) + 바다(0x2255aa 파랑) 절차적 생성
  - 판 경계선: 실제 지구 판 경계를 따라 밝은 빨간 선(LineGeometry, 0xff4444, linewidth 2)
  - 주요 판 라벨 (교과서 그림 III-12, p114~115):
    - 유라시아판, 태평양판, 북아메리카판, 남아메리카판, 아프리카판, 인도-오스트레일리아판, 남극판, 나스카판, 필리핀판, 코코스판, 카리브판, 아라비아판, 스코샤판
  - 판 이동 방향: 각 판에 작은 화살표(ArrowHelper)로 이동 방향 표시
  - 지진 분포: 판 경계를 따라 작은 노란 점(Points, 300개, 0xffff00) — 지진대
  - 화산 분포: 판 경계를 따라 빨간 삼각형(ConeGeometry r=0.04, h=0.08, 0xff2200), 50개 — 화산대
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls (지구 회전)
- 조명: AmbientLight(0.3) + DirectionalLight(0.6)

## 주요 판 경계 사례 6개 (교과서 p114~115)

지구 위에 번호 마커(①~⑥)를 배치하고, 클릭 시 하단 단면도에 해당 경계 표시:

### ① 히말라야산맥 (수렴형 — 대륙판+대륙판)
- 위치: 인도-오스트레일리아판 ↔ 유라시아판 경계
- 단면도: 두 대륙판이 서로 충돌 → 습곡 산맥 형성
  - 좌측 판: BoxGeometry(폭 5, 높이 1, 깊이 3), 0xaa8866 (대륙 갈색), "인도판" 라벨
  - 우측 판: BoxGeometry 동일, "유라시아판" 라벨
  - 충돌 부위: 판 가장자리가 접히며 위로 솟아오름 (CurvedGeometry, 산 형태)
  - 습곡 산맥(히말라야): ConeGeometry(r=2, h=3), 0x886644
- 특징: "지진 자주 발생, 화산 활동 없음"

### ② 마리아나 해구 (수렴형 — 해양판+해양판)
- 위치: 태평양판 ↔ 필리핀판 경계
- 단면도: 태평양판이 필리핀판 아래로 섭입
  - 좌측: 해양판(0x334466 짙은 파랑) "태평양판", 아래로 꺾어짐 (경사면)
  - 우측: 해양판 "필리핀판"
  - 해구: V자형 깊은 골 (0x001133), "마리아나 해구" 라벨
  - 호상열도: 섭입 위에 작은 화산섬들(ConeGeometry 4~5개, 0xff4422)
- 특징: "지진과 화산 활동 활발, 해구와 호상열도 형성"

### ③ 안데스산맥 (수렴형 — 해양판+대륙판)
- 위치: 나스카판 ↔ 남아메리카판 경계
- 단면도: 나스카판(해양)이 남아메리카판(대륙) 아래로 섭입
  - 좌측: 해양판(0x334466) "나스카판", 아래로 꺾어짐
  - 우측: 대륙판(0xaa8866) "남아메리카판"
  - 해구: 판 앞 깊은 골, 산맥: 대륙판 위 습곡 산맥(ConeGeometry)
  - 화산: 산맥 위 화산(0xff4422) 3~4개
- 특징: "해구, 습곡 산맥, 화산 분포"

### ④ 산안드레아스 단층 (보존형)
- 위치: 북아메리카판 ↔ 태평양판 경계
- 단면도: 두 판이 서로 반대 방향으로 어긋나게 이동
  - 좌측 판: "태평양판" → 방향 화살표
  - 우측 판: "북아메리카판" ← 방향 화살표
  - 변환 단층: 두 판 사이 세로 틈(0xff8844 주황 점선)
- 특징: "지진 자주 발생, 화산 활동 거의 없음, 판 생성/소멸 없음"

### ⑤ 동태평양 해령 (발산형 — 해양)
- 위치: 태평양판 동쪽
- 단면도: 두 해양판이 서로 멀어짐
  - 좌우 판: 각각 반대 방향으로 이동 (← →)
  - 중앙 틈: 마그마가 솟아올라 새로운 지각 생성 (빨간 용암 파티클 ↑, 0xff4400)
  - 해령: 중앙에 해저 산맥(CylinderGeometry 낮은 봉우리들)
- 특징: "지진과 화산 활동 활발, 새로운 해양 지각 생성"

### ⑥ 동아프리카 열곡대 (발산형 — 대륙)
- 위치: 아프리카판 동부
- 단면도: 대륙판이 갈라짐
  - 좌우: 대륙판 조각 (← →)
  - 중앙: 열곡(깊고 좁은 골짜기), 호수 형성(파란 물)
  - 화산: 열곡 주변 화산 2~3개
- 특징: "먼 미래에 아프리카 대륙이 두 부분으로 갈라질 것"

## 하단 단면도 3D
- 폭 전체, 높이 40%
- 선택된 판 경계의 단면을 3D로 렌더링 (위에서 약간 비스듬히 내려다보는 시점)
- 맨틀 대류: 단면도 하단에 뜨거운 물질 상승(빨간 화살표 ↑) + 차가운 물질 하강(파란 화살표 ↓)
- 각 단면도에서 판 이동 애니메이션: 판이 느리게 이동하며 경계 현상 발생
- 섭입 시: 해양판이 느리게 아래로 기울어지며 들어감

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "판 경계 사례" 드롭다운: ① 히말라야 / ② 마리아나 / ③ 안데스 / ④ 산안드레아스 / ⑤ 동태평양 해령 / ⑥ 동아프리카 열곡대
  - 선택 시 상단 지구에서 해당 위치 강조 + 하단 단면도 전환
- "판 경계 유형" 드롭다운: 전체 / 발산형만 / 수렴형만 / 보존형만
  - 해당 유형 경계선만 밝게, 나머지 페이드
- "지진대 표시" 토글: ON/OFF (기본값 ON)
- "화산대 표시" 토글: ON/OFF (기본값 ON)
- "판 이동 방향" 토글: ON/OFF (기본값 ON) — 화살표 표시
- "판 이동 애니메이션" 토글: ON/OFF (기본값 OFF) — 단면도에서 판 이동 애니메이션
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
🌍 판 구조론
━━━━━━━━━━
선택 경계:     [사례명]
경계 유형:     [발산형/수렴형/보존형]
관련 판:       [판1] ↔ [판2]

판 경계 유형:
  발산형: 판↔판 멀어짐 → 새 지각 생성, 해령/열곡대
  수렴형: 판↔판 충돌/섭입 → 해구/습곡 산맥/호상열도
  보존형: 판↔판 어긋나게 이동 → 변환 단층, 지진

판 운동 원동력: 맨틀 대류
  암석권(판) — 단단한 암석
  연약권 — 유동성, 맨틀 대류 발생
```

## 코드 규칙
- 단일 HTML 파일로 작성 (외부 파일 없음)
- Three.js는 CDN import: https://unpkg.com/three@0.160.0/build/three.module.js
- OrbitControls CDN: https://unpkg.com/three@0.160.0/examples/jsm/controls/OrbitControls.js
- lil-gui CDN: https://unpkg.com/lil-gui@0.19.2/dist/lil-gui.esm.js
- ES6 모듈 방식 (type="module")
- 한국어 주석, 반응형, 모바일 터치 지원
- 지구 판 경계: BufferGeometry Line, 아래 위경도 데이터를 3D 좌표로 변환 (lon,lat → x=r*cos(lat)*cos(lon), y=r*sin(lat), z=r*cos(lat)*sin(lon))
- 주요 판 경계 위경도 포인트 (간략화, 직선 연결):
  - 환태평양 조산대: (50°N,170°E)→(40°N,145°E)→(35°N,140°E)→(20°N,120°E)→(0°,130°E)→(20°S,170°E)→(40°S,175°W)→(55°S,70°W)→(20°S,70°W)→(0°,80°W)→(20°N,105°W)→(40°N,125°W)→(50°N,170°W)
  - 대서양 중앙 해령: (65°N,18°W)→(40°N,30°W)→(20°N,45°W)→(0°,25°W)→(20°S,12°W)→(40°S,15°W)→(55°S,0°)
  - 히말라야 수렴대: (35°N,70°E)→(28°N,85°E)→(25°N,95°E)
  - 동아프리카 열곡: (10°N,40°E)→(0°,35°E)→(10°S,35°E)→(20°S,35°E)
  - 산안드레아스 단층: (40°N,124°W)→(35°N,120°W)→(32°N,115°W)
  - 참고: 정밀한 판 경계는 100+ 포인트가 필요하나, 교육용 시뮬레이션이므로 위 간략 데이터로 주요 경계만 표현
- 지진/화산 분포: Points/ConeGeometry, 판 경계 따라 분포
- 단면도: Group으로 조립, 판 경계별 교체
- 파일명: plate_tectonics_boundaries_sim.html
