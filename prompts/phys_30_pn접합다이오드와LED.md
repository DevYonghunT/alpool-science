너는 Three.js 과학 시뮬레이션 전문 개발자다. 고등학교 물리학 교과서 기반의 인터랙티브 3D 시뮬레이션을 단일 HTML 파일로 제작한다.

## 시뮬레이션 정보
- 과목: 물리학
- 단원: III. 빛과 물질 > 2. 원자와 에너지 준위 > 에너지띠와 반도체
- 토픽: p-n 접합 다이오드와 LED
- 대상: 고등학교 (선택과목)
- 핵심 개념: p형 반도체와 n형 반도체를 접합하면 p-n 접합 다이오드가 된다. 접합면 근처에 공핍층이 형성되며, 순방향 바이어스(p→양극, n→음극)를 걸면 전류가 흐르고, 역방향 바이어스를 걸면 전류가 흐르지 않는 정류 작용을 한다. LED(발광 다이오드)는 순방향 전류가 흐를 때 전자와 양공이 재결합하면서 빛을 방출하는 소자이다.

## 씬 구성
- 화면 좌측: 3D p-n 접합 다이오드 내부 구조 (p형 + n형 + 공핍층 + 전자/양공)
- 화면 우측: I-V 특성 곡선 + LED 응용 (Canvas 2D / 3D)
- 3가지 모드: 무바이어스 / 순방향 바이어스 / 역방향 바이어스
- 배경: 0x0a0a20
- 카메라: PerspectiveCamera(FOV 50) + OrbitControls
- 조명: AmbientLight(0.3) + DirectionalLight(0.5)

## 3D p-n 접합 다이오드 내부

### p형 영역 (좌측)
- BoxGeometry(4, 3, 2), 0xff6644 반투명 opacity 0.3, "p형" 라벨
- 양공(hole): RingGeometry(0.1, 0.15, 16) × 20개, 0xff4444, 랜덤 분포
  - 양공은 (+) 전하 운반자
- 붕소(B) 불순물 원자: SphereGeometry(r=0.12), 0x44aaff, 3~4개
- 격자 배경: 작은 회색 구 격자

### n형 영역 (우측)
- BoxGeometry(4, 3, 2), 0x4466ff 반투명 opacity 0.3, "n형" 라벨
- 전자: SphereGeometry(r=0.08) × 20개, 0xffff44 (노랑), 랜덤 분포
  - 전자는 (−) 전하 운반자
- 인(P) 불순물 원자: SphereGeometry(r=0.12), 0xff6644, 3~4개
- 격자 배경

### 접합면 (중앙)
- PlaneGeometry(0.05, 3, 2), 0xffffff opacity 0.2 (접합면 표시)

### 공핍층 (depletion region)
- BoxGeometry(1.5, 3, 2), 0x888888 반투명 opacity 0.15
- 접합면 양측에 걸침
- 공핍층 내부: 전자도 양공도 없음 (비어있음)
- 전기장 화살표: n→p 방향 (내장 전위에 의한 전기장), 0xaaaaaa
- 라벨: "공핍층 — 전자와 양공이 재결합하여 비어있는 영역"

## 3가지 바이어스 모드

### 모드 1: 무바이어스 (전원 없음)
- 공핍층이 형성된 상태
- p형 양공과 n형 전자가 접합면 근처에서 재결합 → 공핍층
- 내장 전위에 의한 전기장이 추가 확산 방지
- 전류 = 0

### 모드 2: 순방향 바이어스 (p→양극(+), n→음극(−))
- 외부 전원: p형에 (+), n형에 (−) 연결
- **공핍층 축소** (외부 전기장이 내장 전기장을 상쇄)
  - 공핍층 BoxGeometry width 감소 애니메이션
- **양공이 p→n 방향으로 이동**, **전자가 n→p 방향으로 이동**
  - 양공/전자 파티클이 접합면을 통과하여 반대편으로 이동
- **접합면에서 재결합** → 전류 흐름!
  - 재결합 순간 작은 플래시(빛) 효과
- 전류 세기: 전압에 따라 지수적 증가
- 순방향 전압 슬라이더: 0~3V

### 모드 3: 역방향 바이어스 (p→음극(−), n→양극(+))
- 외부 전원: p형에 (−), n형에 (+) 연결
- **공핍층 확대** (외부 전기장이 내장 전기장 강화)
  - 공핍층 BoxGeometry width 증가 애니메이션
- **양공은 p형 쪽으로, 전자는 n형 쪽으로 밀려남** (접합면에서 멀어짐)
- 전류 = 거의 0 (매우 작은 누설 전류만)
- "❌ 역방향 — 전류 거의 안 흐름" 라벨
- **정류 작용**: 순방향만 전류 통과 = 교류→직류 변환 가능

## LED (발광 다이오드) 시각화

### 순방향에서 발광
- 재결합 시 방출되는 에너지 = 띠틈 에너지 → 빛으로 방출
- LED 3D 모형: CylinderGeometry(0.3, 0.3, 0.5) + SphereGeometry(0.3) 상단 투명 렌즈
- 재결합 위치에서 빛 파티클 발사 (색상은 띠틈 에너지에 따라)
  - 빨간 LED (GaAsP, Eg≈1.8eV): 0xff0000
  - 초록 LED (GaP, Eg≈2.3eV): 0x00ff00
  - 파란 LED (GaN, Eg≈2.7eV): 0x0000ff
  - 백색 LED: 파란 LED + 형광체 (0xffffff)
- LED 색상 선택 드롭다운

### 에너지 다이어그램
- 가전자띠 → 전도띠 전자가 떨어지면서 빛 방출
- ΔE = hf = hc/λ
- 띠틈이 클수록 짧은 파장(높은 에너지 = 파란빛)

## I-V 특성 곡선 (우측, Canvas 2D)
- X축: 전압 V (-5~+3V)
- Y축: 전류 I (0~수백 mA)
- 순방향: 문턱 전압(~0.7V Si, ~2V LED) 이상에서 전류 급증 (지수 곡선)
- 역방향: 전류 ≈ 0 (매우 작은 누설 전류)
- 현재 (V, I) 위치에 빨간 점 마커
- "정류 작용: 순방향만 전류 통과" 강조

## 인터랙션 (GUI 패널)
lil-gui 라이브러리(CDN)로 우측 상단에 컨트롤 패널을 배치한다:
- "바이어스" 드롭다운: 무바이어스 / 순방향 / 역방향
- "전압" 슬라이더: -5~+3V (기본값 0, 0.1 단계)
- "LED 모드" 토글: ON/OFF (순방향에서만)
- "LED 색상" 드롭다운 (LED 모드): 빨강 / 초록 / 파랑 / 백색
- "공핍층 표시" 토글: ON/OFF (기본값 ON)
- "전자/양공 이동" 토글: ON/OFF (기본값 ON)
- "I-V 곡선" 토글: ON/OFF (기본값 ON)
- "에너지띠 다이어그램" 토글: ON/OFF
- "초기화" 버튼

## 수식 패널
화면 좌측 하단에 반투명 HTML overlay 패널로 표시:

```
💡 p-n 접합 다이오드와 LED
━━━━━━━━━━━━━━━━━━
바이어스: [무/순방향/역방향]
전압: [값]V
전류: [값]mA
공핍층 폭: [좁음/보통/넓음]

p-n 접합:
  p형(양공) + n형(전자) 접합
  → 공핍층 형성 (내장 전위)

순방향: 공핍층↓, 전류 흐름 ✅
역방향: 공핍층↑, 전류 차단 ❌
→ 정류 작용 (AC → DC)

LED:
  순방향 재결합 시 빛 방출
  ΔE = hf = hc/λ
  띠틈 ↑ → 짧은 파장(파란빛)
  [색상]: λ ≈ [값]nm, Eg ≈ [값]eV
```

## 교육적 요소
화면 상단에 토글 가능한 "📖 학습 가이드" 패널:
- 핵심 원리: "p형과 n형 반도체를 접합하면 다이오드가 됩니다. 순방향 바이어스에서는 전류가 흐르고, 역방향에서는 차단됩니다(정류 작용). LED는 순방향에서 전자-양공 재결합 시 빛을 방출합니다."
- 탐구 질문:
  1. "p-n 접합 다이오드에서 순방향 바이어스를 걸면 공핍층은 어떻게 변하나요?"
  2. "LED에서 방출되는 빛의 색은 무엇에 의해 결정되나요?"
  3. "다이오드의 정류 작용을 이용하면 교류를 직류로 변환할 수 있는 원리를 설명해 보세요."
- 실생활 예시: LED 조명, 스마트폰 충전기(AC→DC 변환), 태양 전지, 신호등, 디스플레이
- 교과서 연계: "물리학 III편 p170~175 — p-n 접합, 다이오드, LED, 정류 작용"
