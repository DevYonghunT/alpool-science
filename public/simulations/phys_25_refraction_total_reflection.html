빛의 굴절과 전반사

입사각과 매질 조합을 바꾸며 스넬 법칙, 임계각, 전반사를 한 화면에서 비교하고 광섬유 내부의 빛 전달까지 연결해 보세요.

빛은 굴절률이 큰 매질일수록 더 느리게 진행하며, 그 속도 차이 때문에 경계면에서 경로가 꺾입니다.

학습 가이드

빛이 한 매질에서 다른 매질로 진행할 때 경계면에서 굴절합니다. 이때 스넬 법칙 n₁sinθ₁ = n₂sinθ₂가 성립하며, 굴절률이 큰 매질에서 작은 매질로 진행할 때 입사각이 임계각보다 크면 전반사가 일어납니다. 광섬유는 이 전반사를 이용해 빛 신호를 멀리 전달합니다.

탐구 질문

  • 물속에서 수면 위를 올려다볼 때 일정 각도 이상에서 수면이 거울처럼 보이는 까닭은 무엇인가요?
  • 다이아몬드가 유리보다 더 반짝이는 까닭을 전반사와 연결해 설명할 수 있나요?
  • 광섬유가 구부러져 있어도 빛이 전달될 수 있는 까닭은 무엇인가요?

실생활 예시

  • 광케이블을 이용한 인터넷 통신
  • 내시경과 의료용 광섬유
  • 다이아몬드 커팅과 보석의 반짝임
  • 수영장에서 수면이 거울처럼 보이는 현상
  • 렌즈와 프리즘을 이용한 광학 기술

🔍 빛의 굴절과 전반사

모드굴절 법칙
매질 1공기 (n₁ = 1.00)
매질 2물 (n₂ = 1.33)
스넬 법칙: n₁sinθ₁ = n₂sinθ₂
입사각θ₁ = 30.0°
굴절각θ₂ = 22.1°
전반사 조건 (n₁ > n₂): 현재 조합에서는 임계각이 정의되지 않습니다.
현재 상태굴절
광섬유: 코어(n = 1.60) > 클래딩(n = 1.40)
전반사를 이용해 빛 신호를 전달합니다.
빛의 속도는 v = c / n 입니다. 굴절률이 클수록 빛의 속도가 느려지고, 파면의 한쪽이 먼저 늦어지며 경로가 꺾입니다.

광섬유 활용

코어의 굴절률이 클래딩보다 더 크기 때문에, 빛은 코어 내부 경계면에서 전반사를 반복하며 멀리 전달됩니다.

  • 인터넷 통신: 광케이블로 대용량 데이터를 빠르게 전송
  • 의료: 내시경과 레이저 전달 장치
  • 조명: 굽은 경로에서도 빛을 안내하는 광섬유 조명
